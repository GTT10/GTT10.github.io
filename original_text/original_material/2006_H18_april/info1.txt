### **【問題1】柱の圧縮**

この問題は、柱が自分自身の重さ（自重）と、上に載せられた円板の重さによって、どのように縮むかを考えます。

**この問題の最大のポイント** 💡
自重を考える場合、柱の上の方ほど、その下にある部分全体の重さが加わるため、かかる力が大きくなります。この力の変化を考慮して計算する必要があります。

\<img src="[https://storage.googleapis.com/gemini-prod/images/40669d51-c063-4554-aa6c-59d08e50f55e](https://www.google.com/search?q=https://storage.googleapis.com/gemini-prod/images/40669d51-c063-4554-aa6c-59d08e50f55e)" alt="Diagrams for Problem 1" width="500"/\>

#### **(1) 自重を考慮しない場合の応力と全縮み**

**考え方**

  * **応力**: 柱の自重を無視する場合、柱のどの部分にも、上に載っている円板の重さ `Mg` が一様にかかります。応力は「力 ÷ 断面積」で計算します。
  * **全縮み**: 応力が一定なので、縮みは基本公式 `λ = (応力 / E) × 長さ` で簡単に計算できます。

**解答**

  * **応力 `σ`**:
    $$\sigma = \frac{Mg}{A} = \frac{4Mg}{\pi d^2}$$
  * **全縮み `λ`**:
    $$\lambda = \frac{\sigma l}{E} = \frac{4Mgl}{\pi d^2 E}$$

-----

#### **(2) 自重を考慮する場合の応力と全縮み**

**考え方**

  * **応力 `σ(x)`**: 柱の上端から `x` の距離にある断面を考えます。この断面には、「円板の重さ `Mg`」と「その断面より上にある柱部分の重さ」の両方がかかります。
  * **全縮み `λ`**: 応力が場所によって変わるため、微小部分の縮みを考えて、それを柱全体で合計（積分）する必要があります。

**解答**

  * **位置 `x` における応力 `σ(x)`**:
    $$\sigma(x) = \frac{Mg}{A} + \rho g x = \frac{4Mg}{\pi d^2} + \rho g x$$
  * **柱の全縮み `λ`**:
    $$\lambda = \int_0^l \frac{\sigma(x)}{E} dx = \frac{l}{E} \left( \frac{4Mg}{\pi d^2} + \frac{\rho g l}{2} \right)$$

-----

#### **(3) 応力が一定になる柱の断面積と全縮み**

**考え方**

  * **断面積 `A(x)`**: 応力を一定 `σ₀` に保つためには、下に行くほど重くなる力 `F(x)` を支えるために、断面積 `A(x)` も下に行くほど指数関数的に大きくする必要があります。この関係は、微小部分の力のつり合いから微分方程式を立てて解くことで導出できます。
  * **全縮み `λ`**: 応力が柱のどこでも一定 `σ₀` なので、ひずみも一定 `ε = σ₀/E` となります。したがって、全体の縮みは単純に `λ = ε × l` で計算できます。

**解答**

  * **位置 `x` における断面積 `A(x)`**:
    $$A(x) = A(0) \exp\left(\frac{\rho g}{\sigma_0} x\right)$$
    （ここで `A(0) = Mg/σ₀` は柱の上端の断面積、`σ₀` は一定の応力値です）
  * **柱の全縮み `λ`**:
    $$\lambda = \frac{\sigma_0 l}{E}$$

-----

#### **(4) 縮みが生じないようにするための温度上昇**

**考え方**
柱を温めると、熱膨張によって伸びようとします。この「熱による伸び」と、(3)で計算した「力による縮み」がちょうど打ち消しあうようにすれば、結果として柱の長さは変わりません。

**解答**
縮みが生じないようにするための温度上昇 `ΔT` は以下の通りです。
$$\Delta T = \frac{\lambda}{\alpha l} = \frac{\sigma_0}{E \alpha}$$

### **【問題2】3点支持はりの曲げ**

この問題は、3つの点で支えられた「連続はり」に、一様な力がかかる場合を扱います。これも、力のつり合いだけでは解けない「不静定問題」です。

**キーポイント**: はりの形状や荷重が左右対称であるため、「対称性」を利用して解くと計算が簡略化できます。

\<img src="[https://storage.googleapis.com/gemini-prod/images/5f048259-3dae-4c7b-b384-0a37e584a754](https://www.google.com/search?q=https://storage.googleapis.com/gemini-prod/images/5f048259-3dae-4c7b-b384-0a37e584a754)" alt="Diagram for Problem 2" width="500"/\>

#### **(1) 曲げモーメント M(x)**

**考え方**
はりの左端Aから `x` の距離にある断面で、モーメントのつり合いを考えます。

  * **AC間 (`0 ≤ x ≤ l`)**: `x` の左側には、反力 `RA` と、長さ `x` にわたる分布荷重 `w` があります。
  * **CB間 (`l ≤ x ≤ 2l`)**: `x` の左側には、反力 `RA`、反力 `RC`、そして長さ `x` にわたる分布荷重 `w` があります。

**解答**

  * **AC間 (`0 \le x \le l`)**:
    $$M(x) = R_A x - \frac{wx^2}{2}$$
  * **CB間 (`l \le x \le 2l`)**:
    $$M(x) = R_A x + R_C(x-l) - \frac{wx^2}{2}$$

-----

#### **(2) 支点の反力**

**考え方**
不静定問題を解くため、はりの変形（たわみ）の条件を使います。

1.  **対称性の利用**: はり全体が左右対称なので、`RA = RB` となります。また、中央のC点では傾きがゼロになります。
2.  **境界条件**:
      * **A点 (`x=0`)**: 単純支持なので、**たわみ `y` はゼロ**。
      * **C点 (`x=l`)**: 対称性の中心なので、**傾き `y'` はゼロ**。
3.  **計算**: (1)の `M(x)` の式を2回積分して、たわみの式 `y(x)` を導出します。そして、上記の境界条件を適用することで、未知数だった反力を決定できます。

**解答**

  * **A点、B点の反力**:
    $$R_A = R_B = \frac{3}{8}wl$$
  * **C点の反力**:
    $$R_C = \frac{10}{8}wl = \frac{5}{4}wl$$

#### **(4) 最大の曲げ応力**

**考え方**
曲げ応力 `σ` は、公式 `σ = My/I` で計算でき、曲げモーメント `M` の**絶対値が最大**になる場所で最大になります。

1.  **最大モーメントの特定**: (3)のBMDから、モーメントの絶対値が最大になるのは中央のC点であり、その大きさは `|M|_max = wl²/8` です。
2.  **応力の計算**: `y` は中立軸からの距離で、丸棒の表面で最大 `y_max = d/2` となります。これらを公式に代入します。

**解答**

  * **最大の曲げ応力が生じる位置**: 中央の支持点 **C**
  * **最大の曲げ応力 `σ_max`**:
    $$\sigma_{max} = \frac{|M|_{max} y_{max}}{I} = \frac{(wl^2/8)(d/2)}{I} = \frac{wl^2 d}{16I}$$

### **【問題3】薄肉円筒の応力とひずみ**

この問題は、内側から圧力を受けたり、軸方向に引っ張られたりする薄い円筒（圧力容器など）の応力とひずみの関係を扱います。

**キーポイント** 💡
軸方向の力と内圧は、それぞれ**軸方向**と**円周方向**の両方にひずみを生じさせます（ポアソン効果）。全体のひずみは、これらの効果を**重ね合わせる**ことで求まります。

\<img src="[https://storage.googleapis.com/gemini-prod/images/a639668d-8c10-4228-98e9-d368e5d070b4](https://www.google.com/search?q=https://storage.googleapis.com/gemini-prod/images/a639668d-8c10-4228-98e9-d368e5d070b4)" alt="Diagram for Problem 3" width="500"/\>

#### **(1) 軸方向荷重Wのみによるひずみ**

**考え方**
これは、単純な棒の引張と同じです。

  * **軸方向ひずみ**: フックの法則 `ε = σ/E` を使います。
  * **円周方向ひずみ**: 軸方向に伸びると、ポアソン効果で円周方向（半径方向）には縮みます。`ε_θ = -ν * ε_z`

**解答**

  * **軸方向のひずみ `ε_z1`**:
    $$\epsilon_{z1} = \frac{\sigma_z}{E} = \frac{W}{2\pi rtE}$$
  * **円周方向のひずみ `ε_θ1`**:
    $$\epsilon_{\theta1} = -\nu \epsilon_{z1} = -\frac{\nu W}{2\pi rtE}$$

-----

#### **(2) 内圧pのみによるひずみ**

**考え方**
内圧 `p` は、円筒を引き裂こうとする「円周方向応力 `σθ`」と、両端の蓋を吹き飛ばそうとする「軸方向応力 `σz`」を発生させます。

  * `σθ = pr/t`
  * `σz = pr/(2t)`
    これらの2つの応力が、それぞれの方向にひずみを生じさせます。計算には「一般化フックの法則」を使います。

**解答**

  * **軸方向のひずみ `ε_z2`**:
    $$\epsilon_{z2} = \frac{1}{E}(\sigma_z - \nu\sigma_\theta) = \frac{pr}{Et}\left(\frac{1}{2} - \nu\right)$$
  * **円周方向のひずみ `ε_θ2`**:
    $$\epsilon_{\theta2} = \frac{1}{E}(\sigma_\theta - \nu\sigma_z) = \frac{pr}{Et}\left(1 - \frac{\nu}{2}\right)$$

-----

#### **(3) 半径が変化しないためのWとpの関係**

**考え方**
円筒の半径が変化しない、ということは、**円周方向のひずみがゼロ**になるということです。
重ね合わせの原理により、(1)で求めた円周方向ひずみ（縮み）と、(2)で求めた円周方向ひずみ（伸び）の合計がゼロになればよい、という式を立てます。
`ε_θ1 + ε_θ2 = 0`

**解答**
半径が変化しないための `W` と `p` の関係は以下の通りです。
$$-\frac{\nu W}{2\pi rtE} + \frac{pr}{Et}\left(1 - \frac{\nu}{2}\right) = 0$$これを `W` について整理すると、$$W = \frac{2\pi r^2 p}{\nu}\left(1 - \frac{\nu}{2}\right)$$

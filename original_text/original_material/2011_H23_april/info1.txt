### **【問題1】はめあい構造の熱応力**

この問題は、性質の異なる2つの部品（中空円筒と中実円柱）を組み合わせ、温度を上げたときに、内部にどのような力（応力）が発生するかを考える問題です。部品間に初期すきまがあるのが特徴です。

**この問題の最大のポイント** 💡
温度が上がると、熱膨張係数が大きい中実円柱Bのほうが、中空円筒Aよりもたくさん伸びようとします。最初はすきまがあるので自由に伸びますが、すきまが無くなった後は、**AとBがお互いを拘束し合いながら一緒に伸びていく**ようになります。

\<img src="[https://storage.googleapis.com/gemini-prod/images/03b1e779-7973-455b-b9f2-2b28c5c7d013](https://storage.googleapis.com/gemini-prod/images/03b1e779-7973-455b-b9f2-2b28c5c7d013)" alt="Diagram for Problem 1" width="400"/\>

#### **(1) すきまがちょうど無くなるときの温度上昇と長さ**

**考え方**

1.  **伸びの差**: 中実円柱Bが、中空円筒Aよりも、ちょうどすきま `Δl` の分だけ多く伸びた瞬間に、Bの先端が上の円板Cに接触します。
2.  **それぞれの伸び**:
      * Aの伸び: `δ_A = α_A * l * ΔT₁`
      * Bの伸び: `δ_B = α_B * (l-Δl) * ΔT₁`
3.  **条件式**: Bの伸びがAの伸びより `Δl` 大きくなるので、`δ_B - δ_A = Δl` という式が成り立ちます。
4.  **解を求める**: この式を `ΔT₁` について解きます。また、そのときのAの長さ `l'` は、元の長さ `l` にAの伸び `δ_A` を足したものです。

**解答**

  * すきまがちょうど無くなるときの温度上昇 `ΔT₁`:
    $$\Delta T_1 \approx \frac{\Delta l}{l(\alpha_B - \alpha_A)}$$
    （ただし、`l >> Δl` と近似しています）
  * そのときの中空円筒Aの長さ `l'`:
    $$l' = l(1 + \alpha_A \Delta T_1) = l + \frac{\alpha_A \Delta l}{\alpha_B - \alpha_A}$$

-----

#### **(2) さらに温度上昇させたときの応力と伸び**

**考え方**
接触後、さらに温度が `ΔT₂` 上昇すると、Bはもっと伸びたいのにAがそれを妨げ、Aはあまり伸びたくないのにBに引っ張られる、という状態になります。その結果、**Bには圧縮応力**が、**Aには引張応力**が発生します。

1.  **力のつり合い**: Aに発生する引張力とBに発生する圧縮力の大きさは等しくなります。
2.  **変形のつり合い（キーポイント！）**: 最終的に、AとBは一体となって変形するため、この `ΔT₂` の温度上昇による**最終的な伸びの量は両者で等しくなります**。
3.  **伸びの内訳**:
      * Aの伸び: `(熱による伸び) + (引張力による伸び)`
      * Bの伸び: `(熱による伸び) - (圧縮力による縮み)`
4.  **連立して解く**: 「力のつり合い」と「変形のつり合い」から、発生する応力と最終的な伸びを計算します。

**解答**

  * A材に生じる軸応力 `σ_A` (引張):
    $$\sigma_A = \frac{(\alpha_B - \alpha_A)E_A E_B S_B \Delta T_2}{E_A S_A + E_B S_B}$$
  * B材に生じる軸応力 `σ_B` (圧縮):
    $$\sigma_B = -\frac{(\alpha_B - \alpha_A)E_A S_A E_B \Delta T_2}{E_A S_A + E_B S_B}$$
  * A材の伸び `λ_A`:
    $$\lambda_A = \frac{(\alpha_A E_A S_A + \alpha_B E_B S_B)l \Delta T_2}{E_A S_A + E_B S_B}$$

### **【問題2】両端固定はりの中央支持**

この問題は、はりの両端が壁に完全に固定され、さらに中央が支えられているという、非常に拘束の強い「不静定はり」です。はりの形状や荷重が左右対称であるため、**対称性**を利用して解くのが鍵となります。

\<img src="[https://storage.googleapis.com/gemini-prod/images/4566c1f1-b962-421f-bc45-866465492d1c](https://storage.googleapis.com/gemini-prod/images/4566c1f1-b962-421f-bc45-866465492d1c)" alt="Diagram for Problem 2" width="500"/\>

#### **(1) 曲げモーメント M(x)**

**考え方**
はりの左端Aから `x` の距離にある断面で、モーメントのつり合いを考えます。この断面より左側には、壁からの反力 `RA`、壁からの固定モーメント `MA`、そして長さ `x` にわたる分布荷重 `w` が作用しています。これらの力がつくるモーメントをすべて足し合わせます。

**解答**
曲げモーメント `M(x)` は、`0 ≤ x ≤ l` の範囲で以下のようになります。
$$M(x) = R_A x - M_A - \frac{wx^2}{2}$$

-----

#### **(2) 反力 RA と曲げモーメント MA**

**考え方**
この問題を解くには、はりの変形（たわみ）の条件を使います。

1.  **たわみの基礎式**: `EIy'' = M(x)` という微分方程式を立てます。
2.  **境界条件**:
      * **A点 (`x=0`)**: 壁に固定されているので、**たわみ `y` も傾き `y'` もゼロ**です。
      * **C点 (`x=l`)**: 支えられており、かつ対称性の中心なので、**たわみ `y` も傾き `y'` もゼロ**になります。
3.  **計算**: `M(x)` の式を2回積分して、たわみの式 `y(x)` を導出します。そして、上記の境界条件を適用することで、未知数だった `RA` と `MA` を決定できます。

**解答**

  * 反力 `RA`:
    $$R_A = \frac{wl}{2}$$
  * 曲げモーメント `MA`:
    $$M_A = \frac{wl^2}{12}$$

-----

#### **(3) C点の反力 RC、曲げモーメント MC、およびSFD, BMD**

**考え方**

  * **RCの計算**: (2)で `RA` が求まりました。対称性から `RB = RA` です。はり全体にかかる力（`w × 2l`）と、支点が支える力（`RA + RB + RC`）がつり合うことから `RC` を求めます。
  * **MCの計算**: (1)の `M(x)` の式に、(2)で求めた `RA`, `MA` の値と `x=l` を代入して計算します。
  * **SFD（せん断力図）**: `V(x) = RA - wx` の式から、力の分布を描きます。
  * **BMD（曲げモーメント図）**: `M(x)` の式から、モーメントの分布を描きます。

**解答**

  * **C点の反力 `RC`**:
    $$R_C = wl$$
  * **C点の曲げモーメント `MC`**:
    $$M_C = -\frac{wl^2}{12}$$
  * **SFDとBMD**:
      * **SFD**: A点で `wl/2` から始まり、C点で `-wl/2` まで直線的に減少。C点で反力 `RC` により `wl/2` まで跳ね上がり、B点で `-wl/2` まで再び減少します。
      * **BMD**: A点とB点で `-wl²/12`、C点でも `-wl²/12` となり、AC間とCB間のそれぞれの中央 (`x=l/2`, `x=3l/2`) で `+wl²/24` の最大値をとる、W字型の曲線になります。

### **【問題3】中実丸棒と中空丸棒の比較**

この問題は、同じ外径を持つ「中実（中身が詰まった）丸棒」と「中空（中が空洞の）丸棒」について、引張とねじりに対する強さや変形のしやすさを比較する問題です。

#### **(1) 丸棒Aの伸び λA**

**考え方**
棒の伸びは、基本公式 `λ = PL / (AE)` で計算します。`A` は中実丸棒の断面積 `πd²/4` です。

**解答**
$$\lambda_A = \frac{Pl}{E(\pi d^2 / 4)} = \frac{4Pl}{\pi d^2 E}$$

-----

#### **(2) 中空丸棒Bの伸び λB**

**考え方**
(1)と同じ公式を使いますが、断面積 `A` は中空断面のものを使います。中空断面の面積は「（外側の円の面積）-（内側の円の面積）」で計算します。

**解答**
中空丸棒Bの断面積は `A_B = π/4 * (d² - (d/2)²) = 3πd²/16` なので、
$$\lambda_B = \frac{Pl}{E(3\pi d^2 / 16)} = \frac{16Pl}{3\pi d^2 E}$$

-----

#### **(3) 伸びの比 λB/λA**

**考え方**
(2)で求めた `λB` を(1)で求めた `λA` で割ります。

**解答**
$$\frac{\lambda_B}{\lambda_A} = \frac{16Pl / (3\pi d^2 E)}{4Pl / (\pi d^2 E)} = \frac{16}{3} \times \frac{1}{4} = \frac{4}{3}$$
（中空にすると、同じ荷重でも伸びやすくなることがわかります）

-----

#### **(4) 丸棒Aのねじりモーメント T**

**考え方**
ねじりモーメント `T` と、単位長さあたりのねじれ角 `θ` の関係は `T = G * Jp * θ` で表されます。`Jp` は断面二次極モーメントといい、ねじりにくさを表す断面の指標です。中実丸棒の `Jp` は `πd⁴/32` です。

**解答**
$$T = G \theta_A J_p = G \theta_A \frac{\pi d^4}{32}$$

-----

#### **(5) 中空丸棒Bのねじりモーメント T**

**考え方**
(4)と同様に `T = G * Jp * θ` の公式を使いますが、`Jp` は中空丸棒のものを計算します。「（外側の円のJp）-（内側の円のJp）」で求めます。

**解答**
中空丸棒Bの断面二次極モーメントは `Jp_B = π/32 * (d⁴ - (d/2)⁴) = 15πd⁴/512` なので、
$$T = G \theta_B J_{pB} = G \theta_B \frac{15\pi d^4}{512}$$

-----

#### **(6) 比ねじれ角の比 θB/θA**

**考え方**
(4)と(5)で求めた `T` の式は、どちらも同じトルク `T` を表しています。したがって、`(4)の式の右辺 = (5)の式の右辺` という等式を立て、`θB/θA` について解きます。

**解答**
$$\frac{\theta_B}{\theta_A} = \frac{J_{pA}}{J_{pB}} = \frac{\pi d^4 / 32}{15\pi d^4 / 512} = \frac{1}{32} \times \frac{512}{15} = \frac{16}{15}$$
（同じトルクをかけた場合、中空の方が少しだけねじれにくい（比ねじれ角が小さい）ことを示唆していますが、差はわずかです）

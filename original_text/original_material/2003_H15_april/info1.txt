### 【１】組み合わせ棒の問題

**問題文**
図1に示すように、長さ `l` の中実円柱Aと中空円筒Bの軸を一致させて配置し、両端に剛体円板 C, D を固定した組合わせ棒について、次の問に答えよ。
ただし、A材およびB材の縦弾性係数、線膨張係数、断面積をそれぞれ `E₁`, `α₁`, `A₁` および `E₂`, `α₂`, `A₂` とし、各部材の自重は考慮しないものとする。

(1) 剛体円板 C, D に軸方向に引張荷重 `P` が作用する場合、A材およびB材の軸応力 `σ₁` および `σ₂` を求め、さらに組合わせ棒の伸び `λ` を求めよ。
(2) 荷重の作用しない状態で、中空円筒Bだけが温度 `Δt` 上昇するとき、A材およびB材の軸応力 `σ₁` および `σ₂` を求め、さらに組合わせ棒の伸び `λ` を求めよ。

#### (1) 引張荷重が作用する場合

**方針・考え方**
この問題は、異なる材料でできた2つの部材が一体となって変形する「不静定問題」の一種です。このタイプの問題を解くには、以下の2つの条件から連立方程式を立てるのが定石です。

1.  **力のつり合い**: 各部材が分担する力の合計は、外力に等しい。
2.  **変形の適合条件**: 両端が剛体円板で固定されているため、A材とB材の伸びは等しい。

**途中式と計算過程**
1.  **力のつり合いの式を立てる**
    A材が分担する力を `P₁`、B材が分担する力を `P₂` とすると、全体の引張荷重 `P` との間に次の関係が成り立ちます。
    `P₁ + P₂ = P`  --- (式a)

2.  **変形の適合条件の式を立てる**
    A材の伸びを `λ₁`、B材の伸びを `λ₂` とします。両部材は一体なので、伸びは等しくなります。これを `λ` とおきます。
    `λ₁ = λ₂ = λ`
    棒の伸びの公式 `λ = (PL) / (AE)` を使うと、
    `λ₁ = (P₁ * l) / (A₁ * E₁)`
    `λ₂ = (P₂ * l) / (A₂ * E₂)`
    よって、
    `(P₁ * l) / (A₁ * E₁) = (P₂ * l) / (A₂ * E₂)`
    両辺から `l` を消去して、
    `P₁ / (A₁E₁) = P₂ / (A₂E₂)`
    この式を `P₂` について解くと、
    `P₂ = (A₂E₂ / A₁E₁) * P₁` --- (式b)

3.  **連立方程式を解いて P₁, P₂ を求める**
    (式b)を(式a)に代入します。
    `P₁ + (A₂E₂ / A₁E₁) * P₁ = P`
    `P₁ * (1 + A₂E₂ / A₁E₁) = P`
    `P₁ * ( (A₁E₁ + A₂E₂) / A₁E₁ ) = P`
    これを解いて `P₁` が求まります。
    `P₁ = (A₁E₁ / (A₁E₁ + A₂E₂)) * P`

    同様に、`P₂` も求めます。(式a)から `P₁ = P - P₂` として代入しても良いですし、対称性から類推してもOKです。
    `P₂ = (A₂E₂ / (A₁E₁ + A₂E₂)) * P`

4.  **軸応力 σ₁, σ₂ を求める**
    応力は「力 ÷ 断面積」なので、
    `σ₁ = P₁ / A₁ = (E₁ / (A₁E₁ + A₂E₂)) * P`
    `σ₂ = P₂ / A₂ = (E₂ / (A₁E₁ + A₂E₂)) * P`

5.  **伸び λ を求める**
    伸び `λ` は `λ₁` または `λ₂` のどちらかを計算すれば求まります。
    `λ = λ₁ = (P₁ * l) / (A₁E₁) = { (A₁E₁ / (A₁E₁ + A₂E₂)) * P * l } / (A₁E₁)`
    `λ = (P * l) / (A₁E₁ + A₂E₂)`

**解答**
*   軸応力: `σ₁ = (E₁ * P) / (A₁E₁ + A₂E₂)`, `σ₂ = (E₂ * P) / (A₁E₁ + A₂E₂)`
*   伸び: `λ = (P * l) / (A₁E₁ + A₂E₂)`

**要点・チップス**
分母の `A₁E₁ + A₂E₂` は「組合せ棒全体の引張剛性」と見なすことができます。全体の伸び `λ = P l / (AE)` の形と見比べると理解しやすいですね。

#### (2) B材のみ温度上昇する場合

**方針・考え方**
今度は熱応力の問題です。これも不静定問題で、考え方は(1)と似ています。

1.  **もし拘束がなかったら？**: まず、B材が自由に熱膨張した場合を想像します。その伸びは `λ_therm = α₂ * l * Δt` です。
2.  **拘束による力の発生**: 実際にはA材と一体なので、B材は自由に伸びることができません。B材が伸びようとすると、それを妨げるようにA材から**圧縮力**を受けます。作用反作用の法則により、A材はB材から**引張力**を受けます。
3.  **力のつり合い**: 外力は無いので、A材に生じる引張力とB材に生じる圧縮力の大きさは等しくなります。この内力を `P_internal` とします。
4.  **変形の適合条件**: 最終的に、A材が引張力で伸びる量と、B材が「熱膨張する量」から「圧縮力で縮む量」を引いた量が等しくなります。

**途中式と計算過程**
1.  **力のつり合い**
    A材に生じる引張力を `P₁`、B材に生じる圧縮力を `P₂` とすると、その大きさは等しいので、
    `P₁ = P₂ = P_internal` (大きさを `P_i` と書くことにします)

2.  **変形の適合条件**
    最終的な棒の伸び `λ` は、A材、B材で等しくなります。
    `λ = λ₁ = λ₂`
    *   A材の伸び `λ₁` は、引張力 `P_i` による機械的な伸びのみです。
        `λ₁ = (P_i * l) / (A₁E₁)`
    *   B材の伸び `λ₂` は、熱膨張と圧縮力 `P_i` による縮みの合計です。
        `λ₂ = (熱による伸び) - (圧縮による縮み) = α₂*l*Δt - (P_i * l) / (A₂E₂)`

    `λ₁ = λ₂` なので、
    `(P_i * l) / (A₁E₁) = α₂*l*Δt - (P_i * l) / (A₂E₂)`

3.  **内力 P_i を求める**
    上の式の両辺から `l` を消去し、`P_i` について整理します。
    `P_i / (A₁E₁) + P_i / (A₂E₂) = α₂*Δt`
    `P_i * (1/(A₁E₁) + 1/(A₂E₂)) = α₂*Δt`
    `P_i * ( (A₁E₁ + A₂E₂) / (A₁E₁ * A₂E₂) ) = α₂*Δt`
    `P_i = (A₁E₁ * A₂E₂ / (A₁E₁ + A₂E₂)) * α₂*Δt`

4.  **軸応力 σ₁, σ₂ を求める**
    応力は「力 ÷ 断面積」です。A材は引張なので正、B材は圧縮なので負になることに注意してください。
    `σ₁ = P_i / A₁ = (E₁ * A₂E₂ / (A₁E₁ + A₂E₂)) * α₂*Δt` (引張応力)
    `σ₂ = -P_i / A₂ = -(E₂ * A₁E₁ / (A₁E₁ + A₂E₂)) * α₂*Δt` (圧縮応力)

5.  **伸び λ を求める**
    `λ` は `λ₁` を計算すればOKです。
    `λ = (P_i * l) / (A₁E₁) = { (A₁E₁ * A₂E₂ / (A₁E₁ + A₂E₂)) * α₂*Δt * l } / (A₁E₁)`
    `λ = (A₂E₂ / (A₁E₁ + A₂E₂)) * α₂*l*Δt`

**解答**
*   軸応力: `σ₁ = (A₂E₁E₂α₂Δt) / (A₁E₁ + A₂E₂) ` (引張), `σ₂ = -(A₁E₁E₂α₂Δt) / (A₁E₁ + A₂E₂)` (圧縮)
*   伸び: `λ = (A₂E₂α₂lΔt) / (A₁E₁ + A₂E₂)`

### 【2】一端固定他端支持はりの問題

**問題文**
図2に示すように、AC間に等分布荷重wを受ける長さ2lの一端固定他端支持はりABについて、縦弾性係数をEおよび断面二次モーメントをIとして次の問に答えよ。

(1) 図のようにx軸をとり、B点の反力 `R_B` を未知数として、はりの曲げモーメント `M(x)` を表せ。
(2) たわみの基礎式および境界条件を示して、B点の反力 `R_B` を求めよ。
(3) A点の反力 `R_A` および曲げモーメント `M_A` を求め、はりのせん断力線図（SFD）および曲げモーメント線図（BMD）を描け。

#### (1) 曲げモーメント M(x) の表現

**方針・考え方**
このはりは不静定なので、力のつり合いだけでは反力を決められません。そこで、ひとまずB点の反力 `R_B` を未知数（まだ分からない値）として扱います。
A点からxの位置での曲げモーメント `M(x)` を、`R_B` を使って表現します。このとき、A点の反力 `R_A` とモーメント `M_A` も未知数ですが、これらも力のつり合いから `R_B` で表すことで、`M(x)` を `R_B` だけの式で表現できます。

**途中式と計算過程**
1.  **A点の反力 `R_A` と `M_A` を `R_B` で表す**
    *   **上下方向の力のつり合い (ΣF_y = 0):**
        `R_A + R_B - w*l = 0`  =>  `R_A = wl - R_B`
    *   **A点周りのモーメントのつり合い (ΣM_A = 0):**
        固定端の反力モーメントを `M_A`（時計回りを正とする）、反時計回りのモーメントを正として式を立てると、
        `-M_A + R_B*(2l) - (w*l)*(l/2) = 0`
        `M_A = 2l*R_B - wl²/2`

2.  **曲げモーメント M(x) を求める**
    区間ACとCBで分布荷重の状況が違うため、場合分けして考えます。曲げモーメントは下に凸を正とします。
    *   **区間AC (0 ≦ x ≦ l):**
        A点からxの位置で切断して左側を考えます。
        `M(x) = R_A * x - M_A - w*x²/2`
        ここに、先ほど求めた `R_A` と `M_A` の式を代入します。
        `M(x) = (wl - R_B)x - (2l*R_B - wl²/2) - wx²/2`
        `M(x) = wlx - R_Bx - 2lR_B + wl²/2 - wx²/2`
    *   **区間CB (l ≦ x ≦ 2l):**
        B点から `(2l-x)` の位置で切断して右側を考えた方が簡単です。
        `M(x) = R_B * (2l - x)`

**解答**
はりの曲げモーメント `M(x)` は以下のようになります。
*   **区間AC (0 ≦ x ≦ l):** `M(x) = (wl - R_B)x - (2lR_B - wl²/2) - wx²/2`
*   **区間CB (l ≦ x ≦ 2l):** `M(x) = R_B (2l - x)`

**(注)** 解き方によっては `M(x) = R_B(2l-x) - w(l-x)²/2` (for 0≤x≤l) のように、`x` の右側で考える方法もありますが、結果は同じです。

#### (2) B点の反力 RB の計算

**方針・考え方**
たわみの微分方程式（基礎式）`EI * d²y/dx² = M(x)` を解いて反力 `R_B` を求めます。(1)で求めた `M(x)` は区間によって式が異なるため、積分も区間ごとに行い、「連続条件」を使ってつなぎ合わせます。

**たわみの基礎式:**
`EI * (d²y / dx²) = M(x)`
ここで `y` はたわみ、`E` は縦弾性係数、`I` は断面二次モーメントです。

**境界条件:**
物理的な拘束条件を数式で表したものです。
1.  `y(0) = 0` （A点は固定端なので、たわみはゼロ）
2.  `y'(0) = 0` （A点は固定端なので、たわみ角（傾き）もゼロ）
3.  `y(2l) = 0` （B点は支持端なので、たわみはゼロ）

**計算過程 (概要)**
この計算は非常に長くなるため、ここでは一般的な解法である**重ね合わせの原理**を用いて `R_B` を求め、(2)の問いに答えます。直接積分法でも同じ結果が得られます。

**重ね合わせの原理による解法:**
この問題を、以下の2つの単純な片持ちはりの問題の足し合わせと考えます。
*   **ケース1:** 長さ2lの片持ちはりのAC間に等分布荷重 `w` が作用する場合。このときのB点(先端)の下向きのたわみを `δ_w` とする。
*   **ケース2:** 長さ2lの片持ちはりの先端Bに上向きの力 `R_B` が作用する場合。このときのB点の上向きのたわみを `δ_R` とする。

実際のはりではB点のたわみはゼロなので、`δ_w = δ_R` となります。

1.  **`δ_w` の計算 (分布荷重によるたわみ):**
    分布荷重による先端たわみの公式を積分して求めます。
    `δ_w = ∫[0 to l] (wx'²/ (6EI)) * (3*(2l) - x') dx'` （ここで `x'` はA点からの距離）
    計算すると、`δ_w = (7wl⁴) / (24EI)` となります。

2.  **`δ_R` の計算 (RBによるたわみ):**
    片持ちはりの先端荷重によるたわみの公式 `PL³/3EI` を使います。
    `δ_R = (R_B * (2l)³) / (3EI) = (8l³ * R_B) / (3EI)`

3.  **`δ_w = δ_R` から R_B を求める:**
    `(7wl⁴) / (24EI) = (8l³ * R_B) / (3EI)`
    両辺の `EI`, `l³` を消去して `R_B` について解きます。
    `(7wl) / 24 = (8 * R_B) / 3`
    `R_B = (7wl / 24) * (3 / 8) = 21wl / 192 = 7wl / 64`

**解答**
たわみの基礎式 `EIy''=M(x)` と境界条件 `y(0)=0`, `y'(0)=0`, `y(2l)=0` を用いて解くと、B点の反力は以下のように求められる。
`R_B = 7wl / 64`

#### (3) R_A, M_A, SFD, BMD

**方針・考え方**
`R_B` の値が確定したので、(1)で立てた力のつり合いの式に戻って `R_A` と `M_A` を計算します。その後、各区間のせん断力 `V(x)` と曲げモーメント `M(x)` を計算し、SFD（せん断力線図）とBMD（曲げモーメント線図）を描きます。

**途中式と計算過程**
1.  **R_A と M_A の計算**
    `R_B = 7wl / 64` を代入します。
    *   `R_A = wl - R_B = wl - 7wl/64 = 57wl / 64`
    *   `M_A = 2l*R_B - wl²/2 = 2l*(7wl/64) - wl²/2 = 14wl²/64 - 32wl²/64 = -18wl²/64 = -9wl² / 32`
        (マイナスは、仮定した向きと逆、つまりはりを上に凸にするモーメントであることを示す)

2.  **SFD (せん断力線図)**
    *   **A点 (x=0):** `V_A = R_A = 57wl / 64`
    *   **区間AC (0<x<l):** `V(x) = R_A - wx = 57wl/64 - wx` (直線的に減少)
    *   **C点 (x=l):** `V_C = 57wl/64 - wl = -7wl / 64`
    *   **区間CB (l<x<2l):** 分布荷重がないのでせん断力は一定。`V(x) = -7wl / 64`
    *   **B点 (x=2l):** `R_B = 7wl/64` の上向き反力で `0` に戻る。

3.  **BMD (曲げモーメント線図)**
    *   **A点 (x=0):** 固定端モーメント `M(0) = -M_A = -9wl² / 32`
    *   **区間AC (0<x<l):** SFDを積分した形（二次関数、下に凸の放物線）。
    *   **C点 (x=l):** `M(l) = M(0) + ∫[0 to l] V(x)dx = -9wl²/32 + ∫[0 to l](57wl/64 - wx)dx = 7wl² / 64`
    *   **区間CB (l<x<2l):** SFDが一定値なので、BMDは直線。
        `M(x) = M(l) + V_C * (x-l) = 7wl²/64 - (7wl/64)(x-l)`
    *   **B点 (x=2l):** `M(2l) = 7wl²/64 - (7wl/64)(2l-l) = 0`。支持端なので `0` になり、計算が合っていることが確認できる。

**解答**
*   A点の反力: `R_A = 57wl / 64`
*   A点の曲げモーメント: `M_A = 9wl² / 32` (時計回り、上に凸)
*   **SFD:**
    *   x=0で `+57wl/64` から始まり、x=lで `-7wl/64` まで直線的に減少。
    *   x=lからx=2lまでは `-7wl/64` で一定。
*   **BMD:**
    *   x=0で `-9wl²/32` から始まり、x=lで `+7wl²/64` まで二次曲線で変化。
    *   x=lからx=2lまでは `0` まで直線的に減少。

### 【3】中実丸棒のねじりの問題

**問題文**
図3に示すように、直径dの中実丸棒にねじりモーメントTが作用している場合について、次の問に答えよ。

(1) 丸棒の円形断面の中心に関する断面二次極モーメント `Ip` (= ∫r²dA, A: 丸棒の断面積、r: 丸棒の中心から微小面積要素 dA までの距離)を求めよ。
(2) 丸棒の比ねじれ角 `θ` を、丸棒の直径 `d`、ねじりモーメント `T` および丸棒材料の横弾性係数 `G` で表せ。
(3) 丸棒表面における応力状態について説明し、最大せん断応力 `τ_max` を求めよ。
(4) 丸棒表面における最大主応力 `σ₁` を求めよ。

#### (1) 断面二次極モーメント Ip

**方針・考え方**
定義式 `Ip = ∫r²dA` に従って、断面全体で積分計算を行います。円形断面なので、極座標を使うと計算が簡単になります。

**途中式と計算過程**
1.  **微小面積 dA を極座標で表す**
    中心からの距離 `r`、角度 `φ` の位置にある微小な面積 `dA` は、`dA = r dr dφ` と表せます。

2.  **定義式を二重積分で計算する**
    積分範囲は、半径方向が `0` から `d/2` まで、角度方向が `0` から `2π` までです。
    `Ip = ∫[A] r² dA = ∫[0 to 2π] ∫[0 to d/2] r² * (r dr dφ)`
    `Ip = ∫[0 to 2π] dφ * ∫[0 to d/2] r³ dr`
    それぞれの積分を計算します。
    *   `∫[0 to 2π] dφ = [φ]_0^2π = 2π`
    *   `∫[0 to d/2] r³ dr = [r⁴ / 4]_0^(d/2) = ( (d/2)⁴ / 4 ) - 0 = (d⁴/16) / 4 = d⁴/64`
    これらを掛け合わせます。
    `Ip = 2π * (d⁴ / 64) = πd⁴ / 32`

**解答**
`Ip = πd⁴ / 32`

#### (2) 比ねじれ角 θ

**方針・考え方**
ねじりに関する基本公式 `T = G * Ip * θ` を使います。ここで `θ` は「比ねじれ角」または「単位長さあたりのねじれ角」と呼ばれる量です。この式を `θ` について解くだけです。

**途中式と計算過程**
1.  **公式の変形**
    `T = G * Ip * θ`  より  `θ = T / (G * Ip)`

2.  **(1)の結果を代入**
    `Ip = πd⁴ / 32` を代入します。
    `θ = T / ( G * (πd⁴ / 32) )`
    `θ = 32T / (Gπd⁴)`

**解答**
`θ = 32T / (Gπd⁴)`

#### (3) 表面の応力状態と最大せん断応力 τ_max

**方針・考え方**
*   **応力状態:** 純粋なねじりモーメントを受ける棒の断面には、せん断応力のみが発生します。このせん断応力は、棒の中心でゼロ、中心からの距離 `r` に比例して大きくなり、表面で最大となります。
*   **最大せん断応力:** せん断応力を求める公式 `τ = Tr / Ip` において、`r` が最大となる表面 (`r = d/2`) での値が `τ_max` となります。

**途中式と計算過程**
1.  **公式に r = d/2 を代入**
    `τ_max = T * (d/2) / Ip`

2.  **(1)の結果を代入**
    `Ip = πd⁴ / 32` を代入します。
    `τ_max = (T * d/2) / (πd⁴ / 32) = (Td / 2) * (32 / πd⁴)`
    `τ_max = 16T / (πd³)`

**解答**
*   **応力状態:** 丸棒表面には、棒の軸に垂直な断面と平行な断面に、互いに直交するせん断応力のみが作用している（純せん断応力状態）。
*   **最大せん断応力:** `τ_max = 16T / (πd³)`

#### (4) 表面の最大主応力 σ₁

**方針・考え方**
丸棒表面は、せん断応力 `τ_max` だけを受ける「純せん断応力状態」です。この状態の主応力は、応力の座標変換式またはモール円を描くことで求められます。
応力成分は `σ_x = 0`, `σ_y = 0`, `τ_xy = τ_max` となります。

**途中式と計算過程**
1.  **主応力の公式**
    `σ₁,₂ = (σ_x + σ_y)/2 ± √[ ( (σ_x - σ_y)/2 )² + τ_xy² ]`

2.  **値を代入**
    `σ_x=0`, `σ_y=0`, `τ_xy = τ_max` を代入します。
    `σ₁,₂ = (0 + 0)/2 ± √[ ( (0 - 0)/2 )² + τ_max² ]`
    `σ₁,₂ = 0 ± √[ τ_max² ] = ± τ_max`

3.  **最大主応力 σ₁ を求める**
    主応力のうち大きい方を最大主応力 `σ₁` とするので、
    `σ₁ = τ_max`

4.  **(3)の結果を代入**
    `σ₁ = 16T / (πd³)`

**解答**
最大主応力 `σ₁ = 16T / (πd³)`

**要点・チップス**
純せん断状態では、最大主応力（引張）と最小主応力（圧縮）の絶対値が、最大せん断応力の大きさに等しくなります。これらの主応力は、棒の軸線に対して45°傾いた面に作用します。これは、チョークをねじって折ると45°のらせん状に破断することからも直感的に理解できます。

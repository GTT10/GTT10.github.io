### **【問題1】棒の熱応力**

この問題は、温度変化によって物体が伸び縮みしようとするのを、壁などで拘束したときに内部に発生する力（応力）について考えます。

**この問題の最大のポイント** 💡
「もし拘束がなかったら自由に伸び縮みするはずの量」が、「拘束があるために発生する力によって無理やり押し縮められる量」と等しくなる、という考え方で式を立てます。

#### **(1) 拘束された棒1本に生じる応力**

**考え方**

1.  **自由な熱膨張**: もし棒が壁に固定されていなければ、温度が `ΔT` 上昇すると、`δ_T = α₁ * l₁ * ΔT` だけ伸びるはずです [cite: 337]。
2.  **壁による拘束**: 実際には両端が壁で固定されているため、この伸びは許されません。壁は、伸びようとする棒を `δ_T` と同じ量だけ押し縮める力（圧縮力 `P`）を及ぼします。
3.  **力による圧縮**: この力 `P` による縮み量は、公式 `δ_P = P * l₁ / (S₁ * E₁)` で表せます [cite: 337]。
4.  **力の決定**: `δ_T = δ_P` なので、`α₁ * l₁ * ΔT = P * l₁ / (S₁ * E₁)` となります。この式から力 `P` を求めます。
5.  **応力の計算**: 応力 `σ` は「力 ÷ 断面積」なので、`σ₁ = P / S₁` を計算します。

**解答**

棒内に生じる応力 `σ₁`（圧縮応力）は以下の通りです。
$$\sigma_1 = \alpha_1 E_1 \Delta T$$

-----

#### **(2) 連結された棒に生じる応力**

**考え方**

今度は2本の棒が繋がっていますが、基本的な考え方は同じです。棒1だけが温められますが、両端が固定されているため、やはり全体の長さは変わりません。

1.  **各棒の変形を考える**:
      * **棒1**: 温度上昇で `α₁ * l₁ * ΔT` だけ伸びようとしますが、同時に全体に発生する圧縮力 `P` によって `P*l₁/(S₁E₁)` だけ縮みます [cite: 337, 339]。
      * **棒2**: 温度は変わりませんが、棒1から押される力 `P` によって `P*l₂/(S₂E₂)` だけ縮みます [cite: 337, 339]。
2.  **全体の変形がゼロ**: 棒1の正味の変形量と、棒2の変形量（縮み）の合計がゼロになります。
    `(α₁ l₁ ΔT - P l₁/(S₁E₁)) - P l₂/(S₂E₂) = 0`
3.  **応力の計算**: この式を `P` について解き、求まった力 `P` をそれぞれの断面積 `S₁`, `S₂` で割ることで、応力 `σ₁` と `σ₂` を求めます。

**解答**

  * 棒1に生じる応力 `σ₁` (圧縮):
    $$\sigma_1 = \frac{\alpha_1 l_1 \Delta T}{S_1 \left( \frac{l_1}{S_1 E_1} + \frac{l_2}{S_2 E_2} \right)}$$
  * 棒2に生じる応力 `σ₂` (圧縮):
    $$\sigma_2 = \frac{\alpha_1 l_1 \Delta T}{S_2 \left( \frac{l_1}{S_1 E_1} + \frac{l_2}{S_2 E_2} \right)}$$

#### **(3) 連結部Bの変位 λB**

**考え方**

連結部Bがどれだけ動くかは、固定された壁Aから見た「棒1の最終的な長さの変化」を計算すれば分かります。

1.  **棒1の変形の内訳**: 棒1は、熱で伸びようとする一方で、力で縮められています。
2.  **正味の変形量**: 棒1の実際の変形量 `λB` は、「熱による伸び」から「力による縮み」を引いたものです。
    `λB = (熱による伸び) - (力による縮み) = α₁ * l₁ * ΔT - P * l₁/(S₁E₁)`
3.  **計算**: (2)で求めた力 `P` の式をここに代入して整理します。

**解答**

連結部Bの右方向への変位 `λB` は以下の通りです。
$$\lambda_B = \frac{\alpha_1 l_1 l_2 \Delta T}{S_2 E_2 \left( \frac{l_1}{S_1 E_1} + \frac{l_2}{S_2 E_2} \right)}$$

### **【問題2】ばねで支えられた片持ちはり**

この問題は、はりの曲げ（材料力学）と、ばねの伸び（物理学）が組み合わさった問題です。

**この問題の最大のポイント** 💡
外から加えた力 `P` は、**はりが支える分 (`P₁`)** と、**ばねが支える分 (`P₂`)** に分担されます。そして、最終的に「**はりの先端のたわみ**」と「**ばねの伸び**」が**同じになる**点でつり合います。

\<img src="[https://storage.googleapis.com/gemini-prod/images/ffb801a2-63dd-4b3f-8463-b09e23652611](https://storage.googleapis.com/gemini-prod/images/ffb801a2-63dd-4b3f-8463-b09e23652611)" alt="Diagram of a cantilever beam supported by a spring" width="400"/\>

#### **(1) はりのたわみ y₁**

**考え方**
これは、先端に集中荷重 `P₁` がかかる「片持ちはり」のたわみの公式そのものです。

**解答**
はりの自由端におけるたわみ `y₁` は、`P₁` を用いて以下のように表せます。
$$y_1 = \frac{P_1 l^3}{3EI}$$

-----

#### **(2) ばねの伸び λ₂**

**考え方**
これは、力 `P₂` を受けるばねの「フックの法則」そのものです。

**解答**
ばねの伸び `λ₂` は、`P₂` を用いて以下のように表せます。
$$\lambda_2 = \frac{P_2}{k}$$

-----

#### **(3) 自由端Bにおける変位**

**考え方**
(1)と(2)で求めた関係と、力のつり合い、変形のつり合いを組み合わせて解きます。

1.  **力のつり合い**: `P = P₁ + P₂`
2.  **変形のつり合い（キーポイント！）**: はりのたわみとばねの伸びは等しくなります。この最終的な変位を `y_B` とすると、
    `y_B = y₁ = λ₂`
3.  **連立して解く**:
      * `y_B = P₁l³ / (3EI)` より `P₁ = 3EI * y_B / l³`
      * `y_B = P₂ / k` より `P₂ = k * y_B`
      * これらの `P₁` と `P₂` を力のつり合いの式に代入します。
      * `P = (3EI/l³)*y_B + k*y_B = y_B * (k + 3EI/l³)`
4.  **変位を求める**: この式を `y_B` について解きます。

**解答**
自由端Bにおける変位 `y_B` は以下の通りです。
$$y_B = \frac{P}{k + \frac{3EI}{l^3}}$$

### **【問題3】ねじりと曲げを同時に受ける丸棒**

この問題は、1本の棒に「ねじり」と「曲げ」が同時にかかったときに、内部に生じる応力がどうなるかを考えます。

**この問題の最大のポイント** 💡
ねじりによる「せん断応力」と、曲げによる「垂直応力（引張・圧縮）」をそれぞれ計算し、それらを**合成**して、本当に危険な方向の最大の応力（**主応力**）を求める必要があります。

\<img src="[https://storage.googleapis.com/gemini-prod/images/43797c55-b4cc-42b7-8ccb-a3d82c442220](https://storage.googleapis.com/gemini-prod/images/43797c55-b4cc-42b7-8ccb-a3d82c442220)" alt="Diagram of a shaft under combined bending and torsion" width="450"/\>

#### **(1) ねじりのみによる最大せん断応力**

**考え方**
丸棒をねじったときに生じるせん断応力 `τ` は、公式 `τ = Tr / J` で計算できます。応力は中心でゼロ、外表面で最大になります。

  * `T`: ねじりモーメント
  * `r`: 半径 (`d/2`)
  * `J`: 断面二次極モーメント (`πd⁴/32`)

**解答**
最大せん断応力 `τ_max` は以下の通りです。
$$\tau_{max} = \frac{T(d/2)}{\pi d^4 / 32} = \frac{16T}{\pi d^3}$$

-----

#### **(2) 曲げのみによる軸方向の応力**

**考え方**
丸棒を曲げたときに生じる軸方向の応力 `σx` は、公式 `σx = My / I` で計算できます。

  * `M`: 曲げモーメント
  * `y`: 中立面からの距離。角度 `θ` の位置なので `y = (d/2)sinθ` となります。
  * `I`: 断面二次モーメント (`πd⁴/64`)

**解答**
中立面から角度 `θ` だけ傾いた丸棒外表面に生じる軸方向の応力 `σx` は以下の通りです。
$$\sigma_x = \frac{M y}{I} = \frac{M (d/2 \sin\theta)}{\pi d^4 / 64} = \frac{32M \sin\theta}{\pi d^3}$$

-----

#### **(3) 同時作用時の最大主応力と最大せん断応力**

**考え方**
「ねじりによるせん断応力 `τ`」と「曲げによる垂直応力 `σ`」が同時にかかる点の応力状態を考えます。この合成された応力状態から、その点における最大の引張応力（最大主応力）と最大のせん断応力を公式で求めます。

**解答**

  * **最大主応力 `σ_max`**:
    $$\sigma_{max} = \frac{\sigma_x}{2} + \sqrt{\left(\frac{\sigma_x}{2}\right)^2 + \tau^2} = \frac{16M}{\pi d^3} + \sqrt{\left(\frac{16M}{\pi d^3}\right)^2 + \left(\frac{16T}{\pi d^3}\right)^2} = \frac{16}{\pi d^3} \left( M + \sqrt{M^2 + T^2} \right)$$
  * **最大せん断応力 `τ_max`**:
    $$\tau_{max} = \sqrt{\left(\frac{\sigma_x}{2}\right)^2 + \tau^2} = \sqrt{\left(\frac{16M}{\pi d^3}\right)^2 + \left(\frac{16T}{\pi d^3}\right)^2} = \frac{16}{\pi d^3} \sqrt{M^2 + T^2}$$

#### **(4) モールの応力円と主応力の方向**

**考え方**

  * **モールの応力円**: 複雑な応力状態を円でグラフィカルに表現したものです。横軸に垂直応力 `σ`、縦軸にせん断応力 `τ` をとります。2つの点 `(σx, τxy)` と `(σy, -τxy)` をプロットし、それらを結ぶ線分を直径とする円を描きます。
  * **主応力の方向**: 応力円上で、円と横軸が交わる点が主応力を表します。元の状態から主応力の状態まで、応力円上で回転した角度の半分が、実際の材料要素が主応力方向に傾いている角度になります。

**解答**

  * **モールの応力円**:
      * **中心C**: `(σx/2, 0) = (16M/(πd³), 0)`
      * **半径R**: `R = (16/(πd³)) * sqrt(M² + T²)`
  * **最大主応力の方向 `θp`**:
    最大主応力の方向（元の軸からの傾き）`θp` は、以下の関係式で求められます。
    $$\tan(2\theta_p) = \frac{2\tau}{\sigma_x} = \frac{T}{M}$$
    したがって、角度は
    $$\theta_p = \frac{1}{2} \arctan\left(\frac{T}{M}\right)$$
    となります。

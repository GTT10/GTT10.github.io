## 【問題1】 3本の丸棒の圧縮

\<img src="/api/download\_file?file\_path=preppy-433b9b-83c9c9/user\_uploads/PAGE\_2\_materials\_2021\_R3\_question.pdf.jpg" alt="Problem 1 Diagram" width="500"/\>

この問題は、材質や断面積が異なる可能性のある3本の棒が、剛体板を介して圧縮される不静定問題です。剛体板によって、3本の棒の縮み量が等しくなるのがポイントです。

### (1) 丸棒の縮み

#### 考え方

1.  **変形の適合条件**: 問題の指示通り、3本の丸棒A, B, Cはすべて同じ量だけ縮みます [cite: 106]。この縮み量を $\\delta$ とします。

2.  **力と変形の関係**: 各丸棒に作用する圧縮力をそれぞれ $P\_A, P\_B, P\_C$ とすると、フックの法則から縮み $\\delta$ は次のように表せます。
    $\\delta = \\frac{P\_A l}{E\_1 A\_1} = \\frac{P\_B l}{E\_2 A\_2} = \\frac{P\_C l}{E\_3 A\_3}$

3.  **力のつり合い**: 外力 $P$ は、各丸棒が支える内力の合計とつり合います。
    $P = P\_A + P\_B + P\_C$

4.  **連立方程式を解く**: 上記の式から各内力 $P\_A, P\_B, P\_C$ を縮み $\\delta$ で表し、力のつり合い式に代入します。

      * $P\_A = \\frac{E\_1 A\_1 \\delta}{l}$
      * $P\_B = \\frac{E\_2 A\_2 \\delta}{l}$
      * $P\_C = \\frac{E\_3 A\_3 \\delta}{l}$

    $P = \\frac{E\_1 A\_1 \\delta}{l} + \\frac{E\_2 A\_2 \\delta}{l} + \\frac{E\_3 A\_3 \\delta}{l} = \\frac{\\delta}{l} (E\_1 A\_1 + E\_2 A\_2 + E\_3 A\_3)$

5.  **縮みを求める**: この式を $\\delta$ について解きます。

#### 解答

丸棒の縮み $\\delta$ は以下のようになります。
$$\delta = \frac{Pl}{E_1 A_1 + E_2 A_2 + E_3 A_3}$$

### (2) 外力Pの作用位置 x

#### 考え方

1.  **モーメントのつり合い**: 剛体板は傾かずに平行に下降するため、どの点を中心に考えてもモーメントはつり合っています。ここでは、計算を簡単にするため、丸棒Aの中心を基準にモーメントのつり合いを考えます。

2.  **力の作用点**: 外力 $P$ は丸棒Aから距離 $x$ の位置に作用します。丸棒B、Cからの反力 $P\_B, P\_C$ は、それぞれ距離 $a, 2a$ の位置に作用します。

3.  **つり合い式**: （反時計回りのモーメント）=（時計回りのモーメント）より、
    $P\_B \\cdot a + P\_C \\cdot (2a) = P \\cdot x$

4.  **内力の代入**: (1)の考え方で導いた $P\_B, P\_C$ と $P$ の関係を代入して $x$ を求めます。

      * $P\_B = \\frac{P E\_2 A\_2}{E\_1 A\_1 + E\_2 A\_2 + E\_3 A\_3}$
      * $P\_C = \\frac{P E\_3 A\_3}{E\_1 A\_1 + E\_2 A\_2 + E\_3 A\_3}$

    これらの式をつり合い式に代入すると、両辺の $P$ が消去できます。

#### 解答

外力Pを作用させた位置 $x$ は以下の通りです。
$$x = \frac{a(E_2 A_2 + 2E_3 A_3)}{E_1 A_1 + E_2 A_2 + E_3 A_3}$$

### (3) 破壊時の外力 Pmax

#### 考え方

1.  **破壊条件**: 丸棒Aがせん断破壊するのは、Aに生じる最大せん断応力がせん断強さ $\\tau\_A$ に達したときです [cite: 109]。一軸圧縮状態において、最大せん断応力は圧縮応力 $\\sigma\_A$ の半分 ($\\tau\_{max} = \\sigma\_A / 2$) です。
    $\\tau\_A = \\frac{\\sigma\_{A,max}}{2}$
2.  **応力と内力の関係**: 丸棒Aの圧縮応力は $\\sigma\_A = P\_A / A\_1$ なので、破壊時の内力 $P\_{A,max}$ は、
    $P\_{A,max} = \\sigma\_{A,max} A\_1 = (2\\tau\_A) A\_1$
3.  **外力との関係**: 設問(1)の考え方から、$P\_A$ と $P$ の間には次の関係があります。
    $P\_A = \\frac{P E\_1 A\_1}{E\_1 A\_1 + E\_2 A\_2 + E\_3 A\_3}$
4.  **最大外力を求める**: この式に $P\_A = P\_{A,max}$ と $P = P\_{max}$ を代入し、$P\_{max}$ について解きます。

#### 解答

丸棒Aがせん断破壊したときの外力 $P\_{max}$ は以下となります。
$$P_{max} = \frac{P_{A,max} (E_1 A_1 + E_2 A_2 + E_3 A_3)}{E_1 A_1} = \frac{2\tau_A A_1 (E_1 A_1 + E_2 A_2 + E_3 A_3)}{E_1 A_1} = 2\tau_A \frac{E_1 A_1 + E_2 A_2 + E_3 A_3}{E_1}$$

## 【問題2】 段付きはりの曲げ

\<img src="/api/download\_file?file\_path=preppy-433b9b-83c9c9/user\_uploads/PAGE\_3\_materials\_2021\_R3\_question.pdf.jpg" alt="Problem 2 Diagram" width="500"/\>

この問題は、両端を単純支持された段付きはりに、三角形の分布荷重が作用する場合の反力、せん断力、曲げモーメント、たわみを求める問題です。系の対称性を利用すると計算が簡略化できます。

### (1) A点の支点反力 $R\_A$

#### 考え方

1.  **荷重の対称性**: はりの形状、支持条件、そして作用する分布荷重が、はりの中心線 ($x=2l$) に関して左右対称です。
2.  **反力の対称性**: このため、両端の支点AとEに生じる反力 $R\_A$ と $R\_E$ は等しくなります。
3.  **全荷重の計算**: 全荷重 $W\_{total}$ は、三角形分布荷重の面積に等しくなります。
      * 底辺の長さ: $3l - l = 2l$
      * 高さ: $x=2l$ での $w$ の値で $w\_0$
      * $W\_{total} = \\frac{1}{2} \\times (\\text{底辺}) \\times (\\text{高さ}) = \\frac{1}{2} \\times (2l) \\times w\_0 = w\_0 l$
4.  **反力の計算**: $R\_A$ と $R\_E$ で全荷重を等しく分担するため、
    $R\_A = R\_E = \\frac{W\_{total}}{2}$

#### 解答

A点に作用する支点反力 $R\_A$ は以下の通りです。
$$R_A = \frac{w_0 l}{2}$$

### (2) BC間のせん断力 $F\_{BC}$

#### 考え方

1.  **せん断力の定義**: はりのある断面におけるせん断力は、その断面より左側にあるすべての鉛直方向の力の総和です。
2.  **力の計算**: BC間 ($l \\le x \\le 2l$) のある位置 $x$ において、その左側には反力 $R\_A$（上向き）と、区間 $[l, x]$ にかかる分布荷重（下向き）があります。
3.  **分布荷重の合力**: 区間 $[l, x]$ の分布荷重の合力は、その部分の面積を積分して求めます。
    $W(x) = \\int\_l^x w(t) dt = \\int\_l^x \\frac{w\_0}{l}(t-l) dt = \\frac{w\_0}{2l}(x-l)^2$
4.  **せん断力の式**: したがって、せん断力 $F\_{BC}(x)$ は $R\_A$ から $W(x)$ を引いたものになります。

#### 解答

BC間のせん断力 $F\_{BC}$ は以下となります。
$$F_{BC}(x) = R_A - \frac{w_0}{2l}(x-l)^2 = \frac{w_0 l}{2} - \frac{w_0}{2l}(x-l)^2$$

### (3) BC間の曲げモーメント $M\_{BC}$

#### 考え方

1.  **モーメントとせん断力の関係**: 曲げモーメント $M(x)$ は、せん断力 $F(x)$ を積分することで求められます。
    $M\_{BC}(x) = \\int F\_{BC}(x) dx + C$ （$C$ は積分定数）
2.  **積分**: (2)で求めた $F\_{BC}(x)$ を積分します。
    $\\int \\left(\\frac{w\_0 l}{2} - \\frac{w\_0}{2l}(x-l)^2\\right) dx = \\frac{w\_0 l}{2}x - \\frac{w\_0}{6l}(x-l)^3 + C$
3.  **積分定数の決定**: 積分定数 $C$ は、境界条件から求めます。点B ($x=l$) におけるモーメント $M\_B$ を考えます。AB間には分布荷重がないため、モーメントは $M\_{AB}(x) = R\_A x = \\frac{w\_0 l}{2}x$ となります。よって、$M\_B = M\_{AB}(l) = \\frac{w\_0 l^2}{2}$ です。
4.  $M\_{BC}(l) = M\_B$ の関係を使って $C$ を求めると、$C=0$ となります。

#### 解答

BC間の曲げモーメント $M\_{BC}$ は以下となります。
$$M_{BC}(x) = \frac{w_0 l}{2}x - \frac{w_0}{6l}(x-l)^3$$

### (4) AB間のたわみ $v\_{AB}$

#### 考え方

1.  **たわみとモーメントの関係**: たわみ $v(x)$ は、たわみの方程式 $EI \\frac{d^2v}{dx^2} = M(x)$ を2回積分することで求められます。
2.  **対称性の利用**: はり全体の対称性から、中央点 ($x=2l$) でのたわみ角（傾き）はゼロになります。この条件を使うと積分定数を効率的に決定できます。
3.  **積分定数の決定**:
      * AB間、BC間のたわみ角の式をそれぞれ求めます（BC間は断面二次モーメントが $2I$ であることに注意）。
      * $x=l$ でのたわみ角の連続条件と、$x=2l$ でのたわみ角ゼロの条件を使って、AB間のたわみ角の式に含まれる積分定数 $C\_2$ を求めます。
      * A点 ($x=0$) でたわみがゼロであることから、たわみの式に含まれるもう一つの積分定数 $C\_3$ がゼロであることがわかります。

#### 解答

AB間のたわみ $v\_{AB}$ は以下の通りです。
$$v_{AB}(x) = \frac{w_0}{48EI}(4lx^3 - 29l^3x)$$

### (5) BC間のたわみ $v\_{BC}$

#### 考え方

1.  **たわみ角の積分**: (4)の途中で求めたBC間のたわみ角の式をさらに積分し、たわみの式を導出します。
    $E(2I) v\_{BC}(x) = \\int \\left( \\frac{w\_0 l}{4}x^2 - \\frac{w\_0}{24l}(x-l)^4 - \\frac{23w\_0 l^3}{24} \\right) dx$
2.  **積分定数の決定**: 新たな積分定数 $C\_5$ は、点B ($x=l$) でのたわみの連続条件 ($v\_{AB}(l) = v\_{BC}(l)$) を使って求めます。(4)で求めた $v\_{AB}(x)$ の式に $x=l$ を代入し、BC間のたわみの式と等しいとおくことで $C\_5$ が決まります。

#### 解答

BC間のたわみ $v\_{BC}$ は以下の通りです。
$$v_{BC}(x) = \frac{1}{2EI} \left( \frac{w_0 l}{12} x^3 - \frac{w_0}{120l}(x-l)^5 - \frac{23 w_0 l^3}{24} x - \frac{w_0 l^4}{6} \right)$$

### **【問題1】3本のワイヤで吊るされた剛体棒**

この問題は、3本のワイヤで吊られた「剛体（変形しない硬い棒）」に力がかかった場合を考えます。ワイヤが3本あるため、単純な力のつり合いだけでは解けない「不静定問題」です。

**この問題の最大のポイント** 💡
棒が「剛体」であるため、力がかかって傾いた後も、A, B, Cの各点の変位（ワイヤの伸び）は、一直線上に並びます。この「**変形の幾何学的な関係**」を追加の条件として使うことで、問題を解くことができます。

\<img src="[https://storage.googleapis.com/gemini-prod/images/5d3c8c73-424d-4c3e-bce1-d7031649931b](https://storage.googleapis.com/gemini-prod/images/5d3c8c73-424d-4c3e-bce1-d7031649931b)" alt="Diagram of suspended rigid bar" width="400"/\>

#### **(1) 各ワイヤの張力**

**考え方**

1.  **力のつり合い**: まず、棒が静止しているための条件を考えます。

      * **上下方向の力のつり合い**: 上向きの力（3本の張力）と下向きの力 `P` がつり合います。
        `Q_AD + Q_BE + Q_CF = P`  --- (式1)
      * **モーメントのつり合い**: 棒が回転しないための条件です。A点を中心に考えると、
        `Q_BE * a + Q_CF * 2a = P * x` --- (式2)
        これだけでは未知数が3つ（`Q_AD`, `Q_BE`, `Q_CF`）あるので解けません。

2.  **変形のつり合い**: ここでポイントを使います。ワイヤAD, BE, CFの伸びをそれぞれ `δ_A`, `δ_B`, `δ_C` とします。棒が剛体なので、これらの伸びの間には `2 * δ_B = δ_A + δ_C` という関係が成り立ちます（中央の伸びは両端の伸びの平均になる）。

3.  **伸びと力の関係**: ワイヤの伸びは張力に比例します (`δ = Ql/AE`)。ワイヤは全て同じなので、`2 * Q_BE = Q_AD + Q_CF` という張力だけの関係式が得られます。--- (式3)

4.  **連立方程式を解く**: (式1), (式2), (式3) の3つの式を連立方程式として解くことで、各張力が求まります。

**解答**

  * ワイヤADの張力 `Q_AD`:
    $$Q_{AD} = \frac{P}{6a}(5a - 3x)$$
  * ワイヤBEの張力 `Q_BE`:
    $$Q_{BE} = \frac{P}{3}$$
  * ワイヤCFの張力 `Q_CF`:
    $$Q_{CF} = \frac{P}{6a}(3x - a)$$

-----

#### **(2) すべてのワイヤがたるまない x の範囲**

**考え方**
ワイヤが「たるまない」とは、ワイヤに引っ張る力（張力）がかかっている状態、つまり**張力が0以上**であるということです。(1)で求めた3つの張力の式が、すべて0以上になるような `x` の範囲を求めます。

**解答**
すべてのワイヤがたるまない `x` の範囲は以下の通りです。
$$\frac{a}{3} \le x \le \frac{5a}{3}$$

-----

#### **(3) 剛体棒の傾き角**

**考え方**
剛体棒の傾き角 `θ` は、棒の両端でのワイヤの伸びの差から計算できます。角度が非常に小さいと仮定すると `θ ≈ tanθ` と近似できます。

1.  **張力の計算**: まず、`x = a/2` のときの各ワイヤの張力を(1)の式から計算します。
2.  **伸びの計算**: 次に、A点とC点でのワイヤの伸び `δ_A` と `δ_C` を「伸び = 張力 × 長さ / (断面積 × 縦弾性係数)」の公式で求めます。
3.  **傾き角の計算**: 傾き角は `θ = (δ_A - δ_C) / 2a` で計算できます。

**解答**
`x=a/2` のときの剛体棒の傾き角 `θ` は以下の通りです。（大きさで回答）
$$\theta = \frac{Pl}{4aAE} \text{ [rad]}$$

### **【問題2】一端固定・他端支持はりの曲げ**

この問題は、片方が壁に固定され、もう片方が支えられている「不静定はり」に、部分的に力がかかる場合の曲げモーメントや反力を求める問題です。これも、力のつり合いだけでは解けないため、はりの変形（たわみ）を考慮する必要があります。

\<img src="[https://storage.googleapis.com/gemini-prod/images/42e8bb33-4f9e-4e36-a320-f4728d82d921](https://storage.googleapis.com/gemini-prod/images/42e8bb33-4f9e-4e36-a320-f4728d82d921)" alt="Diagram of a propped cantilever beam" width="500"/\>

#### **(1) 曲げモーメント M の表現**

**考え方**
B点から `x` の距離にある断面の曲げモーメントを考えます。はりの右側半分 (`0 ≤ x ≤ l/2`) と左側半分 (`l/2 ≤ x ≤ l`) で、力の掛かり方が違うため、式が異なります。

  * **右側半分 (`0 ≤ x ≤ l/2`)**: この区間には外力がかかっていません。`x` の断面から右側を見ると、B点の反力 `RB` しかありません。したがって、モーメントは `RB * x` となります。
  * **左側半分 (`l/2 ≤ x ≤ l`)**: この区間には分布荷重 `w` がかかります。モーメントは、`RB` によるものと、分布荷重 `w` によるものの合計になります。

**解答**
支点Bから距離 `x` の位置における曲げモーメント `M` は以下の通りです。

  * **CB間 (`0 \le x \le l/2`)**:
    $$M(x) = R_B x$$
  * **AC間 (`l/2 \le x \le l`)**:
    $$M(x) = R_B x - \frac{w}{2}\left(x-\frac{l}{2}\right)^2$$

-----

#### **(2) 支点Bの反力 RB**

**考え方**
不静定問題を解くため、はりの「たわみ」の条件を使います。

1.  **境界条件**: 壁A (`x=l`) は完全に固定されているため、「たわみ」も「傾き」もゼロです。また、支点B (`x=0`) は支えられているため、「たわみ」はゼロです。
2.  **たわみの計算**: (1)で求めた曲げモーメントの式を2回積分すると、はりのたわみの式が求まります。
3.  **反力の決定**: 上記の3つの境界条件をたわみの式に適用することで、未知数だった反力 `RB` を求めることができます。この計算は大学の専門レベルの内容を含むため、ここでは結果のみを示します。

**解答**
支点Bの反力 `RB` は以下の通りです。
$$R_B = \frac{7}{128}wl$$

-----

#### **(3) 固定端Aにおける曲げモーメント MA**

**考え方**
(2)で反力 `RB` が求まったので、問題は「静定問題」として扱えるようになります。はり全体についてのモーメントのつり合いを考えることで `MA` を求めます。A点を中心にモーメントのつり合いを考えます。

**解答**
固定端Aにおける曲げモーメント `MA` は以下の通りです。
$$M_A = \frac{9}{128}wl^2$$

### **【問題3】両端が固定された丸棒のねじり**

この問題は、両端を固定された丸棒の途中に、ねじりの力（トルク）をかけたときの変形（ねじれ角）や、壁からの反力を考える問題です。これも「不静定問題」です。

**この問題の最大のポイント** 💡
棒は両端を固定されているため、**全体のねじれ角はゼロ**になります。つまり、「AC間のねじれ角」と「CB間のねじれ角」を足すとゼロになる、という条件を使います。

\<img src="[https://storage.googleapis.com/gemini-prod/images/418f0a0d-2b47-49f3-80f0-3ab6542713f0](https://storage.googleapis.com/gemini-prod/images/418f0a0d-2b47-49f3-80f0-3ab6542713f0)" alt="Diagram of a fixed-fixed shaft with torque" width="350"/\>

#### **(1) 断面二次極モーメント Ip**

**考え方**
断面二次極モーメント `Ip` は、断面の形状がねじりに対してどれだけ強いかを示す値です。直径 `d` の中実円の場合、公式があります。

**解答**
丸棒ABの断面二次極モーメント `Ip` は以下の通りです。
$$I_p = \frac{\pi d^4}{32}$$

-----

#### **(2) C点のねじれ角 (TA で表現)**

**考え方**
ねじれ角 `θ` は、公式 `θ = TL / (GIp)` で計算できます。ここでは、AC区間について考えます。この区間の内部にかかるトルクは、壁Aからの反力トルク `TA` に等しいです。

**解答**
位置Cのねじれ角 `θc` は、A点を基準に考えると以下のようになります。
$$\theta_c = \frac{T_A l_1}{G I_p}$$

-----

#### **(3) ねじりモーメントの比**

**考え方**
ポイントである「全体のねじれ角がゼロ」という条件を使います。
`θ_AC + θ_CB = 0`

  * AC間のねじれ角 `θ_AC` は、トルク `TA` によって生じます。
  * CB間のねじれ角 `θ_CB` は、壁Bからの反力トルク `TB` によって生じます（向きが逆なのでマイナス）。
    この関係式を立てることで、`TA` と `TB` の比が求まります。

**解答**
大きさの比 `|TA| / |TB|` は以下の通りです。
$$\frac{|T_A|}{|T_B|} = \frac{l_2}{l_1}$$
（反力トルクは、距離が遠い方ほど小さくなることがわかります）

-----

#### **(4) C点のねじれ角 (Tc で表現)**

**考え方**
(2)で求めた `θc` の式には未知数 `TA` が含まれています。これを、問題で与えられたトルク `Tc` を使って表します。

1.  **力のつり合い**: `TA + TB = Tc` の関係が成り立ちます。
2.  **TAを求める**: (3)で求めた比の関係 `TB = TA * (l₁/l₂)` を、上のつり合い式に代入して `TA` を `Tc` で表します。
3.  **代入**: 求まった `TA` の式を、(2)の `θc` の式に代入します。

**解答**
位置Cのねじれ角 `θc` を `Tc` で示すと以下の通りです。
$$\theta_c = \frac{T_c l_1 l_2}{G I_p (l_1+l_2)} = \frac{T_c l_1 l_2}{G I_p l}$$

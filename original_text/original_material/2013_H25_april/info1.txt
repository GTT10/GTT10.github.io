### **【問題1】片持ちはりのたわみと重ね合わせ**

この問題は、まず基本的な片持ちはりのたわみ（しなり）を計算させ、その結果を利用して、より複雑な支持条件のはりの問題を解く、という流れになっています。これは「重ね合わせの原理」という、材料力学の非常に重要な考え方を学ぶための問題です。

\<img src="[https://storage.googleapis.com/gemini-prod/images/ff006871-084e-4f52-875f-25c7e39a38f3](https://storage.googleapis.com/gemini-prod/images/ff006871-084e-4f52-875f-25c7e39a38f3)" alt="Diagrams for problem 1" width="600"/\>

#### **(1) 自由端Bのたわみ δB₁** [cite: 423]

**考え方**
これは、先端に集中荷重 `P` がかかる「片持ちはり」のたわみを求める、最も基本的な問題です。この解は公式として扱われることが多いです。

**解答**
はりの先端（自由端）Bにおけるたわみ `δ_B1` は、以下の公式で与えられます。
$$\delta_{B1} = \frac{Pl^3}{3EI}$$

-----

#### **(2) 点Cのたわみ δC₁** [cite: 424]

**考え方**
同じく図1(a)の状況で、今度は途中にあるC点のたわみを求めます。これには、はりのどの位置 `x` でもたわみを計算できる「たわみ曲線」の式を導き、その式に `x=a` を代入します。たわみ曲線は、曲げモーメントの式を2回積分することで得られます。

**解答**
固定端Aから距離 `a` のところにある点Cのたわみ `δ_C1` は以下の通りです。
$$\delta_{C1} = \frac{Pa^2(3l - a)}{6EI}$$

-----

#### **(3) 点Cのたわみ δC₂** [cite: 425]

**考え方**
今度は、はりの途中のC点に上向きの力 `Q` がかかった場合に、そのC点自身がどれだけたわむかを考えます。これも(2)と同様にたわみ曲線を求めることで計算できますが、結果はよく知られた公式の形になります。

**解答**
点Cに上向きの集中荷重 `Q` が作用しているとき、点Cのたわみ `δ_C2` は以下の通りです。（上向きなので、下向きを正とする座標系ではマイナスになります）
$$\delta_{C2} = -\frac{Qa^3}{3EI}$$

-----

#### **(4) 図1(c)の自由端Bにおけるたわみ δB₃** [cite: 426]

**考え方**
これが「重ね合わせの原理」の本番です。図1(c)の状態は、以下の2つの状態を「足し合わせた」ものと考えることができます。

1.  **状態1**: はりの先端Bに下向きの力 `P` がかかる状態（図1(a)）。
2.  **状態2**: はりのC点に、支えからの上向きの反力 `R_C` がかかる状態（図1(b)の `Q` を `R_C` と考える）。

**解法のステップ**

1.  **反力 `R_C` の決定**: 図1(c)では、C点は支えられているので、最終的なC点のたわみはゼロです。つまり、「状態1によるC点の下向きのたわみ」と「状態2によるC点の上向きのたわみ」が、ちょうど同じ大きさになって打ち消しあいます。
    `δ_C1(P) + δ_C2(R_C) = 0`
    この式に(2)と(3)の結果を代入して、未知の反力 `R_C` を `P` を用いて表します。
2.  **B点のたわみの計算**: 最後に、B点のたわみを計算します。これも重ね合わせで、「状態1によるB点のたわみ」と「状態2によるB点のたわみ」を足し合わせます。

**解答**
自由端Bにおけるたわみ `δ_B3` は以下の通りです。
$$\delta_{B3} = \frac{Pl^3}{3EI} - \frac{R_C a^2(3l - a)}{6EI} = \frac{Pl^3}{3EI} - \frac{P(3l-a)}{2a} \frac{a^2(3l-a)}{6EI} = \frac{P(4al-a^2-3l^2)}{12EI}$$
*（計算過程で `R_C = P(3l-a)/(2a)` が求まります）*

### **【問題2】異材接合棒の熱応力**

この問題は、性質の異なる2つの材料を繋げた棒を温めたときに、内部にどのような力（応力）が発生するかを考える問題です。これも「不静定問題」の一種です。

**この問題の最大のポイント** 💡
(1)のように両端が自由な場合は、それぞれの材料が持つ「熱膨張係数」に従って自由に伸びます。しかし、(2)のように両端を壁で固定すると、伸びようとするのを壁が妨げるため、棒の内部に「圧縮する力」が発生します。

\<img src="[https://storage.googleapis.com/gemini-prod/images/05be8d90-b8c7-43ca-a1dd-a030c6fa4d58](https://storage.googleapis.com/gemini-prod/images/05be8d90-b8c7-43ca-a1dd-a030c6fa4d58)" alt="Diagram of a composite rod between two walls" width="500"/\>

#### **(1) 両端自由のときの棒の伸び** [cite: 455]

**考え方**
両端が自由な場合、各棒は他の影響を受けずに、温度上昇 `ΔT` に応じてそれぞれ伸びます。全体の伸びは、単純に2つの棒の伸びの合計です。

  * 棒1の伸び: `α₁ * a * ΔT`
  * 棒2の伸び: `α₂ * b * ΔT`

**解答**
棒の伸び `Δl` は以下の通りです。
$$\Delta l = \alpha_1 a \Delta T + \alpha_2 b \Delta T = (\alpha_1 a + \alpha_2 b)\Delta T$$

-----

#### **(2) 両端固定のときの圧縮応力** [cite: 456]

**考え方**

1.  **自由な熱膨張**: もし壁がなければ、棒は(1)で求めた `Δl` だけ伸びるはずです。
2.  **壁による拘束**: 両端が壁で固定されているため、この `Δl` の伸びが完全に妨げられます。壁は、伸びた分 `Δl` をちょうど元に戻すだけの圧縮力 `P` を棒に及ぼします。
3.  **力による圧縮**: この力 `P` によって、棒1と棒2はそれぞれ縮みます。全体の縮み量は `P*a/(S*E₁) + P*b/(S*E₂)` です。
4.  **つり合い**: 「熱による伸びの合計」と「力による縮みの合計」が等しくなります。
    `Δl = P(a/(SE₁) + b/(SE₂))`
5.  **応力の計算**: この式から力 `P` を求め、断面積 `S` で割ることで圧縮応力 `σc` が求まります。

**解答**
棒に生じる圧縮応力 `σc` は以下の通りです。
$$\sigma_c = \frac{(\alpha_1 a + \alpha_2 b)\Delta T}{\frac{a}{E_1} + \frac{b}{E_2}}$$

-----

#### **(3) 接合部Bの変位** [cite: 457]

**考え方**
接合部Bがどれだけ動くかは、固定された壁Aから見た「棒1の最終的な長さの変化」を計算すれば分かります。

1.  **棒1の変形の内訳**: 棒1は、「熱によって `α₁aΔT` だけ伸びようとする力」と、「(2)で求めた圧縮応力 `σc` によって縮む力」の両方を受けています。
2.  **正味の変形量**: 実際の変位 `δB` は、この2つの変形量の差し引きで決まります。
    `δB = (熱による伸び) - (力による縮み)`

**解答**
接合部Bの水平方向の変位 `δB` は以下の通りです。
$$\delta_B = \alpha_1 a \Delta T - \frac{\sigma_c a}{E_1} = \frac{ab\Delta T(\alpha_1 E_1 - \alpha_2 E_2)}{aE_2 + bE_1}$$

### **【問題3】はりへの物体の落下（衝撃）**

この問題は、静かに力をかけるのではなく、物体を落として「衝撃」を与えたときに、はりにどのような力やエネルギーが生じるかを考えます。

**この問題の最大のポイント** 💡
落下する物体が持っていた「**位置エネルギー**」が、はりを曲げるための「**弾性ひずみエネルギー**」にそっくり変換される、という「**エネルギー保存則**」を使って解きます。

\<img src="[https://storage.googleapis.com/gemini-prod/images/50849929-de95-4672-ad0b-a010461cc153](https://storage.googleapis.com/gemini-prod/images/50849929-de95-4672-ad0b-a010461cc153)" alt="Diagram of a mass falling onto a beam" width="400"/\>

#### **(1) モーメント M(x)** [cite: 468]

**考え方**
衝撃によって、瞬間的にはりに非常に大きな力がかかります。この力の最大値を `P` として、まずは静的な問題として扱います。単純支持ばりの中央に荷重 `P` がかかる場合の曲げモーメントを考えます。

**解答**
はりに生じるモーメント `M(x)` は、区間によって式が異なります。

  * **左半分 (`0 \le x \le l/2`)**:
    $$M(x) = \frac{P}{2}x$$
  * **右半分 (`l/2 \le x \le l`)**:
    $$M(x) = \frac{P}{2}(l-x)$$

-----

#### **(2) 弾性ひずみエネルギー U** [cite: 469]

**考え方**
弾性ひずみエネルギー `U` は、はりが曲がることによって蓄えられるエネルギーです。これは、(1)で求めた曲げモーメント `M(x)` を使って、`U = ∫ [M(x)² / (2EI)] dx` という式を、はりの全長にわたって積分することで計算できます。

**解答**
はりに蓄えられる弾性ひずみエネルギー `U` は以下の通りです。
$$U = \frac{P^2 l^3}{96EI}$$

-----

#### **(3) 最大荷重 P** [cite: 470]

**考え方**
ここでエネルギー保存則を使います。

1.  **物体の位置エネルギー**: 質量 `m` の物体が高さ `h` から落ちるときに失う位置エネルギーは `mg(h + δ)` です。`δ` ははりの最大たわみですが、問題の指示により `h` に比べて十分に小さいとして無視します。したがって、`PE = mgh`。
2.  **エネルギーの変換**: この位置エネルギーが、(2)で求めたはりの弾性ひずみエネルギー `U` に全て変換されます。
    `mgh = U = P²l³ / (96EI)`
3.  **`P` を求める**: この等式を `P` について解きます。

**解答**
最大荷重 `P` は以下の通りです。
$$P = \sqrt{\frac{96EI mgh}{l^3}} = 4\sqrt{\frac{6EImgh}{l^3}}$$

#### **(4) はり中央の最大たわみ δ** [cite: 471]

**考え方**
単純支持ばりの中央に荷重 `P` がかかったときの最大たわみの公式 `δ = Pl³ / (48EI)` に、(3)で求めた `P` を代入します。

**解答**
はり中央の最大たわみ `δ` は以下の通りです。
$$\delta = \sqrt{\frac{mghl^3}{24EI}}$$

#### **(5) 最大曲げ応力** [cite: 472]

**考え方**
最大曲げ応力 `σ_max` は、公式 `σ_max = M_max * y_max / I` で計算します。

  * **最大モーメント `M_max`**: はりの中央で生じ、`M_max = Pl/4` です。
  * **断面係数 `Z = I / y_max`**: 正方形断面の場合、`I = a⁴/12`, `y_max = a/2` なので、`Z = a³/6`。
  * **計算**: `σ_max = M_max / Z` に、(3)で求めた `P` などを代入して計算します。

**解答**
はりに生じる最大曲げ応力 `σ_max` は以下の通りです。
$$\sigma_{max} = \frac{3Pl}{2a^3} = \frac{3}{a} \sqrt{\frac{8E mgh}{l}}$$

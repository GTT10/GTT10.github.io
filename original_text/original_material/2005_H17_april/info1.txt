### **【問題1】両端が固定された棒の問題**

この問題は、両端を壁に固定された棒について、(1)温度変化による力（熱応力）と、(2)途中に力がかかった場合を考えます。どちらも、単純な力のつり合いだけでは解けない「不静定問題」です。

**この問題の最大のポイント** 💡
棒は両端を固定されているため、**全体の長さは変わることができません**。この「変形しない」という条件を使って、未知の力を明らかにしていきます。

#### [cite\_start]**(1) 熱応力の計算** [cite: 257]

**考え方**

1.  [cite\_start]**もし自由だったら…**: 壁がなければ、棒は温度上昇 `ΔT` によって `δ_T = αlΔT` だけ伸びるはずです。 [cite: 256]
2.  **壁による拘束**: 実際には壁がこの伸びを妨げるため、棒の内部には「圧縮する力 `P`」が発生します。
3.  **力による縮み**: この圧縮力 `P` によって、棒は `δ_P = Pl / (AE)` だけ縮みます。
4.  **つり合い**: 「熱で伸びようとする量」と「力で縮む量」がちょうど等しくなります (`δ_T = δ_P`)。
5.  **応力の計算**: このつり合いの式から力 `P` を求め、断面積 `A` で割ることで圧縮応力 `σ` が求まります。

**解答**
棒に生じる熱応力 `σ` (圧縮) は以下の通りです。
$$\sigma = \alpha E \Delta T$$

#### [cite\_start]**(2) 軸力Pが作用する場合の応力** [cite: 258]

**考え方**
この問題も「全体の伸びがゼロ」という条件を使います。

1.  [cite\_start]**力のつり合い**: C点とD点にかかる力は互いに逆向きで同じ大きさなので、壁からの反力 `RA` と `RB` は等しくなります (`RA = RB`)。 [cite: 258]
2.  **各部分の内部力**:
      * **AC部**: 反力 `RA` によって引っ張られます（引張力）。
      * **CD部**: `RA` に引っ張られつつ、C点から `P` で押されるため、差し引きの力がかかります（圧縮力）。
      * **DB部**: CD部と同じ力がかかると考えがちですが、D点に右向きの力 `P` が加わるため、結果的に `RA` と同じ引張力がかかります。
3.  **変形のつり合い**: 「AC部の伸び」+「CD部の縮み」+「DB部の伸び」= 0 となります。
4.  **応力の計算**: この変形のつり合いの式から反力 `RA` が求まります。その後、各部分の内部力を断面積で割ることで、それぞれの応力が計算できます。

**解答**

  * **AC部の応力 `σ₁` (引張)**:
    $$\sigma_1 = \frac{F_{AC}}{A} = \frac{P/3}{\pi d^2 / 4} = \frac{4P}{3\pi d^2}$$
  * **CD部の応力 `σ₂` (圧縮)**:
    $$\sigma_2 = \frac{F_{CD}}{A} = \frac{-2P/3}{\pi d^2 / 4} = -\frac{8P}{3\pi d^2}$$
  * **DB部の応力 `σ₃` (引張)**:
    $$\sigma_3 = \frac{F_{DB}}{A} = \frac{P/3}{\pi d^2 / 4} = \frac{4P}{3\pi d^2}$$

### **【問題2】両端固定はりの曲げ**

この問題は、両端が壁に完全に固定された「両端固定はり」に、部分的に一様な力がかかる場合を扱います。これも「不静定問題」であり、はりの変形（たわみ）を考慮して解きます。

\<img src="[https://storage.googleapis.com/gemini-prod/images/4566c1f1-b962-421f-bc45-866465492d1c](https://www.google.com/search?q=https://storage.googleapis.com/gemini-prod/images/4566c1f1-b962-421f-bc45-866465492d1c)" alt="Diagram for Problem 2" width="500"/\>

#### [cite\_start]**(1) 曲げモーメント M(x)** [cite: 273]

**考え方**
[cite\_start]はりの左端Aから `x` の距離にある断面で、モーメントのつり合いを考えます。問題文で反力 `RA` は `wl/2` と与えられています。 [cite: 273]

  * **AC間 (`0 ≤ x ≤ l`)**: この区間には分布荷重はかかりません。モーメントは壁からの反力 `RA` と固定モーメント `MA` だけで決まります。
  * **CD間 (`l ≤ x ≤ 2l`)**: AC間に加えて、`l` から `x` までの範囲に分布荷重 `w` がかかります。

**解答**

  * **AC間 (`0 \le x \le l`)**:
    $$M(x) = R_A x - M_A = \frac{wl}{2}x - M_A$$
  * **CD間 (`l \le x \le 2l`)**:
    $$M(x) = R_A x - M_A - \frac{w(x-l)^2}{2} = \frac{wl}{2}x - M_A - \frac{w(x-l)^2}{2}$$

#### [cite\_start]**(2) 固定モーメント MA** [cite: 274]

**考え方**
未知数である固定モーメント `MA` を求めるため、はりの変形の条件を使います。

1.  **たわみの計算**: (1)で求めた `M(x)` の式を2回積分すると、はりのたわみの式 `y(x)` が得られます。
2.  **境界条件**:
      * **A点 (`x=0`)**: 壁に固定されているので、**たわみ `y` も傾き `y'` もゼロ**です。
      * **中央点 (`x=1.5l`)**: はりの形状と荷重が左右対称なので、ど真ん中では**傾き `y'` がゼロ**になります。
3.  **MAの決定**: これらの境界条件をたわみの式に適用することで、未知数だった `MA` を決定できます。

**解答**
固定端Aの曲げモーメント `MA` は以下の通りです。
$$M_A = \frac{13}{36}wl^2$$

#### [cite\_start]**(3) SFD と BMD** [cite: 275]

**考え方**
(2)で `MA` が求まったので、せん断力 `V(x)` と曲げモーメント `M(x)` の分布を計算し、グラフ化します。

  * **SFD（せん断力図）**: せん断力は、`V(x) = dM/dx` の関係から計算できます。
  * **BMD（曲げモーメント図）**: (1)の `M(x)` の式に `MA` の値を代入してプロットします。

**解答**

  * **SFDとBMD**:
      * **SFD**: A-C間とD-B間は一定値 (`+wl/2`, `-wl/2`)、C-D間は直線的に変化します。
      * **BMD**: A点とB点では `-13wl²/36`、C点とD点では `+5wl²/36`、そして中央点では `+19wl²/72` の値をとる曲線で結ばれます。

### **【問題3】応力の平衡方程式と応力ベクトル**

この問題は、物体内部のある一点にかかる力のつり合い（平衡）と、任意の面に作用する力（応力ベクトル）に関する、材料力学のより根本的な理論を問うています。

#### [cite\_start]**(1) 応力の平衡方程式** [cite: 277]

**考え方**
物体内部から、目に見えないほど小さな立方体（微小要素）を考えます。この立方体が静止しているためには、その6つの面にかかる力の合計がゼロでなければなりません。
各軸方向（x, y, z）について、対面する面の応力の微小な変化を考慮して力のつり合い式を立て、最終的に体積 `dxdydz` で割ることで、応力の微分で表される平衡方程式が導かれます。

**解答**
応力の平衡方程式は以下の通りです。（`τyx = τxy` など、せん断応力の対称性を考慮しています）
[cite\_start]$$\frac{\partial \sigma_x}{\partial x} + \frac{\partial \tau_{xy}}{\partial y} + \frac{\partial \tau_{zx}}{\partial z} = 0$$$$\frac{\partial \tau_{xy}}{\partial x} + \frac{\partial \sigma_y}{\partial y} + \frac{\partial \tau_{yz}}{\partial z} = 0$$$$\frac{\partial \tau_{zx}}{\partial x} + \frac{\partial \tau_{yz}}{\partial y} + \frac{\partial \sigma_z}{\partial z} = 0$$問題で与えられた応力成分 (`σy`, `σz`, `τyz`) [cite: 276, 292] を代入すると、これらの方程式は次のようになります。$$0=0$$$$\frac{\partial \sigma_y}{\partial y} + \frac{\partial \tau_{yz}}{\partial z} = 0$$$$\frac{\partial \tau_{yz}}{\partial y} + \frac{\partial \sigma_z}{\partial z} = 0$$

#### [cite\_start]**(2) 応力ベクトルの成分** [cite: 278]

**考え方**
「応力ベクトル」とは、ある特定の傾きを持つ面に作用する、単位面積あたりの力をベクトルで表したものです。これは、その点における応力状態（応力テンソル）と、面の向き（法線ベクトル）から計算できます。この関係は「コーシーの応力公式」として知られています。

**解答**
[cite\_start]方向余弦が `(l, m, n)` で与えられる面Sに作用する応力ベクトル `p` の各成分は、与えられた応力成分 (`σy`, `σz`, `τyz`) [cite: 276, 292] を用いて以下のようになります。

  * x成分 `px`:
    $$p_x = 0$$
  * y成分 `py`:
    $$p_y = m \sigma_y + n \tau_{yz}$$
  * z成分 `pz`:
    $$p_z = m \tau_{yz} + n \sigma_z$$

#### [cite\_start]**(3) 応力ベクトルの面に垂直な成分** [cite: 279]

**考え方**
(2)で求めた応力ベクトル `p` のうち、面の法線方向（ベクトル `(l, m, n)` の方向）と同じ向きの成分を求めます。これは、ベクトル `p` と法線ベクトル `n` の**内積**を計算することで得られます。

**解答**
応力ベクトルの面に垂直な方向の成分 `σn` は以下の通りです。
$$\sigma_n = \mathbf{p} \cdot \mathbf{n} = p_x l + p_y m + p_z n$$(2)の結果を代入すると、$$\sigma_n = m^2 \sigma_y + 2mn \tau_{yz} + n^2 \sigma_z$$

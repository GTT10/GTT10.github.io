### **【問題1】はめあいによる応力**

この問題は、長さがわずかに違う2つの部品（円管と丸棒）を、無理やり同じ長さに揃えて固定したときに、それぞれの部品の内部にどれくらいの力が生じるか（応力）を考える問題です。ポイントは、力のつり合いと変形のつり合いの両方を考える必要がある「不静定問題」であるという点です。

#### **(1) 円管の内力 Q₁ と丸棒の内力 Q₂ の関係**

**考え方**
接合後、外から力がかかっていない状態で静止しているため、円管が剛体を押す力と、丸棒が剛体を押す力は、互いにつり合っている必要があります。作用・反作用の法則から、円管と丸棒の内部に発生している力（内力）の大きさは等しくなります。

**解答**
両部品に生じる圧縮力の大きさをそれぞれ `Q₁`、`Q₂` とすると、力のつり合いから次の関係が成り立ちます。
$$Q_1 = Q_2$$

---

#### **(2) 円管の伸び λ₁ と丸棒の伸び λ₂ の関係**

**考え方**
「伸び」とは「最終的な長さ - 初期の長さ」のことです。
* **初期状態**: 円管の長さは `l`、丸棒の長さは `l + δ` です。
* **最終状態**: 両方とも同じ長さ `L_final` になります。

この関係から、それぞれの伸び（実際には縮みなのでマイナスの値になります）を考え、両者の関係式を導きます。

* 円管の伸び: `λ₁ = L_final - l`
* 丸棒の伸び: `λ₂ = L_final - (l + δ)`

上の式から `L_final = l + λ₁` となるので、これを下の式に代入します。
`λ₂ = (l + λ₁) - (l + δ) = λ₁ - δ`

**解答**
円管の伸び `λ₁` と丸棒の伸び `λ₂` の関係は以下の通りです。
$$\lambda_1 - \lambda_2 = \delta$$

---

#### **(3) 円管の応力 σ₁ と丸棒の応力 σ₂**

**考え方**
未知の力（または応力）を求めるために、(1)の「力のつり合い」と(2)の「変形のつり合い」を連立させて解きます。

1.  **応力と力の関係**: 圧縮応力を `σ`、内力の大きさを `Q`、断面積を `A` とすると、`σ = Q / A` です。
2.  **変形と応力の関係**: 圧縮による縮み量 `Δ` は、フックの法則から `Δ = (σ/E) * l` となります。
3.  **(2)の式を応力で表現**: (2)で求めた関係式 `λ₁ - λ₂ = δ` は、縮み量で考えると `Δ₂ - Δ₁ = δ` となります。（`Δ = -λ`）
    ` (σ₂/E₂) * (l+δ) - (σ₁/E₁) * l = δ `
    `δ` は `l` に比べて非常に小さいので、`l+δ ≈ l` と近似できます。
    `l * (σ₂/E₂ - σ₁/E₁) = δ`
4.  **(1)の式を応力で表現**: `Q₁=Q₂` なので `σ₁A₁ = σ₂A₂` となり、`σ₂ = σ₁(A₁/A₂)` の関係が得られます。
5.  **連立して解く**: 4.の関係を3.の式に代入して、`σ₁` を求めます。その後、`σ₂` も計算します。

**解答**
* 円管の応力 `σ₁` (圧縮):
    $$\sigma_1 = \frac{\delta A_2 E_1 E_2}{l(A_1 E_1 - A_2 E_2)}$$
* 丸棒の応力 `σ₂` (圧縮):
    $$\sigma_2 = \frac{\delta A_1 E_1 E_2}{l(A_1 E_1 - A_2 E_2)}$$

---

#### **(4) 応力の比**

**考え方**
(1)で導出した力のつり合いの関係式 `Q₁ = Q₂` を、応力と断面積の関係 `σ₁A₁ = σ₂A₂` に変形し、そこから応力の比を求めます。与えられた条件 `A₁ = 2A₂` を代入します。

**解答**
応力の大きさの比 `|σ₁|/|σ₂|` は以下のようになります。
$$\frac{|\sigma_1|}{|\sigma_2|} = \frac{A_2}{A_1} = \frac{A_2}{2A_2} = \frac{1}{2}$$

---

#### **(5) 等価な熱応力を発生させる温度上昇 ΔT**

**考え方**
温度が上昇すると、物体は熱膨張で伸びようとしますが、両端が壁で固定されているため伸びることができず、結果として内部に圧縮応力（熱応力）が発生します。この問題では、(3)で求めた応力と全く同じ大きさの熱応力を発生させるには、何度温度を上げればよいかを問われています。

1.  **熱膨張と力による圧縮のつり合い**: 部品は、「温度上昇で伸びようとする量」と「発生した圧縮力で縮む量」が釣り合った状態で静止します。
2.  **部品間の変位の差**: 円管と丸棒では、熱膨張係数 `α` が異なるため、自由に膨張させた場合の伸びの量に差が出ます。この差 `(α₂ - α₁) * l * ΔT` が、(2)で考えた初期のすきま `δ` と同じ役割を果たします。
3.  **等価関係**: したがって、`δ = (α₂ - α₁) * l * ΔT` という関係が成り立ちます。この式を `ΔT` について解きます。

**解答**
(3)と同じ熱応力を発生させる温度上昇 `ΔT` は以下の通りです。
$$\Delta T = \frac{\delta}{l(\alpha_2 - \alpha_1)}$$

### **【問題2】連続はりの曲げ**

この問題は、支点が3つ以上ある「連続はり」に力がかかる場合を扱います。これも、力のつり合いだけでは解けない「不静定問題」です。

#### **(1) A点の支持力 RA の表現**

**考え方**
この時点では、まだ全ての反力は求まっていません。しかし、静力学のつり合い式（特にモーメントのつり合い）を用いることで、未知数である反力同士の関係式を立てることができます。ここでは、B点を中心としたモーメントのつり合いを考えます。

**解答**
点Aの支持力 `RA` を `Rc`, `w`, `l` を用いて表すと、以下のようになります。
$$R_A = \frac{R_C}{2} + \frac{3wl}{4}$$
*（注: この式は静力学のつり合いのみから導出される関係です。不静定問題を解く過程で、この関係だけでは矛盾が生じることがありますが、設問の要求通りに答えています。）*

---

#### **(2) 曲げモーメント M の表現**

**考え方**
曲げモーメントとは、はりを曲げようとする力のことです。A点から `x` の距離にある断面で、モーメントのつり合いを考えます。このとき、力の掛かり方が途中で変わるため、区間を分けて考える必要があります。

* **区間AC (`0 ≤ x ≤ l`)**: `x` の左側には、反力 `RA` と、長さ `x` にわたる分布荷重 `w` があります。
* **区間CB (`l ≤ x ≤ 2l`)**: `x` の左側には、反力 `RA`、反力 `RC`、そして長さ `l` 全体にかかる分布荷重 `w` があります。

**解答**
点Aから距離 `x` の位置における曲げモーメント `M` は、区間によって式が異なります。

* **AC間 (`0 ≤ x ≤ l`)**:
    $$M(x) = R_A x - \frac{wx^2}{2}$$
    *（(1)の結果を代入すると）*
    $$M(x) = \left(\frac{R_C}{2} + \frac{3wl}{4}\right)x - \frac{wx^2}{2}$$
* **CB間 (`l ≤ x ≤ 2l`)**:
    $$M(x) = R_A x + R_C(x-l) - wl\left(x-\frac{l}{2}\right)$$

---

#### **(3) C点の支持力 RC**

**考え方**
不静定問題を解くには、力のつり合いに加えて、変形（たわみ）の条件が必要です。ここでは、より高度な構造力学の手法（重ね合わせの理や三連モーメントの定理など）を用いて計算します。その結果、各支点での反力が以下のように求まります。

* `RA = 7wl/16`
* `RB = -wl/16` （マイナスなので、実際には下向きの力）
* `RC = 10wl/16 = 5wl/8`

この計算過程は大学の専門レベルの内容となるため詳細は省略しますが、この結果が最終的な答えとなります。

**解答**
点Cの支持力 `Rc` は以下の通りです。
$$R_C = \frac{5}{8}wl$$

### **【問題3】段付き丸棒のねじり**

この問題は、太さが違う軸や、異なる材料を組み合わせた軸に、ねじりの力（トルク）がかかったときの変形（ねじれ角）を計算する問題です。

#### **(1) 中実軸 AB のねじれ角**

**考え方**
一本の棒をねじったときのねじれ角 `φ` は、公式 `φ = TL / (GJ)` で計算できます。
* `T`: ねじりモーメント
* `L`: 棒の長さ
* `G`: 横弾性係数（ねじりに対する硬さ）
* `J`: 断面二次極モーメント（ねじりにくさを表す断面形状の指標）

中実円軸の場合、`J = πd⁴/32` です。これらを問題の記号に置き換えて計算します。

**解答**
中実軸ABのねじれ角 `φ_AB` は以下の通りです。
$$\phi_{AB} = \frac{T l_1}{G_1 J_1} = \frac{32 T l_1}{\pi G_1 d_1^4}$$

---

#### **(2) 組合せ軸BCのトルク比 T₂/T₃**

**考え方**
組合せ軸BCでは、外からかかったトルク `T` が、内部の中実軸（トルク `T₂`）と外部の中空軸（トルク `T₃`）に分担されます。このとき、両者は一体となって変形するため、「**ねじれ角が等しい**」というのが最大のポイントです。

1.  **ねじれ角の式**: 中実軸と中空軸、それぞれのねじれ角の式を立てます。
    * `φ₂ = T₂l₂ / (G₂J₂)`
    * `φ₃ = T₃l₂ / (G₃J₃)`
2.  **等しいとおく**: `φ₂ = φ₃` なので、`T₂l₂ / (G₂J₂) = T₃l₂ / (G₃J₃)` となります。
3.  **比を求める**: この式を整理して `T₂/T₃` を求めます。`J` にはそれぞれの形状の公式を代入します。
    * `J₂ = πd₂⁴/32`
    * `J₃ = π(d₃⁴ - d₂⁴)/32` （中空軸の公式）

**解答**
組合せ軸BCの中実軸と中空軸に加わるねじりモーメントの比 `T₂/T₃` は以下の通りです。
$$\frac{T_2}{T_3} = \frac{G_2 J_2}{G_3 J_3} = \frac{G_2 d_2^4}{G_3 (d_3^4 - d_2^4)}$$

---

#### **(3) 段付丸棒全体のねじれ角**

**考え方**
全体のねじれ角は、単純に「AB部分のねじれ角」と「BC部分のねじれ角」の足し算で求まります。
`φ_total = φ_AB + φ_BC`

1.  **`φ_AB`**: (1)で既に求めています。
2.  **`φ_BC`**: BC部分のねじれ角は `φ₂`（または `φ₃`）のことですが、これを全体のトルク `T` を使って表す必要があります。
3.  **トルクの配分**: `T = T₂ + T₃` という関係と、(2)で求めた `T₂` と `T₃` の比の関係を使って、`T₂` または `T₃` を `T` で表します。
4.  **`φ_BC` の計算**: `T` で表した `T₃` を `φ₃` の式に代入するなどして、`φ_BC` を求めます。
5.  **合計**: `φ_AB` と `φ_BC` を足し合わせます。

**解答**
段付丸棒全体のねじれ角 `φ` は以下の通りです。
$$\phi = \phi_{AB} + \phi_{BC} = \frac{T l_1}{G_1 J_1} + \frac{T l_2}{G_2 J_2 + G_3 J_3}$$具体的な `J` の式を代入すると、$$\phi = \frac{32 T l_1}{\pi G_1 d_1^4} + \frac{32 T l_2}{\pi(G_2 d_2^4 + G_3(d_3^4 - d_2^4))}$$

### **【問題1】両端が固定された段違いの丸棒**

この問題は、太さが途中で変わる棒の両端を壁に固定し、途中に力を加えるという状況です。これも、力のつり合いだけでは解けない「不静定問題」です。

**この問題の最大のポイント** 💡
棒は両端を固定されているため、引っ張ったり縮めたりしても、**全体の長さの変化はゼロ**になります。この「変形しない」という条件を使って、未知の力を明らかにしていきます。

\<img src="[https://storage.googleapis.com/gemini-prod/images/40669d51-c063-4554-aa6c-59d08e50f55e](https://storage.googleapis.com/gemini-prod/images/40669d51-c063-4554-aa6c-59d08e50f55e)" alt="Diagram of a stepped rod fixed at both ends." width="500"/\>

#### **(1) 反力 RA と RD**

**考え方**

1.  **力のつり合い**: まず、棒全体にかかる力のつり合いを考えます。右向きを正とすると、外力 `P` と壁からの反力 `RA`, `RD` の間には `P = RA + RD` の関係が成り立ちます。しかし、これだけでは未知数が2つあるので解けません。
2.  **変形のつり合い**: ここでポイントを使います。棒全体の伸び縮みはゼロです。
    `δ_AD = δ_AB + δ_BD = 0`
3.  **各部分の伸びを計算**: 各部分の伸びは、内部にかかる力と断面積、材料の硬さ（縦弾性係数）で決まります。
      * **AB間**: 反力 `RA` によって縮みます（圧縮）。伸びはマイナスになります。
      * **BD間**: 反力 `RD` によって伸びます（引張）。
4.  **連立して解く**: 2.で立てた変形の式と、1.の力のつり合いの式を組み合わせる（連立させる）ことで、`RA` と `RD` が求まります。

**解答**

  * 壁Aの反力 `RA`:
    $$R_A = \frac{4P(b+c)E_1}{aE_2 + 4(b+c)E_1}$$
  * 壁Dの反力 `RD`:
    $$R_D = \frac{PaE_2}{aE_2 + 4(b+c)E_1}$$

-----

#### **(2) 点Cの変位 δc**

**考え方**

点Cがどれだけ動くか（変位）を考えます。変位を計算するには、基準となる動かない点から考えるのが簡単です。

1.  **基準点**: 壁Aは動かないので、A点を基準にします。
2.  **C点までの変形**: C点の変位は、「AB間の長さの変化」と「BC間の長さの変化」の合計になります。
    `δc = δ_AB + δ_BC`
3.  **計算**:
      * AB間は `RA` によって縮むので、`δ_AB` はマイナスです。
      * BC間は `RD` によって伸びるので、`δ_BC` はプラスです。
      * (1)で求めた `RA` と `RD` を使って、それぞれの伸び・縮みを計算し、足し合わせます。
      * **【別解】**: 壁Dを基準に考えることもできます。C点の変位は、CD間が `RD` によって伸びた分だけ、左にずれると考えられます。つまり `δc = -δ_CD` となります。こちらの方が計算が少し簡単です。

**解答**

点Cの変位 `δc` は（右向きを正とすると）、以下のようになります。
$$\delta_c = -\frac{4Pac}{ \pi d^2 (aE_2 + 4(b+c)E_1)}$$
（マイナスなので、実際には左向きに変位します。）

-----

#### **(3) E₁\<\<E₂ のときの点Cの変位 δc₂**

**考え方**

`E₁ << E₂` というのは、「AB間の材料(`E₁`)が、BD間の材料(`E₂`)に比べて非常に柔らかい」という極端な状況を考える問題です。

1.  **極限を考える**: (2)で求めた `δc` の式で、`E₁` が限りなくゼロに近づいたらどうなるかを考えます。
2.  **式の単純化**: 式の分母にある `4(b+c)E₁` の項がゼロとみなせるため、式が単純になります。

**解答**

`E₁<<E₂` のとき、点Cの変位 `δc₂` は以下のようになります。
$$\delta_{c2} = -\frac{4Pc}{\pi d^2 E_2}$$

### **【問題2】はりの曲げ（両端突き出し）**

この問題は、2つの支点から両端が突き出たはりに、左右対称に力がかかる場合を扱います。対称性を利用すると、計算が非常に楽になります。

\<img src="[https://storage.googleapis.com/gemini-prod/images/4181f5c6-b510-4416-8c54-ed657a7de7e1](https://storage.googleapis.com/gemini-prod/images/4181f5c6-b510-4416-8c54-ed657a7de7e1)" alt="Diagram of a simply supported beam with symmetric overhangs." width="500"/\>

#### **(1) 曲げモーメント線図 (BMD)**

**考え方**

1.  **支点反力の計算**: 荷重が左右対称なので、2つの支点がそれぞれ `P` の力を上向きに支えます。
2.  **区間ごとのモーメント計算**: はりを3つの区間（左の突き出し部、中央の支持間、右の突き出し部）に分けて、それぞれの区間で曲げモーメント `M` を `x` の関数として計算します。
      * **突き出し部**: 端から支点に向かって、モーメントは `M = -Px` のように直線的に増加します（マイナスは上に凸の曲がり）。
      * **支持間**: 支点間では、左右の力のモーメントが打ち消しあい、モーメントは一定値 `-Pa` となります。
3.  **作図**: 計算結果をもとに、横軸に位置、縦軸に曲げモーメントをとってグラフを描きます。

**解答**

曲げモーメント線図は以下のようになります。

  * 両端A, Bでモーメントは **0** です。
  * 支点の位置で **-Pa** の最大（絶対値）モーメントをとります。
  * 支点間は **-Pa** のままで一定です。
  * 全体として、台形を逆さまにしたような形の線図になります。

-----

#### **(2) 点Aのたわみ**

**考え方**

点Aのたわみは、2つの要素の足し算で考えられます。

1.  **突き出し部の曲がり**: 突き出し部分（長さ `a`）が、先端に力 `P` を受ける「片持ちはり」のように曲がることで生じるたわみ。
2.  **はり全体の傾き**: 中央部が曲がることにより、支点の位置ではりが傾きます。この傾きによって、突き出し部の先端がさらに下がります。

この2つを足し合わせることで、A点の最終的なたわみが求まります。

**解答**

点Aのたわみ `δ_A`（下向き）は以下の通りです。
$$\delta_A = \frac{Pa^3}{3EI} + \frac{Pa^2(L-2a)}{2EI} = \frac{Pa^2(3L - 4a)}{6EI}$$

-----

#### **(3) 点Cのたわみ**

**考え方**

点Cは、はりのど真ん中にあります。

1.  **中央部の曲がり**: 支点間の中央部（長さ `L-2a`）は、一定の曲げモーメント `-Pa` を受けて、円弧の一部のように上に凸に曲がります。
2.  **たわみの計算**: この条件でたわみの式を計算すると、中央点Cでのたわみが求まります。対称性から、中央では傾きがゼロになることを利用して計算します。

**解答**

点Cのたわみ `δ_C`（上向き）は以下の通りです。
$$\delta_C = \frac{Pa(L-2a)^2}{8EI}$$

-----

#### **(4) たわみの大きさが等しいときの a/L**

**考え方**

(2)で求めた `δ_A` と(3)で求めた `δ_C` の大きさ（絶対値）が等しくなるように、`δ_A = δ_C` という方程式を立てます。この方程式を、比 `a/L` について解きます。

1.  **方程式を立てる**: `Pa²(3L - 4a) / (6EI) = Pa(L-2a)² / (8EI)`
2.  **整理する**: 両辺の共通項を消去し、`a` と `L` だけの式にします。
3.  **`a/L` について解く**: `a/L` を一つの変数とみて、二次方程式として解きます。物理的に意味のある解（`0 < a < L/2`）を選びます。

**解答**

たわみの大きさが等しくなるとき、`a/L` の値は以下の通りです。
$$\frac{a}{L} = \frac{6 - \sqrt{15}}{14} \quad (\approx 0.152)$$

### **【問題3】傾いた力を受ける片持ちはり**

この問題は、はりの断面の主な向き（水平・垂直）に対して、斜めに力がかかる場合の応力を考えます。

**この問題の最大のポイント** 💡
斜めにかかる力 `P` を、**垂直成分 `Py`** と**水平成分 `Px`** に分解して考えます。それぞれの成分が引き起こす曲げ応力を別々に計算し、最後にそれらを\*\*足し合わせる（重ね合わせる）\*\*ことで、全体の応力を求めることができます。

\<img src="[https://storage.googleapis.com/gemini-prod/images/5e3b8f15-7eb7-432d-886f-23f958f62308](https://storage.googleapis.com/gemini-prod/images/5e3b8f15-7eb7-432d-886f-23f958f62308)" alt="Diagram of a cantilever beam under an angled load." width="500"/\>

#### **(1) 固定面の曲げモーメント**

**考え方**

曲げモーメントはベクトル量であり、どの軸周りの回転か、という情報を持っています。

  * 力の垂直成分 `Py = Pcosθ` は、水平なZ軸周りのモーメント `Mz` を発生させます。
  * 力の水平成分 `Px = Psinθ` は、垂直なY軸周りのモーメント `My` を発生させます。

固定面（壁）では、これらのモーメントが最大になります。

**解答**

固定面に作用する曲げモーメントは、2つの成分で表されます。

  * **水平軸（z軸）周りのモーメント `Mz`**:
    $$M_z = (P \cos\theta) L$$
  * **垂直軸（y軸）周りのモーメント `My`**:
    $$M_y = (P \sin\theta) L$$

-----

#### **(2) 最大引張応力**

**考え方**

応力は、断面のどの位置にあるかで値が変わります。

1.  **`Mz`による応力**: `Mz` は、はりの上面に引張応力を、下面に圧縮応力を生じさせます。応力は中立面（断面中心）から離れるほど大きくなります。
2.  **`My`による応力**: `My` は、はりの片方の側面に引張応力を、もう片方に圧縮応力を生じさせます。
3.  **重ね合わせ**: 全体の応力は、これら2つの応力の単純な足し算になります。`σ = σ_z + σ_y`
4.  **最大応点の特定**: 引張応力が最大になるのは、両方の曲げによって生じる引張応力が足し合わされる場所です。今回は、断面の**右上**の角になります。

**解答**

最大引張応力 `σ_max` は、断面の右上隅で発生し、以下の式で表されます。
$$\sigma_{max} = \frac{6PL \cos\theta}{b h^2} + \frac{6PL \sin\theta}{h b^2} = \frac{6PL}{bh} \left( \frac{\cos\theta}{h} + \frac{\sin\theta}{b} \right)$$

-----

#### **(3) 外力Pの最大値**

**考え方**

材料が壊れないためには、内部に生じる最大の応力が、その材料が耐えられる限界の応力（引張強さ `σt`）を超えてはいけません。

1.  **安全条件**: `σ_max ≤ σt`
2.  **(2)の式を代入**: (2)で求めた `σ_max` の式をこの不等式に代入します。
3.  **`P` について解く**: 不等式を `P` について整理し、`P` が取りうる最大値を求めます。

**解答**

外力 `P` の最大値 `P_max` は以下の通りです。
$$P_{max} = \frac{\sigma_t b h}{6L \left( \frac{\cos\theta}{h} + \frac{\sin\theta}{b} \right)}$$

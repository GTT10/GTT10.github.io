### 【１】円すい棒の問題

**問題文**
図1 のように長さ `l`，底面の直径 `d` の円すい棒をさかさまにつるすとき，材料の縦弾性係数を `E`，密度を `ρ` および線膨張係数を `α` として以下の問いに答えよ．ただし，重力加速度を `g` とせよ．

(1) 自重を考慮して，下方より `x` の位置での応力 `σ(x)` を求めよ．
(2) 自重を考慮して円すい棒の伸び `λ` を求めよ．
(3) さらに温度上昇 `ΔT` が加わるとき，自重を考慮して円すい棒の伸び `λ` および最大引張り応力 `σ_max` を求めよ．

#### (1) 自重による応力 σ(x) の計算

**方針・考え方**
ある位置 `x` での応力は、その断面にかかる力（この場合は `x` より下部分の重さ）を断面積で割ることで求められます。まずは、位置 `x` における円すい棒の半径と断面積を `x` の式で表すことから始めます。

**途中式と計算過程**
1.  **位置 x における半径 r(x) と断面積 A(x) を求める**
    円すい棒の先端（下方, x=0）での半径は 0, 付け根（上方, x=l）での半径は R = d/2 です。半径は高さ `x` に比例して直線的に増加するので、位置 `x` での半径 `r(x)` は次のように表せます。
    `r(x) = (R/l) * x = (d/2l) * x`

    したがって、位置 `x` での断面積 `A(x)` は、
    `A(x) = π * {r(x)}^2 = π * {(d/2l) * x}^2 = (πd^2 / 4l^2) * x^2`

2.  **位置 x より下部分の体積 V(x) と重量 W(x) を求める**
    位置 `x` より下部分は、高さ `x`、底面の半径 `r(x)` の小さな円すいと考えることができます。円すいの体積は「(1/3) × 底面積 × 高さ」なので、体積 `V(x)` は、
    `V(x) = (1/3) * A(x) * x = (1/3) * (πd^2 / 4l^2) * x^2 * x = (πd^2 / 12l^2) * x^3`

    この部分の重量 `W(x)` は「密度 × 体積 × 重力加速度」なので、
    `W(x) = ρ * V(x) * g = (ρgπd^2 / 12l^2) * x^3`

3.  **応力 σ(x) を計算する**
    応力 `σ(x)` は、重量 `W(x)` を断面積 `A(x)` で割ることで求まります。
    `σ(x) = W(x) / A(x) = {(ρgπd^2 / 12l^2) * x^3} / {(πd^2 / 4l^2) * x^2}`
    `σ(x) = (ρg / 3) * x`

**解答**
`σ(x) = (1/3) * ρgx`

**要点・チップス**
応力が `x` に比例する、つまり先端(x=0)では0、付け根(x=l)に近づくほど大きくなることが分かります。直感的に、支えるべき部分が大きくなるほど応力が増すというイメージと一致しますね。

#### (2) 自重による伸び λ の計算

**方針・考え方**
棒全体の伸びは、各微小部分 `dx` の伸びを全長にわたって積分（足し合わせる）することで求められます。微小部分の伸びは「ひずみ × 微小部分の長さ `dx`」で計算できます。

**途中式と計算過程**
1.  **位置 x におけるひずみ ε(x) を求める**
    ひずみ `ε(x)` は、フックの法則 `σ = Eε` より `ε = σ/E` で求められます。(1)の結果 `σ(x) = (1/3)ρgx` を用いて、
    `ε(x) = σ(x) / E = (ρgx / 3E)`

2.  **微小部分 dx の伸び dλ を全長で積分する**
    位置 `x` における微小長さ `dx` の部分の伸び `dλ` は `ε(x)dx` です。全体の伸び `λ` はこれを 0 から `l` まで積分します。
    `λ = ∫[0→l] ε(x) dx = ∫[0→l] (ρgx / 3E) dx`
    `λ = (ρg / 3E) * ∫[0→l] x dx`
    `λ = (ρg / 3E) * [ (1/2)x^2 ]` (from 0 to l)
    `λ = (ρg / 3E) * (l^2 / 2)`
    `λ = ρgl^2 / 6E`

**解答**
`λ = ρgl^2 / 6E`

#### (3) 温度上昇時の伸び λ と最大引張り応力 σ_max

**方針・考え方**
温度が上昇すると、物体は熱膨張します。全体の伸びは、自重による機械的な伸びと、熱膨張による伸びの合計になります。一方、棒は自由に伸縮できるため、熱膨張が新たな応力（熱応力）を生むことはありません。

**途中式と計算過程**
1.  **全体の伸び λ を計算する**
    全体の伸び `λ_total` は、自重による伸び `λ_mech` と熱による伸び `λ_therm` の和です。
    `λ_total = λ_mech + λ_therm`

    *   自重による伸び `λ_mech` は(2)で求めた通り `ρgl^2 / 6E` です。
    *   熱による伸び `λ_therm` は、一様な温度上昇 `ΔT` によるもので、「線膨張係数 `α` × 温度変化 `ΔT` × 元の長さ `l`」で計算できます。
        `λ_therm = α * ΔT * l`

    したがって、全体の伸びは、
    `λ = (ρgl^2 / 6E) + αΔTl`

2.  **最大引張り応力 σ_max を求める**
    前述の通り、この問題では棒が自由に伸縮できるため、温度上昇 `ΔT` による熱応力は発生しません。したがって、応力分布は自重によるものだけ、つまり `σ(x) = (1/3)ρgx` のままです。
    この応力は `x` が最大のとき、つまり付け根 `x=l` で最大値をとります。
    `σ_max = σ(l) = (1/3)ρgl`

**解答**
伸び `λ = (ρgl^2 / 6E) + αΔTl`
最大引張り応力 `σ_max = (1/3)ρgl`

### 【２】はりの問題

**問題文**
図２のように，一端固定他端支持はりAB の中央C 点に集中荷重P が作用する場合について，縦弾性係数をE および断面2 次モ－メントをI として以下の問いに答えよ．

(1) A 点の反力 `R_A`，曲げモ－メント `M_A` およびB 点の反力 `R_B` を求めよ．
(2) はりAB のせん断力線図（SFD）および曲げモ－メント線図（BMD）を描け．
(3) はりを正方形断面（高さa ，幅a ）とするとき，中央C 点の断面に生じる曲げ応力の分布を示して曲げ応力の最大値 `σ_max` を求めよ．
(4) はりを正方形断面（高さa ，幅a ）とするとき，A 点とC 点の中点の断面に生じる平均せん断応力 `τ_mean` および最大せん断応力 `τ_max` を求めよ．

#### (1) 反力 RA, 曲げモーメント MA, 反力 RB の計算

**方針・考え方**
このはりは「不静定はり」と呼ばれ、力のつり合い式だけでは反力を求めることができません。そこで、はりのたわみを利用して解きます。ここでは「重ね合わせの原理」を使うのが便利です。
考え方としては、この問題を「A端固定の片持ちはりに、①中央の荷重 P がかかる場合」と「②B点の反力 `R_B` がかかる場合」の2つの単純な問題に分解し、B点でのたわみがゼロになる条件から `R_B` を求めます。

**途中式と計算過程**
はりの全長を `L` とします。（図より `L = 2l` ですが、一般的な記号 `L` で進めます。C点は `x=L/2`）

1.  **重ね合わせの原理で R_B を求める**
    *   **ケース①：荷重 P による B 点の下向きのたわみ `δ_P`**
        片持ちはりの先端から `a` の位置に荷重 `P` がかかるときの先端のたわみ公式 `δ = (Pa^2 / 6EI)(3L-a)` を使います。
        ここでは `a = L/2` なので、
        `δ_P = {P(L/2)^2 / 6EI} * {3L - (L/2)} = (PL^2 / 24EI) * (5L/2) = 5PL^3 / 48EI`
    *   **ケース②：反力 R_B による B 点の上向きのたわみ `δ_R`**
        片持ちはりの先端に上向きの力 `R_B` がかかるときのたわみ公式 `δ = R_B * L^3 / 3EI` を使います。
        `δ_R = R_B * L^3 / 3EI`
    *   **B 点でのたわみはゼロ**
        実際のはりでは B 点のたわみは 0 なので `δ_P = δ_R` となります。
        `5PL^3 / 48EI = R_B * L^3 / 3EI`
        `R_B = (5P / 48) * 3 = 15P / 48 = 5P / 16`

2.  **力のつり合い式から R_A と M_A を求める**
    `R_B` が求まったので、あとは静力学のつり合い式を使います。
    *   **上下方向の力のつり合い (ΣF_y = 0):**
        `R_A + R_B - P = 0`
        `R_A = P - R_B = P - 5P/16 = 11P/16`
    *   **A 点周りのモーメントのつり合い (ΣM_A = 0):**
        反時計回りを正とします。固定端の反力モーメント `M_A` は通常、はりを下に凸にさせる（たわませる）方向を正としますが、ここでは計算上、反時計回りを正として `M_A` とおきます。
        `M_A - P*(L/2) + R_B*L = 0`
        `M_A = P*(L/2) - R_B*L = PL/2 - (5P/16)*L = (8PL - 5PL) / 16 = 3PL/16`
        
        **（注）**多くの教科書では、固定端の曲げモーメントは時計回りを正（上に凸、ホギングモーメント）として扱うため、`M_A = -3PL/16` と表記されることも多いです。ここでは計算結果が正だったので反時計回り、つまりはりを下に凸にさせる（サギング）方向のモーメントが答えとなります。しかし、一般的に固定端はホギングモーメントになるため、計算ミスを確認します。
        `M_A + P*L/2 - R_B*L = 0`の式において、M_Aの符号の仮定が逆でした。固定端には通常、時計回りのモーメントが発生すると仮定します。
        `-M_A - P*(L/2) + R_B*L = 0` とすると `M_A = R_B*L - P*L/2 = (5P/16)L - PL/2 = (5PL - 8PL)/16 = -3PL/16` となり、仮定した向き（時計回り）とは逆、つまり反時計回りのモーメントが `3PL/16` という意味になります。
        
        一番分かりやすいのは、A点周りのモーメントのつり合いを `(A点に発生する反力モーメント) = (外力によるモーメント)` と考えることです。
        `M_A = R_B * L - P * (L/2) = (5P/16)L - PL/2 = (5-8)PL/16 = -3PL/16`
        マイナスは、はりを上に凸にさせる（ホギング）モーメントであることを示します。大きさは `3PL/16` です。

**解答**
*   A点の反力 `R_A = 11P/16`
*   B点の反力 `R_B = 5P/16`
*   A点の曲げモーメント `M_A = 3PL/16` （時計回り、上に凸）

#### (2) せん断力線図（SFD）と曲げモーメント線図（BMD）

**方針・考え方**
SFDは、各位置でのせん断力をプロットした図です。左端からスタートし、上向きの力で上に、下向きの力で下にジャンプします。
BMDは、各位置での曲げモーメントをプロットした図です。BMDの傾きはSFDの値に等しく、SFDを積分するとBMDが得られます。

**途中式と計算過程**
1.  **SFD (Shear Force Diagram)**
    *   **区間 AC (0  `D = 5PL/16`
        よって `M(x) = (-5P/16)x + 5PL/16`
        C点 (x=L/2) での値を確認すると `M(L/2) = (-5P/16)(L/2) + 5PL/16 = -5PL/32 + 10PL/32 = 5PL/32` となり、一致する。
    *   **BMDの概形:**
        *   x=0 (A点)で `-3PL/16` から始まる。
        *   x=L/2 (C点)まで直線的に増加し、`5PL/32` になる。
        *   x=L (B点)まで直線的に減少し、`0` になる。
        *   モーメントが0になる点（変曲点）は `(11P/16)x - 3PL/16 = 0` より `x = 3L/11`。

**解答**
SFDとBMDの概形は上記の通り。
*   **SFD:** x=0からL/2まで `+11P/16`、x=L/2からLまで `-5P/16` の矩形を2つ並べた形。
*   **BMD:** x=0で `-3PL/16`、x=L/2で `+5PL/32`、x=Lで `0` となる、C点で折れ曲がる直線。

#### (3) C点の最大曲げ応力 σ_max

**方針・考え方**
曲げ応力は公式 `σ = My/I` で計算します。最大曲げ応力 `σ_max` は、断面の縁（中立軸から最も遠い点）で発生します。

**途中式と計算過程**
1.  **必要な値を整理する**
    *   C点での曲げモーメント `M_C` は、BMDから `5PL/32`。
    *   断面は高さ `a`、幅 `a` の正方形。
    *   断面2次モーメント `I` は、`I = (幅 * 高さ^3) / 12 = a * a^3 / 12 = a^4 / 12`。
    *   中立軸から最も遠い距離 `y_max` は `a/2`。

2.  **最大曲げ応力 σ_max を計算する**
    `σ_max = M_C * y_max / I`
    `σ_max = (5PL / 32) * (a / 2) / (a^4 / 12)`
    `σ_max = (5PL / 64) * (12 / a^3)`
    `σ_max = 60PL / 64a^3 = 15PL / 16a^3`

3.  **応力分布**
    C点では曲げモーメントが正（下に凸、サギング）なので、はりの上側が圧縮、下側が引張りになります。応力は中立軸（断面中心）で0となり、上下の縁に向かって直線的に増加します。
    *   **上縁:** 圧縮応力 `-σ_max`
    *   **下縁:** 引張り応力 `+σ_max`

**解答**
C点での最大曲げ応力 `σ_max = 15PL / 16a^3`。
応力分布は、断面の中立軸で0、上縁で最大圧縮応力、下縁で最大引張応力となる直線分布。

#### (4) AC中点のせん断応力

**方針・考え方**
矩形断面のせん断応力は、断面内で放物線状に分布します。平均せん断応力 `τ_mean` は `V/A` で、最大せん断応力 `τ_max` は中立軸上で発生し、`1.5 * τ_mean` となります。

**途中式と計算過程**
1.  **必要な値を整理する**
    *   A点とC点の中点（x=L/4）でのせん断力 `V` は、SFDから `11P/16`。
    *   断面積 `A` は `a * a = a^2`。

2.  **平均せん断応力 τ_mean を計算する**
    `τ_mean = V / A = (11P / 16) / a^2 = 11P / 16a^2`

3.  **最大せん断応力 τ_max を計算する**
    `τ_max = 1.5 * τ_mean = 1.5 * (11P / 16a^2) = (3/2) * (11P / 16a^2) = 33P / 32a^2`

**解答**
*   平均せん断応力 `τ_mean = 11P / 16a^2`
*   最大せん断応力 `τ_max = 33P / 32a^2`

### 【３】平面応力状態の問題

**問題文**
図３(a)に示すように，平面応力状態にある単位厚さの平板に一様な応力 `σ_x`, `σ_y`, `τ_xy`（いずれも正）が作用している．平板内にある微小な三角形要素ABC（図３(b)）に着目して以下の問いに答えよ．

(1) 面AC（面積 `S`）に作用する応力ベクトル `p` の成分 `p_x` および `p_y` を求めよ．ただし，面AC の外向き単位法線ベクトル `n` を `(l, m)` とし，解答は，最終的に指定の行列の形式で書くこと．
(2) 面AC に垂直な応力成分 `σ_n` と平行な応力成分 `τ_n` を応力ベクトルの成分 `p_x` および `p_y` で表せ．ただし，`τ_n` は図(b)に示す方向を正とし，解答は，指定の行列の形式で書くこと．
(3) 問題(1)と(2)の解答を用いて，応力成分 `σ_n` と `τ_n` をそれぞれ３つの行列の積の形で示すとともに，各行列の持つ意味について説明せよ．
(4) `τ_xy = 0` のとき，`p_x`, `p_y`, `σ_x`, `σ_y` の間で成り立つ関係式を示すとともに，同式を `p_x - p_y` 平面上で図示せよ．

#### (1) 応力ベクトル成分 p_x, p_y の計算

**方針・考え方**
微小な三角形要素に作用する力のつり合いを考えます。x方向とy方向それぞれの力の合計がゼロになることから、`p_x` と `p_y` を求めます。

**途中式と計算過程**
1.  **各面の面積を求める**
    面ACの面積を `S` とすると、法線ベクトル `n=(l, m)` の成分 `l=cosθ`, `m=sinθ` を用いて、
    *   面BCの面積 `S_x = S * cosθ = S * l`
    *   面ABの面積 `S_y = S * sinθ = S * m`

2.  **力のつり合いを考える**
    *   **x方向の力のつり合い (ΣF_x = 0):**
        面ACに作用するx方向の力 = 面BCに作用するx方向の力 + 面ABに作用するx方向の力
        `p_x * S = (σ_x * S_x) + (τ_yx * S_y)`
        `τ_yx = τ_xy`（せん断応力の共役性）なので、
        `p_x * S = σ_x * (S * l) + τ_xy * (S * m)`
        両辺を `S` で割って、
        `p_x = σ_x * l + τ_xy * m`
    *   **y方向の力のつり合い (ΣF_y = 0):**
        `p_y * S = (τ_xy * S_x) + (σ_y * S_y)`
        `p_y * S = τ_xy * (S * l) + σ_y * (S * m)`
        `p_y = τ_xy * l + σ_y * m`

3.  **行列形式で表現する**
    上記2つの式を行列でまとめると、以下のようになります。

**解答**
```
( p_x )   ( σ_x  τ_xy ) ( l )
( p_y ) = ( τ_xy  σ_y ) ( m )
```

#### (2) 垂直応力 σ_n とせん断応力 τ_n の計算

**方針・考え方**
`σ_n` は、応力ベクトル `p = (p_x, p_y)` を、面ACの法線方向 `n = (l, m)` に射影した成分です。`τ_n` は、`p` を接線方向 `t` に射影した成分です。ベクトルの内積を使って計算します。

**途中式と計算過程**
1.  **垂直応力 σ_n**
    `σ_n` は、ベクトル `p` と単位法線ベクトル `n` の内積です。
    `σ_n = p ⋅ n = p_x * l + p_y * m`
    これを行列で書くと、

    `σ_n = ( l  m ) ( p_x )`
                `( p_y )`

2.  **せん断応力 τ_n**
    `τ_n` は、ベクトル `p` と単位接線ベクトル `t` の内積です。`n=(l, m)=(cosθ, sinθ)` に対し、接線ベクトル `t` は `t=(-m, l)=(-sinθ, cosθ)` となります。
    `τ_n = p ⋅ t = p_x * (-m) + p_y * l = l*p_y - m*p_x`
    これを行列で書くと、

    `τ_n = ( -m  l ) ( p_x )`
                  `( p_y )`

**解答**
垂直応力 `σ_n`:
```
      ( l  m ) ( p_x )
σ_n =        ( p_y )
```
せん断応力 `τ_n`:
```
      ( -m  l ) ( p_x )
τ_n =         ( p_y )
```

#### (3) 行列の積での表現と各行列の意味

**方針・考え方**
(2)の式に(1)の結果を代入することで、`σ_n` と `τ_n` を3つの行列の積で表します。

**途中式と計算過程**
1.  **σ_n の表現**
    `σ_n = ( l  m ) ( p_x )` に、(1)の `(p_x, p_y)` の式を代入します。
                      `( p_y )`
    `σ_n = ( l  m ) ( σ_x  τ_xy ) ( l )`
                     `( τ_xy  σ_y ) ( m )`

2.  **τ_n の表現**
    `τ_n = ( -m  l ) ( p_x )` に、(1)の `(p_x, p_y)` の式を代入します。
                       `( p_y )`
    `τ_n = ( -m  l ) ( σ_x  τ_xy ) ( l )`
                      `( τ_xy  σ_y ) ( m )`

**各行列の意味**
上記の `A * B * C` のような積の形で考えたとき、各行列の意味は以下の通りです。

*   **一番右の行列 (C): 単位法線ベクトル n**
    ` ( l ) `
    ` ( m ) `
    これは、応力を考えたい面の**向き**を定義するベクトルです。どの傾きの面について考えるかを示します。

*   **中央の行列 (B): 応力テンソル σ**
    ` ( σ_x  τ_xy ) `
    ` ( τ_xy  σ_y ) `
    これは、ある点の**応力状態**を完全に記述する行列（テンソル）です。この行列が、面の向き（ベクトル `n`）を受け取って、その面に作用する力（応力ベクトル `p`）に変換する役割を果たします。材料内部のどの点でも、このテンソルが一つ決まれば、あらゆる方向の面の応力が計算できます。

*   **一番左の行列 (A): 座標変換・射影**
    `σ_n` の場合: `( l  m )` （法線ベクトル `n` の転置 `n^T`）
    `τ_n` の場合: `( -m  l )` （接線ベクトル `t` の転置 `t^T`）
    これは、(B)と(C)の積で得られた応力ベクトル `p` を、新しい座標軸（法線方向と接線方向）に**射影**して、垂直成分とせん断成分に分解する役割を持ちます。一種の座標変換と見なせます。

**解答**
`σ_n` と `τ_n` の行列積表現は上記の通り。各行列の意味は、右から順に「①面の向きの定義」「②応力状態の定義」「③成分への分解（射影）」となります。

#### (4) τ_xy = 0 の場合の関係式と図示

**方針・考え方**
(1)で求めた式に `τ_xy = 0` を代入します。得られた2つの式から、媒介変数である `l` と `m` を消去することで、`p_x` と `p_y` の関係式を導きます。

**途中式と計算過程**
1.  **関係式を導く**
    `τ_xy = 0` のとき、(1)の式は以下のようになります。
    `p_x = σ_x * l`  --- (a)
    `p_y = σ_y * m`  --- (b)

    法線ベクトルの成分 `l` と `m` には、`l^2 + m^2 = 1` という関係があります。
    式(a)から `l = p_x / σ_x`
    式(b)から `m = p_y / σ_y`
    これらを `l^2 + m^2 = 1` に代入すると、

    `(p_x / σ_x)^2 + (p_y / σ_y)^2 = 1`

2.  **p_x - p_y 平面上での図示**
    この式は、`p_x` を横軸、`p_y` を縦軸とする平面上で**楕円**を表す方程式です。
    *   中心は原点 (0, 0)。
    *   `p_x` 軸との交点（`p_y=0` のとき）は `p_x = ±σ_x`。
    *   `p_y` 軸との交点（`p_x=0` のとき）は `p_y = ±σ_y`。
    つまり、長半径が `σ_x`、短半径が `σ_y`（あるいはその逆）の楕円を描くことになります。この楕円は「ラメの応力楕円」として知られています。

**解答**
関係式: `(p_x / σ_x)^2 + (p_y / σ_y)^2 = 1`
図: `p_x-p_y` 平面上に、中心が原点で、x軸方向に `±σ_x`、y軸方向に `±σ_y` の範囲で広がる楕円を描きます。

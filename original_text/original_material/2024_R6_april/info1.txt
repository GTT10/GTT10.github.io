### **【問題1】自重による伸びの計算**

この問題は、棒自身の重さ（自重）によって、棒がどれくらい伸びるかを計算する問題です。形が違う3つのケース（円柱、円錐、途中で切れた円錐）について考えていきます。基本的な考え方は、「ある位置にかかる力は、それより下にある部分の重さに等しい」ということです。

#### **(1) 円柱 AB の応力と伸び** 🗼

\<img src="[https://storage.googleapis.com/gemini-prod/images/4741639d-2104-4c6e-81fa-2c8c6db27f4d](https://storage.googleapis.com/gemini-prod/images/4741639d-2104-4c6e-81fa-2c8c6db27f4d)" alt="Cylinder diagram with axis" width="200"/\>

**考え方**

1.  **応力（σ）とは？**: 応力とは、部材の内部で発生する「力の混み具合」のことで、「力 ÷ 断面積」で計算します。
2.  **どの位置の応力を考えるか**: 下端（B点）から `x` の高さにある断面を考えます。この断面には、それより下にある部分（長さ `x` の円柱）の重さが、下向きに引っ張る力としてのしかかります。
3.  **重さの計算**:
      * 断面積 `A` は、直径 `d0` なので `A = π(d0/2)² = πd0²/4` です [cite: 12]。
      * 下半分の体積 `V(x)` は「断面積 × 高さ」なので `V(x) = A * x` です。
      * 重さ `W(x)` は「密度(ρ) × 体積(V) × 重力加速度(g)」なので `W(x) = ρ * g * A * x` となります。
4.  **応力の計算**: 応力 `σ_AB(x)` は、この重さ `W(x)` を断面積 `A` で割ることで求まります。
    `σ_AB(x) = W(x) / A = (ρ * g * A * x) / A = ρgx`
5.  **伸び（δ）とは？**: 部材がどれだけ長くなったかを示す量です。全体の伸びは、微小な部分 `dx` の伸びを足し合わせる（積分する）ことで求まります。
6.  **微小な伸びの計算**: 長さ `dx` の部分の伸び `dδ` は、フックの法則から「(応力 / 縦弾性係数E) × dx」で計算できます。
    `dδ = (σ_AB(x) / E) * dx = (ρgx / E) * dx`
7.  **全体の伸びの計算**: この微小な伸びを、棒全体（`x=0` から `x=l` まで）で合計（積分）します。

**解答**

  * **下端から `x` の位置における応力 `σ_AB(x)`**:
    $$\sigma_{AB}(x) = \rho g x$$
  * **円柱全体の伸び `δ_AB`**:
    $$\delta_{AB} = \int_{0}^{l} \frac{\rho g x}{E} dx = \frac{\rho g}{E} \left[ \frac{x^2}{2} \right]_{0}^{l} = \frac{\rho g l^2}{2E}$$

#### **(2) 円錐 CD の応力と伸び** 🍦

\<img src="[https://storage.googleapis.com/gemini-prod/images/e6f16c14-2c0b-4d76-857c-87d0c3dd0d15](https://storage.googleapis.com/gemini-prod/images/e6f16c14-2c0b-4d76-857c-87d0c3dd0d15)" alt="Cone diagram with axis" width="200"/\>

**考え方**

円錐は円柱と違い、場所によって断面積が変わるのがポイントです。

1.  **断面積 `A(x)` の変化**:
      * 下端（D点, `x=0`）で直径は0、上端（C点, `x=l`）で直径は `d0` です [cite: 12]。
      * 直径 `d(x)` は高さ `x` に比例して大きくなるので、`d(x) = (d0/l) * x` と表せます。
      * したがって、断面積 `A(x)` は `A(x) = π(d(x)/2)² = π(d0*x / (2l))²` となります。
2.  **重さの計算**:
      * 下端から `x` までの部分（高さ `x` の小さな円錐）の体積 `V(x)` は「1/3 × 底面積 × 高さ」で計算できます。
      * `V(x) = (1/3) * A(x) * x`
      * 重さ `W(x)` は `W(x) = ρ * g * V(x) = ρ * g * (1/3) * A(x) * x` です。
3.  **応力の計算**: 応力 `σ_CD(x)` は、重さ `W(x)` を断面積 `A(x)` で割ります。
    `σ_CD(x) = W(x) / A(x) = (ρ * g * (1/3) * A(x) * x) / A(x) = (1/3)ρgx`
4.  **全体の伸びの計算**: 円柱の時と同様に、応力の式を積分して全体の伸びを求めます。

**解答**

  * **下端から `x` の位置における応力 `σ_CD(x)`**:
    $$\sigma_{CD}(x) = \frac{1}{3}\rho g x$$
  * **円錐全体の伸び `δ_CD`**:
    $$\delta_{CD} = \int_{0}^{l} \frac{\sigma_{CD}(x)}{E} dx = \int_{0}^{l} \frac{\rho g x}{3E} dx = \frac{\rho g}{3E} \left[ \frac{x^2}{2} \right]_{0}^{l} = \frac{\rho g l^2}{6E}$$

#### **(3) 切れた円錐 EF の応力と伸び** 🏺

\<img src="[https://storage.googleapis.com/gemini-prod/images/42e97a39-4d10-482a-a92c-63b5f92ff480](https://storage.googleapis.com/gemini-prod/images/42e97a39-4d10-482a-a92c-63b5f92ff480)" alt="Truncated cone diagram with axis" width="200"/\>

**考え方**

この形は、大きな円錐から小さな円錐を切り取ったものと考えるのがコツです。

1.  **断面積 `A(x)` の変化**:
      * 下端（F点, `x=0`）で直径は `d0/2`、上端（E点, `x=l`）で直径は `d0` です [cite: 12]。
      * 直径 `d(x)` は高さ `x` と共に直線的に増加するので、`d(x) = (d0/2) + (d0/(2l))x` と表せます。
2.  **重さの計算**:
      * 円錐の時と同様に、下端から `x` までの部分の体積 `V(x)` を求め、その重さ `W(x)` を計算します。この計算は少し複雑になりますが、`x` の位置の断面積 `A(x)` と、その下の部分の体積 `V(x)` を積分で求めます。
3.  **応力の計算**: 応力 `σ_EF(x)` は、重さ `W(x)` をその位置の断面積 `A(x)` で割ることで求まります。計算すると、興味深いことに、円錐の時とは違う少し複雑な形になります。
    `σ_EF(x) = (ρg/3) * [ (x+l) - l³ / (x+l)² ]`
4.  **全体の伸びの計算**: この応力の式を積分して、全体の伸び `δ_EF` を計算します。

**解答**

  * **下端から `x` の位置における応力 `σ_EF(x)`**:
    $$\sigma_{EF}(x) = \frac{\rho g}{3} \left( (x+l) - \frac{l^3}{(x+l)^2} \right)$$
  * **切れた円錐全体の伸び `δ_EF`**:
    $$\delta_{EF} = \int_{0}^{l} \frac{\sigma_{EF}(x)}{E} dx = \frac{\rho g}{3E} \int_{0}^{l} \left( (x+l) - \frac{l^3}{(x+l)^2} \right) dx = \frac{\rho g l^2}{3E}$$

### **【問題2】はりの曲げ**

この問題は、片方が固定され、もう片方が単純に支えられている「不静定はり」に、一様な力がかかる場合の反力や曲げモーメントを求める問題です。通常のつり合い式だけでは解けないため、はりの「たわみ（しなり）」を考慮する必要があります。

\<img src="[https://storage.googleapis.com/gemini-prod/images/f38a901d-55e1-4541-ba66-83395d826c71](https://storage.googleapis.com/gemini-prod/images/f38a901d-55e1-4541-ba66-83395d826c71)" alt="Propped cantilever beam" width="500"/\>

#### **(1) y方向の力の釣合い式**

**考え方**

これは、物体が上下に動かないための条件式です。「上向きの力の合計」と「下向きの力の合計」が等しい、と考えます。

  * **上向きの力**: A点の支点反力 `RA` と B点の支点反力 `RB` です [cite: 38, 45]。
  * **下向きの力**: 単位長さあたり `q` の力が、長さ `l` にわたってかかっているので、合計の力は `q * l` となります [cite: 33, 40]。

**解答**

力のつり合い式は以下の通りです。
$$R_A + R_B - ql = 0$$

#### **(2) 曲げモーメント M**

**考え方**

曲げモーメントとは、はりを曲げようとする回転の力のことです。A点から `x` の距離にある断面で、モーメントのつり合いを考えます。

1.  **左側の部分を考える**: `x` の断面から左側を見ると、A点に上向きの力 `RA` が、そして `x` の範囲に下向きの分布荷重 `q` がかかっています [cite: 33, 38]。
2.  **モーメントの計算**:
      * `RA` は `x` の距離で、はりを上に凸に曲げようとします（モーメント：`RA * x`）。
      * 分布荷重 `q` の合力 `qx` は、その中心（`x/2` の位置）に作用し、はりを下に凸に曲げようとします（モーメント：`- (qx) * (x/2)`）。
3.  **合計**: これらを足し合わせると、その断面の曲げモーメント `M` が求まります。

**解答**

点Aから任意の位置 `x` における曲げモーメント `M` は以下の通りです。
$$M(x) = R_A x - \frac{q x^2}{2}$$

#### **(3) 支持力 RA, RB, 支持モーメント MB**

**考え方**

このはりは「不静定」なので、力のつり合いだけでは `RA`, `RB`, `MB` を決められません。そこで、「はりのたわみ」に関する条件を使います。

1.  **たわみの計算**: (2)で求めた `M(x)` の式を2回積分すると、はりのたわみの式 `v(x)` が得られます。このとき、積分定数 `C1`, `C2` が現れます。
2.  **境界条件**: はりの支えられ方から、たわみに関する以下の条件がわかります。
      * A点 (`x=0`)：単純支持なので、たわみは **0** [cite: 33]。
      * B点 (`x=l`)：固定端なので、たわみは **0** [cite: 33]。
      * B点 (`x=l`)：固定端なので、はりの傾き（たわみ角）も **0** [cite: 33]。
3.  **連立方程式を解く**: これら3つの条件をたわみの式に代入すると、未知数であった `RA`, `C1`, `C2` を求めることができます。
4.  **残りの反力を計算**: `RA` がわかれば、(1)の力のつり合い式から `RB` が求まります。また、はり全体のモーメントのつり合いから `MB` を計算できます。

**解答**

  * **支持力 `RA`**:
    $$R_A = \frac{3}{8}ql$$
  * **支持力 `RB`**:
    $$R_B = \frac{5}{8}ql$$
  * **支持モーメント `MB`**:
    $$M_B = \frac{1}{8}ql^2$$

#### **(4) 曲げモーメントの大きさの最大値 |M|max**

**考え方**

曲げモーメントが最大になる場所は、はりが最も強く曲げられる場所であり、破壊の危険性が最も高い点です。

1.  **最大値の候補**: 曲げモーメントが最大値をとる可能性があるのは、以下の場所です。
      * せん断力（はりを断ち切ろうとする力）が **0** になる点。
      * はりの端（特に固定端）。
2.  **せん断力が0になる点**: せん断力 `V(x)` は `RA - qx` と表せます。`V(x)=0` とすると、`x = RA/q = (3/8)l` となります。この点でのモーメントを計算します。
3.  **固定端のモーメント**: B点 (`x=l`) でのモーメントは、(3)で求めた `MB` です。
4.  **比較**: 上記2点で計算したモーメントの「大きさ（絶対値）」を比較し、大きい方が最大値となります。計算すると、固定端Bでのモーメントが最も大きくなります。

**解答**

曲げモーメントの大きさの最大値 `|M|_max` は、固定端Bで生じます。
$$|M|_{max} = \frac{1}{8}ql^2$$

### **【問題1】段付き棒・テーパ棒の変形と熱応力**

この問題は、太さが一定の棒と、太さが徐々に変わる「テーパ棒」を組み合わせた部材について、引張による伸びと、温度変化による応力を考えます。

**この問題の最大のポイント** 💡
テーパ棒のように断面積が連続的に変化する場合、全体の伸びを求めるには、**薄い円盤状の微小部分**を考え、その部分の伸びを計算し、それを\*\*棒全体で足し合わせる（積分する）\*\*という手法を使います。

\<img src="[https://storage.googleapis.com/gemini-prod/images/53702a63-2f08-410a-8672-000c0f8bb1fd](https://www.google.com/search?q=https://storage.googleapis.com/gemini-prod/images/53702a63-2f08-410a-8672-000c0f8bb1fd)" alt="Diagram for Problem 1" width="500"/\>

#### **(1) 丸棒Ⅰの伸び λ₁ とテーパ棒Ⅱの伸び λ₂**

**考え方**

  * **丸棒Ⅰ**: 断面積が一定なので、伸びの基本公式 `λ = PL / (AE)` を使います。
      * `L = l₁`、`A = πd₁²/4` です。
  * **テーパ棒Ⅱ**: 断面積が場所によって変わるため、積分計算が必要です。
    1.  **直径d(x)の式**: まず、テーパ棒の左端からの距離 `x` における直径 `d(x)` を `x` の一次関数として表します。
    2.  **断面積A(x)の式**: `A(x) = π[d(x)]²/4` を計算します。
    3.  **微小部分の伸び**: 厚さ `dx` の薄い円盤の伸び `dλ₂` は、`dλ₂ = P*dx / (E*A(x))` となります。
    4.  **積分**: この `dλ₂` を、テーパ棒の全長（`x=0` から `x=l₂` まで）にわたって合計（積分）することで、全体の伸び `λ₂` を求めます。

**解答**

  * **丸棒Ⅰの伸び `λ₁`**:
    $$\lambda_1 = \frac{P l_1}{E (\pi d_1^2 / 4)} = \frac{4Pl_1}{\pi E d_1^2}$$
  * **テーパ棒Ⅱの伸び `λ₂`**:
    $$\lambda_2 = \int_0^{l_2} \frac{P dx}{E A(x)} = \frac{4Pl_2}{\pi E d_1 d_2}$$

#### **(2) 温度上昇させたときの丸棒Ⅰの軸応力**

**考え方**
これは、両端が壁で固定された棒の「熱応力」を考える問題です。

1.  **もし自由だったら**: 壁がなければ、棒全体は温度上昇 `ΔT` によって `δ_T = α(l₁+l₂)ΔT` だけ伸びるはずです。
2.  **壁による拘束**: 実際には壁がこの伸びを妨げるため、棒の内部には「圧縮する力 `Pc`」が発生します。
3.  **力による縮み**: この圧縮力 `Pc` によって、棒全体は `δ_P = λ₁ + λ₂`（ただし `P=Pc` としたもの）だけ縮みます。
4.  **つり合い**: 「熱で伸びようとする量」と「力で縮む量」が等しくなります (`δ_T = δ_P`)。
5.  **応力の計算**: このつり合い式から圧縮力 `Pc` を求め、それを丸棒Ⅰの断面積 `A₁` で割ることで、求めたい応力 `σ₁` が計算できます。

**解答**
丸棒Ⅰに生じる軸応力 `σ₁` (圧縮) は以下の通りです。
$$\sigma_1 = \frac{\alpha E (l_1+l_2) \Delta T}{l_1 + l_2 \frac{d_1}{d_2}}$$

### **【問題2】一端固定・他端支持はりの曲げ**

この問題は、片方が壁に固定され、もう片方が支えられている「不静定はり」に、三角形の分布荷重がかかる場合を扱います。これも、力のつり合いだけでは解けないため、はりの変形（たわみ）を考慮する必要があります。

\<img src="[https://storage.googleapis.com/gemini-prod/images/41d5b1da-ebf0-45f8-a006-25816ef06277](https://www.google.com/search?q=https://storage.googleapis.com/gemini-prod/images/41d5b1da-ebf0-45f8-a006-25816ef06277)" alt="Diagram for Problem 2" width="500"/\>

#### **(1) 曲げモーメント M(x)**

**考え方**
はりの左端Aから `x` の距離にある断面で、モーメントのつり合いを考えます。この断面より左側には、A点の反力 `RA`（未知数）と、`x` の範囲にかかる三角形の分布荷重が作用しています。

1.  **分布荷重の大きさ**: `x` の位置での荷重の強さ `w(x)` は、`w(x) = (w₁/l)x` と表せます。
2.  **分布荷重の合力**: `x` までにかかる分布荷重の合計の力は、三角形の面積なので `(1/2) * x * w(x) = w₁x²/(2l)` となります。
3.  **モーメントの計算**: この合力は、Aから `2x/3` の位置（三角形の重心）に作用します。これらの力からモーメントを計算します。

**解答**
曲げモーメント `M(x)` は、未知数 `RA` を用いて以下のように表せます。
$$M(x) = R_A x - \frac{w_1 x^3}{6l}$$

-----

#### **(2) 反力 RA**

**考え方**
不静定問題を解くため、はりの「たわみ」の条件を使います。

1.  **たわみの計算**: (1)で求めた `M(x)` の式を2回積分すると、はりのたわみの式 `y(x)` が得られます。このとき、積分定数が2つ出てきます。
2.  **境界条件**: はりの支えられ方から、以下の3つの条件がわかります。
      * **A点 (`x=0`)**: 単純支持なので、**たわみ `y` はゼロ**。
      * **B点 (`x=l`)**: 壁に固定されているので、**たわみ `y` も傾き `y'` もゼロ**。
3.  **反力の決定**: これらの3つの条件をたわみの式に適用することで、未知数だった反力 `RA` と2つの積分定数を決定できます。

**解答**
反力 `RA` は以下の通りです。
$$R_A = \frac{w_1 l}{10}$$

-----

#### **(3) 反力 RB, モーメント MB と SFD, BMD**

### **【問題3】平面応力状態**

この問題は、一枚の薄い板のように、厚み方向の力を無視できる「平面応力状態」において、座標系を回転させると応力がどのように変化して見えるかを扱う問題です。

**この問題の最大のポイント** 💡
物体のある一点における応力の状態は、見る角度（座標系）によって、垂直応力とせん断応力の組み合わせが変化して見えます。この変換は、**応力変換式**や**モールの応力円**を使って計算・理解することができます。

\<img src="[https://storage.googleapis.com/gemini-prod/images/457f00d3-2f03-4c55-b461-163f5383f982](https://www.google.com/search?q=https://storage.googleapis.com/gemini-prod/images/457f00d3-2f03-4c55-b461-163f5383f982)" alt="Diagram for Problem 3" width="500"/\>

#### **(1) 回転後の応力成分**

**考え方**
元のxy座標系での応力 (`σx`, `σy`, `τxy`) がわかっているとき、座標系を `θ` だけ回転させたx'y'座標系での応力 (`σx'`, `σy'`, `τx'y'`) を求めるには、**応力変換の公式**を使います。この問題では、`τxy=0`、`θ=π/4` の場合を考えます。

**解答**

  * **垂直応力 `σx'`**:
    $$\sigma_{x'} = \frac{\sigma_x + \sigma_y}{2}$$
  * **垂直応力 `σy'`**:
    $$\sigma_{y'} = \frac{\sigma_x + \sigma_y}{2}$$
  * **せん断応力 `τx'y'`**:
    $$\tau_{x'y'} = -\frac{\sigma_x - \sigma_y}{2}$$

-----

#### **(2) 特別な応力状態**

**考え方**
(1)で求めた式に、`σx = -σ`, `σy = σ` という具体的な値を入れると、x'y'座標系ではどのような応力状態になるかを計算します。

**解答**

  * **応力成分**:
      * `σx' = (-σ + σ) / 2 = 0`
      * `σy' = (-σ + σ) / 2 = 0`
      * `τx'y' = -(-σ - σ) / 2 = σ`
  * **応力状態の名称**:
    この、垂直応力がゼロで、せん断応力のみが作用している状態を「**純せん断**」と呼びます。

-----

#### **(3) 最大主応力と最小主応力**

**考え方**
**主応力**とは、その点で生じる垂直応力の最大値と最小値のことです。主応力が生じる面では、せん断応力はゼロになります。
元のxy座標系では、せん断応力がゼロなので、`σx` と `σy` 自身が主応力となります。

**解答**

  * **最大主応力 `σmax`**:
    $$\sigma_{max} = \sigma_y = \sigma$$
  * **最小主応力 `σmin`**:
    $$\sigma_{min} = \sigma_x = -\sigma$$

-----

#### **(4) 最大主ひずみと最小主ひずみ**

**考え方**
**主ひずみ**は、主応力の方向に生じるひずみです。平面応力状態の「一般化されたフックの法則」を使って、各主応力から主ひずみを計算します。
`ε = (1/E) * (σ₁ - νσ₂)`

**解答**

  * **最大主ひずみ `εmax`**:
    $$\epsilon_{max} = \frac{1}{E}(\sigma_y - \nu\sigma_x) = \frac{1}{E}(\sigma - \nu(-\sigma)) = \frac{\sigma(1+\nu)}{E}$$
  * **最小主ひずみ `εmin`**:
    $$\epsilon_{min} = \frac{1}{E}(\sigma_x - \nu\sigma_y) = \frac{1}{E}(-\sigma - \nu\sigma) = -\frac{\sigma(1+\nu)}{E}$$

-----

#### **(5) 最大せん断ひずみの大きさ**

**考え方**
最大せん断ひずみ `γmax` は、最大せん断応力 `τmax` との関係式 `τ = Gγ` から求められます。

1.  **最大せん断応力を求める**: `τmax = (σmax - σmin) / 2`
2.  **横弾性係数Gを変換する**: `G = E / (2(1+ν))` の関係を使います。
3.  **γmaxを計算する**: `γmax = τmax / G`

**解答**
最大せん断ひずみの大きさ `|γmax|` は以下の通りです。
$$|\gamma_{max}| = \frac{\tau_{max}}{G} = \frac{(\sigma - (-\sigma))/2}{E/(2(1+\nu))} = \frac{\sigma}{E/(2(1+\nu))} = \frac{2\sigma(1+\nu)}{E}$$

### **【問題1】はりのたわみと重ね合わせ**

この問題は、まず基本的な「片持ちはり」の変形を計算し、その結果を利用して、より複雑な「両端固定はり」の問題を解くという構成になっています。これは**重ね合わせの原理**という、材料力学の強力なツールを学ぶための典型的な問題です。

\<img src="[https://storage.googleapis.com/gemini-prod/images/50f9a562-b942-4f36-9b57-617a26c48375](https://storage.googleapis.com/gemini-prod/images/50f9a562-b942-4f36-9b57-617a26c48375)" alt="Diagrams for problem 1" width="700"/\>

#### **(1) 図1(a)の反力と曲げモーメント**

**考え方**
これは、静力学の基本的なつり合いの問題です。はりが静止しているためには、「力の合計がゼロ」かつ「モーメント（回転させる力）の合計がゼロ」でなければなりません。

  * **力のつり合い**: 下向きの力 `P` と、壁がはりを支える上向きの反力 `RB₁` がつり合います。
  * **モーメントのつり合い**: B点を中心に考えると、力 `P` がはりを時計回りに回転させようとするモーメント (`P × l`) と、壁がそれを防ぐために反時計回りに加えるモーメント `MB₁` がつり合います。

**解答**

  * **反力 `RB₁`**:
    $$R_{B1} = P$$
  * **曲げモーメント `MB₁`**:
    $$M_{B1} = Pl$$

-----

#### **(2) 図1(a)のたわみ角とたわみ**

**考え方**
これは、片持ちはりの先端に荷重がかかった場合の変形量を求める問題で、材料力学で最も基本的な公式の一つです。これらの公式は、曲げモーメントの式を積分することで導出されます。

**解答**

  * **自由端Aにおけるたわみ角 `θA₁`**:
    $$\theta_{A1} = \frac{Pl^2}{2EI}$$
  * **自由端Aにおけるたわみ `yA₁`**:
    $$y_{A1} = \frac{Pl^3}{3EI}$$

-----

#### **(3) 図1(b)の反力と曲げモーメント**

**考え方**
今度は、はりの先端に直接モーメント `M` がかかる場合です。これも(1)と同様に、静力学のつり合いで考えます。

  * **力のつり合い**: 上下方向の力はかかっていないので、壁からの反力 `RB₂` はゼロです。
  * **モーメントのつり合い**: 先端のモーメント `M` と、壁からの反力モーメント `MB₂` がつり合います。

**解答**

  * **反力 `RB₂`**:
    $$R_{B2} = 0$$
  * **曲げモーメント `MB₂`**:
    $$M_{B2} = M$$

-----

#### **(4) 図1(b)のたわみ角とたわみ**

**考え方**
これも(2)と同様に、片持ちはりの先端にモーメントがかかった場合の変形量を求める公式です。

**解答**

  * **自由端Aにおけるたわみ角 `θA₂`**:
    $$\theta_{A2} = \frac{Ml}{EI}$$
  * **自由端Aにおけるたわみ `yA₂`**:
    $$y_{A2} = \frac{Ml^2}{2EI}$$

-----

#### **(5) 図1(c)の反力と曲げモーメント**

**考え方**
ここが**重ね合わせの原理**の応用です。図1(c)の状態は、単純な片持ちはり（壁Bに固定）に対して、A点に上向きの力 `RA` と、反時計回りのモーメント `MA` を同時に加えることで、「**最終的にA点のたわみが `δ` になり、かつ傾きがゼロになる**」状態を作り出した、と考えることができます。

1.  **重ね合わせる**:
      * 力 `RA` によるたわみと傾き（(2)の結果を利用）
      * モーメント `MA` によるたわみと傾き（(4)の結果を利用）
2.  **条件式を立てる**:
      * たわみの合計 = `δ`
      * 傾きの合計 = `0`
3.  **連立方程式を解く**: 上記の2つの条件式を、未知数 `RA` と `MA` についての連立方程式として解きます。

**解答**

  * **反力 `RA`**:
    $$R_A = \frac{12EI\delta}{l^3}$$
  * **曲げモーメント `MA`**:
    $$M_A = \frac{6EI\delta}{l^2}$$

### **【問題2】自重による棒の応力**

この問題は、棒自身の重さ（自重）によって、棒の内部にどのような応力（引張力）が発生するかを考えます。

**この問題の最大のポイント** 💡
ある断面にかかる力は、その**断面よりも下にある部分すべての重さの合計**に等しくなります。したがって、応力は棒の上部ほど大きくなります。

#### **(1) 棒1の垂直応力 σ₁(x)**

**考え方**
棒の下端から `x` の距離にある断面を考えます。この断面には、それより下にある部分（長さ `x` の棒）の重さがぶら下がっていることになります。

  * **下部の重さ**: `W(x) = (比重量 γ) × (体積 V) = γ × (断面積 A₁ × 長さ x)`
  * **応力**: `σ₁(x) = W(x) / A₁`

**解答**
$$\sigma_1(x) = \frac{\gamma A_1 x}{A_1} = \gamma x$$

-----

#### **(2) 棒1の最大応力 σ₁max**

**考え方**
(1)で求めた式 `σ₁(x) = γx` からわかるように、応力は `x` が大きいほど、つまり\*\*棒の最上部（根元）\*\*で最大になります。

**解答**
$$\sigma_{1max} = \gamma l_1$$

-----

#### **(3) 棒2（重り付き）の垂直応力 σ₂(x)**

**考え方**
(1)と考え方は同じですが、今回は棒の下にさらに重り `W` がぶら下がっています。

  * **かかる力の合計**: `F(x) = (棒の下部の重さ) + (重りの重さ W) = γA₂x + W`
  * **応力**: `σ₂(x) = F(x) / A₂`

**解答**
$$\sigma_2(x) = \frac{W + \gamma A_2 x}{A_2} = \frac{W}{A_2} + \gamma x$$

-----

#### **(4) 棒2の最大応力 σ₂max**

**考え方**
(3)の式から、応力はやはり**棒の最上部 (`x=l₂`)** で最大になります。

**解答**
$$\sigma_{2max} = \frac{W}{A_2} + \gamma l_2$$

-----

#### **(5) 段付棒の最大長さ lmax**

**考え方**
この段付き棒が壊れるのは、棒の中で最も応力が高い部分が、材料の限界強度 `σB` に達したときです。

1.  **最も危険な場所を探す**: 応力は常に上部ほど高くなります。特に、断面積が細くなる棒1の最上部（根元）が最も応力が大きくなり、最も危険な箇所となります。
2.  **最大応力の式を立てる**: 棒1の最上部にかかる力は、「棒1全体の重さ」と「棒2全体の重さ」の合計です。この力から最大応力を計算します。
3.  **最大長さを求める**: 「最大応力 = 引張強さ `σB`」という式を立て、全体の長さ `l = l₁ + l₂` が最大になる条件を求めます。計算すると、応力が大きい細い棒1 (`l₁`) をできるだけ短くし、太い棒2 (`l₂`) を長くするほど全体の長さを稼げることがわかります。極限として、`l₁=0` のとき、つまり**全体が太い棒2だけでできているとき**に長さが最大になります。

**解答**
両棒を合わせた長さが最大となる `lmax` は、細い棒1の長さ `l₁` をゼロにした場合に相当します。
$$l_{max} = \frac{\sigma_B}{\gamma}$$

### **【問題3】曲げとねじりを同時に受けるL字レバー**

この問題は、L字型の棒に力がかかったときに、場所によって「曲げ」だけが作用したり、「曲げ」と「ねじり」が同時に作用したりする状況を扱います。

**この問題の最大のポイント** 💡
ある点にかかる力は、その点まで**どのように伝わってきたか**を考えることで、曲げモーメントやねじりモーメントを計算できます。特に、力をその力の作用線と平行に移動させるときは、**モーメント（回転力）も一緒に移動する**と考えると分かりやすいです。

#### **(1) 断面Cの曲げモーメント Mc**

**考え方**
アームABの部分だけを見ると、これは壁Aに固定された長さ `w` の片持ちはりと考えることができます。B点にかかる力 `P` は、C点に対して `a` の距離で曲げモーメントを発生させます。

**解答**
$$M_c = P a$$

-----

#### **(2) 断面Dの曲げモーメント MD とねじりモーメント TD**

**考え方**
B点にかかっている力 `P` を、垂直なアームOA上の断面Dまで伝えていきます。

  * **ねじりモーメント `TD`**: B点にかかる力 `P` は、アームOAの軸に対して `w` の距離分離れています。この「力 `P` × 腕の長さ `w`」が、アームOAを雑巾のようにねじる「ねじりモーメント」になります。このねじりモーメントは、OAのどこでも一定です。
  * **曲げモーメント `MD`**: ねじりとは別に、力 `P` はアームOAを曲げようとします。断面Dから見ると、力 `P` は `h-b` の高さでかかっているのと同じなので、これが腕の長さとなって曲げモーメントを発生させます。

**解答**

  * **曲げモーメント `MD`**:
    $$M_D = P(h-b)$$
  * **ねじりモーメント `TD`**:
    $$T_D = Pw$$

-----

#### **(3) 断面Dの応力**

**考え方**
曲げモーメントは「曲げ応力（引張・圧縮）」を、ねじりモーメントは「せん断応力（ずれ）」を発生させます。それぞれを標準的な公式で計算します。応力はどちらも棒の外表面で最大になります。

**解答**

  * **曲げ応力 `σD_max`**:
    $$\sigma_{Dmax} = \frac{M_D (d/2)}{I} = \frac{P(h-b)(d/2)}{\pi d^4 / 64} = \frac{32P(h-b)}{\pi d^3}$$
  * **せん断応力 `τD_max`**:
    $$\tau_{Dmax} = \frac{T_D (d/2)}{J_p} = \frac{Pw(d/2)}{\pi d^4 / 32} = \frac{16Pw}{\pi d^3}$$

-----

#### **(4) 断面Dの最大主応力**

**考え方**
ある点に「曲げ応力 `σ`」と「せん断応力 `τ`」が同時にかかっているとき、それらを合成した結果、本当に最も大きくなる引張応力（最大主応力）は、斜め方向に発生します。これは、モールの応力円や、以下の合成応力の公式を使って計算します。

**解答**
最大主応力 `σ₁` は以下の公式で求まります。
$$\sigma_1 = \frac{\sigma_D}{2} + \sqrt{\left(\frac{\sigma_D}{2}\right)^2 + \tau_D^2} = \frac{16P}{\pi d^3} \left( (h-b) + \sqrt{(h-b)^2 + w^2} \right)$$

-----

#### **(5) 最小の丸棒直径**

**考え方**
レバーが壊れないためには、「(4)で求めた最大主応力 `σ₁`」が、「材料が耐えられる限界の応力（許容応力 `σa`）」を超えなければ良い、という条件式を立てます。
`σ₁ ≤ σa`
この式を、直径 `d` について解きます。

**解答**
力 `P` に耐える最小の丸棒直径 `d_min` は以下の通りです。
$$d_{min} = \sqrt[3]{\frac{16P}{\pi \sigma_a} \left( (h-b) + \sqrt{(h-b)^2 + w^2} \right)}$$

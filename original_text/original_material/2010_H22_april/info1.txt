### **【問題1】段付き棒の引張**

この問題は、太さが途中で変わる「段付き棒」を引っ張ったときに、内部にどのような応力が発生し、全体としてどれだけ伸びるかを考えます。棒自身の重さ（自重）を考える場合と、考えない場合を比較します。

\<img src="[https://storage.googleapis.com/gemini-prod/images/07d581fd-4b41-477d-8152-320c291244e8](https://storage.googleapis.com/gemini-prod/images/07d581fd-4b41-477d-8152-320c291244e8)" alt="Diagram of a stepped rod under tension" width="200"/\>

#### **(1) 自重を考慮しない場合の応力と伸び**

**考え方**
自重を無視する場合、棒のどの部分にも、先端にかかる引張荷重 `P` と同じ大きさの力がかかっている、と考えることができます。

1.  **応力の計算**: 応力 `σ` は「力 ÷ 断面積」です。直径 `d` の丸棒の断面積は `A = πd²/4` です。
      * 細い棒（直径`d₁`）：`σ₁ = P / A₁ = 4P / (πd₁²)`
      * 太い棒（直径`d₂`）：`σ₂ = P / A₂ = 4P / (πd₂²)`
2.  **伸びの計算**: 棒全体の伸びは、「細い棒の伸び」と「太い棒の伸び」の足し算になります。棒の伸びの公式は `λ = PL / (AE)` です。

**解答**

  * **応力**:
    $$\sigma_1 = \frac{4P}{\pi d_1^2}, \quad \sigma_2 = \frac{4P}{\pi d_2^2}$$
  * **全体の伸び**:
    $$\lambda = \lambda_1 + \lambda_2 = \frac{Pl}{E A_1} + \frac{Pl}{E A_2} = \frac{4Pl}{\pi E} \left( \frac{1}{d_1^2} + \frac{1}{d_2^2} \right)$$

-----

#### **(2) 自重を考慮した場合の最大応力**

**考え方**
自重を考慮すると、棒の上の方の断面ほど、その下にある部分全体の重さが加わるため、かかる力が大きくなります。

1.  **力の分布**: 太い棒（直径 `d₂`）のある断面にかかる力は、「先端の荷重`P`」+「細い棒全体の重さ」+「その断面より下にある太い棒部分の重さ」の合計です。
2.  **最大応力の発生場所**: この力は、一番上、つまり**天井との接合部で最大**になります。
3.  **最大応力の計算**: この最大の力（`P` + 棒全体の重さ）を、太い棒の断面積 `A₂` で割ることで、最大応力が求まります。

**解答**
直径 `d₂` の棒に生じる最大応力 `(σ_max)₂` は、棒の最上部で発生し、以下のようになります。
$$(\sigma_{max})_2 = \frac{4P}{\pi d_2^2} + \rho g l \left( 1 + \frac{d_1^2}{d_2^2} \right)$$

-----

#### **(3) 自重を考慮した場合の全体の伸び**

**考え方**
自重を考えると、棒の内部の力は場所によって continuously に変化します。そのため、全体の伸びを求めるには、微小な部分の伸びを考えて、それを棒全体で合計（積分）する必要があります。

1.  **伸びの内訳**: 全体の伸び `λ` は、「荷重 `P` による伸び」と「自重による伸び」の合計です。
2.  **荷重Pによる伸び**: これは(1)で計算したものと同じです。
3.  **自重による伸び**: 細い棒と太い棒、それぞれの自重による伸びを計算して足し合わせます。
4.  **合計**: 2.と3.を足し合わせることで、全体の伸びが求まります。

**解答**
自重を考慮した場合の段付き棒全体の伸び `λ` は以下の通りです。
$$\lambda = \frac{4Pl}{\pi E} \left( \frac{1}{d_1^2} + \frac{1}{d_2^2} \right) + \frac{\rho g l^2}{E} \left( 1 + \frac{d_1^2}{d_2^2} \right)$$

### **【問題2】片側突出しはり**

この問題は、片方が壁に固定され、途中が支えられ、もう一端が突き出している「不静定はり」です。力のつり合いだけでは解けないため、はりの変形（たわみ）を考慮する必要があります。

\<img src="[https://storage.googleapis.com/gemini-prod/images/8c06e123-b1d6-4e56-b072-5a242fca1e3a](https://storage.googleapis.com/gemini-prod/images/8c06e123-b1d6-4e56-b072-5a242fca1e3a)" alt="Diagram for Problem 2" width="500"/\>

#### **(1) B点の曲げモーメント MB**

**考え方**
B点より右側の突き出し部分（BC間）だけを見ると、これは長さ `l/2` の片持ちはりと見なせます。B点はその固定端にあたります。この部分にかかる分布荷重 `w` が、B点に曲げモーメントを発生させます。

**解答**
B点の曲げモーメント `MB` は、突き出し部分の荷重によって生じ、上に凸の曲げ（負のモーメント）となるため、以下のようになります。
$$M_B = -w \left(\frac{l}{2}\right) \left(\frac{l}{4}\right) = -\frac{wl^2}{8}$$

-----

#### **(2) 曲げモーメント M(x)**

**考え方**
はりの左端Aから `x` の距離にある断面で、モーメントのつり合いを考えます。この断面より左側には、壁からの反力 `RA`、壁からの固定モーメント `MA`、そして長さ `x` にわたる分布荷重 `w` が作用しています。

**解答**
曲げモーメント `M(x)` は、未知数 `RA`, `MA` を用いて以下のように表せます。
$$M(x) = R_A x - M_A - \frac{wx^2}{2}$$

-----

#### **(3) 反力 RA と曲げモーメント MA**

**考え方**
不静定問題を解くため、はりの「たわみ」の条件を使います。

1.  **たわみの計算**: (2)で求めた `M(x)` の式を2回積分すると、はりのたわみの式 `y(x)` が得られます。
2.  **境界条件**:
      * **A点 (`x=0`)**: 壁に固定されているので、**たわみ `y` も傾き `y'` もゼロ**です。
      * **B点 (`x=l`)**: 支えられているので、**たわみ `y` はゼロ**です。
3.  **連立方程式**: これらの条件と、(1)で求めた `M(l) = -wl²/8` という条件を使うことで、未知数だった `RA` と `MA` を決定できます。

**解答**

  * **反力 `RA`**:
    $$R_A = \frac{7}{16}wl$$
  * **曲げモーメント `MA`**:
    $$M_A = \frac{1}{16}wl^2$$

-----

#### **(4) 反力 RB と SFD, BMD**

**考え方**

  * **RBの計算**: (3)で `RA` が求まったので、はり全体（ABC）の力のつり合いを考えれば `RB` が求まります。
  * **SFD（せん断力図）**: はりの各位置で、切断しようとする力（せん断力）がどのように変化するかを描きます。

### **【問題3】棒の引張と体積変化**

この問題は、棒を引っ張ったときに、長さ方向だけでなく、それに垂直な横方向にも変形が起こる「ポアソン効果」と、それに伴う体積の変化について考えます。

\<img src="[https://storage.googleapis.com/gemini-prod/images/c5108221-a2b1-4d32-841c-30c88385bbd1](https://storage.googleapis.com/gemini-prod/images/c5108221-a2b1-4d32-841c-30c88385bbd1)" alt="Diagram for Problem 3" width="400"/\>

#### **(1) 軸方向のひずみ εL**

**考え方**
ひずみとは「元の長さに対する伸びの割合」です。フックの法則 `σ = Eε` から、`ε = σ/E` となります。応力 `σ` は `P/A` です。

**解答**
$$\epsilon_L = \frac{\sigma}{E} = \frac{P/a^2}{E} = \frac{P}{a^2 E}$$

-----

#### **(2) 軸に垂直な方向のひずみ εT**

**考え方**
棒を縦に伸ばすと、横には縮みます。この「縦のひずみ」と「横のひずみ」の比が**ポアソン比 `ν`** です。`ν = - (横ひずみ) / (縦ひずみ)` という関係があります。

**解答**
$$\epsilon_T = -\nu \epsilon_L = -\frac{\nu P}{a^2 E}$$

-----

#### **(3) 体積増加率 ΔV/V**

**考え方**
体積 `V = l * a * a` が、ひずみによって `V' = l(1+εL) * a(1+εT) * a(1+εT)` に変化します。体積増加率 `ΔV/V = (V'-V)/V` を計算します。ひずみは非常に小さい値なので、`ε²` や `ε³` のような項は無視して近似計算します。

**解答**
$$\frac{\Delta V}{V} \approx \epsilon_L + 2\epsilon_T = \frac{P}{a^2 E} - \frac{2\nu P}{a^2 E} = \frac{P(1-2\nu)}{a^2 E}$$

-----

#### **(4) 側面を拘束したときの軸方向ひずみ**

**考え方**
側面を拘束されると、横方向に縮もうとしても縮めなくなります（`εT = 0`）。しかし、縮もうとする力は発生しており、それが逆に軸方向の伸びやすさに影響します。この複雑な関係は「一般化されたフックの法則」を使って解きます。

**解答**
$$\epsilon_L' = \frac{P}{a^2 E} \frac{(1+\nu)(1-2\nu)}{1-\nu}$$

-----

#### **(5) 側面を拘束したときの体積増加率**

**考え方**
体積増加率は、各方向のひずみの和で近似できます (`ΔV/V ≈ εx + εy + εz`)。この問題では、側面方向のひずみ（`εy`, `εz`）は拘束によってゼロなので、体積増加率は軸方向のひずみ `εL'` そのものになります。

**解答**
$$\frac{\Delta V}{V} = \epsilon_L' = \frac{P}{a^2 E} \frac{(1+\nu)(1-2\nu)}{1-\nu}$$

### **【問題1】両端が固定された棒に働く力**

この問題は、両端を壁にがっちり固定された一本の棒に、軸方向（棒の長さ方向）の力がかかったときに、壁がどのように棒を支えるか（反力）、そして棒がどれだけ伸び縮みするかを考える問題です。

**この問題の最大のポイント** 💡
棒は両端を固定されているため、**全体の長さは変わることができません**。つまり、「左側の部分の伸び」と「右側の部分の伸び」を足し合わせると、必ずゼロになる、という「**変形のつり合い**」を考える必要があります。

#### [cite_start]**(1) 壁の反力 RA と RB の計算 (力が1つの場合) [cite: 13]**

**考え方**

1.  [cite_start]**力のつり合い**: まず、単純な力のつり合いを考えます。右向きを正とすると、壁Aからの反力 `RA`、外力 `P`、壁Bからの反力 `RB` の間には `P = RA + RB` の関係が成り立ちます [cite: 13]。しかし、これだけでは未知数が2つ（`RA`, `RB`）あるので解けません。
2.  **変形のつり合い**: ここでポイントの登場です。棒全体の伸び縮みはゼロです。
    * AC間は、壁Aからの反力 `RA` によって引っ張られ、伸びます。伸びる量は `δ_AC = (RA * a) / (S * E)` です。
    * CB間は、壁Bからの反力 `RB` によって圧縮され、縮みます。縮む量は `δ_CB = -(RB * (l-a)) / (S * E)` です。（マイナスは縮みを表します）
    * 全体の伸びがゼロなので、`δ_AC + δ_CB = 0` となります。
3.  **連立方程式を解く**: 上記の2つの式を組み合わせることで、`RA` と `RB` を求めることができます。

**解答**

* 壁Aから受ける反力 `RA`:
    $$R_A = \frac{P(l-a)}{l}$$
* 壁Bから受ける反力 `RB`:
    $$R_B = \frac{Pa}{l}$$

---

#### [cite_start]**(2) AC間とCB間の伸び [cite: 14]**

**考え方**

(1)で求めた反力 `RA` と `RB` を、それぞれの区間の伸びの式に代入するだけです。

**解答**

* AC間の伸び `δ_AC`:
    $$\delta_{AC} = \frac{R_A a}{SE} = \frac{P a (l-a)}{lSE}$$
* CB間の伸び `δ_CB`:
    $$\delta_{CB} = -\frac{R_B (l-a)}{SE} = -\frac{P a (l-a)}{lSE}$$
    （AC間が伸びた分、CB間がちょうど同じだけ縮んでいることがわかります）

---

#### [cite_start]**(3) 壁の反力 RA と RB の計算 (力が2つの場合) [cite: 15]**

**考え方**

力が2つになっても基本的な考え方は同じですが、「**重ね合わせの原理**」を使うとより簡単に解くことができます。これは、「`P1` だけがかかった場合」と「`P2` だけがかかった場合」を別々に計算し、その結果を単純に足し合わせればよい、という便利な原理です。

1.  **`P1` による反力**: (1)の結果を使い、`P` を `P1` に、`a` を `a1` に置き換えます。
    * `RA1 = P1(l-a1)/l`
    * `RB1 = P1*a1/l`
2.  **`P2` による反力**: `P2` は左向き（マイナスの力）であることに注意し、(1)の結果を使います。
    * `RA2 = (-P2)(l-a2)/l`
    * `RB2 = (-P2)*a2/l`
3.  **合計**: それぞれの反力を足し合わせます。`RA = RA1 + RA2`, `RB = RB1 + RB2`。

**解答**

* 壁Aから受ける反力 `RA`:
    $$R_A = \frac{P_1(l-a_1) - P_2(l-a_2)}{l}$$
* 壁Bから受ける反力 `RB`:
    $$R_B = \frac{P_1 a_1 - P_2 a_2}{l}$$

---

#### [cite_start]**(4) 壁の反力 RA と RB の計算 (分布する力の場合) [cite: 16, 17]**

**考え方**

力が棒全体に分布している場合、微小な部分 `dx` にかかる力を考えて、それを全体で合計（積分）するというアプローチをとります。

1.  **内部の力 `T(x)`**: 棒の左端から `x` の位置での内部の力（張力）は、壁の反力 `RA` から、`x` までにかかる分布荷重の合計を引いたものになります。
2.  **微小な伸び `dδ`**: 長さ `dx` の微小部分の伸びは `dδ = T(x) dx / (SE)` です。
3.  **全体の伸びがゼロ**: この `dδ` を棒全体（`x=0` から `x=l` まで）で積分したものがゼロになる、という式を立てて `RA` を求めます。
4.  **RBを求める**: `RA` がわかれば、力のつり合い（`RA + RB = 分布荷重の総和`）から `RB` が求まります。

**解答**

* 壁Aから受ける反力 `RA`:
    $$R_A = \frac{1}{l} \int_{0}^{l} \left( \int_{0}^{x} q(t) dt \right) dx$$
* 壁Bから受ける反力 `RB`:
    $$R_B = \int_{0}^{l} q(x) dx - R_A$$

---

#### [cite_start]**(5) 壁の反力 RA と RB の計算 (具体的な分布荷重の場合) [cite: 18]**

**考え方**

(4)で求めた一般式に、具体的な関数 `q(x) = q₀(x/l)` を代入して積分計算を実行します。

**解答**

* 壁Aから受ける反力 `RA`:
    $$R_A = \frac{1}{6} q_0 l$$
* 壁Bから受ける反力 `RB`:
    $$R_B = \frac{1}{3} q_0 l$$

### **【問題2】両端が固定された段付きはりの曲げ**

この問題は、両端を固定された、途中で太さが変わる「段付きはり」の中央に力がかかったときの曲げモーメントやたわみを求める問題です。

**この問題の最大のポイント** 💡
はりの形と力の掛かり方が**左右対称**であるため、「中央での傾きはゼロ」「左右の壁での反力や固定モーメントは同じ」といった対称性を利用することで、計算を大幅に簡略化できます。

#### [cite_start]**(1) 曲げモーメント M の計算 [cite: 49]**

**考え方**

1.  **対称性の利用**: はり全体が左右対称なので、A点とB点での垂直な反力は、中央の荷重 `P` を半分ずつ分担します。つまり `RA = P/2` です。
2.  **自由体図**: はりの左端から `x` の位置で切断したと想像し、左側の部分だけを取り出します（これを自由体図といいます）。
3.  **モーメントのつり合い**: この自由体図において、切断面に発生する曲げモーメント `M(x)` は、A点に作用する反力 `RA` と固定モーメント `MA` とつり合います。

**解答**

A点から距離 `x` (ただし `0 ≤ x ≤ 2l`) の位置における曲げモーメント `M` は以下の通りです。
$$M(x) = \frac{P}{2}x - M_A$$

---

#### [cite_start]**(2) C点のたわみ vC の計算 [cite: 50]**

**考え方**

1.  **たわみの計算**: たわみは、曲げモーメントの式を2回積分することで求められます。
2.  **境界条件**: 壁A (`x=0`) は完全に固定されているため、「たわみ `v`」も「傾き `dv/dx`」も両方ゼロです。
3.  **計算の実行**: (1)で求めたモーメントの式 `M(x)` を `0 ≤ x ≤ l` の範囲（AC間）で2回積分し、上記の境界条件を適用すると、この区間のたわみの式が `MA` を含んだ形で求まります。その式に `x=l` を代入すれば、C点のたわみが得られます。

**解答**

C点におけるたわみ `vC` は以下の通りです。
$$v_C = \frac{1}{EI} \left( \frac{Pl^3}{12} - \frac{M_A l^2}{2} \right)$$

---

#### [cite_start]**(3) 固定モーメント MA の計算 [cite: 51]**

**考え方**

(2)の時点では、まだ壁の固定モーメント `MA` が未知数として残っています。これを決定するために、対称性から得られるもう一つの条件を使います。

1.  **中央での傾きがゼロ**: はりが左右対称であるため、中央点O (`x=2l`) での傾きはゼロになるはずです。
2.  **傾きの計算**: AC間 (`0 ≤ x ≤ l`) とCO間 (`l ≤ x ≤ 2l`) のそれぞれの区間で、モーメントの式を1回積分してたわみ角（傾き）の式を求めます。このとき、CO間では断面二次モーメントが `2I` になることに注意します。
3.  **連続条件**: C点 (`x=l`)で、はりは滑らかにつながっているので、AC間の傾きとCO間の傾きは等しくなります。この条件を使って積分定数を決めます。
4.  **MAの決定**: 最後に、CO間の傾きの式に `x=2l` を代入し、その値がゼロになるという条件から `MA` を求めます。

**解答**

点Aに作用する曲げモーメント `MA` は以下の通りです。
$$M_A = \frac{5}{12}Pl$$

### **【問題3】フランジ軸継手**

この問題は、2本の軸を「フランジ」と呼ばれる円盤とボルトでつなぎ、トルク（ねじりの力）を伝達する際の強度計算です。「軸本体がねじり切れないか」「ボルトがせん断破壊されないか」という2つの観点から、伝達できる最大のトルクを考えます。

#### [cite_start]**(1) 軸が伝達可能な最大トルク Ts [cite: 66]**

**考え方**

軸が伝えられる最大のトルクは、軸の材料の許容せん断応力 `τs` と、軸の断面形状（断面二次極モーメント `Jp`）によって決まります。

1.  **トルクと応力の関係式**: `T = τ * Zp` という関係があります。`Zp` は断面係数で、`Zp = Jp / r`（`r`は半径）です。
2.  **円形断面のZp**: 直径 `D` の中実円軸の場合、断面係数 `Zp` は `πD³/16` となります。
3.  **最大トルクの計算**: 許容せん断応力 `τs` を使って、軸が耐えられる最大トルク `Ts` を計算します。

**解答**

この軸が伝達可能な最大トルク `Ts` は以下の通りです。
$$T_s = \frac{\pi D^3}{16} \tau_s$$

---

#### [cite_start]**(2) ボルトが伝達可能な最大トルク Tf [cite: 67]**

**考え方**

トルクは、4本のボルトに生じる「せん断力」によって伝えられます。

1.  **1本のボルトが支える力**: 直径 `d` のボルト1本が耐えられる最大のせん断力 `F` は、「ボルトの断面積 × 許容せん断応力 `τf`」で計算できます。`F = (πd²/4) * τf`。
2.  **モーメントへの変換**: このせん断力 `F` が、回転中心から `R` の距離で作用します。したがって、ボルト1本が伝えるトルクは `F * R` です。
3.  **合計トルク**: ボルトは4本あるので、全体のトルク `Tf` はこれを4倍します。

**解答**

フランジが伝達可能な最大トルク `Tf` は以下の通りです。
$$T_f = 4 \times \left( \frac{\pi d^2}{4} \tau_f \right) \times R = \pi d^2 R \tau_f$$

---

#### [cite_start]**(3) Ts = Tf のときのボルト直径 d [cite: 68]**

**考え方**

「軸が耐えられるトルク」と「ボルトが耐えられるトルク」が等しくなる、という条件で式を立て、ボルトの直径 `d` について解きます。これは、継手全体としてバランスの取れた設計をするための条件設定です。

1.  **(1)と(2)の式を等しいとおく**: `Ts = Tf` なので、`πD³τs / 16 = πd²Rτf` となります。
2.  **dについて解く**: この式を整理して `d` を求めます。

**解答**

`Ts = Tf` のとき、`d` は以下の式で表せます。
$$d = \sqrt{\frac{D^3 \tau_s}{16 R \tau_f}} = \frac{D}{4} \sqrt{\frac{D \tau_s}{R \tau_f}}$$

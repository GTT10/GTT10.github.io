### **【問題1】段付き丸棒の引張と熱応力**

この問題は、太さが途中で変わる「段付き棒」について、(1)単純に引っ張った場合と、(2)両端を固定して温めた場合を考えます。

\<img src="[https://storage.googleapis.com/gemini-prod/images/42e8bb33-4f9e-4e36-a320-f4728d82d921](https://www.google.com/search?q=https://storage.googleapis.com/gemini-prod/images/42e8bb33-4f9e-4e36-a320-f4728d82d921)" alt="Diagrams for Problem 1" width="600"/\>

#### **(1) 引張荷重が作用するときの応力と伸び**

**考え方**
これは、静力学的に解ける基本的な問題です。

  * **応力**: 棒のどの部分にも、同じ引張荷重 `P` がかかります。応力は「力 ÷ 断面積」なので、断面積が異なる各部分で応力を計算します。
  * **伸び**: 棒全体の伸びは、「細い部分の伸び」と「太い部分の伸び」の足し算になります。棒は左右対称なので、「細い部分の伸び × 2」と「太い部分の伸び」を合計します。

**解答**

  * **応力**:
      * 細い部分（直径 `d₁`）の応力 `σ₁`:
        $$\sigma_1 = \frac{P}{A_1} = \frac{4P}{\pi d_1^2}$$
      * 太い部分（直径 `d₂`）の応力 `σ₂`:
        $$\sigma_2 = \frac{P}{A_2} = \frac{4P}{\pi d_2^2}$$
  * **棒全体の伸び `λ`**:
    $$\lambda = 2\lambda_1 + \lambda_2 = \frac{4P}{\pi} \left( \frac{2l_1}{E_1 d_1^2} + \frac{l_2}{E_2 d_2^2} \right)$$

-----

#### **(2) 両端を固定して温度上昇させたときの応力**

**考え方**
これは、力のつり合いだけでは解けない「不静定問題」です。
**キーポイント**: 棒は両端を固定されているため、**全体の長さは変われません**。

1.  **もし自由だったら…**: 壁がなければ、棒全体は温度上昇によって伸びるはずです（**熱膨張**）。この伸びる量は、各部分の熱膨張の合計です。
2.  **壁による拘束**: 実際には壁がこの伸びを妨げるため、棒の内部には「圧縮する力 `Pc`」が発生します。
3.  **力による縮み**: この圧縮力 `Pc` によって、棒全体は縮みます（**弾性変形**）。この縮む量は、(1)で計算した伸びの式と同じ形で計算できます。
4.  **つり合い**: 「熱で伸びようとする量」と「力で縮む量」がちょうど等しくなります。この等式を立てることで、未知の圧縮力 `Pc` を求めることができます。
5.  **応力の計算**: 求まった力 `Pc` をそれぞれの断面積で割ることで、圧縮応力が求まります。

**解答**
温度が1K上昇したときに生じる圧縮応力 `σ₁`, `σ₂` は以下の通りです。

  * **細い部分の応力 `σ₁`**:
    $$\sigma_1 = \frac{2\alpha_1 l_1 + \alpha_2 l_2}{\frac{2l_1}{E_1} + \frac{l_2 d_1^2}{E_2 d_2^2}}$$
  * **太い部分の応力 `σ₂`**:
    $$\sigma_2 = \sigma_1 \frac{d_1^2}{d_2^2} = \frac{d_1^2(2\alpha_1 l_1 + \alpha_2 l_2)}{d_2^2 \left(\frac{2l_1}{E_1} + \frac{l_2 d_1^2}{E_2 d_2^2}\right)}$$

### **【問題2】ばねで支えられた片持ちはり**

この問題は、壁に固定された「片持ちはり」の先端を、さらにばねで支えるという設定です。これも、力のつり合いだけでは解けない「不静定問題」です。

**キーポイント**: はりの先端がたわむ量と、ばねが縮む（または伸びる）量が密接に関係している点に着目します。

\<img src="[https://storage.googleapis.com/gemini-prod/images/42e8bb33-4f9e-4e36-a320-f4728d82d921](https://www.google.com/search?q=https://storage.googleapis.com/gemini-prod/images/42e8bb33-4f9e-4e36-a320-f4728d82d921)" alt="Diagram for Problem 2" width="500"/\>

#### **(1) 曲げモーメント M(x)**

**考え方**
はりの左端Aから `x` の距離にある断面で、モーメントのつり合いを考えます。この断面より左側には、壁からの反力 `RA`、壁からの固定モーメント `MA`、そして長さ `x` にわたる分布荷重 `w` が作用しています。これらの力がつくるモーメントをすべて足し合わせます。

**解答**
曲げモーメント `M(x)` は、未知数 `RA`, `MA` を用いて以下のように表せます。
$$M(x) = R_A x - M_A - \frac{wx^2}{2}$$

-----

#### **(2) たわみの基礎式と境界条件**

**考え方**

  * **基礎式**: はりのたわみ `y` と曲げモーメント `M` の関係は、材料力学における最も基本的な微分方程式で表されます。
  * **境界条件**: はりの支えられ方から、たわみに関する条件を数式で表現します。
      * **A点 (`x=0`)**: 壁に完全に固定されているため、たわみ（位置）も傾きもゼロです。
      * **B点 (`x=l`)**: ばねで支えられています。ばねの力は「ばね定数 `k` × 縮み量」です。この力が、はりのB点における反力 `RB` となります。そして、はりのB点のたわみ `y(l)` が、そのままばねの縮み量になります。

**解答**

  * **たわみの基礎式**:
    $$EI \frac{d^2y}{dx^2} = M(x) = R_A x - M_A - \frac{wx^2}{2}$$
  * **A点の境界条件 (`x=0`)**:
    $$y(0) = 0, \quad \frac{dy}{dx}(0) = 0$$
  * **B点の境界条件 (`x=l`)**:
    はりのB点でのたわみ `y(l)` と、ばねによる反力 `RB` の間には、`RB = k \cdot y(l)` の関係が成り立ちます。

-----

#### **(3) 反力、モーメント、およびSFD, BMD**

**考え方**
(2)で立てた微分方程式と境界条件を解くことで、未知数である `RA`, `MA` が求まります。`RB` は `RA` から力のつり合いで計算できます。これらの値が確定すれば、せん断力と曲げモーメントの分布が計算でき、図にすることができます。

**解答**

  * **反力とモーメント**:
    $$R_B = \frac{3k w l^4}{8(3EI + kl^3)}$$   $$R_A = wl - R_B = wl - \frac{3k w l^4}{8(3EI + kl^3)}$$   $$M_A = R_B l - \frac{wl^2}{2} = \frac{3k w l^5}{8(3EI + kl^3)} - \frac{wl^2}{2}$$
  * **SFD（せん断力図）とBMD（曲げモーメント図）**:
      * **SFD**: A点で `RA` から始まり、B点に向かって直線的に減少します。
      * **BMD**: A点で `-MA` から始まり、下に凸の放物線を描きます。B点ではモーメントがゼロになります。

### **【問題3】平面応力状態**

この問題は、一枚の薄い板のように、厚み方向の力を無視できる「平面応力状態」における、応力とひずみの基本的な関係を問うています。

\<img src="[https://storage.googleapis.com/gemini-prod/images/a639668d-8c10-4228-98e9-d368e5d070b4](https://www.google.com/search?q=https://storage.googleapis.com/gemini-prod/images/a639668d-8c10-4228-98e9-d368e5d070b4)" alt="Diagram for Problem 3" width="400"/\>

#### **(1) 一般化フックの法則**

**考え方**
ある方向に物体を引っ張ると（垂直応力）、その方向には伸び（垂直ひずみ）ますが、同時にそれと直角な方向には縮みます（ポアソン効果）。また、せん断応力をかけると、物体はひし形に変形します（せん断ひずみ）。これらの関係をまとめたものが「一般化フックの法則」です。

**解答**
平面応力状態における応力とひずみの関係式は以下の通りです。
$$\epsilon_x = \frac{1}{E}(\sigma_x - \nu \sigma_y)$$$$\epsilon_y = \frac{1}{E}(\sigma_y - \nu \sigma_x)$$$$\gamma_{xy} = \frac{1}{G}\tau_{xy}$$

-----

#### **(2) 厚み方向のひずみ εz**

**考え方**
平面応力状態では、厚み方向（z方向）の応力 `σz` はゼロですが、x方向とy方向の応力によるポアソン効果で、厚み方向にもひずみ `εz` が生じます。

**解答**
$$\epsilon_z = \frac{1}{E}(\sigma_z - \nu(\sigma_x + \sigma_y)) = -\frac{\nu}{E}(\sigma_x + \sigma_y)$$

-----

#### **(3) x軸に垂直な面の応力ベクトル**

**考え方**
「応力ベクトル」とは、ある面に作用する単位面積あたりの力をベクトルで表したものです。x軸に垂直な面には、その面に垂直な応力 `σx` と、面に沿った（y方向の）せん断応力 `τxy` が作用しています。

**解答**

  * **成分**: `p₁x = σx`, `p₁y = τxy`
  * **大きさ**: `|p₁| = sqrt(σx² + τxy²)`

-----

#### **(4) y軸に垂直な面の応力ベクトル**

**考え方**
(3)と考え方は同じです。y軸に垂直な面には、垂直な応力 `σy` と、面に沿った（x方向の）せん断応力 `τyx` (ただし `τyx = τxy`) が作用しています。

**解答**

  * **成分**: `p₂x = τxy`, `p₂y = σy`
  * **大きさ**: `|p₂| = sqrt(τxy² + σy²)`

-----

#### **(5) 任意の面の応力ベクトル**

**考え方**
これは、任意の傾きを持つ面に作用する応力ベクトルを、元の座標系の応力成分で表す「コーシーの応力公式」です。

**解答**
法線の方向余弦が `(l, m)` である面に作用する応力ベクトル `pn` は以下の通りです。

  * **成分**:
    $$p_{nx} = \sigma_x l + \tau_{xy} m$$   $$p_{ny} = \tau_{xy} l + \sigma_y m$$
  * **大きさ**:
    $$|p_n| = \sqrt{p_{nx}^2 + p_{ny}^2}$$

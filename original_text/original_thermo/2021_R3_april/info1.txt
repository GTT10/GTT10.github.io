### 【1】問題文
質量 \(m\) の理想気体が、状態1から状態2に断熱圧縮され、等圧変化で外部から熱量 \(Q_1\) を受け取り、状態3になる。状態3から断熱膨張して状態4になり、定容変化で熱量 \(Q_2\) を放出して状態1に戻るとき、以下の問いに答えよ。ただし、状態 \(i\) における状態量にはそれぞれ添字 \(i\) を付ける。また、気体の圧力を \(p\)、容積を \(V\)、温度を \(T\)、エントロピーを \(S\)、比熱比を \(k\)、定圧比熱を \(c_p\)、定容比熱を \(c_v\)、圧縮比を \(e = V_1 / V_2\)、噴射締切比を \(\rho = V_3 / V_2\) とする。

#### (1) 問題文
このサイクルの \(p-V\)（圧力-容積）線図および \(T-S\)（温度-エントロピー）線図を描け。また、図中に状態1, 2, 3, 4を明記するとともに熱の出入りを示せ。

**方針・考え方**  
- サイクルの過程を整理する：  
  - 1→2：断熱圧縮（可逆断熱過程、エントロピー一定）  
  - 2→3：等圧変化（圧力一定、熱吸収）  
  - 3→4：断熱膨張（可逆断熱過程、エントロピー一定）  
  - 4→1：定容変化（容積一定、熱放出）  
- \(p-V\) 線図：圧力 \(p\) と容積 \(V\) の関係をプロット。断熱過程は \(pV^k = \text{一定}\) の曲線、等圧過程は水平線、定容過程は垂直線。  
- \(T-S\) 線図：温度 \(T\) とエントロピー \(S\) の関係をプロット。断熱過程は垂直線（\(S\) 一定）、等圧・定容過程は曲線（エントロピー変化あり）。  
- 熱の出入り：吸熱過程（2→3）と放熱過程（4→1）を矢印などで明示。  
- 状態点：与えられたパラメータ（\(e = V_1 / V_2\), \(\rho = V_3 / V_2\)）から容積を決定。\(V_4 = V_1\)（定容過程4→1のため）。  
- 図はテキストで描写できないため、詳細な説明と特徴を記述する。

**解答**  
**\(p-V\) 線図**  
- **過程の特徴**:  
  - 1→2（断熱圧縮）：\(V\) 減少、\(p\) 増加（曲線、\(pV^k = \text{一定}\))。  
  - 2→3（等圧変化）：\(p\) 一定、\(V\) 増加（水平線）。熱吸収 \(Q_1 > 0\)。  
  - 3→4（断熱膨張）：\(V\) 増加、\(p\) 減少（曲線、\(pV^k = \text{一定}\))。  
  - 4→1（定容変化）：\(V\) 一定、\(p\) 減少（垂直線）。熱放出 \(Q_2 < 0\)（図では放出を明示）。  
- **状態点の容積**:  
  - \(V_1\)：初期容積（与えられた値）。  
  - \(V_2 = V_1 / e\)（圧縮比 \(e = V_1 / V_2\) より）。  
  - \(V_3 = \rho V_2 = \rho (V_1 / e)\)（噴射締切比 \(\rho = V_3 / V_2\) より）。  
  - \(V_4 = V_1\)（定容過程4→1のため容積変化なし）。  
- **図の概形**:  
  - 状態1 (\(V_1, p_1\)) から開始。  
  - 1→2：右上がりの曲線（\(V\) 減少、\(p\) 増加）。  
  - 2→3：右方向の水平線（\(V_2\) から \(V_3\) へ）。  
  - 3→4：右下がりの曲線（\(V_3\) から \(V_4 = V_1\) へ）。  
  - 4→1：下方向の垂直線（\(V_1\) で \(p\) 減少）。  
- **熱の出入り表示**: 2→3 に「吸熱 \(Q_1\)」、4→1 に「放熱 \(Q_2\)」と記入。

**\(T-S\) 線図**  
- **過程の特徴**:  
  - 1→2（断熱圧縮）：\(S\) 一定、\(T\) 増加（垂直線）。  
  - 2→3（等圧変化）：\(S\) 増加、\(T\) 増加（曲線、理想気体の等圧線）。  
  - 3→4（断熱膨張）：\(S\) 一定、\(T\) 減少（垂直線）。  
  - 4→1（定容変化）：\(S\) 減少、\(T\) 減少（曲線、理想気体の定容線）。  
- **状態点のエントロピー**:  
  - \(S_1 = S_2\)（断熱過程1→2）。  
  - \(S_3 = S_4\)（断熱過程3→4）。  
  - \(S_3 > S_2\)（等圧過程2→3で吸熱）。  
  - \(S_1 = S_4\)（サイクル閉じるため）。  
- **図の概形**:  
  - 状態1 (\(S_1, T_1\)) から開始。  
  - 1→2：上方向の垂直線（\(S\) 一定、\(T\) 増加）。  
  - 2→3：右上がりの曲線（\(S\) 増加、\(T\) 増加）。  
  - 3→4：下方向の垂直線（\(S\) 一定、\(T\) 減少）。  
  - 4→1：左下がりの曲線（\(S\) 減少、\(T\) 減少）。  
- **熱の出入り表示**: 2→3 に「吸熱 \(Q_1\)」、4→1 に「放熱 \(Q_2\)」と記入。

**補足**: 実際の線図は曲線を滑らかに描き、状態点をラベル付けする。最高温度は状態3（後述の(3)参照）で、\(T-S\) 線図では状態3が最高点となる。

#### (2) 問題文
受け取る熱量 \(Q_1\)、放出する熱量 \(Q_2\) を求めよ。

**方針・考え方**  
- \(Q_1\) は等圧過程（2→3）での吸熱量：\(Q_1 = m c_p (T_3 - T_2)\)。  
- \(Q_2\) は定容過程（4→1）での放熱量（放出量は正の値として扱う）：\(Q_2 = m c_v (T_4 - T_1)\)（\(T_4 > T_1\) のため正）。  
- 未知の温度 \(T_2, T_3, T_4\) を \(T_1\) とパラメータ（\(e, \rho, k\)）で表す。  
  - 断熱過程の関係式 \(T V^{k-1} = \text{一定}\) を使用。  
  - 等圧過程2→3では圧力一定より \(V / T = \text{一定}\)。  
- 導出手順：  
  1. 状態1→2（断熱圧縮）：\(T_1 V_1^{k-1} = T_2 V_2^{k-1}\) から \(T_2\) を求める。  
  2. 状態2→3（等圧変化）：\(V_2 / T_2 = V_3 / T_3\) から \(T_3\) を求める。  
  3. 状態3→4（断熱膨張）：\(T_3 V_3^{k-1} = T_4 V_4^{k-1}\) から \(T_4\) を求める（\(V_4 = V_1\)）。  
  4. \(Q_1, Q_2\) に代入。

**途中式**  
- \(T_2\) の計算（1→2断熱圧縮）:  
  \[
  T_1 V_1^{k-1} = T_2 V_2^{k-1}
  \]  
  \(V_2 = V_1 / e\) より、  
  \[
  T_1 V_1^{k-1} = T_2 \left( \frac{V_1}{e} \right)^{k-1} \implies T_2 = T_1 \left( \frac{V_1}{V_2} \right)^{k-1} = T_1 e^{k-1}
  \]

- \(T_3\) の計算（2→3等圧変化）:  
  圧力一定より \(\frac{V_2}{T_2} = \frac{V_3}{T_3}\)。  
  \(V_3 = \rho V_2\) より、  
  \[
  T_3 = T_2 \frac{V_3}{V_2} = T_2 \rho = T_1 e^{k-1} \rho
  \]

- \(T_4\) の計算（3→4断熱膨張）:  
  \[
  T_3 V_3^{k-1} = T_4 V_4^{k-1}
  \]  
  \(V_4 = V_1\), \(V_3 = \rho V_2 = \rho (V_1 / e)\) より、  
  \[
  T_3 \left( \rho \frac{V_1}{e} \right)^{k-1} = T_4 V_1^{k-1} \implies T_4 = T_3 \left( \frac{\rho}{e} \right)^{k-1}
  \]  
  \(T_3 = T_1 e^{k-1} \rho\) を代入、  
  \[
  T_4 = (T_1 e^{k-1} \rho) \left( \frac{\rho}{e} \right)^{k-1} = T_1 e^{k-1} \rho \cdot \rho^{k-1} e^{-(k-1)} = T_1 \rho^k e^{0} = T_1 \rho^k \quad (\text{since } e^{k-1} \cdot e^{-(k-1)} = 1, \ \rho \cdot \rho^{k-1} = \rho^k)
  \]

- \(Q_1\) の計算:  
  \[
  Q_1 = m c_p (T_3 - T_2) = m c_p (T_1 e^{k-1} \rho - T_1 e^{k-1}) = m c_p T_1 e^{k-1} (\rho - 1)
  \]

- \(Q_2\) の計算（放出量のため正の値）:  
  \[
  Q_2 = m c_v (T_4 - T_1) = m c_v (T_1 \rho^k - T_1) = m c_v T_1 (\rho^k - 1)
  \]

**解答**  
\[
Q_1 = m c_p T_1 e^{k-1} (\rho - 1)
\]  
\[
Q_2 = m c_v T_1 (\rho^k - 1)
\]

#### (3) 問題文
状態1の温度 \(T_1\) を用いて、このサイクルの最高温度を求めよ。

**方針・考え方**  
- 各状態の温度：  
  - \(T_1\)：与えられた初期温度。  
  - \(T_2 = T_1 e^{k-1}\)  
  - \(T_3 = T_1 e^{k-1} \rho\)  
  - \(T_4 = T_1 \rho^k\)  
- 最高温度は \(T_3\) または \(T_4\) のいずれか。一般に \(e > 1\), \(\rho > 1\), \(k > 1\) の条件下で比較：  
  - \(T_3 = T_1 e^{k-1} \rho\)  
  - \(T_4 = T_1 \rho^k\)  
- \(e^{k-1} \rho\) と \(\rho^k\) の大小：ディーゼルサイクルでは通常 \(T_3 > T_4\)（状態3が最高温度）。  
  - 例：\(e = 10\), \(\rho = 2\), \(k = 1.4\) とすると、\(e^{k-1} = 10^{0.4} \approx 2.51\), \(T_3 / T_1 = 2.51 \times 2 = 5.02\), \(T_4 / T_1 = 2^{1.4} \approx 2.64\) → \(T_3 > T_4\)。  
- したがって、最高温度は \(T_3\)。

**解答**  
最高温度は状態3の温度 \(T_3\) であり、  
\[
T_3 = T_1 e^{k-1} \rho
\]

#### (4) 問題文
このサイクルの熱効率を \(\kappa\)、\(\varepsilon\) および \(\rho\) を用いて求めよ。

**方針・考え方**  
- 熱効率 \(\eta\) の定義：\(\eta = 1 - \frac{\text{放出熱量}}{\text{吸収熱量}} = 1 - \frac{Q_2}{Q_1}\)（\(Q_1, Q_2\) は正の値）。  
- (2) の結果を使用：  
  \[
  Q_1 = m c_p T_1 e^{k-1} (\rho - 1), \quad Q_2 = m c_v T_1 (\rho^k - 1)
  \]  
- 比熱比 \(k = \frac{c_p}{c_v}\) より、\(\frac{c_v}{c_p} = \frac{1}{k}\)。  
- 問題文では熱効率を \(\kappa\)（比熱比）、\(\varepsilon\)（圧縮比）、\(\rho\)（噴射締切比）で表すように指示。定義より \(\kappa = k\), \(\varepsilon = e\) と解釈（問題文の \(\kappa\) は比熱比 \(k\) を、\(\varepsilon\) は圧縮比 \(e\) を指す）。  
- \(\eta\) を計算し、\(\kappa, \varepsilon, \rho\) で表す。

**途中式**  
\[
\eta = 1 - \frac{Q_2}{Q_1} = 1 - \frac{m c_v T_1 (\rho^\kappa - 1)}{m c_p T_1 \varepsilon^{\kappa-1} (\rho - 1)} = 1 - \frac{c_v}{c_p} \cdot \frac{\rho^\kappa - 1}{\varepsilon^{\kappa-1} (\rho - 1)}
\]  
\(\frac{c_v}{c_p} = \frac{1}{\kappa}\) を代入、  
\[
\eta = 1 - \frac{1}{\kappa} \cdot \frac{\rho^\kappa - 1}{\varepsilon^{\kappa-1} (\rho - 1)}
\]

**解答**  
熱効率 \(\eta\) は、  
\[
\eta = 1 - \frac{1}{\kappa} \cdot \frac{\rho^\kappa - 1}{\varepsilon^{\kappa-1} (\rho - 1)}
\]

#### (5) 問題文
(4)を考慮して、実際の4ストローク機関において、熱効率を向上させるにはどのようにすれば良いか、2点挙げよ。

**方針・考え方**  
- (4) の熱効率 \(\eta = 1 - \frac{1}{\kappa} \cdot \frac{\rho^\kappa - 1}{\varepsilon^{\kappa-1} (\rho - 1)}\) を分析。  
- \(\eta\) を増加させる要因：  
  - \(\varepsilon\)（圧縮比）を大きくする → \(\varepsilon^{\kappa-1}\) が増加し、\(\frac{1}{\varepsilon^{\kappa-1}}\) が減少 → \(\eta\) 増加。  
  - \(\rho\)（噴射締切比）を小さくする → \(\rho^\kappa - 1\) が減少（\(\rho > 1\) で \(\rho^\kappa - 1 > 0\)）、分母 \(\rho - 1\) も減少するが、一般に \(\frac{\rho^\kappa - 1}{\rho - 1}\) は \(\rho\) の増加関数（例：\(\kappa > 1\) で \(\rho\) 増加とともに増加）。したがって \(\rho\) を小さくすると \(\frac{\rho^\kappa - 1}{\rho - 1}\) が減少 → \(\eta\) 増加。  
- 実際の4ストローク機関（ディーゼルエンジンなど）での具体策を挙げる。

**解答**  
実際の4ストローク機関で熱効率を向上させるには、以下の2点が有効である。  
1. **圧縮比 \(\varepsilon\) を大きくする**：燃焼室の設計を変更し、混合気の圧縮を高めることで熱効率が向上する。ただし、ノッキング（異常燃焼）のリスクがあるため材料強度や燃料選択に注意。  
2. **噴射締切比 \(\rho\) を小さくする**：燃料噴射時期を調整し、膨張行程の開始点（状態3）での容積増加を抑える。これにより高温ガスの有効利用が増し熱効率が向上する。

### 【2】問題文
質量 \(m\)、比熱 \(c\) が同じで温度が異なる 2 つの熱源がある。図 1 に示すように、高温熱源（温度 \(T_1\)）に低温熱源（温度 \(T_2\)）を接触させ、熱移動を行う。接触後、熱平衡になったとき、以下の問いに答えよ。  
（図1: 高温熱源 \(m, c, T_1\) → 熱量移動 ← 低温熱源 \(m, c, T_2\)）

#### (1) 問題文
接触後の熱平衡状態の温度を求めよ。

**方針・考え方**  
- 2つの熱源は断熱系で熱交換。熱平衡温度 \(T_f\) では、高温熱源の失った熱量と低温熱源の得た熱量が等しい。  
- 熱容量は両方とも \(m c\)（質量 \(m\)、比熱 \(c\)）。  
- 熱量保存則：高温熱源の放出熱量 = 低温熱源の吸収熱量。

**途中式**  
高温熱源の温度変化：\(T_1 \to T_f\)、放出熱量 \(= m c (T_1 - T_f)\)。  
低温熱源の温度変化：\(T_2 \to T_f\)、吸収熱量 \(= m c (T_f - T_2)\)。  
熱量保存：  
\[
m c (T_1 - T_f) = m c (T_f - T_2)
\]  
\(m c \ne 0\) で割り、  
\[
T_1 - T_f = T_f - T_2 \implies 2 T_f = T_1 + T_2
\]  
\[
T_f = \frac{T_1 + T_2}{2}
\]

**解答**  
熱平衡状態の温度 \(T_f\) は、  
\[
T_f = \frac{T_1 + T_2}{2}
\]

---

#### (2) 問題文
高温熱源および低温熱源のエントロピー変化を求めよ。

**方針・考え方**  
- エントロピー変化 \(\Delta S\) は状態量。実際の過程は不可逆だが、可逆経路で計算。  
- 高温熱源：温度 \(T_1\) から \(T_f\) に冷却。可逆過程として、微小熱量 \(dQ_{\text{rev}} = -m c  dT\)（熱を失うため負）。  
  \[
  \Delta S_{\text{high}} = \int \frac{dQ_{\text{rev}}}{T} = \int_{T_1}^{T_f} \frac{-m c  dT}{T}
  \]  
- 低温熱源：温度 \(T_2\) から \(T_f\) に加熱。可逆過程として、微小熱量 \(dQ_{\text{rev}} = m c  dT\)（熱を得るため正）。  
  \[
  \Delta S_{\text{low}} = \int \frac{dQ_{\text{rev}}}{T} = \int_{T_2}^{T_f} \frac{m c  dT}{T}
  \]

**途中式**  
- 高温熱源のエントロピー変化：  
  \[
  \Delta S_{\text{high}} = -m c \int_{T_1}^{T_f} \frac{dT}{T} = -m c \left[ \ln T \right]_{T_1}^{T_f} = -m c (\ln T_f - \ln T_1) = -m c \ln \left( \frac{T_f}{T_1} \right)
  \]  
- 低温熱源のエントロピー変化：  
  \[
  \Delta S_{\text{low}} = m c \int_{T_2}^{T_f} \frac{dT}{T} = m c \left[ \ln T \right]_{T_2}^{T_f} = m c (\ln T_f - \ln T_2) = m c \ln \left( \frac{T_f}{T_2} \right)
  \]

**解答**  
高温熱源のエントロピー変化：  
\[
\Delta S_{\text{high}} = -m c \ln \left( \frac{T_f}{T_1} \right)
\]  
低温熱源のエントロピー変化：  
\[
\Delta S_{\text{low}} = m c \ln \left( \frac{T_f}{T_2} \right)
\]

---

#### (3) 問題文
系全体のエントロピー変化を求めよ。

**方針・考え方**  
- 系全体 = 高温熱源 + 低温熱源。外部との熱交換なし（断熱系）。  
- エントロピー変化は加算：\(\Delta S_{\text{total}} = \Delta S_{\text{high}} + \Delta S_{\text{low}}\)。  
- (2) の結果を使用し、\(T_f = \frac{T_1 + T_2}{2}\) を代入して整理。

**途中式**  
\[
\Delta S_{\text{total}} = \Delta S_{\text{high}} + \Delta S_{\text{low}} = -m c \ln \left( \frac{T_f}{T_1} \right) + m c \ln \left( \frac{T_f}{T_2} \right) = m c \left[ \ln \left( \frac{T_f}{T_2} \right) - \ln \left( \frac{T_f}{T_1} \right) \right] = m c \ln \left( \frac{T_f / T_2}{T_f / T_1} \right) = m c \ln \left( \frac{T_1}{T_2} \right)
\]  
（\(T_f\) がキャンセルされ、\(T_1\) と \(T_2\) のみに依存。）

**解答**  
系全体のエントロピー変化 \(\Delta S_{\text{total}}\) は、  
\[
\Delta S_{\text{total}} = m c \ln \left( \frac{T_1}{T_2} \right)
\]

---

#### (4) 問題文
この変化は可逆か、不可逆かを説明せよ。

**方針・考え方**  
- エントロピー増大則：断熱系の不可逆過程ではエントロピー増加、可逆過程では不変。  
- (3) の結果：\(\Delta S_{\text{total}} = m c \ln \left( \frac{T_1}{T_2} \right)\)。  
- \(T_1 > T_2\) より \(\ln \left( \frac{T_1}{T_2} \right) > 0\)、したがって \(\Delta S_{\text{total}} > 0\)。  
- 断熱系でエントロピー増加 → 不可逆過程。

**解答**  
この変化は不可逆である。理由は、系全体が断熱でありながらエントロピー変化 \(\Delta S_{\text{total}} > 0\)（正の増加）となるためである。具体的には、温度差がある熱源間の熱移動は自然に進行する不可逆過程であり、エントロピーが増大する。

### 解答：工業熱力学 2014年度 修士課程入試問題

#### 【1】シリンダ内の理想気体

**(1) 状態1の温度 \( T_1 \) と圧力 \( P_1 \)**  
- 過程0→1：定容加熱（体積 \( V_0 \) 一定）  
- 加熱量： \( Q = m c_v (T_1 - T_0) \)  
\[
T_1 = T_0 + \frac{Q}{m c_v}
\]  
- 状態方程式：  
\[
P_1 = \frac{m R T_1}{V_0} = P_0 \frac{T_1}{T_0} \quad \left( \because R = c_v (\kappa - 1) \right)
\]  
\[
\boxed{T_1 = T_0 + \frac{Q}{m c_v}, \quad P_1 = P_0 \left(1 + \frac{Q}{m c_v T_0}\right)}
\]

**(2) 状態0→1のエントロピー変化 \( \Delta S_{01} \)**  
定容過程のエントロピー変化：  
\[
\Delta S_{01} = m c_v \ln \frac{T_1}{T_0}
\]  
\[
\boxed{\Delta S_{01} = m c_v \ln \left(1 + \frac{Q}{m c_v T_0}\right)}
\]

**(3) 状態2の温度 \( T_2 \) と体積 \( V_2 \)**  
- 過程1→2：断熱膨張（系全体が断熱）  
- エネルギー保存則：  
\[
\Delta U = -W \implies m c_v (T_2 - T_1) = -P_0 (V_2 - V_0)
\]  
- 状態方程式（状態2）：  
\[
P_0 V_2 = m R T_2
\]  
- 状態方程式（状態0）：  
\[
P_0 V_0 = m R T_0
\]  
- 連立して解く：  
\[
m c_v (T_2 - T_1) = -P_0 \left( \frac{m R T_2}{P_0} - V_0 \right)
\]  
\[
c_v (T_2 - T_1) = -R T_2 + R T_0 \quad \left( \because V_0 = \frac{m R T_0}{P_0} \right)
\]  
\[
c_v T_2 - c_v T_1 = -R T_2 + R T_0
\]  
\[
T_2 (c_v + R) = c_v T_1 + R T_0
\]  
\[
T_2 = \frac{c_v T_1 + R T_0}{c_p} \quad (\because c_p = c_v + R)
\]  
\[
\boxed{T_2 = \frac{c_v}{c_p} T_1 + \frac{R}{c_p} T_0}, \quad \boxed{V_2 = \frac{m R T_2}{P_0}
\]

**(4) 状態1→2のエントロピー変化 \( \Delta S_{12} \)**  
理想気体のエントロピー変化（一般式）：  
\[
\Delta S_{12} = m c_p \ln \frac{T_2}{T_1} - m R \ln \frac{P_2}{P_1}
\]  
\( P_2 = P_0 \), \( P_1 = P_0 \frac{T_1}{T_0} \) より：  
\[
\Delta S_{12} = m c_p \ln \frac{T_2}{T_1} - m R \ln \frac{T_0}{T_1}
\]  
\[
\boxed{\Delta S_{12} = m \left( c_p \ln \frac{T_2}{T_1} + R \ln \frac{T_1}{T_0} \right)}
\]

**(5) ピストンが外部にした仕事 \( W \)**  
\[
W = P_0 (V_2 - V_0)
\]  
\[
\boxed{W = P_0 \left( \frac{m R T_2}{P_0} - \frac{m R T_0}{P_0} \right) = m R (T_2 - T_0)}
\]

**(6) P-V線図とT-S線図**  
**P-V線図**:  
```
圧力 P ↑
|  P₀: ・状態0────・状態2
|      |           /
|      |         /
| P₁: ・状態1   /
+----------------→ 容積 V
```
- **過程**:  
  - 0→1: 定容加熱（垂直線）  
  - 1→2: 断熱膨張（曲線）  

**T-S線図**:  
```
温度 T ↑
|  T₁: ・状態1
|      | 
|      | 
|  T₂: ・状態2
|      |
|  T₀: ・状態0
+----------------→ エントロピー S
```
- **過程**:  
  - 0→1: エントロピー増加（右向き）  
  - 1→2: 断熱変化（エントロピー一定？ 不可逆のため増加）

#### 【2】再生サイクル（冷凍サイクル）

**(1) T-S線図**  
```
温度 T ↑
|  T_H: ・状態1────・状態4
|        |           |
|        |           |
|  T_L: ・状態2────・状態3
+---------------------→ エントロピー S
```
- **熱の流れ**:  
  - 吸熱: 3→4（等温膨張）  
  - 放熱: 1→2（等温圧縮）  
  - 再生熱: 2→3の放熱 = 4→1の吸熱  

**(2) 2→3の熱量 \( Q_{23} \) と仕事 \( W_{23} \)**  
- 定容過程 → 仕事 \( W_{23} = 0 \)  
- 熱量:  
\[
Q_{23} = m c_v (T_3 - T_2) \quad (\text{放熱})
\]  
\[
\boxed{Q_{23} = m c_v (T_3 - T_2), \quad W_{23} = 0}
\]

**(3) 3→4の熱量 \( Q_{34} \) と仕事 \( W_{34} \)**  
- 等温膨張 → 内部エネルギー変化 \( \Delta U = 0 \)  
- 仕事:  
\[
W_{34} = \int_{V_3}^{V_4} P  dV = m R T_L \ln \frac{V_4}{V_3}
\]  
- 熱力学第一法則:  
\[
Q_{34} = W_{34} \quad (\text{吸熱})
\]  
\[
\boxed{Q_{34} = m R T_L \ln \frac{V_4}{V_3}, \quad W_{34} = m R T_L \ln \frac{V_4}{V_3}}
\]

**(4) 成績係数（COP）**  
- 冷凍効果（吸熱）:  
\[
Q_{\text{冷凍}} = Q_{34} = m R T_L \ln \frac{V_4}{V_3}
\]  
- 正味仕事入力:  
\[
W_{\text{net}} = W_{12} + W_{34} \quad (\text{過程2→3,4→1は仕事なし})
\]  
- 1→2（等温圧縮）の仕事:  
\[
W_{12} = m R T_H \ln \frac{V_2}{V_1} = -m R T_H \ln \frac{V_1}{V_2}
\]  
- 再生条件より \( Q_{23} = Q_{41} \)（熱量相殺）  
\[
W_{\text{net}} = W_{12} + W_{34} = m R \left( T_L \ln \frac{V_4}{V_3} - T_H \ln \frac{V_1}{V_2} \right)
\]  
- 圧縮比 \( \varepsilon = V_1 / V_2 \)、膨張比 \( V_4 / V_3 = V_1 / V_2 = \varepsilon \)（サイクル構造より）  
\[
W_{\text{net}} = m R \ln \varepsilon  (T_L - T_H)
\]  
- 成績係数:  
\[
\text{COP} = \frac{Q_{\text{冷凍}}}{|W_{\text{net}}|} = \frac{T_L \ln \varepsilon}{|T_L - T_H| \ln \varepsilon} = \frac{T_L}{T_H - T_L}
\]  
\[
\boxed{\text{COP} = \frac{T_L}{T_H - T_L}}
\]

#### 【3】ノズル流れ

**(1) ノズル出口圧力 \( P_2 \)**  
問題文に条件不足のため、一般的な関係式を導出。  
- 断熱過程: \( P_1 v_1^\kappa = P_2 v_2^\kappa \)  
- エネルギー保存則:  
\[
h_1 + \frac{w_1^2}{2} = h_2 + \frac{w_2^2}{2}
\]  
- 連続の式:  
\[
\frac{w_1 A_1}{v_1} = \frac{w_2 A_2}{v_2}
\]  
※ 具体的な \( P_2 \) を求めるには追加条件が必要。

**(2) 臨界圧力 \( P_c \)**  
\[
\boxed{P_c = P_1 \left( \frac{2}{\kappa + 1} \right)^{\frac{\kappa}{\kappa - 1}}
\]

**(3) 臨界流れの出口速度 \( w_2 \)**  
- 入口速度 \( w_1 = 0 \)  
- エネルギー保存則:  
\[
h_1 = h_2 + \frac{w_2^2}{2}
\]  
- 断熱過程:  
\[
h_1 - h_2 = c_p (T_1 - T_2) = \frac{\kappa R T_1}{\kappa - 1} \left(1 - \frac{T_2}{T_1}\right)
\]  
- 臨界条件:  
\[
\frac{T_2}{T_1} = \frac{2}{\kappa + 1}
\]  
\[
w_2 = \sqrt{2 (h_1 - h_2)} = \sqrt{ \frac{2 \kappa R T_1}{\kappa - 1} \left(1 - \frac{2}{\kappa + 1}\right) } = \sqrt{ \frac{2 \kappa R T_1}{\kappa + 1} }
\]  
\[
\boxed{w_2 = \sqrt{ \frac{2 \kappa R T_1}{\kappa + 1} }}
\]

**(4) 臨界流れの速度 = 音速**  
- 音速 \( a = \sqrt{\kappa R T} \)（局所音速）  
- 臨界点での温度 \( T_2 = T_1 \cdot \frac{2}{\kappa + 1} \)  
- 臨界点での音速:  
\[
a_2 = \sqrt{\kappa R T_2} = \sqrt{ \kappa R T_1 \cdot \frac{2}{\kappa + 1} } = \sqrt{ \frac{2 \kappa R T_1}{\kappa + 1} }
\]  
\[
\boxed{w_2 = a_2} \quad \text{∴ 臨界流れの速度は音速に一致}
\]

**(5) ノズル形状の選択**  
- **(a) 空気（\(\kappa = 1.40\)）**:  
  \[
  \frac{P_2}{P_1} = \frac{0.10}{0.18} \approx 0.5556 > 0.5283 \, \text{(臨界圧力比)}
  \]  
  → **先細ノズル**（亜音速流れ）  
  **理由**: 圧力比が臨界値以上で超音速流れが発生しないため。  

- **(b) 過熱蒸気（\(\kappa = 1.30\)）**:  
  \[
  \frac{P_2}{P_1} = \frac{2.08}{4.00} = 0.52 < 0.5457 \, \text{(臨界圧力比)}
  \]  
  → **末広ノズル**（超音速流れ）  
  **理由**: 圧力比が臨界値未満で超音速流れが発生するため、拡大管が必要。

### まとめ
- 【1】断熱膨張ではエネルギー保存則を適用し、状態量変化を計算。
- 【2】再生サイクルの成績係数はカルノー効率と一致。
- 【3】ノズル形状は圧力比と臨界圧力比の比較で決定。
- 工業熱力学では、状態量の計算と過程の理解が重要！

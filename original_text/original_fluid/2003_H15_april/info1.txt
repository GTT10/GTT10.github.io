### 【１】 トラス構造の問題

**問題文**
図1のように、上端 A, B, C を剛体にピン止めした3本の同じ棒が同一鉛直面内において、下端を一つの節点Oにピン結合されている。棒の長さ `l`、断面積 `A`、縦弾性係数 `E` および線膨張係数 `α` として、次の問に答えよ。ただし、棒の変形は微小として自重を考えない。

(1) 節点Oに鉛直方向下向きに荷重 `P` が作用したとき、各棒に生じる応力および節点Oの変位を求めよ。ただし、温度変化はないとする。
(2) 温度 `t`℃ 上昇させたとき、各棒に生じる応力および節点Oの変位を求めよ。ただし、荷重は作用しないとする。

#### (1) 荷重Pが作用する場合

**方針・考え方**
この問題は、未知の力が3つ（各棒の軸力 `N_A`, `N_B`, `N_C`）あるのに対し、力のつり合い式だけでは解けない「不静定問題」です。このような問題は、以下の2つの条件から連立方程式を立てて解くのが定石です。

1.  **力のつり合いの条件**: 節点Oに作用する力の合計がゼロになる。
2.  **変形の適合条件**: 3本の棒の伸びと、それによって生じる節点Oの変位との間に成り立つ幾何学的な関係。

また、構造が左右対称なので、棒Aと棒Cの軸力や応力は等しくなります (`N_A = N_C`)。これを利用すると計算が少し楽になります。

**途中式と計算過程**
1.  **力のつり合いの式**
    節点Oに作用する力の、鉛直方向（y方向）のつり合いを考えます。各棒に生じる軸力（引張を正）を `N_A`, `N_B`, `N_C` とすると、
    `N_A cosθ + N_B + N_C cosθ = P`
    対称性より `N_A = N_C` なので、
    `2 N_A cosθ + N_B = P`  --- (式①)

2.  **変形の適合条件**
    節点Oが鉛直下向きに `δ` だけ変位したとします。変形は微小なので、各棒の伸び `λ` は、この変位 `δ` の各棒の軸方向成分に等しくなります。
    *   中央の棒Bの伸び: `λ_B = δ`
    *   左右の棒A, Cの伸び: `λ_A = λ_C = δ cosθ`

3.  **伸びと軸力の関係（フックの法則）**
    各棒の伸びは、軸力 `N` を用いて `λ = Nl / (AE)` と表せます。
    *   `λ_B = N_B l / (AE)`  =>  `N_B = (AE/l) * λ_B = (AE/l) * δ`
    *   `λ_A = N_A l / (AE)`  =>  `N_A = (AE/l) * λ_A = (AE/l) * δ cosθ`

4.  **連立方程式を解く**
    上記で求めた `N_A`, `N_B` を、(式①)の力のつり合い式に代入して、まずは変位 `δ` を求めます。
    `2 * { (AE/l) δ cosθ } * cosθ + { (AE/l) δ } = P`
    `(AE/l) * δ * (2cos²θ + 1) = P`
    これを `δ` について解くと、
    `δ = (P * l) / { AE (1 + 2cos²θ) }`

5.  **応力と変位を求める**
    求めた変位 `δ` を使って、各棒の軸力 `N`、そして応力 `σ = N/A` を計算します。
    *   **棒Bの応力 `σ_B`:**
        `N_B = (AE/l) * δ = (AE/l) * (P l) / { AE (1 + 2cos²θ) } = P / (1 + 2cos²θ)`
        `σ_B = N_B / A = P / { A (1 + 2cos²θ) }` (引張)
    *   **棒A, Cの応力 `σ_A`, `σ_C`:**
        `N_A = N_C = N_B * cosθ = P cosθ / (1 + 2cos²θ)`
        `σ_A = σ_C = N_A / A = P cosθ / { A (1 + 2cos²θ) }` (引張)

**解答**
*   **各棒の応力:**
    *   `σ_A = σ_C = (P cosθ) / { A (1 + 2cos²θ) }` (引張応力)
    *   `σ_B = P / { A (1 + 2cos²θ) }` (引張応力)
*   **節点Oの変位:**
    *   `δ = (P * l) / { AE (1 + 2cos²θ) }` (鉛直下向き)

#### (2) 温度がt℃上昇する場合

**方針・考え方**
今度は荷重の代わりに温度変化による熱応力を考えます。これも不静定問題です。もし各棒が自由に伸縮できるなら、応力は発生せずに `αlt` だけ伸びます。しかし、実際には節点Oで結合されているため、各棒の伸び方が拘束され、内部に応力（熱応力）が発生します。
考え方の手順は(1)とほぼ同じですが、「伸び」の式に熱膨張の項が加わります。

**途中式と計算過程**
1.  **力のつり合いの式**
    外力は作用していないので、節点Oでの力のつり合いは以下のようになります。
    `2 N_A cosθ + N_B = 0` --- (式②)

2.  **変形の適合条件**
    (1)と同様に、節点Oが鉛直下向きに `δ` だけ変位したとすると、各棒の伸び `λ` は、
    *   `λ_B = δ`
    *   `λ_A = λ_C = δ cosθ`

3.  **伸びと応力、温度変化の関係**
    各棒の全長変化 `λ` は、「熱による伸び」と「応力による弾性的な伸び」の和になります。
    `λ = (熱膨張) + (弾性変形) = αlt + Nl/(AE)`
    *   `λ_B = αlt + N_B l / (AE)`
    *   `λ_A = αlt + N_A l / (AE)`

4.  **連立方程式を解く**
    変形の適合条件と、伸びの関係式から、
    `δ = λ_B = αlt + N_B l / (AE)`
    これを `λ_A` の式に代入すると、
    `λ_A = δ cosθ = (αlt + N_B l / (AE)) cosθ`
    一方、`λ_A` は `αlt + N_A l / (AE)` でもあるので、この2式は等しくなります。
    `αlt + N_A l / (AE) = (αlt + N_B l / (AE)) cosθ`
    両辺を `l` で割り、整理します。
    `αt + N_A/(AE) = αt cosθ + N_B cosθ / (AE)`
    `N_A - N_B cosθ = AEαt (cosθ - 1)` --- (式③)

    これで、力のつり合い式(②)と、変形に関する式(③)の2つの連立方程式ができました。
    (式②)より `N_B = -2N_A cosθ` を(式③)に代入します。
    `N_A - (-2N_A cosθ)cosθ = AEαt (cosθ - 1)`
    `N_A (1 + 2cos²θ) = AEαt (cosθ - 1)`
    `N_A = AEαt * (cosθ - 1) / (1 + 2cos²θ)`

5.  **応力と変位を求める**
    *   **棒A, Cの応力 `σ_A`, `σ_C`:**
        `σ_A = σ_C = N_A / A = Eαt * (cosθ - 1) / (1 + 2cos²θ)`
        (`cosθ`は1より小さいので、これは**圧縮応力**になります)
    *   **棒Bの応力 `σ_B`:**
        `N_B = -2N_A cosθ = -2 * { AEαt * (cosθ - 1) / (1 + 2cos²θ) } * cosθ`
        `σ_B = N_B / A = -2 Eαt cosθ (cosθ - 1) / (1 + 2cos²θ) = 2 Eαt cosθ (1 - cosθ) / (1 + 2cos²θ)`
        (こちらは**引張応力**になります)
    *   **節点Oの変位 `δ`:**
        `δ = λ_B = αlt + N_B l / (AE) = l(αt + σ_B/E)`
        `δ = l * ( αt + { 2 Eαt cosθ (1 - cosθ) / (E(1 + 2cos²θ))} )`
        `δ = lαt * ( 1 + { 2cosθ(1 - cosθ) } / (1 + 2cos²θ) )`
        `δ = lαt * { (1 + 2cos²θ + 2cosθ - 2cos²θ) / (1 + 2cos²θ) }`
        `δ = lαt * (1 + 2cosθ) / (1 + 2cos²θ)`

**解答**
*   **各棒の応力:**
    *   `σ_A = σ_C = Eαt (cosθ - 1) / (1 + 2cos²θ)` (圧縮応力)
    *   `σ_B = 2 Eαt cosθ (1 - cosθ) / (1 + 2cos²θ)` (引張応力)
*   **節点Oの変位:**
    *   `δ = lαt (1 + 2cosθ) / (1 + 2cos²θ)` (鉛直下向き)

### 【2】 連続はりの問題

**問題文**
図2(a)のように、スパンが等しく3点支持された長さ `2l` の連続はりABCが、図のような集中荷重 `P` を受ける場合に、縦弾性係数 `E` および断面二次モーメント `I` として、次の問に答えよ。

(1) 図2(a)のようにx軸をとり、はりABの曲げモーメント `M(x)` およびたわみを求める基礎式(微分方程式)を示せ。
(2) はりABの境界条件を示して、A点の反力 `R_A`、B点の反力 `R_B` および曲げモーメント `M_B` を求めよ。
(3) はりABCのせん断力線図 (SFD) および曲げモーメント線図 (BMD) を描け。
(4) はりの断面を図2(b)のように一辺 `a` の正方形とするとき、断面係数 `Z` を誘導して、はり上B点の曲げ応力 `σ_B` を求めよ。

#### (1) M(x) とたわみの基礎式

**方針・考え方**
A点からxの位置における曲げモーメント `M(x)` を、反力 `R_A` を使って表します。荷重 `P` の作用点を境に式が変わるため、場合分けが必要です。たわみの基礎式は、曲げモーメントとたわみの関係を表す微分方程式です。

**解答**
*   **曲げモーメント M(x):** (A点を原点とし、下に凸を正とする)
    *   区間 `0 ≦ x ≦ l/2`: `M(x) = R_A * x`
    *   区間 `l/2 ≦ x ≦ l`: `M(x) = R_A * x - P * (x - l/2)`
*   **たわみの基礎式:**
    `EI * (d²y / dx²) = M(x)`
    (ここで `y` はたわみ、`EI` は曲げ剛性)

#### (2) 反力と曲げモーメントの計算

**方針・考え方**
この問題は3つの支点に反力があるため、静力学のつり合いだけでは解けない不静定問題です。たわみの微分方程式を積分し、境界条件を用いて解くのが基本ですが、計算が非常に複雑になります。
ここでは、不静定はりを解くための強力な公式である「**3モーメントの定理**」を用いて解くのが効率的です。

*   **境界条件（微分方程式で解く場合）:**
    1.  `y(0) = 0` （A点は支点なのでたわみゼロ）
    2.  `y(l) = 0` （B点は支点なのでたわみゼロ）
    3.  `y(2l) = 0` （C点は支点なのでたわみゼロ）
    4.  B点(`x=l`)でたわみ角が連続する。

**3モーメントの定理を用いた計算過程**
3モーメントの定理は `M_A l_1 + 2M_B(l_1+l_2) + M_C l_2 = -(右辺の荷重項)` という形で、隣接する3つの支点の曲げモーメントの関係を表します。

1.  **値を代入:**
    *   両端のA, C点はピン支持なのでモーメントはゼロ。`M_A = 0`, `M_C = 0`。
    *   スパンは等しく `l_1 = l`, `l_2 = l`。
    *   定理の左辺: `0 + 2M_B(l+l) + 0 = 4l * M_B`
2.  **荷重項（右辺）の計算:**
    *   スパンAB, BCをそれぞれ単純支持はりと見なしたときの、中央荷重Pによる曲げモーメント図（高さ`Pl/4`の三角形）の面積と重心から計算します。
    *   右辺 = `-6 * ( (A_1 * x̄_1)/l_1 + (A_2 * x̄_2)/l_2 ) = -6 * (2 * ( (1/2 * l * Pl/4) * (l/2) ) / l ) = -3Pl²/4`
3.  **B点の曲げモーメント `M_B` を求める:**
    `4l * M_B = -3Pl²/4`  =>  `M_B = -3Pl / 16`
    (マイナスは、B点で上が凸になるモーメントであることを示します)

4.  **反力 `R_A`, `R_B` を求める:**
    *   **`R_A`**: スパンABについて、B点周りのモーメントのつり合いを考えます。
        `R_A * l - P * (l/2) = M_B`
        `R_A * l = P*l/2 + M_B = Pl/2 - 3Pl/16 = (8-3)Pl/16 = 5Pl/16`
        `R_A = 5P / 16`
    *   **`R_B`**: B点の左右のせん断力の差が反力 `R_B` になります。
        *   B点直左のせん断力 `V_{B, left} = R_A - P = 5P/16 - P = -11P/16`
        *   対称性から `R_C = R_A = 5P/16`。スパンBCを考えると、B点直右のせん断力 `V_{B, right} = P - R_C = P - 5P/16 = 11P/16`
        *   `R_B = V_{B, right} - V_{B, left} = (11P/16) - (-11P/16) = 22P/16 = 11P / 8`

**解答**
*   **はりABの境界条件:** `y(0)=0`, `y(l)=0`
*   **A点の反力:** `R_A = 5P / 16`
*   **B点の反力:** `R_B = 11P / 8`
*   **B点の曲げモーメント:** `M_B = -3Pl / 16` (上に凸)

#### (3) SFD と BMD

**方針・考え方**
(2)で求めた反力を用いて、はりの左端(A点)から順にせん断力と曲げモーメントを計算し、グラフに描きます。

**解答**
*   **せん断力線図 (SFD):**
    *   A-D (0 < x < l/2): `V = R_A = +5P/16`
    *   D-B (l/2 < x < l): `V = R_A - P = -11P/16`
    *   B-E (l < x < 3l/2): `V = R_A - P + R_B = +11P/16`
    *   E-C (3l/2 < x < 2l): `V = R_A - P + R_B - P = -5P/16`
    *   グラフは、上記の区間ごとに一定値をとる階段状の線図になります。

*   **曲げモーメント線図 (BMD):**
    *   A点 (x=0): `M = 0`
    *   D点 (x=l/2): `M = R_A * (l/2) = (5P/16) * (l/2) = +5Pl/32`
    *   B点 (x=l): `M = M_B = -3Pl/16` (計算結果と一致)
    *   E点 (x=3l/2): 対称性から `M = +5Pl/32`
    *   C点 (x=2l): `M = 0`
    *   グラフは、各点を直線で結んだ折れ線グラフになります。

#### (4) 断面係数とB点の曲げ応力

**方針・考え方**
断面係数Zは `I / y_max` で定義されます。曲げ応力は公式 `σ = M / Z` を使って計算します。

**途中式と計算過程**
1.  **断面係数Zの誘導:**
    *   一辺 `a` の正方形断面の断面二次モーメント `I` は、
        `I = (幅) * (高さ)³ / 12 = a * a³ / 12 = a⁴ / 12`
    *   中立軸から最も遠い距離 `y_max` は `a/2`。
    *   断面係数 `Z` は、
        `Z = I / y_max = (a⁴ / 12) / (a / 2) = a³ / 6`

2.  **B点の曲げ応力 `σ_B` の計算:**
    *   B点での曲げモーメントの大きさは `|M_B| = 3Pl / 16`。
    *   このモーメントによる最大曲げ応力（はりの縁で発生）を求めます。
        `σ_B_max = |M_B| / Z = (3Pl / 16) / (a³ / 6) = (18Pl) / (16a³) = 9Pl / (8a³)`
    *   B点ではモーメントが負（上に凸）なので、はりの**上縁で引張応力**、**下縁で圧縮応力**が生じます。問題では単に「曲げ応力」と聞かれているので、最大値（絶対値）を答えます。

**解答**
*   **断面係数:** `Z = a³ / 6`
*   **B点の曲げ応力(最大値):** `σ_B = 9Pl / (8a³)`

### 【3】 連続体力学の基礎に関する問題

**問題文**
ニュートン力学には、物理学で学習する「質点の力学」と、材料力学などで学習する「連続体(変形体)の力学」があることが知られている。すなわち、質点の力学おいては力のつりあいが成立するのに対して、連続体の力学では、応力のつりあい方程式が物体中の任意の点において成立する。これに関して、次の問に答えよ。

(1) 材料力学などの力学において、力および応力はそれぞれどのように定義されるか。また、それらの相違点は何か。
(2) 力, 応力はそれぞれ英語では何と呼ばれるか。
(3) 3次元問題において、直交座標(O-xyz)に対する応力のつりあい方程式(偏微分方程式)を導け。

#### (1) 力と応力の定義と相違点

**解答**
*   **定義:**
    *   **力 (Force):** 物体の運動状態を変化させたり、物体を変形させたりする原因となる物理量。大きさ、向き、作用点を持つベクトル量であり、単位はニュートン[N]で表される。主に物体全体に作用する外力として扱われる。
    *   **応力 (Stress):** 物体が外力を受けた際に、その内部に生じる抵抗力（内力）を、作用している仮想的な断面の単位面積あたりで表した量。物体内部のある「点」の状態を示す量であり、単位はパスカル[Pa] (N/m²)で表される。

*   **相違点:**
    *   **対象:** 「力」が物体全体や質点といったマクロな対象に作用するのに対し、「応力」は物体内部の微小な面に作用するミクロな概念である。
    *   **物理的意味:** 「力」は運動や変形の直接的な原因であるのに対し、「応力」は外力に対する物体内部の抵抗状態を表す。
    *   **次元と単位:** 「力」の単位は[N]、「応力」は圧力と同じ[N/m²]であり、単位面積あたりの力である点が異なる。
    *   **数学的表現:** 力はベクトルで表現されるが、応力は一点においても面の向きによって成分が変わるため、より複雑なテンソルという量で表現される。

#### (2) 英語での呼び方

**解答**
*   力: **Force**
*   応力: **Stress**

#### (3) 3次元応力のつりあい方程式の導出

**方針・考え方**
物体内部に、各辺の長さが `dx, dy, dz` の微小な直方体要素を考えます。この要素に作用する応力と、単位体積あたりの物体力（重力など）を考慮し、x, y, z の各方向について力のつり合いの式を立てます。

**導出過程**
1.  **微小要素の設定:**
    x, y, z 軸に平行な辺を持つ、体積 `dV = dxdydz` の微小直方体要素を考える。

2.  **x方向の力のつり合い:**
    *   **垂直応力 `σ_x` による力:**
        x軸に垂直な2つの面（面積 `dydz`）に作用する力の差を考える。左の面(`x`)の応力を `σ_x` とすると、右の面(`x+dx`)の応力は `σ_x + (∂σ_x/∂x)dx` となる。これらの合力は `(∂σ_x/∂x) dx dy dz` となる。
    *   **せん断応力 `τ_yx`, `τ_zx` による力:**
        同様に、y軸、z軸に垂直な面に作用するせん断応力による力の合力を考えると、それぞれ `(∂τ_yx/∂y) dx dy dz` と `(∂τ_zx/∂z) dx dy dz` になる。
    *   **物体力:**
        単位体積あたりのx方向の物体力を `X` とすると、微小要素に働く物体力は `X dx dy dz` となる。
    *   **つり合い:**
        これらの力の総和はゼロになるので、
        `(∂σ_x/∂x)dxdydz + (∂τ_yx/∂y)dxdydz + (∂τ_zx/∂z)dxdydz + X dxdydz = 0`
        両辺を体積 `dxdydz` で割ると、x方向のつり合い式が得られる。

3.  **y, z方向のつり合い:**
    同様にy方向、z方向の力のつり合いを考えると、以下の3つの式が得られる。

**解答**
直交座標(O-xyz)における応力のつりあい方程式は以下の通りです。ここで、`X, Y, Z` はそれぞれx, y, z方向の単位体積あたりの物体力を示します。
```
∂σ_x/∂x + ∂τ_yx/∂y + ∂τ_zx/∂z + X = 0
∂τ_xy/∂x + ∂σ_y/∂y + ∂τ_zy/∂z + Y = 0
∂τ_xz/∂x + ∂τ_yz/∂y + ∂σ_z/∂z + Z = 0
```
(注: モーメントのつり合いから導かれるせん断応力の共役性 `τ_xy = τ_yx` などを用いると、よりシンプルな表記も可能です。)

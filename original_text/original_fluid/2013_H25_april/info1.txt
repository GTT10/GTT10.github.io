## 【1】

### 問題文

以下の式で表される2次元速度場 $(u, v)$ を考える. $u, v$ はそれぞれx方向速度, y方向速度である。速度場が時刻の経過とともに次のように変化したとする。 [cite: 8]

$0 \\le t \< 5$ [s]
$(u, v) = (0, 1)$ [m/s] [cite: 10, 11]

$5 \\le t \\le 10$ [s]
$(u, v) = (1, 0)$ [m/s] [cite: 12, 13, 14, 15]

このとき、以下の問いに答えよ。 [cite: 16]

-----

### (1) 次の流体力学用語を説明せよ。

#### (a) 流体粒子

**方針・考え方・説明:**
流体力学では、流体の挙動を分析するために、流体を非常に小さな塊として扱います。この「非常に小さな塊」が流体粒子です。ただし、この「小さい」というのは、流体の連続体としての性質が保たれる程度に小さいという意味です。

**要点・チップス:**

  * 流体粒子は、数学的な概念であり、実際に物理的に分離できる粒子を指すわけではありません。
  * 流体粒子一つ一つは、その位置、速度、加速度、圧力、密度などの物理量を持つと仮定されます。
  * 流体の運動を追跡する際に、この流体粒子を追跡することで、流れの挙動を理解します。

**解答:**
流体粒子とは、流体の連続体を構成する仮想的に無限小の要素のことで、その中では流体の物理的性質（密度、速度、圧力など）が均一であるとみなせる微小な塊を指します。個々の流体粒子は、流れの中で時間とともに移動し、その軌跡を追うことで流体の運動を記述することができます。

-----

#### (b) 流線

**方針・考え方・説明:**
流線は、ある瞬間の流れの様子を視覚的に表現するための概念です。流れの中のある点における流体の速度ベクトルに常に接する線として定義されます。

**要点・チップス:**

  * 流線は「ある瞬間」のスナップショットであり、定常流（時間が経っても速度場が変化しない流れ）でない限り、流線は時間とともに変化します。
  * 流線は決して交差しません（よどみ点などの特異点を除く）。なぜなら、もし交差すると、その交点で流体粒子の速度が2つの方向を持つことになり、物理的にありえないからです。
  * 流線は流体の進行方向を示します。

**解答:**
流線とは、ある特定の瞬間において、流体中の各点における速度ベクトルに常に接するように引かれた仮想的な線です。すなわち、流線上のどの点においても、その点での流体の速度方向と流線の方向は一致します。流線は、その瞬間の流れのパターンや方向を視覚的に表現するために用いられます。定常流においては、流線は流体粒子の流跡線と一致します。

### (2) 2つの時間帯 $0 \\le t \< 5$ [s], $5 \\le t \\le 10$ [s] における流線を描け。

**方針・考え方・説明:**
流線は、速度ベクトルに常に接する線です。速度ベクトル $(u, v)$ が与えられている場合、流線の方程式は $dx/u = dy/v$ で表されます。この微分方程式を解くことで流線の式が得られます。

  * **時間帯 $0 \\le t \< 5$ [s]:**
    速度場は $(u, v) = (0, 1)$ です。
    $u=0, v=1$ の場合、流体はy軸正方向にのみ移動します。このとき、流線は$x$座標が変化せず、$y$座標のみが変化します。
    流線の微分方程式は $dx/0 = dy/1$ となります。これは $dx=0$ を意味します。
    したがって、$x = \\text{定数}$ となります。

  * **時間帯 $5 \\le t \\le 10$ [s]:**
    速度場は $(u, v) = (1, 0)$ です。
    $u=1, v=0$ の場合、流体はx軸正方向にのみ移動します。このとき、流線は$y$座標が変化せず、$x$座標のみが変化します。
    流線の微分方程式は $dx/1 = dy/0$ となります。これは $dy=0$ を意味します。
    したがって、$y = \\text{定数}$ となります。

**解答:**

#### $0 \\le t \< 5$ [s] の流線

速度場は $(u, v) = (0, 1)$ [m/s] です。これは流体がy軸の正方向へ一定の速さで流れていることを示します。
流線の方程式は $dx/u = dy/v$ より、 $dx/0 = dy/1$ となります。
この式は $dx = 0$ を意味するため、$x = C\_1$ (定数) となります。
したがって、この時間帯の流線は、x軸に平行な直線、つまり**y軸に平行な直線**の集合となります。

**図示:**
(テキストでの図示のため、概念的な説明となります)
y軸に平行に、上向きに矢印を伴った直線が複数描かれます。例えば、x= -2, x= -1, x=0, x=1, x=2 などで、それぞれy軸に平行な直線が引かれ、上向きの矢印（速度の方向）が付いている状態です。

```
     ^ y
     |
     |
     |
     |  -> (0,1)
     |  |
     |  |
     |  |
<----|--+------> x
     |  |
     |  |
     |  |
     V
```

（正確には、各x座標においてy軸に平行な上向きの線が複数本描かれます。）

#### $5 \\le t \\le 10$ [s] の流線

速度場は $(u, v) = (1, 0)$ [m/s] です。これは流体がx軸の正方向へ一定の速さで流れていることを示します。
流線の方程式は $dx/u = dy/v$ より、 $dx/1 = dy/0$ となります。
この式は $dy = 0$ を意味するため、$y = C\_2$ (定数) となります。
したがって、この時間帯の流線は、y軸に平行な直線、つまり**x軸に平行な直線**の集合となります。

**図示:**
(テキストでの図示のため、概念的な説明となります)
x軸に平行に、右向きに矢印を伴った直線が複数描かれます。例えば、y= -2, y= -1, y=0, y=1, y=2 などで、それぞれx軸に平行な直線が引かれ、右向きの矢印（速度の方向）が付いている状態です。

```
     ^ y
     |
   --->--->--->
     |
   --->--->--->
     |
<----+----------+------> x
     |
   --->--->--->
     |
   --->--->--->
     V
```

（正確には、各y座標においてx軸に平行な右向きの線が複数本描かれます。）

### (3) 時刻 $t=0$ [s]に原点 $O(0,0)$ を出発した流体粒子が $t=0$ [s] から $t=10$ [s] までに描く流跡線を図示せよ。

**方針・考え方・説明:**
流跡線（Pathline）は、特定の流体粒子が時間とともに移動する軌跡です。流体粒子の位置を $(x(t), y(t))$ とすると、その速度は $u = dx/dt$ と $v = dy/dt$ で与えられます。これらの微分方程式を初期条件を用いて解くことで、流体粒子の軌跡を求めることができます。

  * **$0 \\le t \< 5$ [s] の期間:**
    速度は $(u, v) = (0, 1)$ です。
    $dx/dt = 0 \\implies x(t) = C\_x$
    $dy/dt = 1 \\implies y(t) = t + C\_y$
    初期条件は $t=0$ で $(x, y) = (0, 0)$ です。
    $x(0) = 0 \\implies C\_x = 0$
    $y(0) = 0 \\implies C\_y = 0$
    したがって、この期間の流体粒子の位置は $(x(t), y(t)) = (0, t)$ となります。
    $t=5$ [s] のとき、流体粒子の位置は $(0, 5)$ となります。

  * **$5 \\le t \\le 10$ [s] の期間:**
    速度は $(u, v) = (1, 0)$ です。
    $dx/dt = 1 \\implies x(t) = t + C'\_x$
    $dy/dt = 0 \\implies y(t) = C'\_y$
    この期間の初期条件は $t=5$ [s] で流体粒子の位置が $(0, 5)$ であることです。
    $x(5) = 0 \\implies 5 + C'\_x = 0 \\implies C'\_x = -5$
    $y(5) = 5 \\implies C'\_y = 5$
    したがって、この期間の流体粒子の位置は $(x(t), y(t)) = (t-5, 5)$ となります。
    $t=10$ [s] のとき、流体粒子の位置は $(10-5, 5) = (5, 5)$ となります。

**解答:**

時刻 $t=0$ [s] に原点 $O(0,0)$ を出発した流体粒子が $t=10$ [s] までに描く流跡線を求めます。

  * **$0 \\le t \< 5$ [s] の期間:**
    速度場は $(u, v) = (0, 1)$ です。
    $dx/dt = 0 \\implies x(t) = C\_1$
    $dy/dt = 1 \\implies y(t) = t + C\_2$
    初期条件 $t=0, (x,y)=(0,0)$ を代入すると、$C\_1=0, C\_2=0$。
    よって、$x(t)=0, y(t)=t$。
    この期間では、流体粒子はy軸上を原点から上方向に移動します。
    $t=5$ [s] のとき、流体粒子は $(x,y) = (0, 5)$ の位置に到達します。

  * **$5 \\le t \\le 10$ [s] の期間:**
    速度場は $(u, v) = (1, 0)$ です。
    $dx/dt = 1 \\implies x(t) = t + C\_3$
    $dy/dt = 0 \\implies y(t) = C\_4$
    この期間の開始時刻は $t=5$ [s] であり、そのときの流体粒子の位置は $(0, 5)$ です。
    $x(5) = 0 \\implies 5 + C\_3 = 0 \\implies C\_3 = -5$
    $y(5) = 5 \\implies C\_4 = 5$
    よって、$x(t)=t-5, y(t)=5$。
    この期間では、流体粒子は $y=5$ の直線上をx軸の正方向に移動します。
    $t=10$ [s] のとき、流体粒子は $(x,y) = (10-5, 5) = (5, 5)$ の位置に到達します。

**流跡線の図示:**
流体粒子は、まず原点 $(0,0)$ から $(0,5)$ までy軸に沿って直進し、その後 $(0,5)$ から $(5,5)$ までy=5の直線に沿って直進します。

```
     ^ y
     |
(0,5)+----->(5,5)
     |     
     |     
     |     
     |     
     +-----x (0,0)から(0,5)へ移動
     |
     |
     V
```

（図としては、原点からy軸上を$(0,5)$まで上向きの矢印、そして$(0,5)$からx軸に平行に$(5,5)$まで右向きの矢印でつながる軌跡となります。）

### (4) 時刻 $t=0$ [s] から $t=10$ [s] まで連続して原点に色素を注入した $t=10$ [s] においてこの色素が描く線(流脈線)を図示せよ。

**方針・考え方・説明:**
流脈線（Streakline）は、ある固定点から継続的に注入された流体粒子群が、ある瞬間にどこに存在するかを示す線です。つまり、特定の時刻（ここでは $t=10$ [s]）に、原点から過去に注入されたすべての色素粒子がどこにいるかを示す線です。

原点 $O(0,0)$ から時刻 $\\tau$ に注入された流体粒子が、時刻 $t$ においてどこにいるかを追跡します。ここで $0 \\le \\tau \\le 10$ となります。

  * **粒子が $0 \\le \\tau \< 5$ [s] の間に注入された場合:**
    注入された時刻 $\\tau$ から $t=5$ [s] まで、その粒子は $(0,1)$ の速度で移動します。
    時刻 $t=5$ [s] での位置は $(x\_5, y\_5) = (0, 1 \\times (5-\\tau)) = (0, 5-\\tau)$ です。
    その後、$t=5$ [s] から $t=10$ [s] まで、その粒子は $(1,0)$ の速度で移動します。
    時刻 $t=10$ [s] での位置は $(x\_{10}, y\_{10}) = (0 + 1 \\times (10-5), (5-\\tau) + 0 \\times (10-5)) = (5, 5-\\tau)$ です。
    $\\tau$ は $0 \\le \\tau \< 5$ の範囲を取るので、
    $\\tau=0$ のとき、$(x\_{10}, y\_{10}) = (5, 5)$
    $\\tau=5$ に近づくにつれて、$y\_{10}$ は $0$ に近づき、$(x\_{10}, y\_{10}) = (5, 0)$

  * **粒子が $5 \\le \\tau \\le 10$ [s] の間に注入された場合:**
    注入された時刻 $\\tau$ から $t=10$ [s] まで、その粒子は $(1,0)$ の速度で移動します。
    時刻 $t=10$ [s] での位置は $(x\_{10}, y\_{10}) = (0 + 1 \\times (10-\\tau), 0 + 0 \\times (10-\\tau)) = (10-\\tau, 0)$ です。
    $\\tau$ は $5 \\le \\tau \\le 10$ の範囲を取るので、
    $\\tau=5$ のとき、$(x\_{10}, y\_{10}) = (5, 0)$
    $\\tau=10$ のとき、$(x\_{10}, y\_{10}) = (0, 0)$

**解答:**

流脈線は、特定の固定点（この場合、原点 $O(0,0)$）から異なる時刻 $\\tau$ に出発した流体粒子が、特定の時刻 $t$（この場合 $t=10$ [s]）に存在する場所を結んだ線です。

時刻 $t=10$ [s] における流脈線を求めます。原点 $O(0,0)$ から時刻 $\\tau$ に出発した流体粒子の位置を追跡します。

1.  **$0 \\le \\tau \< 5$ [s] の間に注入された粒子:**

      * 時刻 $\\tau$ から $t=5$ [s] までの移動: 速度 $(0,1)$ で $(5-\\tau)$ [s] 移動。
        この時点での位置は $(x\_5, y\_5) = (0, 0 + 1 \\times (5-\\tau)) = (0, 5-\\tau)$。
      * 時刻 $t=5$ [s] から $t=10$ [s] までの移動: 速度 $(1,0)$ で $(10-5)=5$ [s] 移動。
        時刻 $t=10$ [s] での位置は $(x\_{10}, y\_{10}) = (0 + 1 \\times 5, (5-\\tau) + 0 \\times 5) = (5, 5-\\tau)$。
        $\\tau$ の範囲は $0 \\le \\tau \< 5$ なので、
        $y\_{10}$ の範囲は $5-5 \< y\_{10} \\le 5-0 \\implies 0 \< y\_{10} \\le 5$ となります。
        この部分の流脈線は、点 $(5,0)$ から $(5,5)$ までのy軸に平行な線分（$(5,0)$ を除く）となります。

2.  **$5 \\le \\tau \\le 10$ [s] の間に注入された粒子:**

      * 時刻 $\\tau$ から $t=10$ [s] までの移動: 速度 $(1,0)$ で $(10-\\tau)$ [s] 移動。
        時刻 $t=10$ [s] での位置は $(x\_{10}, y\_{10}) = (0 + 1 \\times (10-\\tau), 0 + 0 \\times (10-\\tau)) = (10-\\tau, 0)$。
        $\\tau$ の範囲は $5 \\le \\tau \\le 10$ なので、
        $x\_{10}$ の範囲は $10-10 \\le x\_{10} \\le 10-5 \\implies 0 \\le x\_{10} \\le 5$ となります。
        この部分の流脈線は、点 $(0,0)$ から $(5,0)$ までのx軸に平行な線分となります。

**流脈線の図示:**
時刻 $t=10$ [s] における流脈線は、上記の2つの線分を組み合わせたものです。具体的には、原点 $(0,0)$ から $(5,0)$ までの線分（x軸上）と、そこから $(5,5)$ までの線分（y軸に平行）がつながった「L」字型の線となります。

```
     ^ y
     |
(5,5)x
     |
     |
     |
     |
(0,0)x------x(5,0)----> x
     |
     |
     V
```

（図としては、原点からx軸に沿って$(5,0)$まで、そこからy軸に沿って$(5,5)$までつながる線となります。矢印はつけません。）

## 【2】 問題文

図1に示すような、x軸の正方向に向かう速度Uの一様流と、原点にある強さmの吹き出しの重ね合わせで表される2次元流がある。このとき、以下の問いに答えよ。

(1) 速度Uの一様な流れの複素速度ポテンシャル $W_1$ を示せ。
(2) 原点にある強さmの吹き出しの複素速度ポテンシャル $W_2$ を示せ。
(3) 図1に示す2次元流の、点 $(x,y)$ における速度とよどみ点位置を、複素速度ポテンシャル $W=W_1+W_2$ を用いて求めよ。
(4) 原点から単位時間に湧き出す流量を求めよ。
(5) 複素速度ポテンシャル Wは、無限遠において幅Dとなる物体のまわりの2次元非圧縮流れを表す。物体によって、下流側で押しのけられる流量はいくらか。
(6) 強さmをU,Dを用いて表せ。

### (1) 速度Uの一様な流れの複素速度ポテンシャル $W_1$ を示せ。

**方針・考え方・説明:**
複素速度ポテンシャル $W = \phi + i\psi$ は、速度ポテンシャル $\phi$ と流れ関数 $\psi$ を組み合わせたものです。2次元非圧縮性流れの場合、速度成分 $u, v$ はそれぞれ $\partial \phi / \partial x = u$ および $\partial \phi / \partial y = v$ で表されます。また、複素速度ポテンシャルを複素変数 $z = x + iy$ で微分すると、共役複素速度が得られます: $dW/dz = u - iv$。

x軸の正方向に向かう速度Uの一様な流れの場合、速度ベクトルは $(U, 0)$ です。
したがって、$u=U, v=0$ となります。

**計算過程:**
1. 速度ポテンシャル $\phi$ を求める。
   $u = \partial \phi / \partial x = U \implies \phi = Ux + f(y)$
   $v = \partial \phi / \partial y = 0 \implies \partial (Ux + f(y)) / \partial y = f'(y) = 0 \implies f(y) = C$ (定数)
   よって、$\phi = Ux + C$。定数は通常ゼロとしますので、$\phi = Ux$。

2. 流れ関数 $\psi$ を求める。
   $u = \partial \psi / \partial y = U \implies \psi = Uy + g(x)$
   $v = -\partial \psi / \partial x = 0 \implies -\partial (Uy + g(x)) / \partial x = -g'(x) = 0 \implies g(x) = C'$ (定数)
   よって、$\psi = Uy + C'$。定数は通常ゼロとしますので、$\psi = Uy$。

3. 複素速度ポテンシャル $W_1 = \phi + i\psi$ に代入する。
   $W_1 = Ux + iUy = U(x + iy) = Uz$

**要点・チップス:**
* 一様流の複素速度ポテンシャルは非常に基本的で、様々な流れの重ね合わせの出発点となります。
* $z = x+iy$ の形を覚えておくと便利です。

**解答:**
速度Uの一様な流れの速度成分は $(u, v) = (U, 0)$ です。
速度ポテンシャル $\phi$ は $u = \partial \phi / \partial x = U$ および $v = \partial \phi / \partial y = 0$ から、$\phi = Ux$ となります。
流れ関数 $\psi$ は $u = \partial \psi / \partial y = U$ および $v = -\partial \psi / \partial x = 0$ から、$\psi = Uy$ となります。
したがって、複素速度ポテンシャル $W_1 = \phi + i\psi$ は、
$W_1 = Ux + iUy = U(x + iy)$
複素変数 $z = x + iy$ を用いて、
$W_1 = Uz$

### (2) 原点にある強さmの吹き出しの複素速度ポテンシャル $W_2$ を示せ。

**方針・考え方・説明:**
吹き出し（Source）は、ある点から流体が放射状に湧き出す流れをモデル化したものです。2次元の吹き出しの強さ $m$ は、単位時間あたりに湧き出す流量と関係しています。

吹き出しの複素速度ポテンシャルは、対数関数を用いて表されます。
一般的に、原点にある強さ $m$ の吹き出しの複素速度ポテンシャルは、$W_2 = (m / 2\pi) \ln z$ と表されます。ただし、文献によっては $m$ の定義が異なる場合があるため、注意が必要です。ここでは、流量と $m$ の関係を後で確認します。

**要点・チップス:**
* 吹き出しは正の $m$ で表され、吸い込み（Sink）は負の $m$ で表されます。
* 対数関数 $\ln z = \ln r + i\theta$ の性質を理解しておくことが重要です。

**解答:**
原点にある強さ $m$ の吹き出しの複素速度ポテンシャル $W_2$ は、
$W_2 = \frac{m}{2\pi} \ln z$

### (3) 図1に示す2次元流の、点 $(x,y)$ における速度とよどみ点位置を、複素速度ポテンシャル $W=W_1+W_2$ を用いて求めよ。

**方針・考え方・説明:**
重ね合わせの原理により、全体の複素速度ポテンシャル $W$ は個々の流れの複素速度ポテンシャルの和として表されます。
$W = W_1 + W_2 = Uz + \frac{m}{2\pi} \ln z$

点 $(x,y)$ における速度 $(u, v)$ を求めるには、複素速度ポテンシャルを $z$ で微分し、共役複素速度 $u - iv$ を求めます。
$u - iv = dW/dz$

よどみ点とは、流体の速度がゼロになる点です。したがって、$u=0$ かつ $v=0$ となる点を探します。これは $u - iv = 0$ となる点を探すことと同じです。

**計算過程:**
1. 複素速度 $u-iv$ を求める。
   $W = U(x+iy) + \frac{m}{2\pi} \ln(x+iy)$
   $dW/dz = U + \frac{m}{2\pi} \frac{1}{z} = U + \frac{m}{2\pi} \frac{1}{x+iy}$
   分母を有理化します:
   $u - iv = U + \frac{m}{2\pi} \frac{x-iy}{x^2+y^2} = U + \frac{m x}{2\pi (x^2+y^2)} - i \frac{m y}{2\pi (x^2+y^2)}$

2. 速度成分 $u, v$ を分離する。
   $u = U + \frac{m x}{2\pi (x^2+y^2)}$
   $v = \frac{m y}{2\pi (x^2+y^2)}$

3. よどみ点を探す。
   よどみ点では $u=0$ かつ $v=0$ です。
   まず $v=0$ となる条件を考えます。
   $\frac{m y}{2\pi (x^2+y^2)} = 0$
   $m$ は吹き出しの強さで、通常 $m \ne 0$ です。また、$x^2+y^2 \ne 0$ (原点を除く) なので、$y=0$ となります。
   つまり、よどみ点はx軸上 ($y=0$) に存在します。

   次に、$y=0$ を $u=0$ の式に代入します。
   $U + \frac{m x}{2\pi (x^2+0^2)} = 0$
   $U + \frac{m x}{2\pi x^2} = 0$
   $U + \frac{m}{2\pi x} = 0$
   $\frac{m}{2\pi x} = -U$
   $x = -\frac{m}{2\pi U}$

   よって、よどみ点の位置は $(x,y) = (-\frac{m}{2\pi U}, 0)$ です。

**要点・チップス:**
* 複素速度ポテンシャルを微分すると共役複素速度 $u-iv$ が得られることを忘れないようにしましょう。
* よどみ点では速度がゼロになるので、$u-iv = 0$ となる点を求めます。

**解答:**
全体の複素速度ポテンシャル $W = W_1 + W_2 = Uz + \frac{m}{2\pi} \ln z$ です。

点 $(x,y)$ における複素速度 $u-iv$ は、$W$ を $z$ で微分することで得られます。
$$u - iv = \frac{dW}{dz} = \frac{d}{dz} \left( Uz + \frac{m}{2\pi} \ln z \right) = U + \frac{m}{2\pi z}$$ここで $z = x+iy$ なので、$$u - iv = U + \frac{m}{2\pi(x+iy)} = U + \frac{m(x-iy)}{2\pi(x^2+y^2)} = \left( U + \frac{mx}{2\pi(x^2+y^2)} \right) - i \left( \frac{my}{2\pi(x^2+y^2)} \right)$$したがって、点 $(x,y)$ における速度成分は以下のようになります。$$u = U + \frac{mx}{2\pi(x^2+y^2)}$$
$$v = \frac{my}{2\pi(x^2+y^2)}$$

次に、よどみ点の位置を求めます。よどみ点では速度がゼロとなるので、$u=0$ かつ $v=0$ となります。
まず $v=0$ の式から、
$$\frac{my}{2\pi(x^2+y^2)} = 0$$
$m \ne 0$ であり、$x^2+y^2 \ne 0$ (原点ではない) なので、$y=0$ である必要があります。
次に $y=0$ を $u=0$ の式に代入します。
$$U + \frac{mx}{2\pi(x^2+0^2)} = 0$$$$U + \frac{mx}{2\pi x^2} = 0$$$$U + \frac{m}{2\pi x} = 0$$$$\frac{m}{2\pi x} = -U$$$$x = -\frac{m}{2\pi U}$$
よって、よどみ点の位置は $P_s = \left( -\frac{m}{2\pi U}, 0 \right)$ です。

---

### (4) 原点から単位時間に湧き出す流量を求めよ。

**方針・考え方・説明:**
2次元の吹き出しの強さ $m$ は、通常、単位奥行きあたりの流量として定義されます。流量は、原点を通る任意の閉じた曲線を囲むように流出する流体の量として計算できます。円周 $r$ の円周上での速度の動径成分を積分することで求められます。

半径 $r$ の円周上では、吹き出しによる速度の動径成分は $u_r = \frac{m}{2\pi r}$ です（吹き出しの複素速度ポテンシャル $W_2 = \frac{m}{2\pi} \ln z = \frac{m}{2\pi}(\ln r + i\theta)$ より、速度ポテンシャル $\phi_2 = \frac{m}{2\pi} \ln r$ なので、 $u_r = \partial \phi_2 / \partial r = \frac{m}{2\pi r}$）。

流量 $Q$ は、この動径速度に円周の長さを掛けたものです。
$Q = u_r \times (2\pi r)$

**計算過程:**
$Q = \left( \frac{m}{2\pi r} \right) \times (2\pi r) = m$

**要点・チップス:**
* 2次元の吹き出しの「強さ $m$」は、定義上、単位奥行きあたりの流量そのものであることが多いです。
* この問題では、単位時間に湧き出す流量を尋ねているので、これはそのまま $m$ となります。

**解答:**
原点にある強さ $m$ の吹き出しは、その定義から、原点から単位時間に湧き出す流量が $m$ であることを意味します。
したがって、原点から単位時間に湧き出す流量は $m$ です。

---

### (5) 複素速度ポテンシャル Wは、無限遠において幅Dとなる物体のまわりの2次元非圧縮流れを表す。物体によって、下流側で押しのけられる流量はいくらか。

**方針・考え方・説明:**
この問題は、一様流と吹き出しの重ね合わせによって形成される、いわゆる「半無限体（Rankine Half-Body）」の概念に基づいています。図1もこの形状を示唆しています。この物体は、上流側のよどみ点から始まり、下流側でx軸に沿って広がっていく形をしています。無限遠において幅Dとなるというのは、この物体がx軸正方向へ無限に広がり、最終的に一様流から押しのける流体の幅がDになることを意味します。

物体によって押しのけられる流量とは、物体が存在することによって、本来流れていたであろう空間から外側に迂回させられる流量のことです。これは、物体の上側と下側を通る流量の合計となります。
図1を見ると、上流の一様流が物体に当たって分かれ、最終的に幅Dで広がる流れになっています。この「幅D」は、物体が存在することで流線がずれる最大幅を意味します。このDは、物体が無限遠で押しのける流量 $Q_{displaced}$ と一様流の速度 $U$ を用いて、$Q_{displaced} = U \times D$ と表されます。

**要点・チップス:**
* ランキン半無限体のような流れの場合、物体によって押しのけられる流量は、無限遠での流線の間の幅と一様流の速度の積で表されます。
* 質量保存の法則に基づいています。

**解答:**
複素速度ポテンシャル $W=W_1+W_2$ で表される流れは、一様流と吹き出しの重ね合わせであり、これは典型的な「ランキン半無限体」の流れ場を表します。図1もこの半無限体を示しています。
物体によって下流側で押しのけられる流量とは、物体が存在することで流路が変化し、一様流の流線が物体に沿って流れることによって、本来物体がなかった場合に流れていただろう領域から押し出される流体の量のことを指します。

無限遠において、この物体が周囲の流れから押しのける流体の幅が $D$ であると与えられています。
一様流の速度が $U$ であるため、この幅 $D$ を通って流れる流量は、単純に速度と面積（この場合、2次元なので幅）の積で表されます。
したがって、物体によって下流側で押しのけられる流量は、$U \times D$ です。

---

### (6) 強さmをU,Dを用いて表せ。

**方針・考え方・説明:**
(4)で求めた吹き出しの流量と、(5)で求めた物体によって押しのけられる流量が等しいという関係を利用します。
吹き出しから湧き出した流体は、一様流と合わさって物体を形成し、その物体によって流体は外側に押し出されます。したがって、吹き出しの強さ $m$ は、物体が無限遠で押しのける流量と等しくなります。

**計算過程:**
(4)より、原点から湧き出す流量は $m$ でした。
(5)より、物体によって下流側で押しのけられる流量は $UD$ でした。
これらの流量は等しいので、$m = UD$ となります。

**要点・チップス:**
* 吹き出し（または吸い込み）の強さは、その要素が流れに与える流量変化と直接関連しています。
* ランキン半無限体の場合、吹き出しの流量が、物体を迂回する（押しのけられる）流量となります。

**解答:**
(4)で求めたように、原点から単位時間に湧き出す流量は $m$ です。
(5)で求めたように、物体によって下流側で押しのけられる流量は $UD$ です。
質量保存の法則により、吹き出しによって湧き出した流量は、その物体によって無限遠で押しのけられる流量に等しくなります。
したがって、
$m = UD$

## 【3】

### 問題文

直径 $d$, 密度 $\rho_s$ の小球が、図2に示すように、粘性係数 $\mu$, 密度 $\rho$ ($\rho < \rho_s$) の非圧縮粘性静止流体中を落下する。この小球の運動について以下の問いに答えよ。なお、重力加速度 $g$ はz軸の負方向に働いているとする。

(1) 小球の落下速度をVとするとき、小球のまわりのレイノルズ数 Reはいくらか。
(2) Re が1より小さいとき、流体から小球が受ける抵抗は $3\pi \mu Vd$ となる。この法則の名前を書け。
(3) 問 (2)の条件が満足されるとき、小球に働くすべての力の大きさと方向を求めよ。
(4) 小球に働く力が問 (3)で与えられるとき、時刻 $t$ における小球のz方向の位置 $Z(t)$ が満たす微分方程式を求めよ。
(5) 小球が時刻 $t=0$ で原点 $(0,0)$ から静かに落下を始めたとき、いくら時間が経過しても問(2)の法則が成り立つための条件を求め、最終的に接近するz軸の負方向速度の大きさを求めよ。また、この速度の名前を書け。
(6) 問 (2)の条件が成り立つとき、流体からの抵抗の3分の1は小球の落下方向前後。の圧力差 $\Delta p$ と関係している。この圧力は前(z軸の負方向)と後(z軸の正方向)のどちらが大きいか。また、$\Delta p$ がどの程度の大きさであるか、おおよその値を求めよ。なお、重力に起因する圧力変化は無視する。

---

### (1) 小球の落下速度をVとするとき、小球のまわりのレイノルズ数 Reはいくらか。

**方針・考え方・説明:**
レイノルズ数 (Reynolds number, Re) は、流体の慣性力と粘性力の比を表す無次元量です。流れの性質（層流か乱流かなど）を特徴づける重要な指標となります。
一般的に、レイノルズ数は $Re = \rho VL / \mu$ で与えられます。
ここで、
* $\rho$: 流体の密度
* $V$: 代表速度
* $L$: 代表長さ
* $\mu$: 流体の粘性係数

この問題の場合、
* 流体の密度は $\rho$
* 小球の落下速度が $V$ なので、代表速度は $V$
* 小球の代表長さは直径 $d$
* 流体の粘性係数は $\mu$

**計算過程:**
上記の定義に当てはめるだけです。

**要点・チップス:**
* レイノルズ数の定義は流体力学の基本中の基本です。
* 各物理量が何を指すのかを正確に理解しておくことが重要です。

**解答:**
小球のまわりのレイノルズ数 Re は、慣性力と粘性力の比を表す無次元量であり、以下の式で与えられます。
$$Re = \frac{\rho V d}{\mu}$$
ここで、
$\rho$: 流体の密度
$V$: 小球の落下速度（代表速度）
$d$: 小球の直径（代表長さ）
$\mu$: 流体の粘性係数

---

### (2) Re が1より小さいとき、流体から小球が受ける抵抗は $3\pi \mu Vd$ となる。この法則の名前を書け。

**方針・考え方・説明:**
レイノルズ数が非常に小さい（Re << 1）流れは、粘性力が支配的になる領域であり、「クリープ流れ」または「ストークス流れ」と呼ばれます。このような流れにおける球体に対する抵抗力は、ある特定の法則によって与えられます。

**要点・チップス:**
* 粘性が支配的な低レイノルズ数領域における抵抗力の法則は非常に有名です。
* 球体に対する抵抗力は、速度に比例するという特徴があります。

**解答:**
Re が1より小さい（Re << 1）非常に遅い流れにおいて、流体から小球が受ける抵抗が $3\pi \mu Vd$ となるこの法則は、**ストークスの法則 (Stokes' Law)** と呼ばれます。

---

### (3) 問 (2)の条件が満足されるとき、小球に働くすべての力の大きさと方向を求めよ。

**方針・考え方・説明:**
小球が流体中を落下する際に働く力は主に3つです。
1.  **重力 (Gravity):** 小球自体の質量によって生じる力。z軸の負方向（下向き）に働きます。
2.  **浮力 (Buoyancy):** 小球が押しのけた流体の重さに等しい力。z軸の正方向（上向き）に働きます。
3.  **流体抵抗 (Drag Force):** 流体が小球の運動を妨げる力。速度の向きと逆方向、つまりz軸の正方向（上向き）に働きます。問題文より、ストークスの法則が適用されるので、抵抗力は $3\pi \mu Vd$ です。

それぞれの力の大きさを計算し、方向を考慮してまとめます。

**計算過程:**
* **小球の体積:** 球の体積 $V_s = \frac{4}{3}\pi \left( \frac{d}{2} \right)^3 = \frac{1}{6}\pi d^3$
* **重力 $F_g$:**
    小球の質量 $m_s = \rho_s V_s = \rho_s \left( \frac{1}{6}\pi d^3 \right)$
    $F_g = m_s g = \rho_s \left( \frac{1}{6}\pi d^3 \right) g$
    方向: z軸の負方向

* **浮力 $F_b$:**
    小球が押しのけた流体の体積は小球の体積と同じ $V_s$ です。
    押しのけた流体の質量 $m_f = \rho V_s = \rho \left( \frac{1}{6}\pi d^3 \right)$
    $F_b = m_f g = \rho \left( \frac{1}{6}\pi d^3 \right) g$
    方向: z軸の正方向

* **流体抵抗 $F_d$:**
    問題文で与えられています。
    $F_d = 3\pi \mu Vd$
    方向: 小球の落下方向と逆方向、つまりz軸の正方向

**要点・チップス:**
* 物体にかかる力を正確に特定し、それぞれの力の定義を理解することが重要です。
* 浮力は「押しのけた流体の重さ」であることを忘れないでください。

**解答:**
問 (2)の条件（Re < 1）が満足されるとき、小球に働く力は以下の3つです。

1.  **重力 $F_g$**:
    * 小球の体積は $V_s = \frac{4}{3}\pi \left( \frac{d}{2} \right)^3 = \frac{1}{6}\pi d^3$ です。
    * 重力は小球の質量 $m_s = \rho_s V_s$ に重力加速度 $g$ を掛けたものです。
    * 大きさ: $F_g = \rho_s \left( \frac{1}{6}\pi d^3 \right) g$
    * 方向: z軸の負方向（下向き）

2.  **浮力 $F_b$**:
    * アルキメデスの原理により、小球が押しのけた流体の重さに等しいです。
    * 大きさ: $F_b = \rho \left( \frac{1}{6}\pi d^3 \right) g$
    * 方向: z軸の正方向（上向き）

3.  **流体抵抗 $F_d$ (ストークス抵抗)**:
    * 問 (2)で与えられたストークスの法則による抵抗力です。
    * 大きさ: $F_d = 3\pi \mu Vd$
    * 方向: 小球の運動方向（落下方向）と逆方向、つまりz軸の正方向（上向き）

### (4) 小球に働く力が問 (3)で与えられるとき、時刻 $t$ における小球のz方向の位置 $Z(t)$ が満たす微分方程式を求めよ。

**方針・考え方・説明:**
小球の運動は、ニュートンの運動方程式 $F = ma$ によって記述されます。
ここで、$F$ は小球に働く合力、$m$ は小球の質量、$a$ は小球の加速度です。
加速度 $a$ は、位置 $Z(t)$ の時間に関する2階微分 $d^2Z/dt^2$ であり、また速度 $V = dZ/dt$ の時間に関する1階微分 $dV/dt$ でもあります。
ここではz軸の負方向を正とすると計算が楽ですが、z軸の正方向を正とすると、重力は負の符号、浮力と抵抗は正の符号になります。問題文の図2ではz軸は上向きなので、z軸の正方向を上向きとします。したがって、落下速度 $V$ は $V = -dZ/dt$ となります。ただし、問題文では落下速度を $V$ としているため、$V$ を正の値として抵抗を $3\pi \mu Vd$ としていると解釈します。運動方程式を立てる際には、方向を明確にする必要があります。

ここでは、z軸の正の向きを上向きとし、下向きの力を負、上向きの力を正とします。
* 重力 $F_g = -\rho_s (\frac{1}{6}\pi d^3) g$
* 浮力 $F_b = +\rho (\frac{1}{6}\pi d^3) g$
* 流体抵抗 $F_d = +3\pi \mu Vd$ (ここで $V$ は速度の大きさであり、 $V = -dZ/dt$ とします。つまり $V$ は正の値です。抵抗は上向き。)

小球の質量は $m_s = \rho_s (\frac{1}{6}\pi d^3)$ です。
運動方程式は $m_s \frac{d^2Z}{dt^2} = \Sigma F$ です。

**計算過程:**
合力 $\Sigma F = F_b + F_d + F_g$
$\Sigma F = \rho \left( \frac{1}{6}\pi d^3 \right) g + 3\pi \mu Vd - \rho_s \left( \frac{1}{6}\pi d^3 \right) g$

ここで、$V = -dZ/dt$ です。$dV/dt = -d^2Z/dt^2$ です。
$\rho_s \left( \frac{1}{6}\pi d^3 \right) \frac{d^2Z}{dt^2} = \rho \left( \frac{1}{6}\pi d^3 \right) g + 3\pi \mu \left( -\frac{dZ}{dt} \right) d - \rho_s \left( \frac{1}{6}\pi d^3 \right) g$

整理すると、
$\rho_s \left( \frac{1}{6}\pi d^3 \right) \frac{d^2Z}{dt^2} + 3\pi \mu d \frac{dZ}{dt} = (\rho - \rho_s) \left( \frac{1}{6}\pi d^3 \right) g$

または、Zの負の方向（下方向）を正として加速度を $a = dV/dt$ とすると、
$m_s \frac{dV}{dt} = F_g - F_b - F_d$
$\rho_s \left( \frac{1}{6}\pi d^3 \right) \frac{dV}{dt} = \rho_s \left( \frac{1}{6}\pi d^3 \right) g - \rho \left( \frac{1}{6}\pi d^3 \right) g - 3\pi \mu Vd$
$\rho_s \left( \frac{1}{6}\pi d^3 \right) \frac{dV}{dt} = (\rho_s - \rho) \left( \frac{1}{6}\pi d^3 \right) g - 3\pi \mu Vd$
ここで $V = -dZ/dt$ (zは上向きが正) なので、
$\frac{dV}{dt} = -\frac{d^2Z}{dt^2}$
$\rho_s \left( \frac{1}{6}\pi d^3 \right) \left( -\frac{d^2Z}{dt^2} \right) = (\rho_s - \rho) \left( \frac{1}{6}\pi d^3 \right) g - 3\pi \mu \left( -\frac{dZ}{dt} \right) d$
$\rho_s \left( \frac{1}{6}\pi d^3 \right) \frac{d^2Z}{dt^2} = -(\rho_s - \rho) \left( \frac{1}{6}\pi d^3 \right) g - 3\pi \mu d \frac{dZ}{dt}$
$\rho_s \left( \frac{1}{6}\pi d^3 \right) \frac{d^2Z}{dt^2} + 3\pi \mu d \frac{dZ}{dt} = -(\rho_s - \rho) \left( \frac{1}{6}\pi d^3 \right) g$
$\rho_s \left( \frac{1}{6}\pi d^3 \right) \frac{d^2Z}{dt^2} + 3\pi \mu d \frac{dZ}{dt} = (\rho - \rho_s) \left( \frac{1}{6}\pi d^3 \right) g$
これは最初の式と同じになります。

**要点・チップス:**
* ニュートンの運動方程式を適用する際は、力の方向を正しく設定することが重要です。
* 重力加速度 $g$ の向き（z軸の負方向）に注意しましょう。
* 加速度が速度の微分、速度が位置の微分であることを忘れずに。

**解答:**
小球の運動方程式は、ニュートンの第2法則 $m_s a = \Sigma F$ に基づきます。
ここで、小球の質量 $m_s = \rho_s \left( \frac{1}{6}\pi d^3 \right)$ です。
z軸の正方向を上向きとします。小球の落下速度 $V$ は正の値として与えられているため、$V = -dZ/dt$ となります。
小球に働く力の合力 $\Sigma F$ は、問 (3)で求めた力の和です。
$$\Sigma F = F_b + F_d + F_g$$$$ \Sigma F = \rho \left( \frac{1}{6}\pi d^3 \right) g + 3\pi \mu Vd - \rho_s \left( \frac{1}{6}\pi d^3 \right) g$$
$$\Sigma F = (\rho - \rho_s) \left( \frac{1}{6}\pi d^3 \right) g + 3\pi \mu Vd$$
運動方程式は $m_s \frac{dV}{dt} = \Sigma F_z$ (z方向の速度に関する運動方程式) または $m_s \frac{d^2Z}{dt^2} = \Sigma F_z$ (z方向の位置に関する運動方程式) です。
速度 $V$ を $Z$ で表すと $V = -dZ/dt$ なので、$dV/dt = -d^2Z/dt^2$ となります。

よって、
$$\rho_s \left( \frac{1}{6}\pi d^3 \right) \left( -\frac{d^2Z}{dt^2} \right) = (\rho - \rho_s) \left( \frac{1}{6}\pi d^3 \right) g + 3\pi \mu d \left( -\frac{dZ}{dt} \right)$$
両辺を $-\rho_s (\frac{1}{6}\pi d^3)$ で割るか、項を整理すると、
$$\rho_s \left( \frac{1}{6}\pi d^3 \right) \frac{d^2Z}{dt^2} + 3\pi \mu d \frac{dZ}{dt} = -(\rho - \rho_s) \left( \frac{1}{6}\pi d^3 \right) g$$
$$\rho_s \left( \frac{1}{6}\pi d^3 \right) \frac{d^2Z}{dt^2} + 3\pi \mu d \frac{dZ}{dt} = (\rho_s - \rho) \left( \frac{1}{6}\pi d^3 \right) g$$
これが時刻 $t$ における小球のz方向の位置 $Z(t)$ が満たす微分方程式です。

### (5) 小球が時刻 $t=0$ で原点 $(0,0)$ から静かに落下を始めたとき、いくら時間が経過しても問(2)の法則が成り立つための条件を求め、最終的に接近するz軸の負方向速度の大きさを求めよ。また、この速度の名前を書け。

**方針・考え方・説明:**
* **問(2)の法則が成り立つ条件:** これは、Re < 1、すなわち $\rho Vd/\mu < 1$ であることを意味します。この条件が「いくら時間が経過しても」成り立つということは、小球が最終的に到達する定常速度（終端速度）においてもこの条件が満たされる必要があるということです。

* **最終的に接近するz軸の負方向速度（終端速度）:** 落下が十分に長く続くと、小球にかかる合力がゼロになり、加速度がゼロになります。このとき、小球は一定の速度で落下するようになります。この速度を終端速度（Terminal velocity）と呼びます。運動方程式において $dV/dt = 0$ または $d^2Z/dt^2 = 0$ と置いて計算します。

**計算過程:**
1.  **終端速度 $V_t$ を求める:**
    小球の加速度がゼロになるとき、$dV/dt = 0$ です。
    (4)で求めた運動方程式を使用します。z軸の負方向を速度の正の方向として $V$ を定義すると、
    $\rho_s \left( \frac{1}{6}\pi d^3 \right) \frac{dV}{dt} = (\rho_s - \rho) \left( \frac{1}{6}\pi d^3 \right) g - 3\pi \mu Vd$
    $dV/dt = 0$ のとき、速度を $V_t$ とすると、
    $0 = (\rho_s - \rho) \left( \frac{1}{6}\pi d^3 \right) g - 3\pi \mu V_t d$
    $3\pi \mu V_t d = (\rho_s - \rho) \left( \frac{1}{6}\pi d^3 \right) g$
    $V_t = \frac{(\rho_s - \rho) \left( \frac{1}{6}\pi d^3 \right) g}{3\pi \mu d}$
    $V_t = \frac{(\rho_s - \rho) d^2 g}{18 \mu}$

2.  **問(2)の法則が成り立つための条件:**
    ストークスの法則が成り立つ条件は Re < 1 です。
    $Re = \frac{\rho V d}{\mu} < 1$
    いくら時間が経過しても法則が成り立つということは、最終的な速度である終端速度 $V_t$ においても Re < 1 でなければなりません。
    $\frac{\rho V_t d}{\mu} < 1$
    $V_t$ の式を代入します。
    $\frac{\rho \left( \frac{(\rho_s - \rho) d^2 g}{18 \mu} \right) d}{\mu} < 1$
    $\frac{\rho (\rho_s - \rho) d^3 g}{18 \mu^2} < 1$
    これが条件です。

3.  **速度の名前:**
    終端速度 (Terminal velocity)

**要点・チップス:**
* 「いくら時間が経過しても」というフレーズは、終端速度（定常状態）での条件を指します。
* 運動方程式で加速度がゼロになる状態が終端速度です。
* 終端速度を求めてから、その速度でレイノルズ数 Re < 1 の条件を満たすかを確認します。

**解答:**
1.  **最終的に接近するz軸の負方向速度（終端速度 $V_t$）の大きさ:**
    小球が十分に長い時間落下すると、加速度がゼロになり、重力、浮力、流体抵抗の3つの力が釣り合い、一定の速度で落下するようになります。この速度を終端速度と呼びます。
    運動方程式において $d^2Z/dt^2 = 0$ (または $dV/dt = 0$) と置くと、
    $$0 = (\rho_s - \rho) \left( \frac{1}{6}\pi d^3 \right) g - 3\pi \mu V_t d$$
    これを $V_t$ について解きます。
    $$3\pi \mu V_t d = (\rho_s - \rho) \left( \frac{1}{6}\pi d^3 \right) g$$
    $$V_t = \frac{(\rho_s - \rho) \frac{1}{6}\pi d^3 g}{3\pi \mu d}$$
    $$V_t = \frac{(\rho_s - \rho) d^2 g}{18 \mu}$$
    この速度の大きさは $\frac{(\rho_s - \rho) d^2 g}{18 \mu}$ です。

2.  **この速度の名前:**
    この速度は**終端速度 (Terminal velocity)** と呼ばれます。

3.  **いくら時間が経過しても問(2)の法則が成り立つための条件:**
    問 (2)の法則（ストークスの法則）が成り立つ条件は、レイノルズ数 Re が1より小さい、すなわち $Re < 1$ です。
    この条件が「いくら時間が経過しても」成り立つためには、最終的に達する終端速度 $V_t$ においても Re < 1 である必要があります。
    $$Re = \frac{\rho V_t d}{\mu} < 1$$
    ここに求めた $V_t$ の式を代入します。
    $$\frac{\rho \left( \frac{(\rho_s - \rho) d^2 g}{18 \mu} \right) d}{\mu} < 1$$
    $$\frac{\rho (\rho_s - \rho) d^3 g}{18 \mu^2} < 1$$
    これが、いくら時間が経過しても問(2)の法則が成り立つための条件です。

### (6) 問 (2)の条件が成り立つとき、流体からの抵抗の3分の1は小球の落下方向前後。の圧力差 $\Delta p$ と関係している。この圧力は前(z軸の負方向)と後(z軸の正方向)のどちらが大きいか。また、$\Delta p$ がどの程度の大きさであるか、おおよその値を求めよ。なお、重力に起因する圧力変化は無視する。

**方針・考え方・説明:**
流体抵抗は、主に2つの成分から構成されます。
1.  **圧力抵抗 (Pressure Drag / Form Drag):** 物体の前後における圧力差によって生じる抵抗。
2.  **摩擦抵抗 (Friction Drag / Skin Friction Drag):** 物体表面の粘性摩擦によって生じる抵抗。

ストークスの法則が適用されるRe < 1 の極めて遅い流れでは、通常、粘性力が支配的であり、抵抗の大部分は摩擦抵抗によるものと考えられます。しかし、この問題では「抵抗の3分の1は圧力差と関係している」と具体的に指定されています。

* **圧力の大小:** 物体が流体中を移動する際、進行方向の前面では流体がせき止められ圧力が上昇し、後面では流体が剥離したりして圧力が低下する傾向があります。抵抗は、この圧力差によって物体を後方に押し戻す力として作用します。したがって、進行方向の圧力が高く、その逆方向の圧力が低い場合に抵抗が生じます。

* **$\Delta p$ のおおよその値:**
    問題文の記述から、全抵抗 $F_d = 3\pi \mu Vd$ の3分の1が圧力抵抗に相当します。
    圧力抵抗 $F_p = \frac{1}{3} F_d = \frac{1}{3} (3\pi \mu Vd) = \pi \mu Vd$
    この圧力抵抗は、圧力差 $\Delta p$ と物体の断面積 $A$ の積としておおよそ表されます。
    $F_p = \Delta p \times A$
    球の進行方向に対する断面積は円の面積 $A = \pi (d/2)^2 = \frac{1}{4}\pi d^2$ です。
    したがって、$\Delta p \approx F_p / A$ と計算できます。

**計算過程:**
1.  **圧力の大小:**
    小球はz軸の負方向（下）に落下しています。したがって、小球の前面（落下方向、z軸の負方向）では流体がせき止められて圧力が上昇し、後面（z軸の正方向）では圧力が低下します。抵抗は、この圧力差によって生じるので、前面の圧力の方が後面の圧力よりも大きくなります。

2.  **$\Delta p$ のおおよその値:**
    圧力抵抗は全抵抗の3分の1なので、
    $F_p = \frac{1}{3} (3\pi \mu Vd) = \pi \mu Vd$
    この圧力抵抗は、小球の断面積 $A$ と圧力差 $\Delta p$ の積に等しいと近似できます。
    小球の断面積 $A = \pi (d/2)^2 = \frac{1}{4}\pi d^2$
    よって、$F_p = \Delta p \times A$ より、
    $\Delta p = \frac{F_p}{A} = \frac{\pi \mu Vd}{\frac{1}{4}\pi d^2} = \frac{4\mu V}{d}$

**要点・チップス:**
* 抵抗力は圧力抵抗と摩擦抵抗に分けられることを理解しておく。
* 前方と後方の圧力差が抵抗を生むメカニズムを理解する。
* 力 = 圧力 × 面積 という基本的な関係を使う。

**解答:**

1.  **圧力の大小:**
    小球はz軸の負方向（下方向）に落下しています。流体が物体に衝突する前面（落下方向、z軸の負方向）では流体の運動エネルギーが圧力エネルギーに変換されるため圧力が上昇します。一方、物体の後面（z軸の正方向）では流体が物体から剥がれるため圧力が低下します。この圧力差が抵抗力の一部として働くため、**z軸の負方向（前）の圧力の方がz軸の正方向（後）の圧力よりも大きい**です。

2.  **$\Delta p$ のおおよその値:**
    問題の記述より、流体からの抵抗 $F_d = 3\pi \mu Vd$ の3分の1が圧力差 $\Delta p$ と関係しています。
    したがって、圧力抵抗 $F_p$ は、
    $$F_p = \frac{1}{3} F_d = \frac{1}{3} (3\pi \mu Vd) = \pi \mu Vd$$
    この圧力抵抗は、小球の進行方向に対する断面積 $A$ と圧力差 $\Delta p$ の積としておおよそ表すことができます。
    小球の断面積 $A = \pi \left( \frac{d}{2} \right)^2 = \frac{1}{4}\pi d^2$
    よって、
    $$\Delta p \approx \frac{F_p}{A} = \frac{\pi \mu Vd}{\frac{1}{4}\pi d^2}$$
    $$\Delta p \approx \frac{4\mu V}{d}$$
    したがって、おおよその圧力差 $\Delta p$ の大きさは $\frac{4\mu V}{d}$ です。

### 解答

#### **問題1: 不定積分の計算**

##### (1) \[ \int \frac{dx}{x(\log x)^4} \]

**方針:**  
置換積分を用いる。\( u = \log x \) とおくと、\( du = \frac{1}{x} dx \) となる。  
**解答:**  
\[
u = \log x, \quad du = \frac{dx}{x}
\]
\[
\int \frac{dx}{x(\log x)^4} = \int u^{-4}  du = \frac{u^{-3}}{-3} + C = -\frac{1}{3(\log x)^3} + C
\]

**答え:** \(\boxed{-\dfrac{1}{3(\log x)^{3}} + C\)

##### (2) \[ \int \frac{dx}{(e^x + 1)^2} \]

**方針:**  
\( t = e^x \) と置換し、部分分数分解を行う。  
**解答:**  
\[
t = e^x, \quad dt = e^x dx, \quad dx = \frac{dt}{t}
\]
\[
\int \frac{dx}{(e^x + 1)^2} = \int \frac{1}{(t + 1)^2} \cdot \frac{dt}{t} = \int \frac{dt}{t(t + 1)^2}
\]
部分分数分解:
\[
\frac{1}{t(t + 1)^2} = \frac{A}{t} + \frac{B}{t + 1} + \frac{C}{(t + 1)^2}
\]
両辺に \( t(t + 1)^2 \) を乗じ:
\[
1 = A(t + 1)^2 + B t (t + 1) + C t
\]
- \( t = 0 \) のとき: \( 1 = A(1)^2 \) → \( A = 1 \)
- \( t = -1 \) のとき: \( 1 = C(-1) \) → \( C = -1 \)
- \( t^2 \) の係数比較: \( A + B = 0 \) → \( B = -1 \)
\[
\int \left( \frac{1}{t} - \frac{1}{t + 1} - \frac{1}{(t + 1)^2} \right) dt = \ln |t| - \ln |t + 1| + \frac{1}{t + 1} + C
\]
\( t = e^x \) を代入:
\[
= \ln \left| \frac{e^x}{e^x + 1} \right| + \frac{1}{e^x + 1} + C = x - \ln (e^x + 1) + \frac{1}{e^x + 1} + C
\]

**答え:** \(\boxed{x - \ln (e^{x} + 1) + \dfrac{1}{e^{x} + 1} + C}\)

#### **問題2: 微分方程式**

##### (1) 完全微分形でないことの証明と積分因子の導出
**方針:**  
完全微分形の条件をチェックし、積分因子を求める。  
**解答:**  
\[
M = x^2 + y^2 + x, \quad N = xy
\]
\[
\frac{\partial M}{\partial y} = 2y, \quad \frac{\partial N}{\partial x} = y
\]
\[
\frac{\partial M}{\partial y} \neq \frac{\partial N}{\partial x} \quad (\because 2y \neq y \text{ for } y \neq 0)
\]
積分因子 \(\mu = \mu(x)\) を仮定:
\[
\frac{\partial}{\partial y}(\mu M) = \frac{\partial}{\partial x}(\mu N)
\]
\[
\mu \cdot 2y = y \mu + xy \frac{d\mu}{dx} \implies y \mu = xy \frac{d\mu}{dx}
\]
\[
\frac{d\mu}{\mu} = \frac{dx}{x} \implies \ln |\mu| = \ln |x| + C \implies \mu = x
\]

**答え:**  
完全微分形でない（\(\because \partial M/\partial y \neq \partial N/\partial x\)）。  
積分因子: \(\boxed{\mu = x}\)

##### (2) 一般解の導出
**方針:**  
積分因子 \(\mu = x\) を乗じ、完全微分方程式を解く。  
**解答:**  
\[
x(x^2 + y^2 + x)dx + x \cdot xy  dy = 0 \implies (x^3 + x y^2 + x^2)dx + x^2 y  dy = 0
\]
\[
M = x^3 + x y^2 + x^2, \quad N = x^2 y
\]
\[
\frac{\partial M}{\partial y} = 2x y = \frac{\partial N}{\partial x} \quad (\text{完全微分形})
\]
解 \(F(x,y) = C\):
\[
\frac{\partial F}{\partial y} = x^2 y \implies F = \frac{1}{2} x^2 y^2 + g(x)
\]
\[
\frac{\partial F}{\partial x} = x y^2 + g'(x) = x^3 + x y^2 + x^2
\]
\[
g'(x) = x^3 + x^2 \implies g(x) = \frac{1}{4} x^4 + \frac{1}{3} x^3
\]
\[
F = \frac{1}{2} x^2 y^2 + \frac{1}{4} x^4 + \frac{1}{3} x^3 = C
\]

**答え:** \(\boxed{\dfrac{1}{2} x^{2} y^{2} + \dfrac{1}{4} x^{4} + \dfrac{1}{3} x^{3} = C}\)

#### **問題3: ベクトル解析**

##### (1) 平行四辺形の面積公式の証明
**方針:**  
ベクトル外積のノルムとラグランジュの恒等式を用いる。  
**解答:**  
\[
S = \| \mathbf{A} \times \mathbf{B} \| = \sqrt{ (A_y B_z - A_z B_y)^2 + (A_z B_x - A_x B_z)^2 + (A_x B_y - A_y B_x)^2 }
\]
\[
S^2 = (A_y B_z - A_z B_y)^2 + (A_z B_x - A_x B_z)^2 + (A_x B_y - A_y B_x)^2
\]
一方、
\[
|\mathbf{A}|^2 |\mathbf{B}|^2 - (\mathbf{A} \cdot \mathbf{B})^2 = (A_x^2 + A_y^2 + A_z^2)(B_x^2 + B_y^2 + B_z^2) - (A_x B_x + A_y B_y + A_z B_z)^2
\]
展開すると \(S^2\) に等しい（ラグランジュの恒等式）。  
問題文の行列式表現は誤りで、正しくは:
\[
S^2 = \begin{vmatrix} A_y & A_z \\ B_y & B_z \end{vmatrix}^2 + \begin{vmatrix} A_z & A_x \\ B_z & B_x \end{vmatrix}^2 + \begin{vmatrix} A_x & A_y \\ B_x & B_y \end{vmatrix}^2
\]

**証明完了。**

##### (2) 球面の面積分
**前提:** 球面 \(S: |\mathbf{r}| = a\), 外向き単位法線ベクトル \(\mathbf{n} = \frac{\mathbf{r}}{a}\)。  

**(a)** \[ \iint_S \frac{\mathbf{r} \cdot \mathbf{n}}{r^3} dS \]
**解答:**  
\[
\mathbf{r} \cdot \mathbf{n} = \mathbf{r} \cdot \frac{\mathbf{r}}{a} = \frac{r^2}{a} = a \quad (\because r = a)
\]
\[
\frac{\mathbf{r} \cdot \mathbf{n}}{r^3} = \frac{a}{a^3} = \frac{1}{a^2}
\]
\[
\iint_S \frac{1}{a^2}  dS = \frac{1}{a^2} \times 4\pi a^2 = 4\pi
\]

**答え:** \(\boxed{4\pi}\)

**(b)** \[ \iint_S (x\mathbf{i} + y\mathbf{j}) \cdot \mathbf{n}  dS \]
**解答:**  
発散定理を適用:
\[
\mathbf{F} = x\mathbf{i} + y\mathbf{j}, \quad \nabla \cdot \mathbf{F} = 1 + 1 = 2
\]
\[
\iint_S \mathbf{F} \cdot \mathbf{n}  dS = \iiint_V (\nabla \cdot \mathbf{F})  dV = \iiint_V 2  dV = 2 \times \frac{4}{3}\pi a^3 = \frac{8}{3}\pi a^3
\]

**答え:** \(\boxed{\dfrac{8}{3}\pi a^{3}}\)

#### **問題4: ガウス分布のフーリエ変換**
**方針:**  
フーリエ変換の定義とガウス積分を利用。  
**解答:**  
\[
f(x) = \frac{1}{\sqrt{2\pi b}} \exp\left(-\frac{(x-a)^2}{2b}\right)
\]
フーリエ変換:
\[
\hat{f}(k) = \int_{-\infty}^{\infty} f(x) e^{-ikx}  dx = \frac{1}{\sqrt{2\pi b}} \int_{-\infty}^{\infty} \exp\left(-\frac{(x-a)^2}{2b} - ikx\right) dx
\]
指数部を平方完成:
\[
-\frac{(x-a)^2}{2b} - ikx = -\frac{1}{2b} \left[ x^2 - 2(a - ibk)x + a^2 \right]
\]
\[
= -\frac{1}{2b} \left[ (x - (a - ibk))^2 - (a - ibk)^2 + a^2 \right]
\]
\[
= -\frac{(x - (a - ibk))^2}{2b} - \frac{ -2iabk + b^2 k^2 }{2b} = -\frac{(x - (a - ibk))^2}{2b} - iak - \frac{b k^2}{2}
\]
積分:
\[
\hat{f}(k) = \frac{1}{\sqrt{2\pi b}} \exp\left(-iak - \frac{b k^2}{2}\right) \int_{-\infty}^{\infty} \exp\left(-\frac{(x - (a - ibk))^2}{2b}\right) dx
\]
積分変数を \( u = x - (a - ibk) \) とし、ガウス積分公式を適用:
\[
\int_{-\infty}^{\infty} \exp\left(-\frac{u^2}{2b}\right) du = \sqrt{2\pi b}
\]
\[
\hat{f}(k) = \frac{1}{\sqrt{2\pi b}} \exp\left(-iak - \frac{b k^2}{2}\right) \times \sqrt{2\pi b} = \exp\left(-iak - \frac{b k^2}{2}\right)
\]

**答え:** \(\boxed{\exp\left(-i a k - \dfrac{b k^{2}}{2}\right)}\)

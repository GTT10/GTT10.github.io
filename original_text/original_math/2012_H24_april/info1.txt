# サイクロイドの二階導関数（直接計算法）

## なぜ半角公式を使わないのか？

**理由**：
1. **計算が複雑になる** - 不必要な変形で混乱を招く
2. **直接的な方法の方が確実** - ミスが少なく、理解しやすい  
3. **大学院入試では効率性が重要** - 時間短縮と正確性の両立

## 直接計算による解法

### パラメータ表示
$$x = a(\theta - \sin \theta), \quad y = a(1 - \cos \theta)$$

### 一階導関数
$$\frac{dx}{d\theta} = a(1 - \cos \theta), \quad \frac{dy}{d\theta} = a \sin \theta$$

$$\frac{dy}{dx} = \frac{dy/d\theta}{dx/d\theta} = \frac{a \sin \theta}{a(1 - \cos \theta)} = \frac{\sin \theta}{1 - \cos \theta}$$

### 二階導関数の計算

**Step 1: 一階導関数を直接微分**

$$\frac{d}{d\theta}\left(\frac{\sin \theta}{1 - \cos \theta}\right)$$

**商の微分法**を適用：
$$\frac{d}{d\theta}\left(\frac{f(\theta)}{g(\theta)}\right) = \frac{f'(\theta)g(\theta) - f(\theta)g'(\theta)}{[g(\theta)]^2}$$

ここで：
- $$f(\theta) = \sin \theta \Rightarrow f'(\theta) = \cos \theta$$
- $$g(\theta) = 1 - \cos \theta \Rightarrow g'(\theta) = \sin \theta$$

**Step 2: 分子の計算**
$$\cos \theta(1 - \cos \theta) - \sin \theta \cdot \sin \theta$$
$$= \cos \theta - \cos^2 \theta - \sin^2 \theta$$
$$= \cos \theta - (\cos^2 \theta + \sin^2 \theta)$$
$$= \cos \theta - 1$$

**Step 3: 結果**
$$\frac{d}{d\theta}\left(\frac{\sin \theta}{1 - \cos \theta}\right) = \frac{\cos \theta - 1}{(1 - \cos \theta)^2} = \frac{-(1 - \cos \theta)}{(1 - \cos \theta)^2} = \frac{-1}{1 - \cos \theta}$$

**Step 4: 二階導関数**
$$\frac{d^2y}{dx^2} = \frac{\frac{d}{d\theta}\left(\frac{dy}{dx}\right)}{\frac{dx}{d\theta}} = \frac{-\frac{1}{1 - \cos \theta}}{a(1 - \cos \theta)}$$

$$= \frac{-1}{a(1 - \cos \theta)^2}$$

## 最終解答

$$\boxed{\frac{dy}{dx} = \frac{\sin \theta}{1 - \cos \theta}, \quad \frac{d^2y}{dx^2} = -\frac{1}{a(1 - \cos \theta)^2}}$$


### 【2】  
#### (1) 完全微分形でないことの証明と積分因子  
**微分方程式**:  
\[
\frac{dy}{dx} + \frac{y}{x} = e^x \iff \left( \frac{y}{x} - e^x \right) dx + dy = 0
\]  
- \( M = \frac{y}{x} - e^x \), \( N = 1 \)  
- 偏導関数:  
  \[
  \frac{\partial M}{\partial y} = \frac{1}{x}, \quad \frac{\partial N}{\partial x} = 0
  \]  
- \( \frac{1}{x} \neq 0 \) より完全微分形でない。  

**積分因子** \(\mu(x)\) を求める:  
\[
\frac{\partial}{\partial y} (\mu M) = \frac{\partial}{\partial x} (\mu N) \implies \mu \cdot \frac{1}{x} = \frac{d\mu}{dx}
\]  
変数分離法で解く:  
\[
\frac{d\mu}{\mu} = \frac{dx}{x} \implies \ln |\mu| = \ln |x| + C \implies \mu = x \quad (C=0)
\]  

**最終解答**:  
\[
\boxed{\text{完全微分形でない}, \quad \boxed{\text{積分因子 } \mu = x}
\]  

#### (2) 一般解  
積分因子 \(x\) を乗じた方程式:  
\[
x \left( \frac{y}{x} - e^x \right) dx + x  dy = 0 \implies (y - x e^x) dx + x  dy = 0
\]  
ポテンシャル関数 \(\psi(x,y)\):  
\[
\frac{\partial \psi}{\partial y} = x \implies \psi = x y + h(x)
\]  
\[
\frac{\partial \psi}{\partial x} = y + h'(x) = y - x e^x \implies h'(x) = -x e^x
\]  
積分:  
\[
h(x) = \int -x e^x  dx = -x e^x + e^x + C
\]  
一般解:  
\[
\psi = x y - x e^x + e^x = C \implies x y + e^x (1 - x) = C
\]  

**最終解答**:  
\[
\boxed{x y + e^x (1 - x) = C}
\]

### 【3】 ベクトルの線形従属性

\[
A = \begin{bmatrix}
1 & 3 & 5 \\
2 & 1 & 0 \\
3 & 2 & k
\end{bmatrix}, \quad \det A = 1 \cdot (k) - 3 \cdot (2k) + 5 \cdot (4 - 3) = -5k + 5 = 0 \implies k = 1
\]  

**最終解答**:  
\[
\boxed{k = 1}
\]

\]  
連立方程式:  
\[
\begin{cases}
c_1 + 3c_2 = 5 \\
2c_1 + c_2 = 0 \\
3c_1 + 2c_2 = 1
\end{cases}
\implies c_1 = -1, \quad c_2 = 2
\]  
**最終解答**:  
\[
\boxed{x_3 = -x_1 + 2x_2}
\]

\[
\begin{cases}
a + 2b + 3c = 0 \\
3a + b + 2c = 0 \\
5a + c = 0
\end{cases}
\implies a = t, \quad b = 7t, \quad c = -5t
\]  
方程式:  
\[
t x + 7t y - 5t z = 0 \implies x + 7y - 5z = 0
\]  
**最終解答**:  
\[
\boxed{x + 7y - 5z = 0}
\]

### 【4】 微分方程式とラプラス変換  

**問題文**
$$
y' + y = f(t)
$$
`f(t)` は、区間 `0 ≤ t` で定義され、図1で表される関数とする。次の問いに答えよ。

（図1は、原点`(0,0)`と点`(a,1)`を結ぶ線分と、`t ≥ a` で値が0となる半直線からなる関数 `f(t)` を示すグラフ）

#### (1) `f(t)` を、単位ステップ関数 `u(t)` を用いて表せ。なお `u(t)` は、次式で定義される。
$$
u(t) = \begin{cases}
0 & (t \le 0) \\
1 & (t > 0)
\end{cases}
$$

**方針・考え方**
1.  まず、図1のグラフから `f(t)` を区間ごとに数式で定義します。
2.  次に、特定の区間でのみ値を持つ関数を表現するために、単位ステップ関数 `u(t)` を組み合わせた「窓関数」 `u(t) - u(t-a)` を利用します。

**途中式**
1.  図1のグラフより、`f(t)` は原点と点`(a, 1)` を通る直線であるため、`0 ≤ t < a` の区間での傾きは `1/a` となります。したがって、`f(t)` は以下のように区分的に定義できます。
    $$
    f(t) = \begin{cases}
    \frac{1}{a}t & (0 \le t < a) \\
    0 & (t \ge a)
    \end{cases}
    $$

2.  この関数は、ランプ関数 `g(t) = (1/a)t` が区間 `[0, a)` でのみ「オン」になるものと解釈できます。このような「窓」は単位ステップ関数を用いて `u(t) - u(t-a)` と表現できます。
    これを `g(t)` に掛けることで `f(t)` を一つの式で表現します。
    $$
    f(t) = \left(\frac{1}{a}t\right) \cdot \{u(t) - u(t-a)\}
    $$

**最終解答**
$$
\boxed{ f(t) = \frac{1}{a}t \{u(t) - u(t-a)\} }
$$

#### (2) `f(t)` をラプラス変換せよ。

**方針・考え方**
(1)で求めた `f(t)` の式を展開し、ラプラス変換の線形性を利用して各項を変換します。基本的な変換公式 `L[t] = 1/s^2` と、**時間ずらしの公式** `L[g(t)u(t-a)] = e^{-as}L[g(t+a)]` を適用します。

**途中式**
1.  (1)の `f(t)` を展開します。
    $$
    f(t) = \frac{1}{a}t u(t) - \frac{1}{a}t u(t-a)
    $$

2.  ラプラス変換の線形性より、
    $$
    F(s) = L[f(t)] = \frac{1}{a}L[t u(t)] - \frac{1}{a}L[t u(t-a)]
    $$

3.  各項を計算します。
    -   第1項：
        $$
        \frac{1}{a}L[t u(t)] = \frac{1}{a} \cdot \frac{1}{s^2} = \frac{1}{as^2}
        $$
    -   第2項（時間ずらしの公式を適用）：
        ここで `g(t) = t` とすると `g(t+a) = t+a`。
        $$
        \frac{1}{a}L[t u(t-a)] = \frac{1}{a} e^{-as} L[t+a] = \frac{1}{a} e^{-as} (L[t] + L[a]) = \frac{1}{a} e^{-as} \left(\frac{1}{s^2} + \frac{a}{s}\right) = e^{-as}\left(\frac{1}{as^2} + \frac{1}{s}\right)
        $$

4.  結果を統合します。
    $$
    F(s) = \frac{1}{as^2} - e^{-as}\left(\frac{1}{as^2} + \frac{1}{s}\right) = \frac{1 - e^{-as} - ase^{-as}}{as^2}
    $$

**最終解答**
$$
\boxed{ F(s) = \frac{1 - e^{-as} - ase^{-as}}{as^2} }
$$

#### (3) 微分方程式をラプラス変換し、微分方程式の解 `y(t)` を求めよ。ただし `y(0) = 0` とする。

**方針・考え方**
1.  与えられた微分方程式 `y' + y = f(t)` の両辺をラプラス変換します。初期条件 `y(0) = 0` を用いて、`y(t)` のラプラス変換 `Y(s)` について解きます。
2.  得られた `Y(s)` の式を、逆ラプラス変換が容易になるように部分分数分解します。
3.  逆ラプラス変換を実行して `y(t)` を求めます。解は `f(t)` の定義域に従い、`0 ≤ t < a` と `t ≥ a` の区間で場合分けして示します。

**途中式**
1.  微分方程式をラプラス変換します。
    $$
    L[y'] + L[y] = L[f(t)]
    $$
    $$
    (sY(s) - y(0)) + Y(s) = F(s)
    $$
    初期条件 `y(0) = 0` と(2)の結果を代入し、`Y(s)` について解きます。
    $$
    (s+1)Y(s) = F(s) \implies Y(s) = \frac{F(s)}{s+1} = \frac{1}{s+1} \cdot \frac{1 - e^{-as}(1+as)}{as^2}
    $$

2.  `Y(s)` を逆変換しやすいように整理します。
    $$
    Y(s) = \frac{1}{as^2(s+1)} - e^{-as}\frac{1+as}{as^2(s+1)}
    $$
    ここで、各項を部分分数分解します。
    -   `\frac{1}{s^2(s+1)} = \frac{1}{s^2} - \frac{1}{s} + \frac{1}{s+1}`
    -   `\frac{1+as}{as^2(s+1)} = \frac{1}{as^2} + \frac{a-1}{as} + \frac{1-a}{a(s+1)}`
    
    よって、`Y(s)` は次のように書き換えられます。
    $$
    Y(s) = \frac{1}{a}\left(\frac{1}{s^2} - \frac{1}{s} + \frac{1}{s+1}\right) - e^{-as}\left(\frac{1}{as^2} + \frac{a-1}{as} + \frac{1-a}{a(s+1)}\right)
    $$

3.  逆ラプラス変換を実行します。
    -   `g(t) = L^{-1}\left[\frac{1}{a}\left(\frac{1}{s^2} - \frac{1}{s} + \frac{1}{s+1}\right)\right] = \frac{1}{a}(t - 1 + e^{-t})`
    -   `h(t) = L^{-1}\left[\frac{1}{as^2} + \frac{a-1}{as} + \frac{1-a}{a(s+1)}\right] = \frac{1}{a}(t + a-1 + (1-a)e^{-t})`
    
    `y(t)` は `y(t) = g(t) - u(t-a)h(t-a)` となります。これを区間ごとに記述します。

4.  **`0 ≤ t < a` の場合：**
    $$
    y(t) = g(t) = \frac{1}{a}(t - 1 + e^{-t})
    $$

5.  **`t ≥ a` の場合：**
    $$
    y(t) = g(t) - h(t-a)
    $$
    $$
    y(t) = \frac{1}{a}(t - 1 + e^{-t}) - \frac{1}{a}\{(t-a) + (a-1) + (1-a)e^{-(t-a)}\}
    $$
    $$
    y(t) = \frac{1}{a} [ (t - 1 + e^{-t}) - (t - 1 + (1-a)e^a e^{-t}) ]
    $$
    $$
    y(t) = \frac{1}{a} [ e^{-t} - (1-a)e^a e^{-t} ] = \frac{e^{-t}}{a}(1 - e^a + ae^a)
    $$

**最終解答**
$$
\boxed{ y(t) = \begin{cases} \frac{1}{a}(t - 1 + e^{-t}) & (0 \le t < a) \\ \frac{e^{-t}}{a}(1 - e^a + ae^a) & (t \ge a) \end{cases} }
$$

問題概要

2016年度の材料力学入試では、大問【1】で両端固定段付き丸棒の軸力問題（不静定構造の極限解析）、大問【2】で両端突き出しはりの曲げ問題（対称構造のたわみ最適化）、大問【3】で傾斜荷重を受ける片持ちはりの問題（2軸曲げの重ね合わせ）が出題されました。理論と実用設計を組み合わせた総合的な年度です。

大問【1】

両端固定された段付き丸棒の中間点に集中荷重が作用する不静定構造について、反力、変位、極限条件を解析する。

与えられた条件:

  • 棒AB：長さa、断面積S₁ = πd²/4、ヤング率E₁
  • 棒BD：長さb+c、断面積S₂ = πd²/4、ヤング率E₂
  • 境界条件：両端A、Dで完全固定
  • 荷重：B点に集中荷重P（右向き）
考え方のヒント

両端固定の不静定問題では、全体の変形がゼロ（δAD = 0）という条件を使います。段付き構造では材料特性の違いも考慮して解析します。

途中式と解答

(1) 反力RAとRD

力の釣り合いと変形の適合条件を連立して反力を求めます。

力の釣り合い：

$$R_A + R_D = P$$

変形の適合条件：

全体の変形がゼロ：
$$\delta_{AB} + \delta_{BD} = 0$$ $$-\frac{R_A \cdot a}{S_1 E_1} + \frac{R_D \cdot (b+c)}{S_2 E_2} = 0$$

S₁ = S₂ = πd²/4の条件を用いて連立すると：

解答：

$$R_A = \frac{4P(b+c)E_1}{aE_2 + 4(b+c)E_1}$$ $$R_D = \frac{PaE_2}{aE_2 + 4(b+c)E_1}$$

(2) 点Cの変位δc

C点の変位をD点基準で計算します。

D点基準での計算：

CD間（長さc）がRDによって伸びることで、C点が左向きに変位：
\delta_c = -\frac{R_D \cdot c}{S_2 E_2}

(1)のRDを代入：

$$\delta_c = -\frac{4Pac}{\pi d^2 (aE_2 + 4(b+c)E_1)}$$
（負号は左向きの変位を示す）

(3) E₁≪E₂のときの点Cの変位δc₂

E₁がE₂に比べて非常に小さい極限条件を考えます。

極限条件の適用：

(2)の式で分母のE₁項を無視：
aE_2 + 4(b+c)E_1 ≈ aE_2 \quad (E_1 \ll E_2)

解答：

$$\delta_{c2} = -\frac{4Pc}{\pi d^2 E_2}$$
これは材料の軟らかい部分（AB間）が変形を支配する状況を表しています。
答え: 上記の各計算結果による
全体のポイント
  • 段付き構造では各部分の材料特性を考慮する。
  • 両端固定では全体変形ゼロの条件を活用する。
  • 極限条件では支配的な要因を見極める。
受験材料力学のチップス
  • 段付き棒では断面積と材料定数の両方に注意する。
  • 極限解析では主要項と副次項を区別する。
  • 変位の符号で変形方向を正確に把握する。
大問【2】

両端突き出し梁に対称荷重が作用する場合について、曲げモーメント線図、たわみ、最適設計条件を求める。

与えられた条件:

  • はり：全長L、断面二次モーメントI、ヤング率E
  • 支点：C、D点で単純支持（支間長L-2a）
  • 突き出し：両端A、B各々長さa
  • 荷重：A、B点にそれぞれ集中荷重P（下向き）
考え方のヒント

対称構造では反力も対称になります。突き出し部のたわみは片持ち効果と全体の傾きの重ね合わせで求まります。

途中式と解答

(1) 曲げモーメント線図(BMD)

対称性を利用して反力と曲げモーメントを求めます。

支点反力：

対称性により：
$$R_C = R_D = P$$

各区間の曲げモーメント：

  • AC間（0 ≤ x ≤ a）： M(x) = -Px
  • CD間（a ≤ x ≤ L-a）： M(x) = -Pa
  • DB間（L-a ≤ x ≤ L）： M(x) = -P(L-x)

曲げモーメント線図の特徴：

  • 両端A、Bでモーメントは0
  • 支点C、Dで最大値-Pa
  • 支点間は-Paで一定
  • 逆台形状の線図

(2) 点Aのたわみ

突き出し部の片持ち効果と全体の傾きを重ね合わせます。

2つの効果の重ね合わせ：

\delta_A = \delta_{片持ち} + \delta_{傾き}

片持ち効果：

\delta_{片持ち} = \frac{Pa^3}{3EI}

全体の傾き効果：

\delta_{傾き} = \frac{Pa^2(L-2a)}{2EI}

解答：

$$\delta_A = \frac{Pa^2(3L - 4a)}{6EI}$$

(3) 点Cのたわみ

支点間中央のたわみを求めます。

一定モーメント-Paによる曲率：

対称性により中央で傾きゼロの条件を使用：
$$\delta_C = \frac{Pa(L-2a)^2}{8EI}$$
（上向きのたわみ）

(4) たわみの大きさが等しいときのa/L

|δA| = |δC|となる最適化条件を求めます。

等価条件：

\frac{Pa^2(3L - 4a)}{6EI} = \frac{Pa(L-2a)^2}{8EI}

整理すると：

4a(3L - 4a) = 3(L-2a)^2

a/L = tとして2次方程式を解くと：

$$\frac{a}{L} = \frac{6 - \sqrt{15}}{14} \approx 0.152$$
答え: 上記の各計算結果による
全体のポイント
  • 対称構造では対称性を最大限活用する。
  • 突き出し部のたわみは2つの効果の重ね合わせ。
  • 最適設計では異なる部位の変形を均等化する。
受験材料力学のチップス
  • BMDでは不連続点（荷重点、支点）に注意する。
  • 重ね合わせの原理を正しく適用する。
  • 最適化問題では物理的に妥当な解を選択する。
大問【3】

矩形断面の片持ちはりに傾斜した集中荷重が作用する場合の2軸曲げ問題について、曲げモーメント、最大応力、許容荷重を求める。

与えられた条件:

  • はり：長さL、矩形断面（幅b×高さh）
  • 境界条件：固定端A、自由端B
  • 荷重：B点に集中荷重P（水平面から角度θ傾斜）
  • 材料：引張強さσt
考え方のヒント

傾斜荷重をy、z成分に分解し、各成分による曲げ応力を重ね合わせます。最大応力は両成分の引張応力が重なる断面角部で発生します。

途中式と解答

(1) 固定面の曲げモーメント

傾斜荷重を2成分に分解し、各軸まわりのモーメントを求めます。

荷重の成分分解：

垂直成分：P_y = P\cos\theta
水平成分：P_x = P\sin\theta

各軸まわりの曲げモーメント：

z軸まわり：$$M_z = P\cos\theta \cdot L$$ y軸まわり：$$M_y = P\sin\theta \cdot L$$

(2) 最大引張応力

2軸曲げによる応力を重ね合わせて最大応力を求めます。

各軸まわりの曲げ応力：

z軸まわり：$$\sigma_z = \frac{M_z \cdot (h/2)}{I_z} = \frac{6PL\cos\theta}{bh^2}$$ y軸まわり：$$\sigma_y = \frac{M_y \cdot (b/2)}{I_y} = \frac{6PL\sin\theta}{b^2h}$$

ここで、Iz = bh³/12、Iy = b³h/12

最大引張応力（断面右上隅）：

$$\sigma_{max} = \sigma_z + \sigma_y = \frac{6PL}{bh}\left(\frac{\cos\theta}{h} + \frac{\sin\theta}{b}\right)$$

(3) 外力Pの最大値

材料の引張強さを超えない条件からPの最大値を決定します。

安全条件：

\sigma_{max} \leq \sigma_t

(2)の結果を代入：

\frac{6PL}{bh}\left(\frac{\cos\theta}{h} + \frac{\sin\theta}{b}\right) \leq \sigma_t

解答：

$$P_{max} = \frac{\sigma_t bh}{6L\left(\frac{\cos\theta}{h} + \frac{\sin\theta}{b}\right)}$$
答え: 上記の各計算結果による
全体のポイント
  • 傾斜荷重は直交成分に分解して解析する。
  • 2軸曲げでは各軸の応力を重ね合わせる。
  • 最大応力点は両応力が同符号で重なる箇所。
受験材料力学のチップス
  • 2016年度は実用設計を意識した問題構成。
  • 断面二次モーメントの公式を正確に適用する。
  • 許容応力設計では安全条件を正しく設定する。

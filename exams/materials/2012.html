問題概要

2012年度の材料力学入試では、大問【1】ではりのたわみと重ね合わせの原理の体系的応用、大問【2】で自重による棒の応力分布問題（垂直応力の計算と段付棒の最適設計）、大問【3】でL字レバーにおける曲げとねじりの複合問題（主応力解析と設計）が出題されました。基礎理論から実用設計まで総合的にカバーした年度です。

大問【1】

はりのたわみと重ね合わせの原理について、片持ちはりの基本解から両端固定はりの複合問題への段階的応用を行う。

与えられた条件:

  • はり：長さl、曲げ剛性EI
  • 荷重：集中荷重P、集中モーメントM
  • 境界条件：片持ち（図1a,b）、両端固定（図1c）
  • 目標変位：図1cでのA点たわみδ
考え方のヒント

重ね合わせの原理により、複数の荷重による変形は各荷重による変形の代数和で求まります。境界条件の調整も同じ原理で処理できます。

途中式と解答

(1) 図1(a)の反力と曲げモーメント

片持ちはりの先端荷重による静力学的解析を行います。

力の釣り合い：

$$R_{B1} = P$$

モーメントの釣り合い（B点まわり）：

$$M_{B1} = P \cdot l$$

(2) 図1(a)のたわみ角とたわみ

片持ちはりの先端荷重による変形を求めます。

自由端Aにおけるたわみ角：

$$\theta_{A1} = \frac{Pl^2}{2EI}$$

自由端Aにおけるたわみ：

$$y_{A1} = \frac{Pl^3}{3EI}$$
これらは片持ちはりの基本公式です。

(3) 図1(b)の反力と曲げモーメント

片持ちはりの先端モーメント荷重による解析を行います。

力の釣り合い：

$$R_{B2} = 0$$

モーメントの釣り合い：

$$M_{B2} = M$$

(4) 図1(b)のたわみ角とたわみ

片持ちはりの先端モーメント荷重による変形を求めます。

自由端Aにおけるたわみ角：

$$\theta_{A2} = \frac{Ml}{EI}$$

自由端Aにおけるたわみ：

$$y_{A2} = \frac{Ml^2}{2EI}$$

(5) 図1(c)の反力と曲げモーメント

重ね合わせの原理を用いて両端固定はりの解を求めます。

重ね合わせ条件：

A点で：たわみ = δ、たわみ角 = 0

力RAとモーメントMAによる変形と(2)(4)の結果を重ね合わせ：

変位条件：$$\frac{R_A l^3}{3EI} + \frac{M_A l^2}{2EI} = \delta$$
回転条件：$$\frac{R_A l^2}{2EI} + \frac{M_A l}{EI} = 0$$

この連立方程式を解くと：

解答：

$$R_A = \frac{12EI\delta}{l^3}$$ $$M_A = \frac{6EI\delta}{l^2}$$
答え: 上記の各計算結果による
全体のポイント
  • 重ね合わせの原理は複合荷重問題の基本。
  • 片持ちはりの基本公式を確実に記憶する。
  • 境界条件から未知反力を決定する手順。
受験材料力学のチップス
  • 変形条件と回転条件の両方を考慮する。
  • 連立方程式の解法を正確に実行する。
  • 重ね合わせでは符号の管理が重要。
大問【2】

自重による棒の応力について、垂直応力分布の解析と段付棒の最適設計を行う。

与えられた条件:

  • 棒1：長さl₁、断面積A₁
  • 棒2：長さl₂、断面積A₂、下端に重りW
  • 材料：比重量γ、引張強さσB
  • 座標：下端から上向きにx
考え方のヒント

自重問題では任意断面にかかる力がその下部の重量と等しくなります。応力は上部ほど大きくなることが特徴です。

途中式と解答

(1) 棒1の垂直応力σ₁(x)

下端からx位置での棒1の垂直応力を求めます。

下部重量：

長さxの棒の重量：W(x) = γA₁x

垂直応力：

$$\sigma_1(x) = \frac{W(x)}{A_1} = \frac{\gamma A_1 x}{A_1} = \gamma x$$

(2) 棒1の最大応力σ₁max

棒1で応力が最大となる位置とその値を求めます。

最大応力位置：

σ₁(x) = γxより、x = l₁（棒1の最上端）で最大

最大応力：

$$\sigma_{1max} = \gamma l_1$$

(3) 棒2（重り付き）の垂直応力σ₂(x)

下端に重りWが付いた棒2の垂直応力を求めます。

下部総重量：

長さxの棒の重量 + 重りW：F(x) = γA₂x + W

垂直応力：

$$\sigma_2(x) = \frac{F(x)}{A_2} = \frac{W + \gamma A_2 x}{A_2} = \frac{W}{A_2} + \gamma x$$

(4) 棒2の最大応力σ₂max

重り付き棒2の最大応力を求めます。

最大応力位置：

x = l₂（棒2の最上端）で最大

最大応力：

$$\sigma_{2max} = \frac{W}{A_2} + \gamma l_2$$

(5) 段付棒の最大長さlmax

破壊しない段付棒の最大長さを求めます。

最危険断面の特定：

棒1の最上端が最も危険（断面積小で荷重大）

棒1最上端の応力：

全荷重 = 棒1重量 + 棒2重量 = γA₁l₁ + γA₂l₂
$$\sigma_{1,top} = \frac{\gamma A_1 l_1 + \gamma A_2 l_2}{A_1} = \gamma l_1 + \gamma \frac{A_2}{A_1} l_2$$

最適化条件：

σ₁,top ≤ σB の制約下でl = l₁ + l₂を最大化

A₂ > A₁より、l₁ = 0（細い棒を除去）で最大：

$$l_{max} = \frac{\sigma_B}{\gamma}$$
答え: 上記の各計算結果による
全体のポイント
  • 自重応力は位置に比例して増加する。
  • 段付構造では細い部分が破壊の起点。
  • 最適設計では制約条件を正しく設定。
受験材料力学のチップス
  • 応力分布の形状を図示して理解する。
  • 最適化では物理的制約を見極める。
  • 段付棒では断面変化点に注意する。
大問【3】

L字レバーにおける曲げとねじりの複合問題について、モーメント解析、応力計算、主応力による設計を行う。

与えられた条件:

  • L字レバー：水平アームAB（長さw）、垂直アームOA（高さh）
  • 荷重：B点に集中荷重P（下向き）
  • 断面：円形断面（直径d）
  • 材料：許容応力σa
考え方のヒント

L字構造では荷重の伝達経路により曲げとねじりが複合的に作用します。主応力による破壊条件を考慮した設計が必要です。

途中式と解答

(1) 断面Cの曲げモーメントMc

水平アーム内の断面Cにおける曲げモーメントを求めます。

片持ちはりとしての解析：

断面CからB点までの距離がaの場合：
$$M_c = P \cdot a$$

(2) 断面Dの曲げモーメントMDとねじりモーメントTD

垂直アーム内の断面Dにおける複合モーメントを求めます。

ねじりモーメント：

B点の荷重Pが水平アームの長さwでねじりを発生：
$$T_D = P \cdot w$$

曲げモーメント：

断面DからB点までの高さ（h-b）で曲げを発生：
$$M_D = P(h-b)$$

(3) 断面Dの応力

断面Dにおける曲げ応力とせん断応力を求めます。

曲げ応力（最大）：

$$\sigma_{Dmax} = \frac{M_D \cdot (d/2)}{I} = \frac{P(h-b)(d/2)}{\pi d^4/64} = \frac{32P(h-b)}{\pi d^3}$$

せん断応力（最大）：

$$\tau_{Dmax} = \frac{T_D \cdot (d/2)}{J_p} = \frac{Pw(d/2)}{\pi d^4/32} = \frac{16Pw}{\pi d^3}$$
ここで、I = πd⁴/64（断面二次モーメント）、Jp = πd⁴/32（断面二次極モーメント）

(4) 断面Dの最大主応力

曲げ応力とせん断応力の合成により最大主応力を求めます。

主応力の公式：

$$\sigma_1 = \frac{\sigma_D}{2} + \sqrt{\left(\frac{\sigma_D}{2}\right)^2 + \tau_D^2}$$

(3)の結果を代入：

$$\sigma_1 = \frac{16P(h-b)}{2\pi d^3} + \sqrt{\left(\frac{16P(h-b)}{2\pi d^3}\right)^2 + \left(\frac{16Pw}{\pi d^3}\right)^2}$$

解答：

$$\sigma_1 = \frac{16P}{\pi d^3} \left( (h-b) + \sqrt{(h-b)^2 + w^2} \right)$$

(5) 最小の丸棒直径

許容応力による設計条件から最小直径を求めます。

設計条件：

$$\sigma_1 \leq \sigma_a$$

(4)の結果を代入：

$$\frac{16P}{\pi d^3} \left( (h-b) + \sqrt{(h-b)^2 + w^2} \right) \leq \sigma_a$$

dについて解くと：

$$d_{min} = \sqrt[3]{\frac{16P}{\pi \sigma_a} \left( (h-b) + \sqrt{(h-b)^2 + w^2} \right)}$$
答え: 上記の各計算結果による
全体のポイント
  • L字構造では曲げとねじりが複合的に作用。
  • 主応力による破壊条件を考慮する。
  • 設計では最も危険な断面で評価する。
受験材料力学のチップス
  • 2012年度は重ね合わせから複合応力まで体系的出題。
  • 荷重の伝達経路を正確に把握する。
  • 主応力の物理的意味を理解する。

大問【1】

両端を壁に固定された棒に軸方向の力が作用する不静定構造について、変形の適合条件を用いて反力とひずみを求める。

与えられた条件:

  • 棒：長さl、断面積S、ヤング率E
  • 境界条件：両端A、Bで完全固定
  • 荷重：集中荷重P（位置a）、分布荷重q(x)
  • 材料：線形弾性体
考え方のヒント

不静定構造では力の釣り合いだけでは不十分。変形の適合条件（全体の伸びがゼロ）を併用する。重ね合わせの原理を活用して複数荷重の影響を計算します。

途中式と解答

(1) 集中荷重による反力（RAとRB）

不静定構造では力の釣り合いと変形の適合条件を同時に考えます。

力の釣り合い：

$$R_A + R_B = P$$

変形の適合条件：

全体の伸びがゼロ：
$$\delta_{AC} + \delta_{CB} = 0$$ $$\frac{R_A \cdot a}{SE} - \frac{R_B \cdot (l-a)}{SE} = 0$$

解答：

$$R_A = \frac{P(l-a)}{l}, \quad R_B = \frac{Pa}{l}$$

(2) AC間とCB間の伸び

(1)で求めた反力を用いて各区間の伸びを計算します。
AC間の伸び：$$\delta_{AC} = \frac{R_A \cdot a}{SE} = \frac{Pa(l-a)}{lSE}$$ CB間の伸び：$$\delta_{CB} = -\frac{R_B \cdot (l-a)}{SE} = -\frac{Pa(l-a)}{lSE}$$
AC間の伸びとCB間の縮みが等しく、全体の変形がゼロになることが確認できます。

(3) 集中荷重2つによる反力（重ね合わせ）

重ね合わせの原理により、各荷重による影響を個別に計算して足し合わせます。
P₁による反力：$$R_{A1} = \frac{P_1(l-a_1)}{l}, \quad R_{B1} = \frac{P_1 a_1}{l}$$ P₂による反力：$$R_{A2} = \frac{(-P_2)(l-a_2)}{l}, \quad R_{B2} = \frac{(-P_2) a_2}{l}$$

合計の反力：

$$R_A = \frac{P_1(l-a_1) - P_2(l-a_2)}{l}$$ $$R_B = \frac{P_1 a_1 - P_2 a_2}{l}$$

(4) 分布荷重による反力（積分解法）

分布荷重に対しては積分を用いて解析します。

内部力の分布：

$$T(x) = R_A - \int_0^x q(t)dt$$

変形の適合条件：

$$\int_0^l \frac{T(x)}{SE}dx = 0$$

一般解：

$$R_A = \frac{1}{l} \int_0^l \left( \int_0^x q(t)dt \right) dx$$ $$R_B = \int_0^l q(x)dx - R_A$$

(5) q(x) = q₀(x/l)の場合

具体的な分布荷重に対して計算を実行します。

積分計算：

$$\int_0^x q(t)dt = \int_0^x q_0 \frac{t}{l}dt = \frac{q_0 x^2}{2l}$$

反力の計算：

$$R_A = \frac{1}{l} \int_0^l \frac{q_0 x^2}{2l}dx = \frac{q_0 l}{6}$$ $$R_B = \frac{q_0 l}{2} - \frac{q_0 l}{6} = \frac{q_0 l}{3}$$
答え: 上記の各計算結果による
全体のポイント
  • 不静定問題では力の釣り合いと変形の適合条件を連立する。
  • 重ね合わせの原理を活用して複数荷重の影響を計算する。
  • 対称構造では対称性を活用して計算を簡略化する。

大問【2】

両端固定の段付きはり（中央部で断面が2倍）の中央に集中荷重が作用する場合の曲げモーメント、たわみ、固定モーメントを求める。

与えられた条件:

  • はり：全長4l、AC間（断面二次モーメントI）、CO間（断面二次モーメント2I）
  • 境界条件：両端A、Bで完全固定
  • 荷重：中央点Oに集中荷重P
  • 対称性：構造と荷重が左右対称
考え方のヒント

左右対称なので、反力は P/2、固定モーメントは両端で等しい。対称性から中央点での傾きがゼロという条件を利用してMAを決定します。

途中式と解答

(1) 曲げモーメントM(x)

対称性を活用して反力と曲げモーメントを求めます。

反力の決定：

対称性により：$$R_A = R_B = \frac{P}{2}$$

曲げモーメント（0 ≤ x ≤ 2l）：

$$M(x) = M_A - \frac{P}{2}x$$
ここで、MAは固定モーメント（未知数）です。

(2) C点のたわみvC

たわみの基本式を用いてC点のたわみを求めます。

たわみの基本式：

$$EI \frac{d^2v}{dx^2} = -M(x)$$

AC間で2回積分：

境界条件（x = 0で v = 0, dv/dx = 0）を適用して：
$$v_C = \frac{1}{EI}\left(\frac{Pl^3}{12} - \frac{M_A l^2}{2}\right)$$

(3) 固定モーメントMA

対称性条件（中央点Oでの傾きがゼロ）を利用してMAを決定します。

CO間の解析：

CO間では断面二次モーメントが2Iなので：
$$2EI \frac{d^2v}{dx^2} = -M(x)$$

連続条件と対称条件：

C点での連続性と、O点での傾きゼロ条件を適用して：
$$M_A = \frac{5Pl}{12}$$
この値を使って、C点のたわみは \(v_C = -\frac{Pl^3}{24EI}\) となります。
答え: 上記の各計算結果による
全体のポイント
  • 段付きはりでは各区間で断面二次モーメントが異なる。
  • 対称構造では中央での傾きがゼロという条件を活用する。
  • 連続条件と境界条件を正確に適用する。
受験材料力学のチップス
  • 両端固定はりでは固定モーメントが発生する。
  • 対称性を最大限に活用して計算を簡略化する。
  • 断面の変化点での連続条件を忘れない。
大問【3】

2本の軸をフランジとボルトで接続する軸継手について、軸とボルトの強度からトルク伝達能力を評価し、最適設計条件を求める。

与えられた条件:

  • 軸：直径D、許容せん断応力τs
  • ボルト：4本、直径d、許容せん断応力τf、ボルト円直径2R
  • トルク伝達：軸のねじりとボルトのせん断で制限
  • 設計目標：バランスの取れた継手設計
考え方のヒント

軸とボルトの強度を等しくする設計により、効率的な継手設計が可能。各要素の限界状態を評価してバランス設計を実現します。

途中式と解答

(1) 軸が伝達可能な最大トルクTs

ねじりの基本式を用いて軸の最大トルクを求めます。

中実円軸の断面係数：

$$Z_p = \frac{\pi D^3}{16}$$

最大トルク：

$$T_s = \tau_s \cdot Z_p = \frac{\pi D^3 \tau_s}{16}$$

(2) ボルトが伝達可能な最大トルクTf

4本のボルトによるせん断力でトルクを伝達します。

1本のボルトの最大せん断力：

$$F = \frac{\pi d^2}{4} \tau_f$$

4本のボルトの合計トルク：

$$T_f = 4 \times F \times R = \pi d^2 R \tau_f$$

(3) Ts = Tfとなるボルト直径d

軸とボルトの強度を等しくする設計条件を求めます。

設計条件：

$$T_s = T_f$$ $$\frac{\pi D^3 \tau_s}{16} = \pi d^2 R \tau_f$$

最終解答：

$$d = \frac{D}{4}\sqrt{\frac{D \tau_s}{R \tau_f}}$$
この条件により、軸とボルトが同時に限界に達する効率的な継手設計が可能です。
答え: 上記の各計算結果による
全体のポイント
  • 軸継手設計では軸とボルトの強度をバランスさせる。
  • 中実円軸のねじり断面係数は\(Z_p = \frac{\pi D^3}{16}\)。
  • ボルトのせん断力でトルクを伝達する。
受験材料力学のチップス
  • 設計問題では各要素の強度を等しくすることが重要。
  • トルク伝達のメカニズムを理解して計算する。
  • ボルトの配置とせん断面積の関係を正確に把握する。

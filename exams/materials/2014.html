問題概要

2014年度の材料力学入試では、大問【1】で温度変化による熱応力の問題（拘束棒の内部応力と連結棒の応力分担）、大問【2】でばね支持片持ちはりの問題（はりとばねの変形適合）、大問【3】でねじりと曲げの組み合わせ応力問題（モールの応力円と主応力解析）が出題されました。基礎理論から応用解析まで包括的にカバーした年度です。

大問【1】

棒の熱応力について、単独拘束棒の応力、連結棒の応力分担、連結部の変位を求める。

与えられた条件:

  • 棒1：長さl₁、断面積S₁、線膨張係数α₁、ヤング率E₁
  • 棒2：長さl₂、断面積S₂、ヤング率E₂（温度変化なし）
  • 温度上昇：棒1のみ ΔT上昇
  • 境界条件：両端完全固定または連結構造
考え方のヒント

熱応力問題では「自由な熱膨張量」と「拘束による圧縮量」が等しくなる条件を活用します。連結された場合は各棒の変形の適合条件が重要です。

途中式と解答

(1) 拘束された棒1本に生じる応力σ₁

両端固定された棒の熱応力を求めます。

熱膨張による伸び：

自由な熱膨張量：
$$\delta_T = \alpha_1 l_1 \Delta T$$

拘束力による適合条件：

壁による圧縮力Pで元の長さに戻すため：
$$\delta_T = \delta_P = \frac{P l_1}{S_1 E_1}$$

これから：

$$\alpha_1 l_1 \Delta T = \frac{P l_1}{S_1 E_1}$$

解答：

$$\sigma_1 = \frac{P}{S_1} = \alpha_1 E_1 \Delta T$$
（圧縮応力）

(2) 連結された棒に生じる応力σ₁とσ₂

2本の棒が連結された場合の応力分担を求めます。

変形の適合条件：

全体の変形がゼロ：
$$(\alpha_1 l_1 \Delta T - \frac{P l_1}{S_1 E_1}) - \frac{P l_2}{S_2 E_2} = 0$$

これを整理すると：

$$\alpha_1 l_1 \Delta T = P\left(\frac{l_1}{S_1 E_1} + \frac{l_2}{S_2 E_2}\right)$$

解答：

$$\sigma_1 = \frac{\alpha_1 l_1 \Delta T}{S_1 \left( \frac{l_1}{S_1 E_1} + \frac{l_2}{S_2 E_2} \right)}$$ $$\sigma_2 = \frac{\alpha_1 l_1 \Delta T}{S_2 \left( \frac{l_1}{S_1 E_1} + \frac{l_2}{S_2 E_2} \right)}$$
（いずれも圧縮応力）

(3) 連結部Bの変位λB

固定端Aから見た連結部Bの変位を求めます。

棒1の正味の変形：

熱膨張による伸びから圧縮力による縮みを差し引く：
$$\lambda_B = \alpha_1 l_1 \Delta T - \frac{P l_1}{S_1 E_1}$$

(2)で求めたPを代入して整理：

$$\lambda_B = \frac{\alpha_1 l_1 l_2 \Delta T}{S_2 E_2 \left( \frac{l_1}{S_1 E_1} + \frac{l_2}{S_2 E_2} \right)}$$
（右向きが正の変位）
答え: 上記の各計算結果による
全体のポイント
  • 熱応力は自由膨張と拘束力の釣り合いで決まる。
  • 連結構造では各部分の変形適合条件を使う。
  • 応力分担は各棒の剛性に依存する。
受験材料力学のチップス
  • 熱応力問題では温度変化がない部分も考慮する。
  • 拘束条件により応力の大きさが決まる。
  • 変位は各部分の伸び縮みの累積として求める。
大問【2】

ばねで支えられた片持ちはりについて、はりのたわみ、ばねの伸び、自由端の変位を求める。

与えられた条件:

  • はり：長さl、曲げ剛性EI
  • 境界条件：固定端、自由端
  • 支持：自由端にばね（ばね定数k）
  • 荷重：自由端に集中荷重P（下向き）
考え方のヒント

外力Pははりが支える分P₁とばねが支える分P₂に分担されます。はりのたわみとばねの伸びが等しくなる点で釣り合います。

途中式と解答

(1) はりのたわみy₁

片持ちはりの先端荷重によるたわみを求めます。

片持ちはりの公式：

先端に集中荷重P₁がかかる場合：
$$y_1 = \frac{P_1 l^3}{3EI}$$

(2) ばねの伸びλ₂

フックの法則によりばねの伸びを求めます。

フックの法則：

ばね力P₂による伸び：
$$\lambda_2 = \frac{P_2}{k}$$

(3) 自由端Bにおける変位yB

力の釣り合いと変形の適合条件から変位を求めます。

力の釣り合い：

$$P = P_1 + P_2$$

変形の適合条件：

はりのたわみ = ばねの伸び = 最終変位：
$$y_B = y_1 = \lambda_2$$

これから：

$$y_B = \frac{P_1 l^3}{3EI} = \frac{P_2}{k}$$

P₁ = 3EI·yB/l³、P₂ = k·yBを力の釣り合いに代入：

$$P = \frac{3EI \cdot y_B}{l^3} + k \cdot y_B = y_B\left(k + \frac{3EI}{l^3}\right)$$

解答：

$$y_B = \frac{P}{k + \frac{3EI}{l^3}}$$
答え: 上記の各計算結果による
全体のポイント
  • はりとばねの変形適合が解法の鍵となる。
  • 外力は構造要素間で分担される。
  • 剛性の並列結合として扱える。
受験材料力学のチップス
  • 変形の適合条件は複合構造の基本原理。
  • 各要素の力学特性を正しく把握する。
  • 連立方程式を体系的に解く。
大問【3】

ねじりと曲げを同時に受ける丸棒について、各応力成分、組み合わせ応力、モールの応力円による主応力解析を行う。

与えられた条件:

  • 丸棒：直径d
  • ねじりモーメント：T
  • 曲げモーメント：M
  • 外表面の角度θ位置での応力状態を解析
考え方のヒント

ねじりによるせん断応力と曲げによる垂直応力を重ね合わせ、モールの応力円により主応力と最大せん断応力を求めます。

途中式と解答

(1) ねじりのみによる最大せん断応力τmax

ねじりによるせん断応力を求めます。

ねじりの公式：

最大せん断応力（外表面）：
$$\tau_{max} = \frac{T \cdot (d/2)}{J} = \frac{T \cdot (d/2)}{\pi d^4/32} = \frac{16T}{\pi d^3}$$
ここで、J = πd⁴/32は断面二次極モーメント

(2) 曲げのみによる軸方向の応力σx

曲げによる軸方向応力を角度θの位置で求めます。

曲げ応力の公式：

中立面からの距離y = (d/2)sinθの位置：
$$\sigma_x = \frac{M \cdot y}{I} = \frac{M \cdot (d/2)\sin\theta}{\pi d^4/64} = \frac{32M \sin\theta}{\pi d^3}$$
ここで、I = πd⁴/64は断面二次モーメント

(3) 同時作用時の最大主応力σmaxと最大せん断応力τmax

ねじりと曲げの組み合わせ応力から主応力を求めます。

応力状態：

σx = 32Msinθ/(πd³)、τ = 16T/(πd³)

主応力の公式：

$$\sigma_{max} = \frac{\sigma_x}{2} + \sqrt{\left(\frac{\sigma_x}{2}\right)^2 + \tau^2}$$
$$= \frac{16M\sin\theta}{\pi d^3} + \sqrt{\left(\frac{16M\sin\theta}{\pi d^3}\right)^2 + \left(\frac{16T}{\pi d^3}\right)^2}$$

解答：

$$\sigma_{max} = \frac{16}{\pi d^3} \left( M\sin\theta + \sqrt{M^2\sin^2\theta + T^2} \right)$$
$$\tau_{max} = \sqrt{\left(\frac{\sigma_x}{2}\right)^2 + \tau^2} = \frac{16}{\pi d^3} \sqrt{M^2\sin^2\theta + T^2}$$

(4) モールの応力円と主応力の方向θp

モールの応力円により主応力の方向を求めます。

モールの応力円：

  • 中心C： (σx/2, 0) = (16Msinθ/(πd³), 0)
  • 半径R： √((σx/2)² + τ²) = (16/(πd³))√(M²sin²θ + T²)

主応力の方向：

$$\tan(2\theta_p) = \frac{2\tau}{\sigma_x} = \frac{2 \cdot 16T/(\pi d^3)}{32M\sin\theta/(\pi d^3)} = \frac{T}{M\sin\theta}$$

解答：

$$\theta_p = \frac{1}{2} \arctan\left(\frac{T}{M\sin\theta}\right)$$
答え: 上記の各計算結果による
全体のポイント
  • ねじりと曲げの重ね合わせによる複合応力状態。
  • モールの応力円は応力変換の強力な道具。
  • 主応力方向は材料の破壊条件に重要。
受験材料力学のチップス
  • 2014年度は熱応力から複合応力まで幅広い出題。
  • モールの応力円の作図法を習得する。
  • 主応力の物理的意味を理解する。

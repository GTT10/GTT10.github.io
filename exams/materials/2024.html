大問【1】自重による伸びの計算

棒自身の重さ（自重）によって生じる伸びを、3つの異なる形状（円柱、円錐、切れた円錐）について計算せよ。各部材の長さはl、密度はρ、縦弾性係数はEとする。

考え方のヒント

自重による伸びの問題では、「ある位置にかかる力は、それより下にある部分の重さに等しい」という考え方が基本です。形状によって断面積が変化する場合は、その変化を正確に把握することが重要です。

途中式と解答

(1) 円柱ABの応力と伸び

考え方：
円柱では断面積が一定なので、計算が比較的単純です。下端からx の高さにある断面には、それより下にある部分（長さxの円柱）の重さがかかります。 応力の計算：
- 断面積：\(A = \frac{\pi d_0^2}{4}\)
- 下端からxまでの重さ：\(W(x) = \rho g A x\)
- 応力：\(\sigma_{AB}(x) = \frac{W(x)}{A} = \rho g x\) 伸びの計算：
微小部分\(dx\)の伸び：\(d\delta = \frac{\sigma_{AB}(x)}{E} dx = \frac{\rho g x}{E} dx\)
全体の伸び： $$\delta_{AB} = \int_0^l \frac{\rho g x}{E} dx = \frac{\rho g}{E} \left[\frac{x^2}{2}\right]_0^l = \frac{\rho g l^2}{2E}$$
答え: 応力 \(\sigma_{AB}(x) = \rho g x\)、伸び \(\delta_{AB} = \frac{\rho g l^2}{2E}\)

(2) 円錐CDの応力と伸び

考え方：
円錐では断面積が位置によって変化します。下端で直径0、上端で直径\(d_0\)なので、断面積は高さに比例して変化します。 断面積の変化：
- 直径：\(d(x) = \frac{d_0}{l} x\)
- 断面積：\(A(x) = \frac{\pi d(x)^2}{4} = \frac{\pi d_0^2 x^2}{4l^2}\) 応力の計算：
- 下端からxまでの体積（小さな円錐）：\(V(x) = \frac{1}{3} A(x) x\)
- 重さ：\(W(x) = \rho g V(x) = \rho g \frac{1}{3} A(x) x\)
- 応力：\(\sigma_{CD}(x) = \frac{W(x)}{A(x)} = \frac{\rho g x}{3}\) 伸びの計算：
$$\delta_{CD} = \int_0^l \frac{\sigma_{CD}(x)}{E} dx = \int_0^l \frac{\rho g x}{3E} dx = \frac{\rho g l^2}{6E}$$
答え: 応力 \(\sigma_{CD}(x) = \frac{\rho g x}{3}\)、伸び \(\delta_{CD} = \frac{\rho g l^2}{6E}\)

(3) 切れた円錐EFの応力と伸び

考え方：
切れた円錐（円錐台）は、大きな円錐から小さな円錐を切り取った形状です。下端で直径\(d_0/2\)、上端で直径\(d_0\)となります。 断面積の変化：
- 直径：\(d(x) = \frac{d_0}{2} + \frac{d_0 x}{2l}\)
- 断面積：\(A(x) = \frac{\pi d(x)^2}{4}\) 応力の計算：
複雑な計算を経て、応力は以下のようになります： $$\sigma_{EF}(x) = \frac{\rho g}{3} \left( (x+l) - \frac{l^3}{(x+l)^2} \right)$$ 伸びの計算：
$$\delta_{EF} = \int_0^l \frac{\sigma_{EF}(x)}{E} dx = \frac{\rho g l^2}{3E}$$
答え: 応力 \(\sigma_{EF}(x) = \frac{\rho g}{3} \left( (x+l) - \frac{l^3}{(x+l)^2} \right)\)、伸び \(\delta_{EF} = \frac{\rho g l^2}{3E}\)
まとめ

自重による伸びは、形状によって大きく異なります。円柱 > 切れた円錐 > 円錐の順で伸びが大きくなることがわかります。これは、応力分布と断面積の変化の関係によるものです。

大問【2】不静定はりの曲げ問題

一端が固定され、他端が単純支持されたはり（プロップトカンチレバー）に、単位長さあたりqの一様分布荷重が作用している。このはりについて以下の問いに答えよ。

考え方のヒント

不静定はりは、力のつり合い式だけでは解けません。はりのたわみに関する境界条件（支点での変位・回転角の条件）を利用して解く必要があります。

途中式と解答

(1) y方向の力のつり合い式

力のつり合い：
上向きの力：A点の支点反力\(R_A\)とB点の支点反力\(R_B\)
下向きの力：分布荷重の合力\(ql\) $$R_A + R_B - ql = 0$$
答え: \(R_A + R_B - ql = 0\)

(2) 曲げモーメントM

曲げモーメントの計算：
A点から距離xにある断面で、左側の部分を考えます。 - \(R_A\)によるモーメント：\(R_A x\)（上に凸） - 分布荷重によるモーメント：\(-\frac{qx^2}{2}\)（下に凸） $$M(x) = R_A x - \frac{q x^2}{2}$$
答え: \(M(x) = R_A x - \frac{q x^2}{2}\)

(3) 支持力\(R_A\), \(R_B\)、支持モーメント\(M_B\)

境界条件を用いた解法：
たわみの基本式：\(EI\frac{d^2v}{dx^2} = -M(x)\)を2回積分し、以下の境界条件を適用します： - A点（x=0）：たわみ = 0（単純支持） - B点（x=l）：たわみ = 0（固定端） - B点（x=l）：たわみ角 = 0（固定端） これらの条件から積分定数と未知反力を決定します。
計算結果：
境界条件を適用して連立方程式を解くと： $$R_A = \frac{3}{8}ql$$ $$R_B = \frac{5}{8}ql$$ $$M_B = \frac{1}{8}ql^2$$
答え: \(R_A = \frac{3}{8}ql\), \(R_B = \frac{5}{8}ql\), \(M_B = \frac{1}{8}ql^2\)

(4) 曲げモーメントの大きさの最大値

最大モーメントの探索：
曲げモーメントの最大値は以下の位置で生じる可能性があります： 1. **せん断力が0になる点：** \(V(x) = R_A - qx = 0\)より\(x = \frac{3l}{8}\) 2. **固定端B：** \(x = l\) 各点でのモーメントの大きさを比較します： - \(x = \frac{3l}{8}\)でのモーメント：\(M = \frac{9ql^2}{128}\) - B点でのモーメント：\(|M_B| = \frac{ql^2}{8}\) $$\frac{ql^2}{8} > \frac{9ql^2}{128}$$
答え: \(|M|_{max} = \frac{1}{8}ql^2\)（固定端Bで発生）
まとめ

不静定はりの解析では、静力学のつり合い式に加えて、変形に関する境界条件を活用することが重要です。特に固定端では大きなモーメントが発生することに注意が必要です。

問題概要

2018年度の材料力学入試では、大問【1】でハメあいによる応力問題（不静定構造の解析）、大問【2】で連続はりの曲げ問題（3点支持の不静定はり）、大問【3】で段付き丸棒のねじり問題（複合軸の変形解析）が出題されました。伝統的な3問構成の最終年度として、全分野をバランスよくカバーした総合的な内容でした。

大問【1】

長さがわずかに異なる円管と丸棒をハメあいで固定した際に生じる応力と、同等の熱応力を発生させる温度上昇について解析する。

与えられた条件:

  • 円管：長さl、断面積A₁、ヤング率E₁
  • 丸棒：長さl+δ、断面積A₂、ヤング率E₂
  • 関係：A₁ = 2A₂
  • 熱膨張係数：α₁、α₂
考え方のヒント

ハメあい問題は不静定問題の典型例です。力の釣り合いと変形の適合条件を同時に考える必要があります。熱応力との等価性も重要な概念です。

途中式と解答

(1) 内力の関係式

接合後、外力がない状態で静止しているため、作用・反作用の法則から両部品の内力は等しくなります。
$$Q_1 = Q_2$$

(2) 伸びの関係式

最終的に両部品は同じ長さになるため、それぞれの伸びから関係式を導きます。

初期状態: 円管の長さはl、丸棒の長さはl+δ

最終状態: 両方とも同じ長さL_finalになる

円管の伸び：$$\lambda_1 = L_{final} - l$$ 丸棒の伸び：$$\lambda_2 = L_{final} - (l + \delta)$$

これらから以下の関係式が得られます：

$$\lambda_1 - \lambda_2 = \delta$$

(3) 各部品の応力

力の釣り合いと変形の適合条件を連立して応力を求めます。

応力と力の関係: σ = Q/A

フックの法則: λ = -σl/E（圧縮のため負号）

関係式を応力で表現すると：

$$l \left(\frac{\sigma_2}{E_2} - \frac{\sigma_1}{E_1}\right) = \delta$$

力の釣り合いから：σ₂ = σ₁(A₁/A₂)

解答:

円管の応力：$$\sigma_1 = \frac{\delta A_2 E_1 E_2}{l(A_1 E_1 - A_2 E_2)}$$ 丸棒の応力：$$\sigma_2 = \frac{\delta A_1 E_1 E_2}{l(A_1 E_1 - A_2 E_2)}$$

(4) 応力の比

A₁ = 2A₂の条件を用いて応力の比を求めます。
$$\frac{|\sigma_1|}{|\sigma_2|} = \frac{A_2}{A_1} = \frac{A_2}{2A_2} = \frac{1}{2}$$

(5) 等価な温度上昇

同じ応力を発生させる温度上昇ΔTを求めます。

熱膨張による変位差がハメあいのすきまδと等価になる条件：

$$\delta = (\alpha_2 - \alpha_1) \cdot l \cdot \Delta T$$

解答:

$$\Delta T = \frac{\delta}{l(\alpha_2 - \alpha_1)}$$
答え: 上記の各計算結果による
全体のポイント
  • ハメあい問題は力の釣り合いと変形の適合条件を連立する不静定問題。
  • 作用・反作用の法則から内力の関係を導く。
  • 熱応力と機械的応力の等価性を理解する。
受験材料力学のチップス
  • 不静定問題では必ず変形の適合条件が必要。
  • 熱応力問題では熱膨張係数の差に注目する。
  • 初期状態と最終状態の長さの関係を明確にする。
大問【2】

3点で支持された連続はりに分布荷重が作用する場合の支持力、曲げモーメントを求める不静定問題。

与えられた条件:

  • はり：全長2l、3点支持（A、C、B）
  • 荷重：AC間に分布荷重w（一様）
  • 支点：A、C、Bで支持（Cは中間支点）
  • 構造：ACとCBが各々長さlの連続はり
考え方のヒント

3点支持の連続はりは1次不静定問題です。モーメントの釣り合いから反力間の関係を求め、たわみの連続条件で完全解を得ます。

途中式と解答

(1) A点の支持力

B点まわりのモーメントの釣り合いから反力間の関係式を導きます。

モーメントの釣り合い（B点まわり）:

$$R_A \cdot 2l + R_C \cdot l - wl \cdot \frac{3l}{2} = 0$$

これを整理すると：

$$R_A = \frac{R_C}{2} + \frac{3wl}{4}$$

(2) 曲げモーメントの表現

A点からの距離xにおける曲げモーメントを区間別に求めます。

AC間（0 ≤ x ≤ l）:

$$M(x) = R_A x - \frac{wx^2}{2}$$

(1)の結果を代入すると：

$$M(x) = \left(\frac{R_C}{2} + \frac{3wl}{4}\right)x - \frac{wx^2}{2}$$

CB間（l ≤ x ≤ 2l）:

$$M(x) = R_A x + R_C(x-l) - wl\left(x-\frac{l}{2}\right)$$

(3) C点の支持力

不静定問題の完全解として、たわみの連続条件等を用いて求めます。

高次の構造解析手法（三連モーメント定理等）により：

$$R_A = \frac{7wl}{16}, \quad R_B = -\frac{wl}{16}, \quad R_C = \frac{5wl}{8}$$

解答:

$$R_C = \frac{5wl}{8}$$
答え: 上記の各計算結果による
全体のポイント
  • 連続はりは静定解析だけでは解けない不静定構造。
  • モーメントの釣り合いから反力間の関係式を導く。
  • たわみの連続条件が追加の拘束条件となる。
受験材料力学のチップス
  • 区間ごとに曲げモーメント式を立てる。
  • 支点での境界条件（たわみ=0）を活用する。
  • 三連モーメント定理は重要な解法手段。
大問【3】

段付き丸棒と複合軸（中実軸+中空軸）からなる構造のねじり変形について、各部のねじれ角と全体の変形を解析する。

与えられた条件:

  • AB部：中実軸（直径d₁、長さl₁、横弾性係数G₁）
  • BC部：複合軸（中実軸+中空軸、長さl₂）
  • 中実軸：直径d₂、横弾性係数G₂
  • 中空軸：外径d₃、内径d₂、横弾性係数G₃
考え方のヒント

複合軸では各軸が一体となって変形するため、ねじれ角が等しいという条件を使います。トルクの分担と変形の適合が重要です。

途中式と解答

(1) 中実軸ABのねじれ角

ねじりの基本公式φ = TL/(GJ)を適用します。

中実円軸の断面二次極モーメント:

$$J_1 = \frac{\pi d_1^4}{32}$$

ねじれ角:

$$\phi_{AB} = \frac{T l_1}{G_1 J_1} = \frac{32 T l_1}{\pi G_1 d_1^4}$$

(2) 複合軸のトルク比

複合軸では中実軸と中空軸のねじれ角が等しい条件を使います。

ねじれ角の等価条件:

$$\phi_2 = \phi_3 \Rightarrow \frac{T_2 l_2}{G_2 J_2} = \frac{T_3 l_2}{G_3 J_3}$$

断面二次極モーメント:

$$J_2 = \frac{\pi d_2^4}{32}, \quad J_3 = \frac{\pi (d_3^4 - d_2^4)}{32}$$

トルク比:

$$\frac{T_2}{T_3} = \frac{G_2 J_2}{G_3 J_3} = \frac{G_2 d_2^4}{G_3 (d_3^4 - d_2^4)}$$

(3) 全体のねじれ角

全体のねじれ角は各部の和として求まります。

トルクの配分: T = T₂ + T₃

(2)の比を用いて：

$$T_2 = \frac{G_2 J_2}{G_2 J_2 + G_3 J_3} T, \quad T_3 = \frac{G_3 J_3}{G_2 J_2 + G_3 J_3} T$$

BC部のねじれ角:

$$\phi_{BC} = \frac{T l_2}{G_2 J_2 + G_3 J_3}$$

全体のねじれ角:

$$\phi = \phi_{AB} + \phi_{BC} = \frac{32 T l_1}{\pi G_1 d_1^4} + \frac{32 T l_2}{\pi(G_2 d_2^4 + G_3(d_3^4 - d_2^4))}$$
答え: 上記の各計算結果による
全体のポイント
  • 複合軸では各軸のねじれ角が等しいという条件を使う。
  • 中空軸の断面二次極モーメントは外径と内径の差で表される。
  • 全体の変形は各部の変形の和として求まる。

大問【1】3本の丸棒による並列支持問題

長さlの3本の丸棒A、B、Cが上下から剛体板で挟まれ、その剛体板に外力Pが作用している。丸棒の中心は互いにaだけ離れており、外力Pによってすべての丸棒は等しく縮むものとする。丸棒A、B、Cの縦弾性係数および断面積は、それぞれ順にE₁、E₂、E₃およびA₁、A₂、A₃とする。

考え方のヒント

この問題は不静定問題です。力のつり合い式だけでは解けないため、変形の適合条件（すべての丸棒が等しく縮む）を利用します。各丸棒の剛性を考慮した力の分配が重要です。

途中式と解答

(1) 丸棒の縮みδを求めよ

考え方：
すべての丸棒が等しく縮むため、各丸棒の縮みは同じδです。フックの法則により、各丸棒の力は剛性×縮みで表されます。 剛性の定義：
各丸棒の剛性を\(k_i = \frac{E_i A_i}{l}\)と定義すると、各丸棒に作用する力は： - \(F_A = k_A \delta = \frac{E_1 A_1}{l} \delta\) - \(F_B = k_B \delta = \frac{E_2 A_2}{l} \delta\) - \(F_C = k_C \delta = \frac{E_3 A_3}{l} \delta\) 力のつり合い：
$$F_A + F_B + F_C = P$$ $$(k_A + k_B + k_C)\delta = P$$ $$\delta = \frac{P}{k_A + k_B + k_C} = \frac{Pl}{E_1 A_1 + E_2 A_2 + E_3 A_3}$$
答え: \(\delta = \frac{Pl}{E_1 A_1 + E_2 A_2 + E_3 A_3}\)

(2) 外力Pを作用させた位置xを求めよ

考え方：
外力Pの作用位置は、モーメントのつり合いから決まります。丸棒Aを基準点として、モーメントのつり合いを考えます。 モーメントのつり合い（点A基準）：
- 丸棒Bの位置：Aからa - 丸棒Cの位置：Aから2a - 外力Pの位置：Aからx $$F_B \cdot a + F_C \cdot 2a = P \cdot x$$ \(F_i = k_i \delta\)を代入： $$k_B \delta \cdot a + k_C \delta \cdot 2a = P \cdot x$$ $$\delta a (k_B + 2k_C) = P \cdot x$$ (1)の結果を代入： $$x = \frac{\delta a (k_B + 2k_C)}{P} = \frac{a(k_B + 2k_C)}{k_A + k_B + k_C}$$ $$x = a \frac{E_2 A_2 + 2E_3 A_3}{E_1 A_1 + E_2 A_2 + E_3 A_3}$$
答え: \(x = a \frac{E_2 A_2 + 2E_3 A_3}{E_1 A_1 + E_2 A_2 + E_3 A_3}\)

(3) 破壊したときの外力P_maxを求めよ

考え方：
丸棒Aがせん断破壊する条件を考えます。一軸応力状態では、最大せん断応力は主応力の半分となります。 破壊条件：
丸棒Aの圧縮応力：\(\sigma = \frac{F_A}{A_1}\)
最大せん断応力：\(\tau_{max} = \frac{\sigma}{2} = \frac{F_A}{2A_1}\) 破壊条件：\(\tau_{max} = \tau_a\)より $$\frac{F_A}{2A_1} = \tau_a$$ $$F_A = 2A_1 \tau_a$$ F_AとPの関係：
(1)より\(F_A = k_A \delta = k_A \frac{P}{k_A + k_B + k_C}\) 破壊条件を代入： $$k_A \frac{P_{max}}{k_A + k_B + k_C} = 2A_1 \tau_a$$ $$P_{max} = 2A_1 \tau_a \cdot \frac{k_A + k_B + k_C}{k_A}$$ $$P_{max} = \frac{2\tau_a}{E_1}(E_1 A_1 + E_2 A_2 + E_3 A_3)$$
答え: \(P_{max} = \frac{2\tau_a}{E_1}(E_1 A_1 + E_2 A_2 + E_3 A_3)\)
まとめ

並列支持問題では、各部材の剛性に応じて力が分配されます。破壊解析では、最も弱い部材（応力が最大となる部材）が先に破壊することを考慮します。

大問【2】段付はりのたわみ問題

縦弾性係数E、長さ4lの両端を支持した段付はりAEがあり、三角形分布荷重が作用している。分布外力wは以下の式で与えられる：

$$w = \begin{cases} \frac{w_0}{l}(x-l) & (l \leq x \leq 2l) \\ \frac{w_0}{l}(3l-x) & (2l \leq x \leq 3l) \end{cases}$$

AB及びDE間の断面二次モーメントをI、長さをl、BD間の断面二次モーメントを2I、長さを2lとする。

考え方のヒント

段付はりの問題では、各区間での断面二次モーメントの変化を考慮します。分布荷重は対称性を持つため、支点反力も対称になります。たわみは微分方程式を区間ごとに解き、境界条件と連続条件を適用します。

途中式と解答

(1) A点に作用する支点反力R_Aを求めよ

分布荷重の合力計算：
第一区間（l ≤ x ≤ 2l）： $$\int_l^{2l} \frac{w_0}{l}(x-l)dx = \frac{w_0}{l} \left[\frac{1}{2}(x-l)^2\right]_l^{2l} = \frac{1}{2}w_0 l$$ 第二区間（2l ≤ x ≤ 3l）： $$\int_{2l}^{3l} \frac{w_0}{l}(3l-x)dx = \frac{1}{2}w_0 l$$ 合力と対称性：
全合力：\(R_w = w_0 l\)（x = 2lで作用）
対称性により：\(R_A = R_E\) 力のつり合い：
$$2R_A = w_0 l$$ $$R_A = \frac{1}{2}w_0 l$$
答え: \(R_A = \frac{1}{2}w_0 l\)

(2) BC間のせん断力F_BCを求めよ

せん断力の計算（l ≤ x ≤ 2l）：
左側の力のつり合いから： $$F(x) = -R_A + \int_l^x w(\xi)d\xi$$ 分布荷重の積分： $$\int_l^x \frac{w_0}{l}(\xi-l)d\xi = \frac{w_0}{2l}(x-l)^2$$ \(R_A = \frac{1}{2}w_0 l\)を代入： $$F_{BC}(x) = -\frac{1}{2}w_0 l + \frac{w_0}{2l}(x-l)^2$$
答え: \(F_{BC}(x) = \frac{w_0}{2l}(x-l)^2 - \frac{1}{2}w_0 l\) (l ≤ x ≤ 2l)

(3) BC間の曲げモーメントM_BCを求めよ

曲げモーメントの計算：
左側のモーメントのつり合いから： $$M(x) = R_A \cdot x - \int_l^x w(\xi)(x-\xi)d\xi$$ 分布荷重によるモーメント： $$\int_l^x \frac{w_0}{l}(\xi-l)(x-\xi)d\xi = \frac{w_0}{6l}(x-l)^3$$ したがって： $$M_{BC}(x) = \frac{w_0 l}{2}x - \frac{w_0}{6l}(x-l)^3$$
答え: \(M_{BC}(x) = \frac{w_0 l}{2}x - \frac{w_0}{6l}(x-l)^3\) (l ≤ x ≤ 2l)

(4) AB間のたわみv_ABを求めよ

たわみの微分方程式（0 ≤ x ≤ l）：
AB区間では荷重なし、\(M(x) = R_A x = \frac{1}{2}w_0 lx\) $$\frac{d^2v}{dx^2} = -\frac{M}{EI} = -\frac{w_0 l x}{2EI}$$ 積分：
$$\frac{dv}{dx} = -\frac{w_0 l}{4EI}x^2 + C_1$$ $$v = -\frac{w_0 l}{12EI}x^3 + C_1 x + C_2$$ 境界条件：
x = 0でv = 0より\(C_2 = 0\)
連続条件と対称性から\(C_1 = \frac{29}{48}\frac{w_0 l^3}{EI}\) $$v_{AB}(x) = \frac{w_0 l}{EI}\left(-\frac{1}{12}x^3 + \frac{29}{48}l^2 x\right)$$
答え: \(v_{AB}(x) = \frac{w_0 l}{EI}\left(-\frac{1}{12}x^3 + \frac{29}{48}l^2 x\right)\) (0 ≤ x ≤ l)

(5) BC間のたわみv_BCを求めよ

BD区間の微分方程式（l ≤ x ≤ 2l）：
断面二次モーメント2Iを考慮： $$\frac{d^2v_2}{dx^2} = -\frac{M}{E \cdot 2I} = -\frac{1}{2EI}\left(\frac{w_0 l}{2}x - \frac{w_0}{6l}(x-l)^3\right)$$ 積分と境界条件：
複雑な計算を経て、連続条件（点Bでのたわみと傾きの連続性）と対称性を適用すると： $$v_{BC}(x) = \frac{w_0}{EI}\left[-\frac{l}{24}x^3 + \frac{1}{240l}(x-l)^5 + \frac{23}{48}l^3 x + \frac{1}{12}l^4\right]$$
答え: \(v_{BC}(x) = \frac{w_0}{EI}\left[-\frac{l}{24}x^3 + \frac{1}{240l}(x-l)^5 + \frac{23}{48}l^3 x + \frac{1}{12}l^4\right]\) (l ≤ x ≤ 2l)
まとめ

段付はりの解析では、各区間の断面二次モーメントの変化を正確に考慮することが重要です。分布荷重の対称性を活用し、連続条件を正しく適用することで複雑な変形状態を解析できます。

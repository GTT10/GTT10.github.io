大問【1】

図1は、壁にピンで固定された剛体棒CDが2本のワイヤーAEとBFで吊り下げられ、先端Dに荷重Pが作用している状況を示しています。

図1：問題1の説明図

考え方のヒント

この問題は不静定問題です。力のつり合い式だけでは解けないため、変形の適合条件を考慮する必要があります。剛体棒の回転によるワイヤーの伸びと、ワイヤーの材料特性による伸びを結びつけて考えましょう。

途中式と解答

(1) 上下方向のつり合い式と，点Cまわりのモーメントのつり合い式を求めよ

物体にはたらく力のつり合いを考えます。支持力\(R\)とワイヤーの張力\(T_a, T_b\)を上向き、外力\(P\)を下向きとすると、垂直方向の力のつり合い式は以下のようになります。 力のつり合い式: $$R + T_a + T_b - P = 0$$
次に、点Cを回転の中心としてモーメントのつり合いを考えます。点C周りのモーメントのつり合い式は以下の通りです。 点Cに関するモーメントのつり合い式: $$a T_a + b T_b - c P = 0$$

(2) ワイヤーAEとBFの応力を求めよ

これは不静定問題なので、力のつり合いに加えて変形の適合条件を考慮する必要があります。剛体棒CDが点Cを中心に微小角度 \(\theta\) だけ回転すると、点Aと点Bの変位 \(\delta_a, \delta_b\) はそれぞれワイヤーの伸びに等しくなります。 変位の幾何学的関係： $$\delta_a = a \theta, \quad \delta_b = b \theta$$
一方、ワイヤーの伸びは材料の性質から以下の式で表されます。 $$\delta_a = \frac{T_a l_a}{E_a S_a}, \quad \delta_b = \frac{T_b l_b}{E_b S_b}$$
これらの式から \(\theta\) を消去して、変形の適合条件式を得ます。 $$\frac{\delta_a}{a} = \frac{\delta_b}{b} \implies \frac{T_a l_a}{a E_a S_a} = \frac{T_b l_b}{b E_b S_b}$$
この式とモーメントのつり合い式 \(a T_a + b T_b = c P\) を連立して、\(T_a\)と\(T_b\)を求めます。
適合条件式から \(T_a\) を \(T_b\) で表すと、 $$T_a = T_b \cdot \frac{a l_b E_a S_a}{b l_a E_b S_b}$$ これをモーメントのつり合い式に代入して、\(T_b\) を求めます。 $$a \left( T_b \frac{a l_b E_a S_a}{b l_a E_b S_b} \right) + b T_b = cP \implies T_b \left( \frac{a^2 l_b E_a S_a}{b l_a E_b S_b} + b \right) = cP$$ $$T_b = \frac{c P b l_a E_b S_b}{a^2 l_b E_a S_a + b^2 l_a E_b S_b}$$ 同様に、\(T_a\) は以下のようになります。 $$T_a = \frac{c P a l_b E_a S_a}{a^2 l_b E_a S_a + b^2 l_a E_b S_b}$$
応力 \(\sigma\) は力 \(T\) を断面積 \(S\) で割ることで求められるので (\(\sigma = T/S\))、各ワイヤーの応力は次のようになります。
ワイヤーAEの応力 \(\sigma_a\): $$\sigma_a = \frac{T_a}{S_a} = \frac{c P a l_b E_a}{a^2 l_b E_a S_a + b^2 l_a E_b S_b}$$
ワイヤーBFの応力 \(\sigma_b\): $$\sigma_b = \frac{T_b}{S_b} = \frac{c P b l_a E_b}{a^2 l_b E_a S_a + b^2 l_a E_b S_b}$$

(3) ワイヤーAEとBFの応力の比を求めよ．ただし，縦弾性係数は等しく，\(2a = b\) , \(l_a = 2l_b\) とする．

変形の適合条件式を応力で書き直します。 $$\frac{(\sigma_a S_a) l_a}{a E_a S_a} = \frac{(\sigma_b S_b) l_b}{b E_b S_b} \implies \frac{\sigma_a l_a}{a E_a} = \frac{\sigma_b l_b}{b E_b}$$ この式から応力の比を求めると、 $$\frac{\sigma_a}{\sigma_b} = \frac{a E_a l_b}{b E_b l_a}$$
問題の条件である \(E_a = E_b\)、 \(b = 2a\)、 \(l_a = 2l_b\) を代入します。 $$\frac{\sigma_a}{\sigma_b} = \frac{a E_a l_b}{(2a) E_a (2l_b)} = \frac{1}{4}$$
答え: 応力 \(\sigma_a\) は \(\sigma_b\) の \(\frac{1}{4}\) 倍になります。
まとめ

この不静定問題は、力のつり合い式と変形の適合条件式を連立することで解くことができました。特に、応力の比を求める際には、与えられた条件を正確に代入することが重要です。最終的に応力 \(\sigma_a\) は \(\sigma_b\) の \(\frac{1}{4}\) 倍という結果になりました。

全体のポイント
  • 不静定問題では、力のつり合い式と変形の適合条件式を連立する。
  • 剛体棒の回転による変位と、ワイヤーの伸びの関係を理解する。
  • 応力とひずみの基本法則を適用する。
受験材料力学のチップス
  • 不静定問題の解法手順をマスターする。
  • 幾何学的関係と物理的関係（フックの法則など）を明確に区別する。
  • 複雑な式の整理と代入を丁寧に行い、計算ミスを防ぐ。
大問【2】

図2は、一端（B）が固定され、他端（A）が支持されたはりに、曲げモーメント \(M_C\) が作用している状況を示しています。

図2：問題2の説明図

(1) 曲げモーメントとたわみを求めよ

考え方のヒント

梁のたわみ問題は、曲げモーメントの微分方程式を積分することで解きます。境界条件と連続条件を正確に適用し、積分定数と未知の反力を決定することが重要です。

途中式と解答
支持力 \(R_A\) を既知の外力とみなし、点Aを原点としてx軸をとります。
曲げモーメント \(M(x)\):
区間AC (\(0 \le x \le a\)) と CB (\(a \le x \le l\)) でモーメントの式は異なります。
  • 区間AC (\(0 \le x \le a\)): $$M_{AC}(x) = R_A x$$
  • 区間CB (\(a \le x \le l\)): $$M_{CB}(x) = R_A x + M_C$$
たわみ \(v(x)\):
たわみの基本微分方程式 \(EI \frac{d^2v}{dx^2} = -M(x)\) を積分して、たわみの式を求めます。
たわみの基礎微分方程式の符号について ▶
\(EI \frac{d^2 v}{dx^2} = -M(x)\) : 上向き正（時計回り）
\(EI \frac{d^2 v}{dx^2} = M(x)\) : 下向き正（反時計回り）
  • 区間AC (\(0 \le x \le a\)): $$EI \frac{d^2v_{AC}}{dx^2} = -R_A x$$ $$EI \frac{dv_{AC}}{dx} = -\frac{1}{2} R_A x^2 + C_1$$ $$EI v_{AC}(x) = -\frac{1}{6} R_A x^3 + C_1 x + C_2$$
  • 区間CB (\(a \le x \le l\)): $$EI \frac{d^2v_{CB}}{dx^2} = -R_A x - M_C$$ $$EI \frac{dv_{CB}}{dx} = -\frac{1}{2} R_A x^2 - M_C x + C_3$$ $$EI v_{CB}(x) = -\frac{1}{6} R_A x^3 - \frac{1}{2} M_C x^2 + C_3 x + C_4$$
積分定数 (\(C_1, C_2, C_3, C_4\)) は、境界条件と連続条件から決定されます。
  1. 支持点A (\(x=0\)): たわみはゼロ \(v_{AC}(0) = 0\)
  2. 固定端B (\(x=l\)): たわみはゼロ \(v_{CB}(l) = 0\)
  3. 固定端B (\(x=l\)): たわみ角はゼロ \(\frac{dv_{CB}}{dx}(l) = 0\)
  4. 点C (\(x=a\)): たわみは連続 \(v_{AC}(a) = v_{CB}(a)\)
  5. 点C (\(x=a\)): たわみ角は連続 \(\frac{dv_{AC}}{dx}(a) = \frac{dv_{CB}}{dx}(a)\)
これらの条件を用いて定数を \(R_A, M_C, a, l\) で表し、たわみの式に代入します。
計算過程 ▶
ステップ 1: 条件の適用
\(v_1(x=0) = 0\) の適用 \(v_1(x)\) の式： $$ EI \cdot v_1(x) = -\frac{1}{6} R_A x^3 + C_1 x + C_2 $$ \(x=0\) を代入： $$ EI \cdot v_1(0) = -\frac{1}{6} R_A \cdot 0^3 + C_1 \cdot 0 + C_2 = C_2 $$ \(v_1(0) = 0\) より： $$ C_2 = 0 $$ \(x=l\) で \(\frac{dv_2}{dx} = 0\) より： $$ -\frac{1}{2} R_A l^2 - M_C l + C_3 = 0 $$ $$ C_3 = \frac{1}{2} R_A l^2 + M_C l \quad \text{(1)} $$
\(v_2(x=l) = 0\) の適用
\(v_2(x)\) の式に \(x=l\) を代入： $$ EI \cdot v_2(l) = -\frac{1}{6} R_A l^3 - \frac{1}{2} M_C l^2 + C_3 l + C_4 = 0 $$ 式 (1) を代入： $$ -\frac{1}{6} R_A l^3 - \frac{1}{2} M_C l^2 + \left(\frac{1}{2} R_A l^2 + M_C l\right) l + C_4 = 0 $$ $$ -\frac{1}{6} R_A l^3 - \frac{1}{2} M_C l^2 + \frac{1}{2} R_A l^3 + M_C l^2 + C_4 = 0 $$ $$ \left(-\frac{1}{6} + \frac{3}{6}\right) R_A l^3 + \left(-\frac{1}{2} + 1\right) M_C l^2 + C_4 = 0 $$ $$ \frac{1}{3} R_A l^3 + \frac{1}{2} M_C l^2 + C_4 = 0 $$ $$ C_4 = -\frac{1}{3} R_A l^3 - \frac{1}{2} M_C l^2 \quad \text{(2)} $$
\(\frac{dv_1}{dx}(x=a) = \frac{dv_2}{dx}(x=a)\) の適用
まず、\(v_1(x)\) の式を \(x\) で微分（\(C_2 = 0\)）： $$ \frac{d}{dx}(EI \cdot v_1(x)) = \frac{d}{dx}\left(-\frac{1}{6} R_A x^3 + C_1 x\right) = -\frac{1}{2} R_A x^2 + C_1 $$ \(v_2(x)\) の微分は既に求めた： $$ \frac{d}{dx}(EI \cdot v_2(x)) = -\frac{1}{2} R_A x^2 - M_C x + C_3 $$ \(x=a\) で等しい： $$ -\frac{1}{2} R_A a^2 + C_1 = -\frac{1}{2} R_A a^2 - M_C a + C_3 $$ $$ C_1 = -M_C a + C_3 $$ 式 (1) を代入： $$ C_1 = -M_C a + \frac{1}{2} R_A l^2 + M_C l = \frac{1}{2} R_A l^2 + M_C (l - a) \quad \text{(3)} $$
\(v_1(x=a) = v_2(x=a)\) の適用
\(v_1(x)\) の式（\(C_2 = 0\)）： $$ EI \cdot v_1(a) = -\frac{1}{6} R_A a^3 + C_1 a $$ \(v_2(x)\) の式： $$ EI \cdot v_2(a) = -\frac{1}{6} R_A a^3 - \frac{1}{2} M_C a^2 + C_3 a + C_4 $$ 等しいので： $$ -\frac{1}{6} R_A a^3 + C_1 a = -\frac{1}{6} R_A a^3 - \frac{1}{2} M_C a^2 + C_3 a + C_4 $$ $$ C_1 a = -\frac{1}{2} M_C a^2 + C_3 a + C_4 $$ 式 (1)、(2)、(3) を代入：
左辺：\(C_1 a = \left[\frac{1}{2} R_A l^2 + M_C (l - a)\right] a = \frac{1}{2} R_A l^2 a + M_C a (l - a)\)
右辺：\(-\frac{1}{2} M_C a^2 + C_3 a + C_4 = -\frac{1}{2} M_C a^2 + \left[\frac{1}{2} R_A l^2 + M_C l\right] a + \left[-\frac{1}{3} R_A l^3 - \frac{1}{2} M_C l^2\right]\) $$ = -\frac{1}{2} M_C a^2 + \frac{1}{2} R_A l^2 a + M_C l a - \frac{1}{3} R_A l^3 - \frac{1}{2} M_C l^2 $$ 等式： $$ \frac{1}{2} R_A l^2 a + M_C a (l - a) = -\frac{1}{2} M_C a^2 + \frac{1}{2} R_A l^2 a + M_C l a - \frac{1}{3} R_A l^3 - \frac{1}{2} M_C l^2 $$ 両辺から \(\frac{1}{2} R_A l^2 a + M_C l a\) を減算： $$ M_C a (l - a) - M_C l a = -\frac{1}{2} M_C a^2 - \frac{1}{3} R_A l^3 - \frac{1}{2} M_C l^2 $$ $$ M_C a l - M_C a^2 - M_C l a = -\frac{1}{2} M_C a^2 - \frac{1}{3} R_A l^3 - \frac{1}{2} M_C l^2 $$ $$ - M_C a^2 = -\frac{1}{2} M_C a^2 - \frac{1}{3} R_A l^3 - \frac{1}{2} M_C l^2 $$ 移項： $$ - M_C a^2 + \frac{1}{2} M_C a^2 + \frac{1}{3} R_A l^3 + \frac{1}{2} M_C l^2 = 0 $$ $$ -\frac{1}{2} M_C a^2 + \frac{1}{3} R_A l^3 + \frac{1}{2} M_C l^2 = 0 $$ 分母を払うため 6 を乗算： $$ 6 \cdot \left(-\frac{1}{2} M_C a^2\right) + 6 \cdot \left(\frac{1}{3} R_A l^3\right) + 6 \cdot \left(\frac{1}{2} M_C l^2\right) = 0 $$ $$ -3 M_C a^2 + 2 R_A l^3 + 3 M_C l^2 = 0 $$ $$ 2 R_A l^3 + 3 M_C (l^2 - a^2) = 0 \quad \text{(4)} $$ ステップ 2: 積分定数の消去 得られた積分定数： - \(C_2 = 0\) - \(C_3 = \frac{1}{2} R_A l^2 + M_C l\) （式 (1)） - \(C_1 = \frac{1}{2} R_A l^2 + M_C (l - a)\) （式 (3)） - \(C_4 = -\frac{1}{3} R_A l^3 - \frac{1}{2} M_C l^2\) （式 (2)） これらを \(v_1(x)\) と \(v_2(x)\) の式に代入する。 \(v_1(x)\) の式： $$ EI \cdot v_1(x) = -\frac{1}{6} R_A x^3 + C_1 x + C_2 = -\frac{1}{6} R_A x^3 + \left[\frac{1}{2} R_A l^2 + M_C (l - a)\right] x + 0 $$ $$ = R_A \left(-\frac{1}{6} x^3 + \frac{1}{2} l^2 x\right) + M_C (l - a) x $$
\(v_2(x)\) の式： $$ EI \cdot v_2(x) = -\frac{1}{6} R_A x^3 - \frac{1}{2} M_C x^2 + C_3 x + C_4 $$ $$ = -\frac{1}{6} R_A x^3 - \frac{1}{2} M_C x^2 + \left[\frac{1}{2} R_A l^2 + M_C l\right] x + \left[-\frac{1}{3} R_A l^3 - \frac{1}{2} M_C l^2\right] $$ $$ = R_A \left(-\frac{1}{6} x^3 + \frac{1}{2} l^2 x - \frac{1}{3} l^3\right) + M_C \left(-\frac{1}{2} x^2 + l x - \frac{1}{2} l^2\right) $$ ステップ 3: 解の整理 積分定数を消去し、解は以下の通りである。ただし、\(R_A\) と \(M_C\) は式 (4) \(2 R_A l^3 + 3 M_C (l^2 - a^2) = 0\) を満たす必要がある。 $$ \boxed{ \begin{aligned} EI \cdot v_{1}(x) &= R_{A} \left( -\frac{1}{6} x^{3} + \frac{1}{2} l^{2} x \right) + M_{C} (l - a) x \\ EI \cdot v_{2}(x) &= R_{A} \left( -\frac{1}{6} x^{3} + \frac{1}{2} l^{2} x - \frac{1}{3} l^{3} \right) + M_{C} \left( -\frac{1}{2} x^{2} + l x - \frac{1}{2} l^{2} \right) \end{aligned} } $$ この解は、初期条件と接続条件を満たし、積分定数が消去されている。\(R_A\) と \(M_C\) は任意ではなく、関係式 \(2 R_A l^3 + 3 M_C (l^2 - a^2) = 0\) によって制約される。
最終的に、たわみの式は以下のようになります。
  • たわみ \(v_{AC}(x)\): $$v_{AC}(x) = \frac{1}{EI} \left[ -\frac{1}{6} R_A x^3 + \left( M_C(l-a) + \frac{1}{2}R_A l^2 \right) x \right]$$
  • たわみ \(v_{CB}(x)\): $$v_{CB}(x) = \frac{1}{EI} \left[ -\frac{1}{6} R_A x^3 - \frac{1}{2} M_C x^2 + \frac{1}{2}\left( 2M_C l + R_A l^2 \right) x - \frac{1}{6}(2R_A l + 3M_C ) l^2 \right]$$
答え:
\(M_{AC}(x) = R_A x\)
\(M_{CB}(x) = R_A x + M_C\)
\(v_{AC}(x) = \frac{1}{EI} \left[ -\frac{1}{6} R_A x^3 + \left( M_C(l-a) + \frac{1}{2}R_A l^2 \right) x \right]\)
\(v_{CB}(x) = \frac{1}{EI} \left[ -\frac{1}{6} R_A x^3 - \frac{1}{2} M_C x^2 + \frac{1}{2}\left( 2M_C l + R_A l^2 \right) x - \frac{1}{6}(2R_A l + 3M_C ) l^2 \right]\)

(2) 支持力 \(R_A\) を求めよ

上記で用いた境界条件をすべて適用することで、未知数であった支持力 \(R_A\) を決定できます。積分定数を求める過程で得られる以下の関係式を用います。 $$-\frac{1}{3}R_A l^3 + \frac{M_C}{2}(a^2-l^2) = 0$$ この式を \(R_A\) について解きます。
  • 支持力 \(R_A\): $$R_A = \frac{3 M_C (a^2-l^2)}{2 l^3}$$
答え: 支持力 \(R_A = \frac{3 M_C (a^2-l^2)}{2 l^3}\)
まとめ

梁のたわみ問題は、曲げモーメントの微分方程式を解き、境界条件と連続条件を丁寧に適用することが鍵となります。積分定数を一つずつ決定していくことで、最終的に未知の反力やたわみの式を求めることができます。

全体のポイント
  • 梁のたわみに関する微分方程式を理解する。
  • 境界条件と連続条件を正確に設定し、適用する。
  • 積分定数を適切に決定する。

問題概要

2015年度の材料力学入試では、大問【1】で両端固定棒の軸力問題（不静定構造の最大荷重設計）、大問【2】で単純支持はりの曲げ問題（対称荷重と自重の影響）、大問【3】で長柱の座屈問題（オイラー座屈と自重座屈）が出題されました。基礎理論から実用的な設計問題まで幅広くカバーした年度です。

大問【1】

両端固定された棒の中間点に軸方向集中荷重が作用する不静定構造について、反力、変位、最大荷重を求める。

与えられた条件:

  • 棒：全長a+b、断面積S、ヤング率E
  • 境界条件：両端A、Cで完全固定
  • 荷重：B点（長さaの位置）に集中荷重P（右向き）
  • 材料：許容せん断応力τa、条件a < b
考え方のヒント

両端固定の不静定問題では、全体の変形がゼロという条件が重要です。AB間の伸びとBC間の縮みが等しくなるという適合条件を活用します。

途中式と解答

(1) 壁から受ける反力RAとRC

力の釣り合いと変形の適合条件を連立して反力を求めます。

力の釣り合い：

$$R_A + R_C = P$$

変形の適合条件：

全体の変形がゼロ：
$$\delta_{AB} = \delta_{BC}$$ $$\frac{R_A \cdot a}{SE} = \frac{R_C \cdot b}{SE}$$

これから RA・a = RC・b の関係が得られます。

解答：

$$R_A = \frac{Pb}{a+b}$$ $$R_C = \frac{Pa}{a+b}$$

(2) 点Bの変位λB

A点を基準としたB点の変位を求めます。

AB間の伸び：

B点の変位はAB間の伸びに等しい：
\lambda_B = \delta_{AB} = \frac{R_A \cdot a}{SE}

(1)のRAを代入：

$$\lambda_B = \frac{Pab}{(a+b)SE}$$
（右向きが正の変位）

(3) 最大荷重Pmax

最大せん断応力による破壊条件から最大荷重を決定します。

最大せん断応力の発生位置：

条件a < bから、(1)より RA > RC となり、AB間の方が危険：
\sigma_{AB} = \frac{R_A}{S} = \frac{Pb}{(a+b)S}

最大せん断応力：

\tau_{max} = \frac{\sigma_{AB}}{2} = \frac{Pb}{2(a+b)S}

破壊条件：

τmax ≤ τa の条件から：
$$P_{max} = \frac{2S\tau_a(a+b)}{b}$$
答え: 上記の各計算結果による
全体のポイント
  • 両端固定では全体変形ゼロの条件を活用する。
  • 反力の大きさは荷重点からの距離に反比例する。
  • 最大せん断応力による破壊を考慮する。
受験材料力学のチップス
  • 不静定問題では変形の適合条件が必須。
  • 最大せん断応力は軸応力の半分である。
  • 危険断面は最大内力が発生する箇所。
大問【2】

単純支持はりに対称集中荷重が作用する場合について、せん断力図・曲げモーメント図、たわみ、応力、自重の影響を求める。

与えられた条件:

  • はり：全長3a、断面は中空円管（外径2d、内径d）
  • 支点：A、B点で単純支持（支間長3a）
  • 荷重：対称位置（a、2aの地点）に集中荷重P
  • 材料：ヤング率E、単位長さ当たり重量w
考え方のヒント

対称荷重では支点反力も対称になります。SFDとBMDは区間ごとに構築し、最大値の位置を特定します。自重は分布荷重として別途考慮します。

途中式と解答

(1) せん断力図(SFD)および曲げモーメント図(BMD)

対称性を利用して反力とせん断力・曲げモーメントを求めます。

支点反力：

対称性により：
$$R_A = R_B = P$$

せん断力分布：

  • A-C間（0 ≤ x ≤ a）： F(x) = P
  • C-D間（a ≤ x ≤ 2a）： F(x) = 0
  • D-B間（2a ≤ x ≤ 3a）： F(x) = -P

曲げモーメント分布：

  • A-C間： M(x) = Px
  • C-D間： M(x) = Pa（一定）
  • D-B間： M(x) = P(3a-x)

線図の特徴：

  • SFD：階段状（P → 0 → -P）
  • BMD：台形状（最大値Pa）

(2) 最大たわみとその位置

対称性から最大たわみは中央で発生します。

最大たわみの位置：

はりの中心：
$$x = \frac{3a}{2} = 1.5a$$

最大たわみ：

たわみの微分方程式を積分して：
$$y_{max} = \frac{23Pa^3}{24EI}$$

(3) 断面二次モーメントIと最大引張応力σmax

中空円管の断面特性と曲げ応力を求めます。

断面二次モーメント：

中空円管（外径2d, 内径d）：
$$I = \frac{\pi}{64}[(2d)^4 - d^4] = \frac{15\pi d^4}{64}$$

最大引張応力：

最大曲げモーメントMmax = Paで：
$$\sigma_{max} = \frac{M_{max} \cdot y_{max}}{I} = \frac{Pa \cdot d}{15\pi d^4/64} = \frac{64Pa}{15\pi d^3}$$

(4) 自重による最大引張応力の増加量

自重を分布荷重として考慮した応力増加を求めます。

自重による最大曲げモーメント：

単純支持はりの中央における分布荷重wによる最大モーメント：
M_{自重} = \frac{w(3a)^2}{8} = \frac{9wa^2}{8}

応力増加量：

$$\Delta\sigma_{max} = \frac{M_{自重} \cdot d}{I} = \frac{(9wa^2/8) \cdot d}{15\pi d^4/64} = \frac{24wa^2}{5\pi d^3}$$
答え: 上記の各計算結果による
全体のポイント
  • 対称荷重では対称性を最大限活用する。
  • 中空円管の断面二次モーメント公式を正確に適用する。
  • 自重は分布荷重として別途考慮する。
受験材料力学のチップス
  • SFD・BMDでは荷重点での不連続に注意する。
  • 最大たわみは対称構造では中央で発生する。
  • 自重の影響は実用設計で重要な考慮事項。
大問【3】

長柱の座屈問題について、オイラー座屈荷重、座屈時のたわみ曲線、自重による座屈条件を求める。

与えられた条件:

  • 柱：長さl、断面積A、断面二次モーメントI
  • 境界条件：下端固定、上端自由
  • 材料：ヤング率E、単位体積当たり重量γ
  • 座屈係数：μ ≈ 7.837（自重座屈用）
考え方のヒント

下端固定・上端自由の柱では有効座屈長さLe = 2lとなります。自重座屈では軸力が位置によって変化することを考慮します。

途中式と解答

(1) 座屈荷重とたわみ曲線

オイラーの座屈理論を適用します。

有効座屈長さ：

下端固定・上端自由の場合：
$$L_e = 2l$$

座屈荷重：

オイラーの座屈公式：
$$P_{cr} = \frac{\pi^2 EI}{L_e^2} = \frac{\pi^2 EI}{(2l)^2} = \frac{\pi^2 EI}{4l^2}$$

たわみ曲線：

座屈時の変形形状（下端から測った距離x）：
$$y(x) = \delta\left(1 - \cos\frac{\pi x}{2l}\right)$$
ここで、δは上端でのたわみ量（座屈開始時は不定）

(2) 自重による座屈

柱の自重だけで座屈が生じる条件を求めます。

自重座屈の特徴：

  • 軸力が高さによって変化（上部ほど小さい）
  • 微分方程式の係数が変数となる
  • 解析的解法は高度で、通常は公式を使用

座屈条件：

下端固定・上端自由の柱における自重座屈の臨界条件：
$$\gamma_{cr} = \frac{\mu EI}{Al^3}$$

ここで、μ ≈ 7.837は境界条件による座屈係数

解答：

$$\gamma_{cr} = \frac{7.837 EI}{Al^3}$$
答え: 上記の各計算結果による
全体のポイント
  • 座屈は細長い柱特有の不安定現象である。
  • 有効座屈長さは境界条件により決まる。
  • 自重座屈では軸力の変化を考慮する。
受験材料力学のチップス
  • 2015年度は基礎から実用設計まで幅広い内容。
  • 座屈問題では境界条件の影響が重要。
  • 自重座屈は高層建築で実用的な問題。

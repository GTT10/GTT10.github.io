問題概要

2013年度の材料力学入試では、大問【1】で片持ちはりのたわみと重ね合わせの原理の応用、大問【2】で異材接合棒の熱応力問題（自由膨張と拘束による応力発生）、大問【3】で物体の落下による衝撃荷重問題（エネルギー保存による弾性解析）が出題されました。基礎理論から実用的な衝撃解析まで幅広い内容を扱った年度です。

大問【1】

片持ちはりのたわみと重ね合わせの原理について、基本的なたわみ計算から複合支持条件への応用を段階的に解析する。

与えられた条件:

  • はり：長さl、曲げ剛性EI
  • 境界条件：固定端A、自由端B
  • 荷重：先端に集中荷重P（下向き）
  • 点C：固定端から距離aの位置
考え方のヒント

重ね合わせの原理により、複数の荷重による変形は各荷重による変形の代数和で求まります。支点条件の変化も同様に扱えます。

途中式と解答

(1) 自由端Bのたわみδ_{B1}

片持ちはりの先端荷重による基本たわみを求めます。

片持ちはりの公式：

先端に集中荷重Pがかかる場合の先端たわみ：
$$\delta_{B1} = \frac{Pl^3}{3EI}$$
これは材料力学の基本公式です。

(2) 点Cのたわみδ_{C1}

同じ荷重条件での途中点Cのたわみを求めます。

たわみ曲線の利用：

固定端から距離aの点でのたわみ：
$$\delta_{C1} = \frac{Pa^2(3l - a)}{6EI}$$

これは先端荷重による一般的なたわみ曲線からx = aを代入した結果です。

(3) 点Cのたわみδ_{C2}（荷重がC点）

C点に上向き荷重Qが作用する場合のC点たわみを求めます。

中間荷重による変形：

C点に上向き集中荷重Qが作用するとき：
$$\delta_{C2} = -\frac{Qa^3}{3EI}$$

負号は上向き荷重により上向き（負の）たわみが生じることを示します。

(4) 自由端Bのたわみδ_{B3}（C点支持）

重ね合わせの原理を用いてC点支持時のB点たわみを求めます。

重ね合わせ条件：

C点で支持されているため、C点でのたわみがゼロ：
$$\delta_{C1} + \delta_{C2} = 0$$
これから反力R_Cを決定：
$$\frac{Pa^2(3l - a)}{6EI} - \frac{R_C a^3}{3EI} = 0$$

これより：

$$R_C = \frac{P(3l - a)}{2a}$$

B点のたわみ：

$$\delta_{B3} = \frac{P(4al - a^2 - 3l^2)}{12EI}$$
答え: 上記の各計算結果による
全体のポイント
  • 重ね合わせの原理は材料力学の基本概念。
  • 支持条件の変化は反力の重ね合わせで表現。
  • 変形の適合条件から未知反力を決定。
受験材料力学のチップス
  • 片持ちはりの基本公式を確実に覚える。
  • 重ね合わせ計算では符号に注意する。
  • 支点反力は変形条件から決定する。
大問【2】

異材接合棒の熱応力について、自由膨張と拘束条件での応力発生、接合部の変位を解析する。

与えられた条件:

  • 棒1：長さa、線膨張係数α₁、ヤング率E₁
  • 棒2：長さb、線膨張係数α₂、ヤング率E₂
  • 断面積：両棒ともS（一定）
  • 温度上昇：両棒ともΔT
考え方のヒント

熱応力問題では自由膨張量と拘束による変形量の釣り合いを考えます。材料定数の違いにより応力分担が決まります。

途中式と解答

(1) 両端自由のときの棒の伸びΔl

両端が自由な場合の各棒の熱膨張量を求めます。

各棒の熱膨張：

  • 棒1の伸び：α₁ · a · ΔT
  • 棒2の伸び：α₂ · b · ΔT

全体の伸び：

$$\Delta l = \alpha_1 a \Delta T + \alpha_2 b \Delta T = (\alpha_1 a + \alpha_2 b) \Delta T$$

(2) 両端固定のときの圧縮応力σc

両端固定による拘束で発生する圧縮応力を求めます。

拘束条件：

自由膨張量Δlを圧縮力Pで元に戻す：
$$\Delta l = \frac{Pa}{SE_1} + \frac{Pb}{SE_2}$$

(1)の結果を代入：

$$(\alpha_1 a + \alpha_2 b) \Delta T = P\left(\frac{a}{SE_1} + \frac{b}{SE_2}\right)$$

圧縮応力：

$$\sigma_c = \frac{P}{S} = \frac{(\alpha_1 a + \alpha_2 b) \Delta T}{\frac{a}{E_1} + \frac{b}{E_2}}$$

(3) 接合部Bの変位δB

固定端Aから見た接合部Bの最終変位を求めます。

棒1の正味変形：

熱膨張から圧縮変形を差し引く：
$$\delta_B = \alpha_1 a \Delta T - \frac{\sigma_c a}{E_1}$$

(2)のσcを代入して整理：

$$\delta_B = \frac{ab \Delta T (\alpha_1 E_1 - \alpha_2 E_2)}{aE_2 + bE_1}$$

材料1の熱膨張が材料2より大きい場合は正（右向き）、小さい場合は負（左向き）の変位となります。

答え: 上記の各計算結果による
全体のポイント
  • 異材料では膨張係数の違いが重要。
  • 拘束により内部応力が発生する。
  • 接合部変位は材料特性により決まる。
受験材料力学のチップス
  • 熱応力は実用構造で重要な設計因子。
  • 材料定数の組み合わせ効果を理解する。
  • 変位の符号は物理的意味を考える。
大問【3】

はりへの物体の落下による衝撃荷重について、エネルギー保存則を用いた弾性解析により最大荷重、たわみ、応力を求める。

与えられた条件:

  • はり：長さl、正方形断面（一辺a）、曲げ剛性EI
  • 境界条件：両端単純支持
  • 落下物：質量m、落下高さh
  • 衝撃位置：はりの中央
考え方のヒント

衝撃問題ではエネルギー保存則を用います。落下物の位置エネルギーがはりの弾性ひずみエネルギーに変換されます。

途中式と解答

(1) モーメントM(x)

中央荷重Pによる曲げモーメント分布を求めます。

支点反力：

対称性により各支点：R = P/2

曲げモーメント：

  • 左半分（0 ≤ x ≤ l/2）：
    • $$M(x) = \frac{P}{2}x$$
  • 右半分（l/2 ≤ x ≤ l）：
    • $$M(x) = \frac{P}{2}(l-x)$$
最大モーメントは中央でPl/4となります。

(2) 弾性ひずみエネルギーU

はりに蓄えられる弾性ひずみエネルギーを求めます。

ひずみエネルギーの公式：

$$U = \int_0^l \frac{M(x)^2}{2EI} dx$$

対称性を利用して左半分の2倍として計算：

$$U = 2 \int_0^{l/2} \frac{(Px/2)^2}{2EI} dx = \frac{P^2}{4EI} \int_0^{l/2} x^2 dx$$

解答：

$$U = \frac{P^2 l^3}{96EI}$$

(3) 最大荷重P

エネルギー保存則から最大荷重を求めます。

エネルギー保存：

位置エネルギー = 弾性ひずみエネルギー
$$mgh = \frac{P^2 l^3}{96EI}$$

これをPについて解くと：

$$P = \sqrt{\frac{96EI \cdot mgh}{l^3}} = 4\sqrt{\frac{6EImgh}{l^3}}$$

(4) はり中央の最大たわみδ

中央荷重による最大たわみを求めます。

中央たわみの公式：

$$\delta = \frac{Pl^3}{48EI}$$

(3)のPを代入：

$$\delta = \frac{4\sqrt{6EImgh/l^3} \cdot l^3}{48EI} = \sqrt{\frac{mghl^3}{24EI}}$$

(5) 最大曲げ応力σmax

正方形断面の最大曲げ応力を求めます。

断面係数：

正方形断面（一辺a）：Z = a³/6

最大曲げ応力：

$$\sigma_{max} = \frac{M_{max}}{Z} = \frac{Pl/4}{a^3/6} = \frac{3Pl}{2a^3}$$

(3)のPを代入：

$$\sigma_{max} = \frac{3 \cdot 4\sqrt{6EImgh/l^3} \cdot l}{2a^3} = \frac{6l}{a^3}\sqrt{\frac{6EImgh}{l^3}}$$

整理すると：

$$\sigma_{max} = \frac{6}{a^3}\sqrt{\frac{6EImghl^3}{l^3}} = \frac{6}{a^3}\sqrt{6EImgh}$$
答え: 上記の各計算結果による
全体のポイント
  • 衝撃問題ではエネルギー保存則を活用。
  • 弾性ひずみエネルギーの積分計算が重要。
  • 静的解析結果を動的問題に応用。
受験材料力学のチップス
  • 2013年度は基礎から衝撃解析まで総合的出題。
  • エネルギー法は動的問題の基本手法。
  • 断面係数の公式を正確に適用する。

問題概要

2017年度の材料力学入試では、大問【1】で3本のワイヤで吊り下げられた剛体棒の問題（不静定構造の変形解析）、大問【2】で一端固定・他端支持はりの曲げ問題（不静定はりの境界条件）、大問【3】で両端固定丸棒のねじり問題（不静定ねじりの反力分布）が出題されました。全て不静定問題で構成された難易度の高い年度です。

大問【1】

3本のワイヤで吊り下げられた剛体棒に集中荷重が作用する不静定構造について、各ワイヤの張力、たるまない条件、剛体棒の傾き角を解析する。

与えられた条件:

  • 剛体棒：長さ2a、A、B、C点で3本のワイヤで吊り下げ
  • ワイヤ：全て同じ材質・寸法（長さl、断面積A、ヤング率E）
  • 荷重：距離xの位置に集中荷重P（垂直下向き）
  • 剛体仮定：棒は変形しない

図1：問題1の説明図

考え方のヒント

剛体棒では3点の変位が一直線上に保たれるため、変形の幾何学的関係「2δB = δA + δC」が成り立ちます。この条件を不静定問題の解法に活用します。

途中式と解答

(1) 各ワイヤの張力

力とモーメントの釣り合いに加えて、剛体の変形の幾何学的関係を利用します。

力の釣り合い：

$$Q_{AD} + Q_{BE} + Q_{CF} = P$$

モーメントの釣り合い（A点まわり）：

$$Q_{BE} \cdot a + Q_{CF} \cdot 2a = P \cdot x$$

変形の適合条件：

剛体の幾何学的関係から：
$$2\delta_B = \delta_A + \delta_C$$ $$2Q_{BE} = Q_{AD} + Q_{CF}$$

解答：

$$Q_{AD} = \frac{P}{6a}(5a - 3x)$$ $$Q_{BE} = \frac{P}{3}$$ $$Q_{CF} = \frac{P}{6a}(3x - a)$$

(2) すべてのワイヤがたるまないxの範囲

全てのワイヤの張力が正の値（QAD ≥ 0、QBE ≥ 0、QCF ≥ 0）となる条件を求めます。

条件の設定：

Q_{AD} = \frac{P}{6a}(5a - 3x) \geq 0 \Rightarrow x \leq \frac{5a}{3}
Q_{BE} = \frac{P}{3} \geq 0 \quad \text{(常に満足)}
Q_{CF} = \frac{P}{6a}(3x - a) \geq 0 \Rightarrow x \geq \frac{a}{3}

解答：

$$\frac{a}{3} \leq x \leq \frac{5a}{3}$$

(3) 剛体棒の傾き角（x = a/2の場合）

x = a/2のときの各ワイヤの張力から変位を求め、傾き角を計算します。

x = a/2での張力：

Q_{AD} = \frac{P}{6a}(5a - \frac{3a}{2}) = \frac{7P}{12}
Q_{CF} = \frac{P}{6a}(\frac{3a}{2} - a) = \frac{P}{12}

各点の変位：

\delta_A = \frac{Q_{AD} l}{AE} = \frac{7Pl}{12AE}
\delta_C = \frac{Q_{CF} l}{AE} = \frac{Pl}{12AE}

傾き角：

$$\theta = \frac{\delta_A - \delta_C}{2a} = \frac{Pl}{4aAE} \text{ [rad]}$$
答え: 上記の各計算結果による
全体のポイント
  • 剛体棒では変形の幾何学的関係が追加の拘束条件となる。
  • 不静定問題では力の釣り合いと変形の適合条件を連立する。
  • 張力の正負で構造物の安定性を評価する。
受験材料力学のチップス
  • 剛体仮定では変形の幾何学的関係を正確に把握する。
  • 吊り構造では張力が負になる場合の物理的意味を理解する。
  • 対称性を活用して計算を簡略化する。
大問【2】

一端固定・他端支持はりに部分分布荷重が作用する不静定構造について、曲げモーメント分布、支点反力、固定端モーメントを求める。

与えられた条件:

  • はり：全長l、断面二次モーメントI、ヤング率E
  • 境界条件：A端で完全固定、B端で単純支持
  • 荷重：AC間（長さl/2）に一様分布荷重w
  • 支点：B点は鉛直反力のみ

図2：問題2の説明図

考え方のヒント

一端固定・他端支持はりでは、支点での境界条件（固定端：変位・傾き=0、支持端：変位=0）を利用して不静定問題を解きます。

途中式と解答

(1) 曲げモーメントMの表現

荷重分布が区間によって異なるため、CB間とAC間で分けて考えます。

CB間（0 ≤ x ≤ l/2）：

この区間には荷重がないため：
$$M(x) = R_B x$$

AC間（l/2 ≤ x ≤ l）：

分布荷重wの影響を含めて：
$$M(x) = R_B x - \frac{w}{2}\left(x-\frac{l}{2}\right)^2$$

(2) 支点Bの反力RB

境界条件を利用してRBを決定します。

境界条件の設定：

  • x = 0（B点）でのたわみ = 0
  • x = l（A点）でのたわみ = 0
  • x = l（A点）での傾き = 0

たわみの微分方程式を2回積分し、上記境界条件を適用すると：

$$R_B = \frac{7}{128}wl$$

(3) 固定端Aにおける曲げモーメントMA

全体のモーメントの釣り合いからMAを求めます。

A点まわりのモーメントの釣り合い：

M_A + R_B \cdot l - w \cdot \frac{l}{2} \cdot \frac{3l}{4} = 0

RBの値を代入して：

$$M_A = \frac{9}{128}wl^2$$
答え: 上記の各計算結果による
全体のポイント
  • 一端固定・他端支持はりは1次不静定構造である。
  • 荷重分布が変わる点で区間を分けて解析する。
  • 境界条件を正確に設定して未知反力を決定する。
受験材料力学のチップス
  • 部分分布荷重では荷重の作用範囲に注意する。
  • 固定端では反力とモーメントの両方が発生する。
  • 境界条件の設定が解法の鍵となる。
大問【3】

両端固定された丸棒の中間点にトルクが作用する不静定ねじり問題について、断面二次極モーメント、ねじれ角、反力トルクの比を求める。

与えられた条件:

  • 丸棒：直径d、全長l = l₁ + l₂
  • 境界条件：両端A、Bで完全固定
  • 荷重：C点（距離l₁の位置）にトルクTc
  • 材料：横弾性係数G

図3：問題3の説明図

考え方のヒント

両端固定のねじりでは全体のねじれ角がゼロという条件を使います。各区間のねじれ角の和がゼロになる関係から反力トルクを決定します。

途中式と解答

(1) 断面二次極モーメントIp

中実円軸の断面二次極モーメントは：
$$I_p = \frac{\pi d^4}{32}$$

(2) C点のねじれ角（TAで表現）

A点を基準としたC点のねじれ角を求めます。

ねじりの基本式：

\theta = \frac{TL}{GI_p}

AC区間（長さl₁）で、内部トルクがTAの場合：

$$\theta_c = \frac{T_A l_1}{G I_p}$$

(3) ねじりモーメントの比|TA|/|TB|

全体のねじれ角がゼロという条件を利用します。

ねじれ角の適合条件：

\theta_{AC} + \theta_{CB} = 0
\frac{T_A l_1}{GI_p} + \frac{(-T_B) l_2}{GI_p} = 0

これから：

T_A l_1 = T_B l_2

解答：

$$\frac{|T_A|}{|T_B|} = \frac{l_2}{l_1}$$

(4) C点のねじれ角（Tcで表現）

トルクの釣り合いを利用してTAをTcで表現します。

トルクの釣り合い：

T_A + T_B = T_c

(3)の関係 TB = TA(l₁/l₂) を代入：

T_A + T_A \frac{l_1}{l_2} = T_c
T_A = \frac{T_c l_2}{l_1 + l_2}

最終解答：

$$\theta_c = \frac{T_c l_1 l_2}{G I_p (l_1+l_2)} = \frac{T_c l_1 l_2}{G I_p l}$$
答え: 上記の各計算結果による
全体のポイント
  • 両端固定ねじりでは全体のねじれ角がゼロという条件を使う。
  • 反力トルクは作用点からの距離に反比例する。
  • 中実円軸の断面二次極モーメントは\\(I_p = \\frac{\\pi d^4}{32}\\)。
受験材料力学のチップス

  • ねじり問題では区間ごとの内部トルクを正確に把握する。
  • 変形の適合条件が不静定問題の解法の鍵となる。

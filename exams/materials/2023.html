大問【1】回転棒の遠心力による伸び

長さl、直径d、密度ρの一様な棒の両端に質量mの重りを取り付け、中央を軸として角速度ωで回転させる。この棒の遠心力による伸びを求めよ。

図1：問題1の説明図

考え方のヒント

回転による遠心力は慣性力の一種で、質量×半径×角速度²で表されます。棒自身の質量を考慮する場合は、位置によって張力が変化することに注意が必要です。

途中式と解答

(1) 端部の重りのみを考慮した場合の伸び

考え方のヒント
棒の質量を無視し、端部の重りによる遠心力のみを考慮します。この場合、棒全体に一定の張力がかかります。
遠心力の計算：
各重りの回転半径：\(r = \frac{l}{2}\)
各重りの遠心力：\(F = mr\omega^2 = m \cdot \frac{l}{2} \cdot \omega^2 = \frac{ml\omega^2}{2}\)
伸びの計算：
- 棒の断面積：\(A = \frac{\pi d^2}{4}\)
- 張力：\(T = F = \frac{ml\omega^2}{2}\)（棒全体で一定）
- 全伸び：\(\delta = \frac{TL}{AE} = \frac{F \cdot l}{AE}\) $$\delta = \frac{\frac{ml\omega^2}{2} \cdot l}{\frac{\pi d^2}{4} \cdot E} = \frac{2ml^2\omega^2}{\pi Ed^2}$$
答え: \(\delta = \frac{2ml^2\omega^2}{\pi Ed^2}\)

(2) 棒の質量も考慮した場合の伸び

考え方のヒント
棒自身の質量による遠心力も考慮します。この場合、張力は位置によって変化するため、積分による解析が必要です。
可変張力の計算：
中心からの距離xにおける張力\(T(x)\)は：
端部重りによる張力：\(\frac{ml\omega^2}{2}\)
棒の外側部分による張力：\(\int_x^{l/2} (\rho A dr) r \omega^2 = \frac{\rho A \omega^2}{2}\left(\frac{l^2}{4} - x^2\right)\)
したがって： $$T(x) = \frac{ml\omega^2}{2} + \frac{\rho A \omega^2}{2}\left(\frac{l^2}{4} - x^2\right)$$
伸びの計算：
微小要素\(dx\)の伸び：\(d\delta = \frac{T(x)}{AE}dx\)
全伸び： $$\delta = \int_{-l/2}^{l/2} \frac{T(x)}{AE} dx = 2\int_0^{l/2} \frac{T(x)}{AE} dx$$ 計算結果： $$\delta = \frac{2ml^2\omega^2}{\pi Ed^2} + \frac{\rho\omega^2 l^3}{12E}$$
答え: \(\delta = \frac{2ml^2\omega^2}{\pi Ed^2} + \frac{\rho\omega^2 l^3}{12E}\)
まとめ

回転による伸びは、端部重りによる項と棒自身の質量による項の和となります。第2項は棒の密度と長さの3乗に比例するため、長い棒では無視できない影響があります。

全体のポイント
  • 遠心力の正確な計算
  • 可変張力の扱い方
  • 積分による全体変形の算出
  • 質量分布の影響の理解
材料力学のチップス
  • 遠心力は質量×半径×角速度²
  • 分布質量では張力が位置の関数になる
  • 対称性を利用して計算を簡略化
  • 積分範囲の設定に注意
大問【2】複合軸のねじり問題（不静定）

両端が固定された複合軸が、2つのトルク\(T_1\)、\(T_2\)を受けている。軸は3つの区間（AC、CD、DB）からなり、AC区間とDB区間の材質は同じ（せん断弾性係数\(G_1\)）、CD区間は異なる材質（せん断弾性係数\(G_2\)）である。各区間の長さはそれぞれa、b、cとする。

図2：問題2の説明図

考え方のヒント

不静定ねじり問題では、静力学的つり合い式と変形適合条件を組み合わせて解きます。両端固定では全体のねじり角が0になることが重要な条件です。

途中式と解答

(1) 円形軸の極断面二次モーメント

円形断面の極断面二次モーメント：
直径dの円形断面の極断面二次モーメント\(J\)は、ねじりに対する抵抗の指標です。 $$J = \frac{\pi d^4}{32}$$
答え: \(J = \frac{\pi d^4}{32}\)

(2) 各区間のトルク

各区間で断面を切り、断面の右側の部分に注目して平衡を考えます
内部トルクの表現：
反力トルク\(T_B\)を基準として、各区間の内部トルクを表現します。

DB間

  • 右側にはB点のT_B（時計回り）のみが作用
  • 平衡条件：内力 + T_B = 0
  • DB間のねじりモーメント = -T_B

CD間

  • 右側にはD点のT₂（反時計回り）とB点のT_B（時計回り）が作用
  • 平衡条件：内力 - T₂ + T_B = 0
  • CD間のねじりモーメント = T₂ - T_B

AC間

  • 右側にはC点のT₁（反時計回り）、D点のT₂（反時計回り）、B点のT_B（時計回り）が作用
  • 平衡条件：内力 - T₁ - T₂ + T_B = 0
  • AC間のねじりモーメント = T₁ + T₂ - T_B

答え: $$T_{DB} = -T_B$$ $$T_{CD} = T_2 - T_B$$ $$T_{AC} = T_1 + T_2 - T_B$$

(3) 各区間のねじり角

ねじり角の計算：
ねじり角は\(\phi = \frac{TL}{GJ}\)の関係で計算されます。
  • 区間AC： \(\phi_{AC} = \frac{(T_1 + T_2 - T_B)a}{G_1 J}\)
  • 区間CD： \(\phi_{CD} = \frac{(T_2 - T_B)b}{G_2 J}\)
  • 区間DB： \(\phi_{DB} = \frac{-T_B c}{G_1 J}\)
答え: \(\phi_{AC} = \frac{(T_1 + T_2 - T_B)a}{G_1 J}\)、\(\phi_{CD} = \frac{(T_2 - T_B)b}{G_2 J}\)、\(\phi_{DB} = \frac{-T_B c}{G_1 J}\)

(4) 反力トルク\(T_A\)、\(T_B\)

適合条件の適用：
両端固定のため、全体のねじり角は0になります： $$\phi_{AC} + \phi_{CD} + \phi_{DB} = 0$$ この条件から\(T_B\)を求めます： $$\frac{(T_1 + T_2 - T_B)a}{G_1 J} + \frac{(T_2 - T_B)b}{G_2 J} - \frac{T_B c}{G_1 J} = 0$$ 整理すると： $$T_B = \frac{aG_2(T_1 + T_2) + bG_1 T_2}{(a+c)G_2 + bG_1}$$
静力学的つり合い：
\(T_A + T_B = T_1 + T_2\)より： $$T_A = \frac{cG_2(T_1 + T_2) + bG_1 T_1}{(a+c)G_2 + bG_1}$$
答え: $$T_A = \frac{cG_2(T_1 + T_2) + bG_1 T_1}{(a+c)G_2 + bG_1}$$ $$T_B = \frac{aG_2(T_1 + T_2) + bG_1 T_2}{(a+c)G_2 + bG_1}$$
まとめ

不静定ねじり問題では、静力学的つり合いに加えて変形適合条件が重要です。異なる材質の複合軸では、各区間のせん断弾性係数の違いが反力分布に大きく影響します。

全体のポイント
  • 不静定次数の正確な把握
  • 適合条件の設定
  • 複合材質の影響理解
  • ねじり角の計算方法
材料力学のチップス
  • 極断面二次モーメントはねじり抵抗の指標
  • 両端固定では全ねじり角が0
  • 材質の違いは弾性係数の比で考慮
  • 内部トルクの符号設定に注意

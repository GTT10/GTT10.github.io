大問【1】: 理想気体サイクル

4過程からなる理想気体サイクル（断熱圧縮→等容加熱→断熱膨張→等容冷却）について、各状態の温度、熱量・仕事、熱効率、図示平均有効圧力を求める問題です。

(1) p-V線図とT-S線図を描き、熱の出入りを示せ。

方針

サイクルの各過程（断熱圧縮、等容加熱、断熱膨張、等容冷却）の特徴を把握し、p-V線図とT-S線図を正確に描く。

考え方

断熱過程では等エントロピー変化（T-S線図で垂直線）、等容過程では体積一定（p-V線図で垂直線）の特徴を活用する。

途中式と解答
手順1: 各過程の特徴を整理し、ASCII図で線図を描く
$$pV^{\kappa} = \text{一定}$$
手順2: 状態方程式と断熱関係式を適用する
$$TV^{\kappa-1} = \text{一定}$$

ポイント

サイクル解析では各過程の物理的特徴を正確に把握することが重要

(2) 状態2,3,4の温度を T₁ を用いて表せ。

方針

断熱関係式と状態方程式を用いて各状態の温度を表現する。

途中式と解答
1→2断熱圧縮: $T_1 V_1^{\kappa-1} = T_2 V_2^{\kappa-1}$より、$T_2 = T_1 \varepsilon^{\kappa-1}$
2→3等容変化: $\frac{p_2}{T_2} = \frac{p_3}{T_3}$より、$T_3 = T_1 \varepsilon^{\kappa-1} \xi$
3→4断熱膨張: $T_4 = T_1 \xi$
$$T_2 = T_1 \varepsilon^{\kappa-1}, \quad T_3 = T_1 \varepsilon^{\kappa-1} \xi, \quad T_4 = T_1 \xi$$
解答
$$\boxed{T_2 = T_1 \varepsilon^{\kappa-1}, \quad T_3 = T_1 \varepsilon^{\kappa-1} \xi, \quad T_4 = T_1 \xi}$$

(3) 2→3の熱量 Q₂₃ と仕事 W₂₃ を求めよ。

方針

等容過程では仕事は0で、熱力学第一法則より熱量は内部エネルギー変化と等しい。

途中式と解答
等容過程: $W_{23} = 0$、$Q_{23} = \Delta U = m c_v (T_3 - T_2) = m c_v T_1 \varepsilon^{\kappa-1} (\xi - 1)$
$$W_{23} = 0, \quad Q_{23} = m c_v T_1 \varepsilon^{\kappa-1} (\xi - 1)$$
解答
$$\boxed{W_{23} = 0, \quad Q_{23} = m c_v T_1 \varepsilon^{\kappa-1} (\xi - 1)}$$
大問1のまとめ

理想気体サイクルの解析では、各過程の物理的特徴を把握し、線図を活用することで体系的に状態量を求めることができます。

重要公式・関係式
  • 断熱関係式: pV^κ = 一定
  • 状態方程式: pV = mRT
  • 熱効率: η = W_net/Q_in
解法のポイント
  • 各過程の特徴を明確に把握する
  • 線図を正確に描いて視覚化する
  • エネルギー収支を厳密に計算する
注意点・間違いやすいポイント
  • 断熱と等容の混同に注意
  • 符号の取り扱いに注意（吸熱・放熱）
熱力学のチップス
  • 理想気体サイクルは内燃機関の基礎
  • エントロピーは不可逆性の指標
大問【2】: ボイラの熱力学
ボイラ内で水を加熱・蒸発させる過程において、燃焼ガスとの熱交換によるエネルギー収支、エントロピー変化、エクセルギー変化を解析する問題です。

(1) 水の温度上昇に要する熱量を求めよ。

方針

比熱の定義を用いて温度上昇による熱量を計算する。

途中式と解答
温度上昇: $\dot{Q}_{\text{temp}} = \dot{m}_w c_w (T_{w2} - T_{w1})$
$$\dot{Q}_{\text{temp}} = \dot{m}_w c_w (T_{w2} - T_{w1})$$
解答
$$\boxed{\dot{Q}_{\text{temp}} = \dot{m}_w c_w (T_{w2} - T_{w1})}$$

(2) 水の蒸発に要する熱量を求めよ。

方針

蒸発潜熱を用いて相変化の熱量を計算する。

途中式と解答
蒸発: $\dot{Q}_{\text{vap}} = \dot{m}_w r$
$$\dot{Q}_{\text{vap}} = \dot{m}_w r$$
解答
$$\boxed{\dot{Q}_{\text{vap}} = \dot{m}_w r}$$
大問2のまとめ

ボイラの熱力学では、エネルギー保存則に基づいた収支計算と、エントロピー・エクセルギー解析により効率評価が可能です。

エネルギー関係式
  • 比熱による加熱: Q = mcΔT
  • 蒸発潜熱: Q = mr
  • エントロピー変化: ΔS = Q/T
実用ポイント
  • ボイラは火力発電の基幹設備
  • エクセルギーは有効エネルギー

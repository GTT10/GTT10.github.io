大問【1】: 容器内理想気体の状態変化

初期状態が$p_1 = 400$ kPa、$V_1 = 1$ m³、$T_1 = 500$ Kの理想気体（$\kappa = 1.4$、$R = 0.25$ kJ/(kg·K)）について、各種状態変化を考える。以下の問いに答えよ。

(1) 容器内に封入されているガスの質量を求めよ。

方針

理想気体の状態方程式$pV = mRT$を用いて質量$m$を求めます。

解答
$$m = \frac{p_1 V_1}{R T_1} = \frac{400 \times 1}{0.25 \times 500} = \frac{400}{125} = 3.2 \text{ kg}$$
解答
$$\boxed{m = 3.2 \text{ kg}}$$

(2) 等圧変化で容積を$V_2 = 2$ m³まで増加させたとき、外部になす仕事$W$、与えられた熱量$Q$、エントロピー変化$\Delta S$を求めよ。

方針

等圧過程では$W = p(V_2 - V_1)$、$Q = mc_p\Delta T$、$\Delta S = mc_p\ln(T_2/T_1)$を使用します。

途中式と解答
外部になす仕事：
$$W = p_1(V_2 - V_1) = 400 \times (2 - 1) = 400 \text{ kJ}$$
与えられた熱量：

定圧比熱：$c_p = \frac{\kappa R}{\kappa - 1} = \frac{1.4 \times 0.25}{0.4} = 0.875$ kJ/(kg·K)

温度変化：$T_2 = T_1 \frac{V_2}{V_1} = 500 \times 2 = 1000$ K

$$Q = mc_p(T_2 - T_1) = 3.2 \times 0.875 \times (1000 - 500) = 1400 \text{ kJ}$$
エントロピー変化：
$$\Delta S = mc_p\ln\frac{T_2}{T_1} = 3.2 \times 0.875 \times \ln 2 = 2.8 \times 0.693 = 1.94 \text{ kJ/K}$$
解答
$$\boxed{ \begin{aligned} W &= 400 \text{ kJ} \\ Q &= 1400 \text{ kJ} \\ \Delta S &= 1.94 \text{ kJ/K} \end{aligned} }$$

(3) 等温変化で容積を$V_2 = 4$ m³まで増加させたとき、外部になす仕事$W$、与えられた熱量$Q$、エントロピー変化$\Delta S$を求めよ。

方針

等温過程では$W = mRT\ln(V_2/V_1)$、$Q = W$、$\Delta S = mR\ln(V_2/V_1)$を使用します。

途中式と解答
外部になす仕事：
$$W = mRT_1\ln\frac{V_2}{V_1} = 3.2 \times 0.25 \times 500 \times \ln 4$$ $$= 400 \times 2\ln 2 = 400 \times 1.386 = 554.4 \text{ kJ}$$
与えられた熱量：

等温変化のため：$Q = W = 554.4$ kJ

エントロピー変化：
$$\Delta S = mR\ln\frac{V_2}{V_1} = 3.2 \times 0.25 \times \ln 4 = 0.8 \times 1.386 = 1.11 \text{ kJ/K}$$
解答
$$\boxed{ \begin{aligned} W &= 554.4 \text{ kJ} \\ Q &= 554.4 \text{ kJ} \\ \Delta S &= 1.11 \text{ kJ/K} \end{aligned} }$$

(4) 可逆断熱変化で容積を$V_2 = 1.93$ m³まで増加させたとき、外部になす仕事$W$、与えられた熱量$Q$、エントロピー変化$\Delta S$を求めよ。ただし$(1.93)^{0.4} \approx 1.3$とする。

方針

断熱過程では$Q = 0$、$\Delta S = 0$、$W = -mc_v\Delta T$を使用します。温度は$TV^{\kappa-1} = \text{一定}$から求めます。

途中式と解答
温度変化：

定容比熱：$c_v = \frac{R}{\kappa - 1} = \frac{0.25}{0.4} = 0.625$ kJ/(kg·K)

$$T_2 = T_1\left(\frac{V_1}{V_2}\right)^{\kappa-1} = 500 \times \left(\frac{1}{1.93}\right)^{0.4} = \frac{500}{1.3} = 384.6 \text{ K}$$
外部になす仕事：
$$W = -mc_v(T_2 - T_1) = -3.2 \times 0.625 \times (384.6 - 500)$$ $$= -2 \times (-115.4) = 230.8 \text{ kJ}$$
解答
$$\boxed{ \begin{aligned} W &= 230.8 \text{ kJ} \\ Q &= 0 \text{ kJ} \\ \Delta S &= 0 \text{ kJ/K} \end{aligned} }$$
重要公式・関係式
  • 理想気体状態方程式: $pV = mRT$
  • 等圧過程: $W = p\Delta V$, $Q = mc_p\Delta T$
  • 等温過程: $W = mRT\ln(V_2/V_1)$, $Q = W$
  • 断熱過程: $Q = 0$, $W = -mc_v\Delta T$
  • エントロピー変化: $\Delta S = mc_p\ln(T_2/T_1) + mR\ln(V_2/V_1)$
解法のポイント
  • 各過程の特徴を正確に把握
  • 比熱比$\kappa$から$c_p$、$c_v$を計算
  • 状態方程式による温度・圧力変化の追跡
  • 第一法則によるエネルギー収支の確認
注意点
  • 各過程で一定に保たれる量の確認
  • 比熱の定義と関係式
  • 断熱過程での$Q = 0$、$\Delta S = 0$
大問【2】: ブレイトンサイクル
理想気体によるブレイトンサイクル（ガスタービンサイクル）について考える。圧力比を$\phi = p_2/p_1$、容積比を$\alpha = V_3/V_2$とし、以下の問いに答えよ。

(1) このサイクルの$p-V$線図および$T-S$線図を描け。

方針

ブレイトンサイクルは1→2（断熱圧縮）、2→3（等圧加熱）、3→4（断熱膨張）、4→1（等圧冷却）の4過程です。

線図の描画と解答

p-V線図

圧力 p ↑
|        2-------3 (等圧加熱)
|       /|       |\
|      / |       | \
|     /  |       |  \
|    / (断熱圧縮) | (断熱膨張)
|   /    |       |   \
|  /     |       |    \
| /      |       |     \
|1-------+-------+-----4----→ 容積 V
 V₁      V₂      V₃    V₄
         (等圧冷却)

T-S線図

温度 T ↑
|        2-------3 (等圧加熱)
|        |       |
|        |       |
| (断熱) |       | (断熱)
|        |       |
|        |       |
|        1-------4 (等圧冷却)
+-------------------------→ エントロピー S
        S₁=S₂   S₃=S₄
解答

p-V線図: 1→2（断熱圧縮曲線）、2→3（水平線）、3→4（断熱膨張曲線）、4→1（水平線）

T-S線図: 1→2（垂直線）、2→3（右上がり曲線）、3→4（垂直線）、4→1（左下がり曲線）

(2) 各過程における熱量の出入りを求めよ。

方針

等圧過程2→3で吸熱、等圧過程4→1で放熱します。断熱過程では熱の出入りはありません。

解答
$$\boxed{ \begin{aligned} Q_{\text{in}} &= mc_p(T_3 - T_2) \quad (\text{2→3の等圧加熱}) \\ Q_{\text{out}} &= mc_p(T_4 - T_1) \quad (\text{4→1の等圧冷却}) \end{aligned} }$$

(3) 状態2、3、4の温度を状態1の温度$T_1$と$\phi$、$\alpha$を用いて表せ。

方針

断熱過程では$T_1V_1^{\kappa-1} = T_2V_2^{\kappa-1}$、等圧過程では$V/T = \text{一定}$の関係を使用します。

途中式と解答
状態2（断熱圧縮）：
$$T_2 = T_1\left(\frac{p_2}{p_1}\right)^{\frac{\kappa-1}{\kappa}} = T_1\phi^{\frac{\kappa-1}{\kappa}}$$
状態3（等圧加熱）：
$$T_3 = T_2\frac{V_3}{V_2} = T_1\phi^{\frac{\kappa-1}{\kappa}}\alpha$$
状態4（断熱膨張）：
$$T_4 = T_3\left(\frac{p_4}{p_3}\right)^{\frac{\kappa-1}{\kappa}} = T_1\phi^{\frac{\kappa-1}{\kappa}}\alpha \cdot \phi^{-\frac{\kappa-1}{\kappa}} = T_1\alpha$$
解答
$$\boxed{ \begin{aligned} T_2 &= T_1\phi^{\frac{\kappa-1}{\kappa}} \\ T_3 &= T_1\phi^{\frac{\kappa-1}{\kappa}}\alpha \\ T_4 &= T_1\alpha \end{aligned} }$$

(4) このサイクルの熱効率を求めよ。

方針

熱効率は$\eta = 1 - Q_{\text{out}}/Q_{\text{in}}$で定義されます。(3)の結果を使用して計算します。

途中式と解答
$$\eta = 1 - \frac{Q_{\text{out}}}{Q_{\text{in}}} = 1 - \frac{T_4 - T_1}{T_3 - T_2}$$ $$= 1 - \frac{T_1\alpha - T_1}{T_1\phi^{\frac{\kappa-1}{\kappa}}\alpha - T_1\phi^{\frac{\kappa-1}{\kappa}}}$$ $$= 1 - \frac{T_1(\alpha - 1)}{T_1\phi^{\frac{\kappa-1}{\kappa}}(\alpha - 1)} = 1 - \frac{1}{\phi^{\frac{\kappa-1}{\kappa}}}$$
解答
$$\boxed{\eta = 1 - \phi^{\frac{1-\kappa}{\kappa}}}$$
ブレイトンサイクルの特徴
  • ガスタービンの理論サイクル
  • 開放サイクルが一般的
  • 圧力比が効率を決定
  • 高温での材料制約が課題
重要な関係式
  • 断熱過程: $Tp^{\frac{1-\kappa}{\kappa}} = \text{一定}$
  • 等圧過程: $V/T = \text{一定}$
  • 効率は圧力比のみに依存
大問【3】: ノズル内の等エントロピー流れ
理想気体がノズル内を等エントロピー（断熱可逆）的に流れる場合を考える。入口状態を1、出口状態を2とし、以下の問いに答えよ。

(1) エネルギー保存則を適用して、出口流速$u_2$を求めよ。ただし、入口流速は無視できるものとし、位置エネルギーの変化も無視する。

方針

定常流れの機械的エネルギー保存則$h_1 + u_1^2/2 = h_2 + u_2^2/2$を適用します。

途中式と解答

入口流速$u_1 \approx 0$として：

$$h_1 + \frac{u_1^2}{2} = h_2 + \frac{u_2^2}{2}$$ $$h_1 = h_2 + \frac{u_2^2}{2}$$ $$u_2^2 = 2(h_1 - h_2)$$
解答
$$\boxed{u_2 = \sqrt{2(h_1 - h_2)}}$$

(2) 理想気体について、比エンタルピー差$h_1 - h_2$を温度で表せ。

方針

理想気体では比エンタルピーは温度の関数のみで、$h = c_pT$の関係を使用します。

解答
$$\boxed{h_1 - h_2 = c_p(T_1 - T_2)}$$

(3) 臨界圧力$p^*$を入口圧力$p_1$と比熱比$\kappa$で表せ。

方針

臨界状態では音速となり、臨界圧力比は比熱比のみの関数として表されます。

解答
$$\boxed{p^* = p_1\left(\frac{2}{\kappa + 1}\right)^{\frac{\kappa}{\kappa-1}}}$$

(4) $p_1 = 0.25$ MPa、$p_2 = 0.10$ MPa、$\kappa = 1.4$のとき、適切なノズル形状を選択し、理由を述べよ。

方針

圧力比と臨界圧力比を比較して、必要なノズル形状を判定します。

計算と判定
圧力比：
$$\frac{p_2}{p_1} = \frac{0.10}{0.25} = 0.4$$
臨界圧力比：
$$\left(\frac{2}{\kappa + 1}\right)^{\frac{\kappa}{\kappa-1}} = \left(\frac{2}{2.4}\right)^{\frac{1.4}{0.4}} = \left(\frac{5}{6}\right)^{3.5} \approx 0.528$$
解答

適切なノズル形状：末広ノズル

理由：$0.4 < 0.528$のため圧力比が臨界値未満であり、超音速流となる。流速を連続的に増加させるには拡大管（末広ノズル）が必要。

$$\boxed{\text{末広ノズル}}$$
ノズル流れの基本
  • 連続の式とエネルギー保存則の適用
  • 等エントロピー過程の仮定
  • 臨界状態での音速達成
  • マッハ数による流れの分類
ノズル形状の選択
  • $p_2/p_1 > p^*/p_1$：先細ノズル
  • $p_2/p_1 < p^*/p_1$：末広ノズル
  • 超音速流では拡大管で加速

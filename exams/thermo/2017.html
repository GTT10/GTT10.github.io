大問【1】: カルノーサイクル（理想気体）

理想気体を作動流体とするカルノーサイクルについて考える。高温熱源温度$T_H$、低温熱源温度$T_L$として、以下の問いに答えよ。

(1) このサイクルの$p-V$線図および$T-S$線図を描け。

方針

カルノーサイクルは2つの等温過程と2つの断熱過程から構成される理想的な可逆サイクルです。

線図の描画と解答

p-V線図

圧力 p ↑
|    1
|     \
|      \2 (等温膨張, T_H)
|       \\
|        \\
|         \\3 (断熱膨張)
|          \\
|           4
|          /
|         /
|        /
|       / (断熱圧縮)
|      /
|     /
|    /
+------------------→ 容積 V

T-S線図

温度 T ↑
|
|  T_H: 1━━━━━━━2
|       |        |
|       | (断熱) | (断熱)
|       |        |
|  T_L: 4━━━━━━━3
|
+-------------------------→ エントロピー S
       S₁=S₄    S₂=S₃
解答

サイクル構成：

  • 1→2: 等温膨張（$T_H$一定、吸熱）
  • 2→3: 断熱膨張（$S$一定）
  • 3→4: 等温圧縮（$T_L$一定、放熱）
  • 4→1: 断熱圧縮（$S$一定）

(2) 各過程における熱量の出入りを求めよ。

方針

等温過程では$Q = mRT\ln(V_2/V_1)$、断熱過程では$Q = 0$を使用します。

解答
$$\boxed{ \begin{aligned} Q_{\text{in}} &= mRT_H\ln\frac{V_2}{V_1} \quad (\text{1→2の等温膨張}) \\ Q_{\text{out}} &= mRT_L\ln\frac{V_3}{V_4} \quad (\text{3→4の等温圧縮}) \end{aligned} }$$

断熱過程（2→3、4→1）では熱の出入りはありません。

(3) 各過程におけるエントロピー変化を求めよ。

方針

エントロピー変化は$\Delta S = Q_{\text{rev}}/T$で計算します。断熱過程では$\Delta S = 0$です。

解答
$$\boxed{ \begin{aligned} \Delta S_{12} &= \frac{Q_{\text{in}}}{T_H} = mR\ln\frac{V_2}{V_1} \quad (\text{増加}) \\ \Delta S_{23} &= 0 \quad (\text{断熱}) \\ \Delta S_{34} &= \frac{Q_{\text{out}}}{T_L} = mR\ln\frac{V_4}{V_3} \quad (\text{減少}) \\ \Delta S_{41} &= 0 \quad (\text{断熱}) \end{aligned} }$$

サイクル全体では$\Delta S_{\text{total}} = 0$（可逆サイクル）

(4) このサイクルの熱効率を求めよ。

方針

熱効率は$\eta = 1 - Q_{\text{out}}/Q_{\text{in}}$で定義されます。断熱過程の関係を用いて簡略化します。

途中式と解答

断熱過程の関係から：$TV^{\kappa-1} = \text{一定}$

$$T_H V_2^{\kappa-1} = T_L V_3^{\kappa-1} \quad \text{と} \quad T_H V_1^{\kappa-1} = T_L V_4^{\kappa-1}$$

これより：$\frac{V_2}{V_1} = \frac{V_3}{V_4}$

$$\eta = 1 - \frac{Q_{\text{out}}}{Q_{\text{in}}} = 1 - \frac{T_L\ln(V_3/V_4)}{T_H\ln(V_2/V_1)}$$ $$= 1 - \frac{T_L}{T_H} \quad (\because V_2/V_1 = V_3/V_4)$$
解答
$$\boxed{\eta = 1 - \frac{T_L}{T_H}}$$

これがカルノー効率で、同一熱源間で動作するすべての熱機関の理論的上限です。

重要公式・関係式
  • 等温過程: $Q = mRT\ln(V_2/V_1)$
  • 断熱過程: $Q = 0$, $TV^{\kappa-1} = \text{一定}$
  • エントロピー変化: $\Delta S = Q_{\text{rev}}/T$
  • カルノー効率: $\eta = 1 - T_L/T_H$
解法のポイント
  • 可逆サイクルの特徴理解
  • 断熱過程での容積比の関係
  • T-S線図での矩形表現
  • 理論的上限としての意義
注意点
  • 可逆過程の前提条件
  • 実用サイクルとの効率差
  • 温度は絶対温度を使用
大問【2】: 混合気体の性質
温度300 K、圧力0.1 MPaで、CH₄（1 m³）、O₂（2 m³）、CO₂（3 m³）、N₂（4 m³）の混合気体について、以下の問いに答えよ。ただし、各気体の分子量は、CH₄: 16、O₂: 32、CO₂: 44、N₂: 28とする。

(1) 混合気体の平均分子量$M_{\text{avg}}$を求めよ。

方針

各気体のモル数を計算し、総質量を総モル数で割って平均分子量を求めます。

途中式と解答
各気体のモル数計算：

理想気体の状態方程式$pV = nRT$より：

$$n_{\text{CH}_4} = \frac{0.1 \times 10^6 \times 1}{8.314 \times 10^3 \times 300} = 0.0401 \text{ kmol}$$ $$n_{\text{O}_2} = \frac{0.1 \times 10^6 \times 2}{8.314 \times 10^3 \times 300} = 0.0802 \text{ kmol}$$ $$n_{\text{CO}_2} = \frac{0.1 \times 10^6 \times 3}{8.314 \times 10^3 \times 300} = 0.1203 \text{ kmol}$$ $$n_{\text{N}_2} = \frac{0.1 \times 10^6 \times 4}{8.314 \times 10^3 \times 300} = 0.1604 \text{ kmol}$$

総モル数：$n_{\text{total}} = 0.0401 + 0.0802 + 0.1203 + 0.1604 = 0.4010$ kmol

各気体の質量計算：
$$m_{\text{CH}_4} = 0.0401 \times 16 = 0.6416 \text{ kg}$$ $$m_{\text{O}_2} = 0.0802 \times 32 = 2.5664 \text{ kg}$$ $$m_{\text{CO}_2} = 0.1203 \times 44 = 5.2932 \text{ kg}$$ $$m_{\text{N}_2} = 0.1604 \times 28 = 4.4912 \text{ kg}$$

総質量：$m_{\text{total}} = 0.6416 + 2.5664 + 5.2932 + 4.4912 = 12.9924$ kg

解答
$$M_{\text{avg}} = \frac{m_{\text{total}}}{n_{\text{total}}} = \frac{12.9924}{0.4010} = \boxed{32.40 \text{ kg/kmol}}$$

(2) 混合気体の平均ガス定数$R_{\text{avg}}$を求めよ。

方針

平均ガス定数は万能ガス定数を平均分子量で割ったものです。

解答
$$R_{\text{avg}} = \frac{R_{\text{universal}}}{M_{\text{avg}}} = \frac{8.314}{32.40} = \boxed{0.2566 \text{ kJ/(kg·K)}}$$

(3) 混合気体の全質量を求めよ。

解答

(1)で計算した総質量：

$$\boxed{13.0 \text{ kg}} \quad (\text{有効数字3桁})$$
混合気体の基本
  • 各成分は理想気体として扱う
  • ダルトンの分圧の法則が成立
  • 平均分子量は質量基準
  • モル分率と体積分率は等しい
計算のポイント
  • 状態方程式による各成分のモル数計算
  • 分子量を用いた質量計算
  • 平均値の定義の正確な適用
  • 有効数字の適切な処理
大問【3】: 再生蒸気サイクル
再生器付き蒸気サイクルについて考える。タービンから一部の蒸気を抽気し、給水を予熱する再生サイクルについて、以下の問いに答えよ。

(1) 抽気量の割合$m$を求めよ。ただし、混合室でのエネルギー収支を考慮せよ。

方針

混合室でのエネルギー収支式を立て、抽気量の割合を求めます。ポンプ仕事は通常無視できます。

途中式と解答

混合室でのエネルギー収支（単位質量の蒸気あたり）：

$$h_6 = m \cdot h_5 + (1-m) \cdot h_2$$

ここで：

  • $m$: 抽気量の割合
  • $h_6$: 混合後の給水エンタルピー
  • $h_5$: 抽気蒸気のエンタルピー
  • $h_2$: 復水器出口の給水エンタルピー
解答

上式を$m$について解くと：

$$\boxed{m = \frac{h_6 - h_2}{h_5 - h_2}}$$

(2) このサイクルの熱効率$\eta$を求めよ。

方針

熱効率は$\eta = W_{\text{net}}/Q_{\text{in}}$で定義されます。正味仕事と吸熱量を求めます。

途中式と解答
吸熱量（ボイラーでの加熱）：
$$Q_{\text{in}} = h_1 - h_6$$
正味仕事（タービン仕事 - ポンプ仕事）：
$$W_{\text{net}} = (h_1 - h_5) + (1-m)(h_5 - h_4)$$

第1項：高圧段タービン仕事、第2項：低圧段タービン仕事

解答
$$\boxed{\eta = \frac{(h_1 - h_5) + (1-m)(h_5 - h_4)}{h_1 - h_6}}$$

(3) タービンが不可逆の場合、$T-S$線図はどのように変化するか説明せよ。

方針

不可逆タービンでは摩擦によりエントロピーが増加し、理想的な等エントロピー線から右側にずれます。

線図と解答

不可逆タービンのT-S線図

温度 T ↑
|           1
|           |\
|           | \
|           |  \5* (不可逆膨張)
|           |   \\
|           |    \\
|           6     4*
|           |     |
|           |     |
|           2-----+---→ エントロピー S
           S₁    S₂>S₁
解答

不可逆タービンの影響：

  • 等エントロピー線（垂直線）から右向きにずれる
  • 摩擦によりエントロピーが増加（$S_2 > S_1$）
  • タービン出口エンタルピーが増加し、仕事量が減少
  • 結果として熱効率が低下する

実際のタービン効率$\eta_T = (h_1 - h_{5*})/(h_1 - h_{5s})$で定義される。

再生サイクルの特徴
  • 給水の予熱により効率向上
  • 抽気量の最適化が重要
  • 複数段の再生も可能
  • 実用発電所で広く採用
解析のポイント
  • 混合室でのエネルギー収支
  • 質量収支の同時考慮
  • タービン効率の影響
  • T-S線図での可視化

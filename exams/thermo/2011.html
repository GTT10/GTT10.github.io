大問【1】: 理想気体サイクル（オットーサイクル）
問題文再現

質量$m$の理想気体が、断熱圧縮（状態1→2）、定容加熱（状態2→3）、断熱膨張（状態3→4）、定容冷却（状態4→1）のサイクルを行う。圧縮比$\varepsilon = V_1/V_2$とする。

(1) このサイクルの名称を答えよ。

方針

各過程の組み合わせから、該当するサイクル名を特定します。

解答
$$\boxed{\text{オットーサイクル}}$$

理由：1→2: 断熱圧縮、2→3: 定容加熱、3→4: 断熱膨張、4→1: 定容冷却の組み合わせは、ガソリンエンジンの基本理論サイクルであるオットーサイクルに一致します。

(2) 熱効率を導出せよ。

方針

熱効率$\eta = 1 - \frac{\text{放出熱量}}{\text{吸収熱量}}$を用い、断熱関係式で温度を整理します。

途中式と解答
吸熱量$Q_{\text{in}}$（2→3: 定容加熱）：
$$Q_{\text{in}} = m c_v (T_3 - T_2)$$
放熱量$Q_{\text{out}}$（4→1: 定容冷却）：
$$Q_{\text{out}} = m c_v (T_4 - T_1)$$
断熱関係式の適用：
  • 断熱過程1→2: $T_2 = T_1 \varepsilon^{k-1}$
  • 断熱過程3→4: $T_3 = T_4 \varepsilon^{k-1}$
温度比の整理：

$V_4 = V_1$、$V_3 = V_2$より、$T_4/T_1 = T_3/T_2$

$$\frac{T_4 - T_1}{T_3 - T_2} = \frac{T_1(T_4/T_1 - 1)}{T_2(T_3/T_2 - 1)} = \frac{T_1}{T_2} = \frac{1}{\varepsilon^{k-1}}$$
解答
$$\eta = 1 - \frac{Q_{\text{out}}}{Q_{\text{in}}} = 1 - \frac{1}{\varepsilon^{k-1}}$$ $$\boxed{\eta = 1 - \frac{1}{\varepsilon^{k-1}}}$$

(3) 実際の4ストローク機関での熱効率向上策を述べよ。

解答

主な向上策：

  1. 圧縮比$\varepsilon$の向上
    • 効率は$\varepsilon$の増加で向上
    • 制約：ノッキング防止のため燃料オクタン価を調整
  2. 比熱比$k$の増加
    • 燃焼ガス組成の最適化
    • 排ガス再循環率の調整
  3. ポンピング損失の低減
    • 可変バルブ機構でバルブタイミング最適化
    • 過給器（ターボチャージャー）の導入
オットーサイクル
  • ガソリンエンジンの理論サイクル
  • 効率は圧縮比と比熱比で決まる
  • 断熱過程：$TV^{k-1} = \text{const}$
  • 定容過程：$P/T = \text{const}$
効率向上のポイント
  • 圧縮比向上が最も効果的
  • ノッキング限界が制約
  • 実機では機械損失も考慮
  • ポンピング損失の影響大
注意点
  • 理論効率と実際の効率は異なる
  • 断熱関係式の正確な適用
  • 温度比の整理に注意
大問【2】: カルノーサイクルと伝熱損失
問題文再現
高温熱源$T_H$、低温熱源$T_L$間のカルノーサイクル。作業媒質の最高温度$T_I$で、熱供給量$Q = K(T_H - T_I)$の伝熱損失がある。

(1) カルノーサイクルの熱効率を求めよ。

解答
$$\boxed{\eta_{\text{carnot}} = 1 - \frac{T_L}{T_H}}$$

理由：カルノーサイクルは可逆サイクルであり、効率は高低温熱源の温度のみで決まります。

(2) 伝熱損失がある場合の熱効率と有効エネルギーを求めよ。

方針

作業媒質の最高温度$T_I$でカルノー効率を適用し、$Q = K(T_H - T_I)$の関係を使います。

途中式と解答
熱効率$\eta$：

カルノー効率を$T_I$で適用：

$$\eta = 1 - \frac{T_L}{T_I}$$

$Q = K(T_H - T_I)$より$T_I = T_H - \frac{Q}{K}$を代入：

$$\eta = 1 - \frac{T_L}{T_H - \frac{Q}{K}}$$
有効エネルギー$W$（仕事出力）：
$$W = \eta Q = Q \left(1 - \frac{T_L}{T_H - \frac{Q}{K}}\right)$$
解答
  • 熱効率：$\boxed{\eta = 1 - \frac{T_L}{T_H - \frac{Q}{K}}}$
  • 有効エネルギー：$\boxed{W = Q \left(1 - \frac{T_L}{T_H - \frac{Q}{K}}\right)}$

(3) エントロピー生成が最大となる$Q$を求めよ。

方針

不可逆性によるエントロピー生成を最大化する条件を求めます。

途中式と解答
エントロピー生成$\sigma$：
  • 高温熱源のエントロピー減少：$-\frac{Q}{T_H}$
  • 低温熱源のエントロピー増加：$\frac{Q-W}{T_L}$
$$\sigma = \frac{Q-W}{T_L} - \frac{Q}{T_H}$$
最適化条件：

エントロピー最大は「仕事出力$W$が最小」と等価で、$T_I$が$T_H$と$T_L$の幾何平均となる点：

$$T_I = \sqrt{T_H T_L}$$
解答

$T_I = T_H - \frac{Q}{K}$より：

$$\boxed{Q = K(T_H - \sqrt{T_H T_L})}$$
カルノーサイクル
  • 理想的な可逆サイクル
  • 効率は温度比のみで決まる
  • 実際には不可逆損失が存在
  • エントロピー生成の概念重要
伝熱損失の影響
  • 作業媒質温度$T_I < T_H$
  • 効率低下の主因
  • 熱交換器設計で重要
  • 最適化問題として扱える
重要な関係式
  • カルノー効率：$\eta = 1 - T_L/T_H$
  • 伝熱関係：$Q = K(T_H - T_I)$
  • エントロピー生成：$\sigma > 0$
  • 最適温度：$T_I = \sqrt{T_H T_L}$
大問【3】: 蒸気表を用いた計算
問題文再現
蒸気表を用いて各種水蒸気の状態量と状態を求める問題。飽和状態、湿り蒸気、過熱蒸気の判定と計算を行う。

(1) 0.5MPaでの沸騰温度を求めよ。

解答

蒸気表より：

$$\boxed{151.8°\text{C}}$$

(2) 0.5MPaでの蒸発潜熱を求めよ。

計算過程
蒸気表より：
  • 飽和液エンタルピー：$h_f = 640.1$ kJ/kg
  • 飽和蒸気エンタルピー：$h_g = 2748.1$ kJ/kg
解答
$$h_{fg} = h_g - h_f = 2748.1 - 640.1 = \boxed{2108.0 \text{ kJ/kg}}$$

(3) 0.5MPa、乾き度0.4の湿り蒸気のエンタルピーを求めよ。

計算過程
湿り蒸気のエンタルピー公式：
$$h = h_f + x \cdot h_{fg}$$

ここで、$x = 0.4$（乾き度）

解答
$$h = 640.1 + 0.4 \times 2108.0 = 640.1 + 843.2 = \boxed{1483.3 \text{ kJ/kg}}$$

(4) 1.0MPa、200℃の蒸気の状態を判定せよ。

判定過程
飽和温度との比較：
  • 1.0MPaの飽和温度：179.9℃
  • 蒸気温度：200℃
  • 200℃ > 179.9℃ → 過熱蒸気
解答
$$\boxed{\text{過熱蒸気}}$$

理由：飽和温度179.9℃より高いため、液体を含まない過熱状態です。

(5) 1.0MPa、100℃の圧縮水を300℃過熱蒸気にする必要熱量を求めよ。

計算過程
各状態のエンタルピー：
  • 圧縮水（100℃, 1.0MPa）：$h_1 \approx 419.1$ kJ/kg（飽和液近似）
  • 過熱蒸気（300℃, 1.0MPa）：$h_2 = 3051.2$ kJ/kg（過熱蒸気表より）
解答
$$Q = h_2 - h_1 = 3051.2 - 419.1 = \boxed{2632.1 \text{ kJ/kg}}$$
蒸気表の読み方
  • 飽和温度・圧力は一対一対応
  • $h_f$：飽和液エンタルピー
  • $h_g$：飽和蒸気エンタルピー
  • $h_{fg} = h_g - h_f$：蒸発潜熱
状態判定のポイント
  • 温度と飽和温度の比較
  • 湿り蒸気：$h = h_f + x h_{fg}$
  • 過熱蒸気：飽和温度より高温
  • 圧縮液：飽和液近似可能
重要な関係式
  • 湿り蒸気：$h = h_f + x h_{fg}$
  • 蒸発潜熱：$h_{fg} = h_g - h_f$
  • 乾き度：$x = \frac{h - h_f}{h_{fg}}$
  • 必要熱量：$Q = \Delta h$

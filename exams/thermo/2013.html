大問【1】: 熱力学の基礎

熱力学第一法則から基本関係式を導出し、理想気体および半理想気体の性質について解析せよ。

(1) $du = c_v dT$および$dh = c_p dT$を導出せよ。

方針

熱力学第一法則とエンタルピーの定義、定容・定圧比熱の定義を用いて導出します。

途中式と解答
比内部エネルギー$du$の導出：

熱力学第一法則：$du = \delta q - \delta w$

定容過程（$dv = 0$）：$\delta w = P dv = 0$

定容比熱の定義：$\delta q = c_v dT$

$$\boxed{du = c_v dT}$$
比エンタルピー$dh$の導出：

エンタルピーの定義：$h = u + Pv$

全微分：$dh = du + P dv + v dP$

定圧過程（$dP = 0$）：$dh = du + P dv$

熱力学第一法則より：$du = \delta q - P dv$

代入：$dh = (\delta q - P dv) + P dv = \delta q$

定圧比熱の定義：$\delta q = c_p dT$

$$\boxed{dh = c_p dT}$$

(2) $c_p$と$c_v$の差を求めよ。

方針

理想気体の状態方程式とエンタルピーの定義を組み合わせます。

途中式と解答

理想気体の状態方程式：$Pv = RT$

エンタルピーの定義：$h = u + Pv = u + RT$

微分：$dh = du + R dT$

(1)の結果を代入：$c_p dT = c_v dT + R dT$

解答
$$\boxed{c_p - c_v = R}$$

(3) 理想気体の比エントロピーを求めよ。
(i) 定容変化　(ii) 定圧変化

方針

エントロピーの定義$ds = \delta q_{\text{rev}}/T$と(1)の結果を使用します。

解答

(i) 定容変化：

$$ds = \frac{\delta q_{\text{rev}}}{T} = \frac{c_v dT}{T}$$ $$\boxed{s = c_v \ln T + \text{const}}$$

(ii) 定圧変化：

$$ds = \frac{c_p dT}{T}$$ $$\boxed{s = c_p \ln T + \text{const}}$$

(4) T-s線図を描き、定容線と定圧線の傾きの違いを説明せよ。

線図と解答

T-s線図

温度 T ↑
|           （定圧線: 傾き緩やか）
|          ／
|         ／（定容線: 傾き急）
|        ／
|       ／
|      ／
|     ／
+-------------------→ 比エントロピー s
解答

傾きの比較：

  • 定容線の傾き：$\frac{dT}{ds} = \frac{T}{c_v}$
  • 定圧線の傾き：$\frac{dT}{ds} = \frac{T}{c_p}$

理由：$c_p > c_v$のため、定圧線の傾きが定容線より緩やかになります。

(5) 半理想気体（$c_v = a + bT + cT^2$）の比エントロピーを求めよ。
(i) 定容変化　(ii) 定圧変化

方針

比熱が温度の関数として与えられているので、積分を実行します。

解答

(i) 定容変化：

$$ds = \frac{c_v dT}{T} = \left(\frac{a}{T} + b + cT\right) dT$$ $$\boxed{s = a \ln T + bT + \frac{c}{2}T^2 + \text{const}}$$

(ii) 定圧変化：

$c_p = c_v + R = a + R + bT + cT^2$より

$$ds = \frac{c_p dT}{T} = \left(\frac{a + R}{T} + b + cT\right) dT$$ $$\boxed{s = (a + R) \ln T + bT + \frac{c}{2}T^2 + \text{const}}$$
重要公式・関係式
  • 熱力学第一法則：$du = \delta q - \delta w$
  • エンタルピー定義：$h = u + Pv$
  • 比熱関係：$c_p - c_v = R$
  • エントロピー定義：$ds = \delta q_{\text{rev}}/T$
解法のポイント
  • 定容・定圧過程の特徴理解
  • 比熱の定義の正確な適用
  • 状態方程式の活用
  • 積分計算の実行
注意点
  • 定容と定圧の区別
  • 理想気体の仮定
  • 積分定数の取り扱い
大問【2】: 有効エネルギーと無効エネルギー
周囲温度$T_0 = 300$ K、平均比熱$c_p = 4.0$ kJ/(kg·K)、$\ln 1.5 = 0.4$として、水の加熱過程における有効・無効エネルギーを計算せよ。

(1) 300 K → 450 Kの水の加熱における各エネルギーを求めよ。

方針

有効エネルギーは$\Delta e = \Delta h - T_0\Delta s$、無効エネルギーは$\Delta i = T_0\Delta s$で計算します。

途中式と解答
エンタルピー変化：
$$\Delta h = c_p(T_2 - T_1) = 4.0 \times (450 - 300) = 600 \text{ kJ/kg}$$
エントロピー変化：
$$\Delta s = c_p \ln\frac{T_2}{T_1} = 4.0 \times \ln\frac{450}{300} = 4.0 \times \ln 1.5 = 4.0 \times 0.4 = 1.6 \text{ kJ/(kg·K)}$$
解答
  • エンタルピー変化：$\boxed{600 \text{ kJ/kg}}$
  • エントロピー変化：$\boxed{1.6 \text{ kJ/(kg·K)}}$
  • 有効エネルギー：$\Delta e = 600 - 300 \times 1.6 = \boxed{120 \text{ kJ/kg}}$
  • 無効エネルギー：$\Delta i = 300 \times 1.6 = \boxed{480 \text{ kJ/kg}}$

(2) 450 K飽和水→飽和蒸気の相変化における各エネルギーを求めよ。蒸発潜熱は1800 kJ/kgとする。

方針

等温過程の相変化では、エントロピー変化は潜熱を温度で割った値になります。

途中式と解答
エンタルピー変化：
$$\Delta h = 1800 \text{ kJ/kg} \quad \text{（蒸発潜熱）}$$
エントロピー変化（等温過程）：
$$\Delta s = \frac{\Delta h}{T_{\text{sat}}} = \frac{1800}{450} = 4.0 \text{ kJ/(kg·K)}$$
解答
  • エンタルピー変化：$\boxed{1800 \text{ kJ/kg}}$
  • エントロピー変化：$\boxed{4.0 \text{ kJ/(kg·K)}}$
  • 有効エネルギー：$\Delta e = 1800 - 300 \times 4.0 = \boxed{600 \text{ kJ/kg}}$
  • 無効エネルギー：$\Delta i = 300 \times 4.0 = \boxed{1200 \text{ kJ/kg}}$

(3) 全体（300 K水→飽和蒸気）の各エネルギーを求めよ。

解答

各過程の値を合計：

  • 総エンタルピー変化：$\Delta h_{\text{total}} = 600 + 1800 = \boxed{2400 \text{ kJ/kg}}$
  • 総エントロピー変化：$\Delta s_{\text{total}} = 1.6 + 4.0 = \boxed{5.6 \text{ kJ/(kg·K)}}$
  • 有効エネルギー：$\Delta e_{\text{total}} = 120 + 600 = \boxed{720 \text{ kJ/kg}}$
  • 無効エネルギー：$\Delta i_{\text{total}} = 480 + 1200 = \boxed{1680 \text{ kJ/kg}}$
有効・無効エネルギー
  • 有効エネルギー：仕事に変換可能
  • 無効エネルギー：周囲との平衡で失われる
  • エクセルギー解析の基礎概念
  • 省エネルギーの指標
計算のポイント
  • 周囲温度の基準設定
  • 等温・等圧過程の区別
  • 相変化でのエントロピー計算
  • エネルギーの質の評価
重要な関係式
  • 有効エネルギー：$\Delta e = \Delta h - T_0\Delta s$
  • 無効エネルギー：$\Delta i = T_0\Delta s$
  • 相変化：$\Delta s = \Delta h/T$
  • エネルギー収支：$\Delta e + \Delta i = \Delta h$
大問【3】: ランキンサイクル
ランキンサイクルについて、h-s線図を参照してタービン仕事と熱効率を求め、損失要因とタービン効率の影響を解析せよ。

(1) タービン仕事を求めよ。

方針

定常流でのエネルギー保存則を適用し、運動・位置エネルギーは無視します。

解答

タービン入口（過熱蒸気）から出口（復水器入口）まで：

$$\boxed{w_T = h_1 - h_2}$$

ここで、$h_1$：タービン入口比エンタルピー、$h_2$：タービン出口比エンタルピー

(2) 熱効率を求めよ。

方針

熱効率は正味仕事を吸熱量で割った値です。ポンプ仕事は無視できるとします。

途中式と解答

ボイラでの吸熱量：

$$q_{\text{in}} = h_1 - h_3$$

ここで、$h_3$：復水器出口の飽和水エンタルピー

解答
$$\eta = \frac{w_T}{q_{\text{in}}} = \boxed{\frac{h_1 - h_2}{h_1 - h_3}}$$

(3) 実際の原動所で生じる損失と不可逆性を列挙せよ。

解答

主な損失と不可逆性：

  1. タービン内部損失：摩擦や乱流によるエントロピー生成
  2. 配管熱損失：断熱不十分による熱逃げ
  3. 復水器過冷却：設計より低温で凝縮、利用可能エネルギー損失
  4. 燃焼不完全：化学エネルギーから熱エネルギーへの変換効率低下
  5. ポンプの機械損失：軸仕事の一部が熱に変換

(4) タービン効率が80%の場合の実際の熱効率を求めよ。

方針

タービン効率は等エントロピー仕事に対する実際の仕事の比として定義されます。

途中式と解答

等エントロピー仕事（理想的なタービン仕事）：

$$w_s = h_1 - h_{2s}$$

ここで、$h_{2s}$：等エントロピー膨張での出口エンタルピー

実際のタービン仕事：

$$w_T = \eta_{\text{turb}} \times w_s = 0.8 \times (h_1 - h_{2s})$$
解答

実際の熱効率：

$$\eta_{\text{actual}} = \frac{w_T}{h_1 - h_3} = \boxed{\frac{0.8(h_1 - h_{2s})}{h_1 - h_3}}$$
ランキンサイクル
  • 蒸気発電所の基本サイクル
  • h-s線図での解析が重要
  • 理想サイクルと実際の差
  • 効率向上の工夫
解析のポイント
  • 各機器でのエネルギー収支
  • タービン効率の定義
  • 不可逆損失の影響
  • h-s線図の読み方
重要な関係式
  • タービン仕事：$w_T = h_1 - h_2$
  • 熱効率：$\eta = w_{\text{net}}/q_{\text{in}}$
  • タービン効率：$\eta_T = (h_1-h_2)/(h_1-h_{2s})$
  • 吸熱量：$q_{\text{in}} = h_1 - h_3$

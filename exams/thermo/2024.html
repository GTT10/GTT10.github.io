大問【1】: 理想気体サイクル

質量mの理想気体が、状態1から状態2に断熱圧縮され、等容変化で外部から熱量 Q23 を受け取り、状態3になる.状態3から断熱膨張して状態4になり、等容変化で 熱量 Q41 を放出して状態1に戻るとき、以下の問いに答えよ。ただし、状態iにおけ る状態量にはそれぞれ添字iを付ける、また、気体の圧力をp, 容積を V, 温度を T, エントロピーをS, 比熱比をκ、定容比熱を cv, 圧縮比をε(=V1/V2), 圧力比をξ(=p3/p2) とする。

(1) p-V線図とT-S線図を描き、熱の出入りを示せ。

方針

サイクルの各過程（断熱圧縮、等容加熱、断熱膨張、等容冷却）の特徴を把握し、p-V線図とT-S線図を正確に描く。

考え方

断熱過程では等エントロピー変化（T-S線図で垂直線）、等容過程では体積一定（p-V線図で垂直線）の特徴を活用する。

途中式と解答

p-V線図とT-S線図

p ↑
|　　　　 3
|　　　　/ \
|　　　2     4
|　　 /       \
|　　/         \
|　1-------------→ V
+----------------

T ↑
|　　　2-------3
|　　　|       |
|　　　|       |
|　　　1-------4
+----------------→ S
熱の出入り:
- 加熱: 2→3（等容加熱）
- 放熱: 4→1（等容冷却）
解答
$$\boxed{\text{上図参照}}$$

(2) 状態2,3,4の温度を T₁ を用いて表せ。

方針

断熱関係式と状態方程式を用いて各状態の温度を表現する。

途中式と解答
1→2断熱圧縮: $T_1 V_1^{\kappa-1} = T_2 V_2^{\kappa-1}$より、圧縮比 $\varepsilon = V_1/V_2$ として
$$T_2 = T_1 \varepsilon^{\kappa-1}$$
2→3等容変化: $\frac{p_2}{T_2} = \frac{p_3}{T_3}$より、圧力比 $\xi = p_3/p_2$ として
$$T_3 = T_1 \varepsilon^{\kappa-1} \xi$$
3→4断熱膨張: サイクルが閉じることから
$$T_4 = T_1 \xi$$
解答
$$\boxed{T_2 = T_1 \varepsilon^{\kappa-1}, \quad T_3 = T_1 \varepsilon^{\kappa-1} \xi, \quad T_4 = T_1 \xi}$$

(3) 2→3の熱量 Q₂₃ と仕事 W₂₃ を求めよ。

方針

等容過程では仕事は0で、熱力学第一法則より熱量は内部エネルギー変化と等しい。

途中式と解答
等容過程: $W_{23} = 0$（体積変化なし）
$Q_{23} = \Delta U = m c_v (T_3 - T_2) = m c_v T_1 \varepsilon^{\kappa-1} (\xi - 1)$
解答
$$\boxed{W_{23} = 0, \quad Q_{23} = m c_v T_1 \varepsilon^{\kappa-1} (\xi - 1)}$$

(4) 3→4の仕事 W₃₄ を求めよ。

方針

断熱過程では熱量は0で、仕事は内部エネルギー変化の負値と等しい。

途中式と解答
断熱過程: $Q_{34} = 0$
$W_{34} = -\Delta U = m c_v (T_3 - T_4) = m c_v T_1 \xi (\varepsilon^{\kappa-1} - 1)$
解答
$$\boxed{W_{34} = m c_v T_1 \xi (\varepsilon^{\kappa-1} - 1)}$$

(5) 4→1の熱量 Q₄₁ を求めよ。

方針

等容過程では仕事は0で、熱量は内部エネルギー変化と等しい。

途中式と解答
等容過程: $W_{41} = 0$
$Q_{41} = m c_v (T_1 - T_4) = m c_v T_1 (1 - \xi)$
解答
$$\boxed{Q_{41} = m c_v T_1 (1 - \xi)}$$

(6) 熱効率を求めよ。

方針

熱効率は正味仕事を吸収熱量で割ったもので、η = W_net/Q_in で計算する。

途中式と解答
正味仕事: $W_{\text{net}} = Q_{23} + Q_{41} = m c_v T_1 (\xi - 1)(\varepsilon^{\kappa-1} - 1)$
吸収熱量: $Q_{\text{in}} = Q_{23}$
$$\eta = \frac{W_{\text{net}}}{Q_{23}} = 1 - \varepsilon^{1-\kappa}$$
解答
$$\boxed{\eta = 1 - \varepsilon^{1-\kappa}}$$

(7) 図示平均有効圧力（IMEP）の定義と物理的意味を説明せよ。

方針

IMEPは正味仕事を容積変位で割ったもので、p-V線図上でサイクルが囲む面積を表現する。

途中式と解答
定義: $\text{IMEP} = \frac{W_{\text{net}}}{V_{\max} - V_{\min}} = \frac{W_{\text{net}}}{V_1 - V_2}$
物理的意味: 仕事を生む等価な平均圧力
解答
$$\boxed{\text{IMEP} = \frac{W_{\text{net}}}{V_1 - V_2}}$$
大問1のまとめ

理想気体サイクルの解析では、各過程の物理的特徴を把握し、線図を活用することで体系的に状態量を求めることができます。

重要公式・関係式
  • 断熱関係式: $pV^κ = $ 一定
  • 状態方程式: $pV = mRT$
  • 熱効率: $η = W_{net}/Q_{in}$
解法のポイント
  • 各過程の特徴を明確に把握する
  • 線図を正確に描いて視覚化する
  • エネルギー収支を厳密に計算する
注意点・間違いやすいポイント
  • 断熱と等容の混同に注意
  • 符号の取り扱いに注意（吸熱・放熱）
熱力学のチップス
  • 理想気体サイクルは内燃機関の基礎
  • エントロピーは不可逆性の指標
大問【2】: ボイラの熱力学
燃焼ガスで水を加熱するボイラにおいて、燃焼ガスは、質量流量 mg, 温度 Tg1でボ イラに入り、温度 Tg2で出ていく、燃焼ガスの圧力は一定であり、定圧比熱はcpで 一定とする、水は、質量流量 mw, 温度 Tw1 でボイラに入り、圧力一定のもと、飽和 温度 Tw2 まで加熱され、さらに蒸発潜熱を得て、乾き飽和蒸気となる、水の比熱は cw で一定とする、周囲温度を To, ボイラ外への放熱はないものとし、以下の問いに 答えよ。

(1) 水の温度上昇に要する熱量を求めよ。

方針

比熱の定義を用いて温度上昇による熱量を計算する。

途中式と解答
水を $T_{w1}$ → $T_{w2}$ に加熱
$$\dot{Q}_{\text{temp}} = \dot{m}_w c_w (T_{w2} - T_{w1})$$
解答
$$\boxed{\dot{Q}_{\text{temp}} = \dot{m}_w c_w (T_{w2} - T_{w1})}$$

(2) 水の蒸発に要する熱量を求めよ。

方針

蒸発潜熱を用いて相変化の熱量を計算する。

途中式と解答
蒸発潜熱 $r$ 使用
$$\dot{Q}_{\text{vap}} = \dot{m}_w r$$
解答
$$\boxed{\dot{Q}_{\text{vap}} = \dot{m}_w r}$$

(3) 燃焼ガスが失う熱量を求めよ。

方針

燃焼ガスの定圧冷却による熱量変化を計算する。

途中式と解答
燃焼ガスの定圧冷却
$$\dot{Q}_g = \dot{m}_g c_p (T_{g1} - T_{g2})$$
解答
$$\boxed{\dot{Q}_g = \dot{m}_g c_p (T_{g1} - T_{g2})}$$

(4) 質量流量比 $\dot{m}_g / \dot{m}_w$ を求めよ。

方針

エネルギー収支（放熱なし）の条件を利用する。

途中式と解答
エネルギー収支: $\dot{Q}_g = \dot{Q}_{\text{temp}} + \dot{Q}_{\text{vap}}$
$$\frac{\dot{m}_g}{\dot{m}_w} = \frac{ c_w (T_{w2} - T_{w1}) + r }{ c_p (T_{g1} - T_{g2}) }$$
解答
$$\boxed{\frac{\dot{m}_g}{\dot{m}_w} = \frac{ c_w (T_{w2} - T_{w1}) + r }{ c_p (T_{g1} - T_{g2}) }}$$

(5) 水の温度上昇のエントロピー変化を求めよ。

方針

可逆加熱と仮定してエントロピー変化を積分で計算する。

途中式と解答
可逆加熱と仮定
$$\Delta \dot{S}_w^{\text{temp}} = \dot{m}_w c_w \ln \frac{T_{w2}}{T_{w1}}$$
解答
$$\boxed{\Delta \dot{S}_w^{\text{temp}} = \dot{m}_w c_w \ln \frac{T_{w2}}{T_{w1}}}$$

(6) 水の蒸発のエントロピー変化を求めよ。

方針

等温蒸発における熱量とエントロピー変化の関係を利用する。

途中式と解答
等温蒸発: $\Delta \dot{S}_w^{\text{vap}} = \frac{\dot{Q}_{\text{vap}}}{T_{w2}}$
$$\Delta \dot{S}_w^{\text{vap}} = \dot{m}_w \frac{r}{T_{w2}}$$
解答
$$\boxed{\Delta \dot{S}_w^{\text{vap}} = \dot{m}_w \frac{r}{T_{w2}}}$$

(7) 燃焼ガスのエントロピー変化を求めよ。

方針

定圧冷却過程のエントロピー変化を積分で計算する。

途中式と解答
定圧冷却
$$\Delta \dot{S}_g = \dot{m}_g c_p \ln \frac{T_{g2}}{T_{g1}}$$
解答
$$\boxed{\Delta \dot{S}_g = \dot{m}_g c_p \ln \frac{T_{g2}}{T_{g1}}}$$

(8) 系全体のエントロピー変化を求めよ。

方針

水と燃焼ガスのエントロピー変化の合計を計算する。

途中式と解答
水と燃焼ガスの合計
$$\Delta \dot{S}_{\text{total}} = \dot{m}_g c_p \ln \frac{T_{g2}}{T_{g1}} + \dot{m}_w c_w \ln \frac{T_{w2}}{T_{w1}} + \dot{m}_w \frac{r}{T_{w2}}$$
解答
$$\boxed{\Delta \dot{S}_{\text{total}} = \dot{m}_g c_p \ln \frac{T_{g2}}{T_{g1}} + \dot{m}_w c_w \ln \frac{T_{w2}}{T_{w1}} + \dot{m}_w \frac{r}{T_{w2}}}$$

(9) 水のエクセルギー変化を求めよ。

方針

エクセルギーの定義 $\Delta E = \Delta H - T_0 \Delta S$ を適用する。

途中式と解答
エンタルピー変化: $\Delta h_w = c_w (T_{w2} - T_{w1}) + r$
エントロピー変化: $\Delta s_w = c_w \ln \frac{T_{w2}}{T_{w1}} + \frac{r}{T_{w2}}$
$$\Delta \dot{E}_w = \dot{m}_w \left[ c_w (T_{w2} - T_{w1}) + r - T_0 \left( c_w \ln \frac{T_{w2}}{T_{w1}} + \frac{r}{T_{w2}} \right) \right]$$
解答
$$\boxed{\Delta \dot{E}_w = \dot{m}_w \left[ c_w (T_{w2} - T_{w1}) + r - T_0 \left( c_w \ln \frac{T_{w2}}{T_{w1}} + \frac{r}{T_{w2}} \right) \right]}$$

(10) 系全体のエクセルギー変化を求めよ。

方針

燃焼ガスと水のエクセルギー変化を合計する。

途中式と解答
燃焼ガス + 水のエクセルギー変化
$$\Delta \dot{E}_{\text{total}} = \dot{m}_g \left[ c_p (T_{g2} - T_{g1}) - T_0 c_p \ln \frac{T_{g2}}{T_{g1}} \right] + \Delta \dot{E}_w$$
解答
$$\boxed{\Delta \dot{E}_{\text{total}} = \dot{m}_g \left[ c_p (T_{g2} - T_{g1}) - T_0 c_p \ln \frac{T_{g2}}{T_{g1}} \right] + \dot{m}_w \left[ c_w (T_{w2} - T_{w1}) + r - T_0 \left( c_w \ln \frac{T_{w2}}{T_{w1}} + \frac{r}{T_{w2}} \right) \right]}$$
大問2のまとめ

ボイラの熱力学では、エネルギー保存則に基づいた収支計算と、エントロピー・エクセルギー解析により効率評価が可能です。

エネルギー関係式
  • 比熱による加熱: $Q = mc\Delta T$
  • 蒸発潜熱: $Q = mr$
  • エントロピー変化: $\Delta S = Q/T$
エクセルギー解析
  • 定義: $E = H - T_0S$
  • 有効エネルギーの指標
  • 効率評価に重要
実用ポイント
  • ボイラは火力発電の基幹設備
  • エクセルギーは有効エネルギー

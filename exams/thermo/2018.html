大問【1】: ポリトロープ変化

理想気体が$pv^n = \text{一定}$に従うポリトロープ変化を行う場合について、以下の問いに答えよ。

(1) 単位質量あたりの仕事$w$と熱量$q$を求めよ。

方針

ポリトロープ変化の関係式$pv^n = \text{一定}$と熱力学第一法則を使用します。

途中式と解答
仕事$w$の計算：
$$w = \int_{v_1}^{v_2} p \, dv = \int_{v_1}^{v_2} \frac{p_1 v_1^n}{v^n} dv$$ $$= p_1 v_1^n \int_{v_1}^{v_2} v^{-n} dv = p_1 v_1^n \left[ \frac{v^{1-n}}{1-n} \right]_{v_1}^{v_2}$$ $$= \frac{p_1 v_1^n}{1-n} (v_2^{1-n} - v_1^{1-n}) = \frac{p_1 v_1 - p_2 v_2}{n-1}$$

理想気体の関係$pv = RT$を用いると：

$$w = \frac{R(T_1 - T_2)}{n-1}$$
熱量$q$の計算：

熱力学第一法則$q = \Delta u + w$より：

$$\Delta u = c_v(T_2 - T_1)$$ $$q = c_v(T_2 - T_1) + \frac{R(T_1 - T_2)}{n-1}$$ $$= (T_2 - T_1)\left(c_v - \frac{R}{n-1}\right)$$
解答
$$\boxed{ \begin{aligned} w &= \frac{R(T_1 - T_2)}{n-1} \\ q &= (T_2 - T_1)\left(c_v - \frac{R}{n-1}\right) \end{aligned} }$$

(2) ポリトロープ比熱$c_n$を求めよ。

方針

比熱の定義$dq = c_n dT$と(1)の結果を使用します。

途中式と解答

(1)の結果から：

$$c_n = \frac{q}{T_2 - T_1} = c_v - \frac{R}{n-1}$$

$c_p - c_v = R$と$\kappa = c_p/c_v$の関係を使用すると：

$$c_n = c_v - \frac{c_p - c_v}{n-1} = c_v\left(1 - \frac{\kappa - 1}{n-1}\right)$$
解答
$$\boxed{c_n = c_v \frac{n - \kappa}{n - 1}}$$

(3) 比エントロピー変化$\Delta s$を求めよ。

方針

エントロピーの定義$ds = dq_{\text{rev}}/T$を積分し、ポリトロープ関係を用いて整理します。

途中式と解答
$$\Delta s = c_v \ln\frac{T_2}{T_1} + R \ln\frac{v_2}{v_1}$$

ポリトロープ関係$T_1 v_1^{n-1} = T_2 v_2^{n-1}$より：

$$\frac{T_2}{T_1} = \left(\frac{v_1}{v_2}\right)^{n-1} = \left(\frac{p_2}{p_1}\right)^{\frac{n-1}{n}}$$

これを代入して整理すると：

$$\Delta s = c_v\left(\frac{n-\kappa}{n}\right)\ln\frac{p_1}{p_2}$$
解答
$$\boxed{\Delta s = c_v\left(\frac{n-\kappa}{n}\right)\ln\frac{p_1}{p_2}}$$

大問【2】: オットーサイクル
理想気体によるオットーサイクル（火花点火機関の理論サイクル）について考える。圧縮比を$\varepsilon = V_1/V_2$とし、以下の問いに答えよ。

(1) このサイクルの$p-V$線図および$T-S$線図を描け。

方針

オットーサイクルは1→2（断熱圧縮）、2→3（等容加熱）、3→4（断熱膨張）、4→1（等容冷却）の4過程です。

|   |\   |  /|
|   | \  | / |
|   |  \ |/  |
|   |   \|   |
|   1----+---+----→ 容積 V
   V₂    V₁   V₁

温度 T ↑
|        2-------3

|        |       |
| (断熱) |       | (断熱)
|        |       |

解答

p-V線図: 1→2（急な断熱圧縮）、2→3（垂直線上昇）、3→4（急な断熱膨張）、4→1（垂直線下降）

T-S線図: 1→2（垂直線上昇）、2→3（右上がり曲線）、3→4（垂直線下降）、4→1（左下がり曲線）

(2) 各過程における熱量の出入りを求めよ。

方針

等容過程2→3で吸熱、等容過程4→1で放熱します。断熱過程では熱の出入りはありません。

解答
$$\boxed{ \begin{aligned} Q_{\text{in}} &= mc_v(T_3 - T_2) \quad (\text{2→3の等容加熱}) \\ Q_{\text{out}} &= mc_v(T_4 - T_1) \quad (\text{4→1の等容冷却}) \end{aligned} }$$

(3) 各状態の温度を状態1の温度$T_1$と圧縮比$\varepsilon$を用いて表せ。

方針

断熱過程では$TV^{\kappa-1} = \text{一定}$の関係を使用します。

途中式と解答
状態2（断熱圧縮）：
$$T_1 V_1^{\kappa-1} = T_2 V_2^{\kappa-1}$$ $$T_2 = T_1\left(\frac{V_1}{V_2}\right)^{\kappa-1} = T_1\varepsilon^{\kappa-1}$$
状態4（断熱膨張）：
$$T_3 V_3^{\kappa-1} = T_4 V_4^{\kappa-1}$$ $$T_4 = T_3\left(\frac{V_3}{V_4}\right)^{\kappa-1} = T_3\varepsilon^{1-\kappa}$$
解答
$$\boxed{ \begin{aligned} T_2 &= T_1\varepsilon^{\kappa-1} \\ T_4 &= T_3\varepsilon^{1-\kappa} \end{aligned} }$$

（$T_3$は等容加熱過程での最高温度として与えられる）

(4) このサイクルの熱効率を求めよ。

方針

熱効率は$\eta = 1 - Q_{\text{out}}/Q_{\text{in}}$で定義されます。(3)の結果を使用します。

途中式と解答
$$\eta = 1 - \frac{Q_{\text{out}}}{Q_{\text{in}}} = 1 - \frac{mc_v(T_4 - T_1)}{mc_v(T_3 - T_2)} = 1 - \frac{T_4 - T_1}{T_3 - T_2}$$

(3)の関係を用いて、$T_4 = T_3\varepsilon^{1-\kappa}$、$T_2 = T_1\varepsilon^{\kappa-1}$を代入すると：

$$\eta = 1 - \frac{T_3\varepsilon^{1-\kappa} - T_1}{T_3 - T_1\varepsilon^{\kappa-1}} = 1 - \varepsilon^{1-\kappa}$$
解答
$$\boxed{\eta = 1 - \varepsilon^{1-\kappa}}$$

(5) 定容比熱が$c_v = a + bT$の場合、2→3過程のエントロピー変化を求めよ。

方針

温度依存比熱の等容過程でのエントロピー変化は$\Delta S = \int c_v \frac{dT}{T}$で計算します。

途中式と解答
$$\Delta S_{23} = m\int_{T_2}^{T_3} \frac{c_v}{T} dT = m\int_{T_2}^{T_3} \frac{a + bT}{T} dT$$ $$= m\int_{T_2}^{T_3} \left(\frac{a}{T} + b\right) dT$$ $$= m\left[a\ln T + bT\right]_{T_2}^{T_3}$$
解答
$$\boxed{\Delta S_{23} = m\left[a\ln\frac{T_3}{T_2} + b(T_3 - T_2)\right]}$$

(6) 断熱過程が不可逆の場合、$T-S$線図はどのように変化するか描け。

方針

不可逆断熱過程ではエントロピーが増加するため、T-S線図で右向きの変化となります。

線図と解答

オットーサイクルの特徴
  • 火花点火機関の理論サイクル
  • 効率は圧縮比のみに依存
  • 高圧縮比ほど高効率
  • ノッキング限界による制約
重要な関係式
  • 断熱過程: $TV^{\kappa-1} = \text{一定}$
  • 等容過程: $Q = mc_v\Delta T$
  • 熱効率: $\eta = 1 - \varepsilon^{1-\kappa}$
  • 不可逆性によるエントロピー増大
大問【3】: 乾き飽和蒸気サイクル
乾き飽和蒸気を使用する蒸気サイクルについて考える。加熱量を$Q_{\text{in}} = 5000$ kJ/kg、高温熱源温度$T_H = 1500$ K、低温熱源温度$T_L = 300$ Kとし、以下の問いに答えよ。

(1) このサイクルの熱効率と正味仕事を求めよ。（蒸気表を使用）

方針

蒸気サイクルの基本的な解析手順に従い、各状態点でのエンタルピーを求めて熱効率を計算します。

解析手順
サイクル構成：
  • A→B: ボイラーでの加熱（等圧）
  • B→C: タービンでの膨張（断熱）
  • C→D: 復水器での冷却（等圧）
  • D→A: ポンプでの圧縮（断熱）
各仕事の計算：
$$w_T = h_B - h_C \quad (\text{タービン仕事})$$ $$w_P = v_f(p_A - p_D) \approx v_f \Delta p \quad (\text{ポンプ仕事})$$ $$w_{\text{net}} = w_T - w_P \approx w_T \quad (\text{ポンプ仕事は通常無視可能})$$
解答

注意：具体的な数値計算には蒸気表が必要です。一般的な解析手順は以下の通りです：

$$\boxed{ \begin{aligned} \eta &= \frac{w_{\text{net}}}{Q_{\text{in}}} = \frac{h_B - h_C}{Q_{\text{in}}} \\ w_{\text{net}} &= \eta \times Q_{\text{in}} \end{aligned} }$$

(2) 同一熱源間で動作するカルノー機関の熱効率と仕事を求めよ。

方針

カルノー効率は温度のみで決定され、$\eta_{\text{Carnot}} = 1 - T_L/T_H$で計算できます。

計算と解答
カルノー効率：
$$\eta_{\text{Carnot}} = 1 - \frac{T_L}{T_H} = 1 - \frac{300}{1500} = 1 - 0.2 = 0.8$$
カルノー機関の仕事：
$$w_{\text{Carnot}} = \eta_{\text{Carnot}} \times Q_{\text{in}} = 0.8 \times 5000 = 4000 \text{ kJ/kg}$$
解答
$$\boxed{ \begin{aligned} \eta_{\text{Carnot}} &= 0.8 \quad (80\%) \\ w_{\text{Carnot}} &= 4000 \text{ kJ/kg} \end{aligned} }$$

(3) エクセルギー効率を求めよ。

方針

エクセルギー効率は実際のサイクルの仕事とカルノー機関の仕事の比で定義されます。

解答
$$\boxed{\eta_{\text{ex}} = \frac{w_{\text{net}}}{w_{\text{Carnot}}}}$$

（$w_{\text{net}}$は(1)で求めた実際のサイクルの正味仕事）

(4) 高温熱源から蒸気への熱移動時の高温熱源のエントロピー変化を求めよ。

方針

高温熱源から熱量$Q_{\text{in}}$が取り去られるときのエントロピー変化は$\Delta S = -Q/T_H$です。

計算と解答
$$\Delta S_{\text{source}} = -\frac{Q_{\text{in}}}{T_H} = -\frac{5000}{1500} = -3.333 \text{ kJ/(kg·K)}$$
解答
$$\boxed{\Delta S_{\text{source}} = -3.33 \text{ kJ/(kg·K)}}$$

（負の値は熱源からエントロピーが取り去られることを示す）

蒸気サイクルの基本
  • ランキンサイクルが基本形
  • 蒸気表による状態量決定
  • ポンプ仕事は通常無視可能
  • カルノー効率が理論的上限
エクセルギー解析
  • エクセルギー効率 = 実仕事/理論最大仕事
  • カルノー機関が理論的基準
  • 不可逆性の定量的評価
  • エントロピー生成との関係

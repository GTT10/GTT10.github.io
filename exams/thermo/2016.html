大問【1】: 流体の流れと熱力学

流体が管内を流れ、その途中でタービンや絞りを通過する場合を考える。以下の問いに答えよ。

(1) 断面Ⅰで気体を押し込む際に必要な単位質量あたりの仕事を答えよ。

方針

流体を管内に押し込む際の流れ仕事（流動仕事）を考えます。

解答
$$\boxed{P_1 v_1}$$

ここで$P_1$は圧力、$v_1$は比容積です。

(2) タービン仕事$W_1$を求めよ。ただし、運動エネルギーと位置エネルギーの変化は無視し、外部への放熱量を$Q$とする。

方針

定常流エネルギー保存則を適用し、エンタルピー変化と放熱量からタービン仕事を求めます。

解答

定常流エネルギー保存則より：

$$mh_1 = mh_2 + W_1 + Q$$

（$Q > 0$は外部への放熱）

解答
$$\boxed{W_1 = m(h_1 - h_2) - Q}$$

(3) 絞り過程（断熱・無仕事）のエネルギー保存式を書け。

方針

絞りでは$W_1 = 0$、熱損失なし（$Q = 0$）、流速遅い（運動エネルギー無視）の条件を適用します。

解答
$$\boxed{h_1 = h_2}$$

絞り過程は等エンタルピー変化となります。

(4) 熱力学第一法則とエンタルピーの定義から$dh = T ds + v dP$を導け。

方針

熱力学第一法則$du = T ds - P dv$とエンタルピーの定義$h = u + Pv$を使用します。

導出過程

熱力学第一法則：

$$du = \delta q - \delta w = T ds - P dv$$

エンタルピー定義：

$$h = u + Pv$$

全微分：

$$dh = du + P dv + v dP = (T ds - P dv) + P dv + v dP$$
解答
$$\boxed{dh = T ds + v dP}$$

(5) $dh = c_p dT + \left[ v - T \left( \frac{\partial v}{\partial T} \right)_P \right] dP$を導け。

方針

エントロピーを$s = s(P, T)$として全微分し、マックスウェルの関係式を適用します。

導出過程

エントロピーの全微分：

$$ds = \left( \frac{\partial s}{\partial T} \right)_P dT + \left( \frac{\partial s}{\partial P} \right)_T dP$$

定圧比熱の関係：$c_p = T \left( \frac{\partial s}{\partial T} \right)_P$

$$ds = \frac{c_p}{T} dT + \left( \frac{\partial s}{\partial P} \right)_T dP$$

マックスウェルの関係式：$\left( \frac{\partial v}{\partial T} \right)_P = -\left( \frac{\partial s}{\partial P} \right)_T$

$$ds = \frac{c_p}{T} dT - \left( \frac{\partial v}{\partial T} \right)_P dP$$

(4)の結果に代入：

$$dh = T \left[ \frac{c_p}{T} dT - \left( \frac{\partial v}{\partial T} \right)_P dP \right] + v dP$$
解答
$$\boxed{dh = c_p dT + \left[ v - T \left( \frac{\partial v}{\partial T} \right)_P \right] dP}$$

(6) ジュール・トムソン係数$\mu$を定義し、その表式を求めよ。

方針

絞り過程では$dh = 0$であることを利用し、(5)の結果からジュール・トムソン係数を定義します。

解答

絞り過程では$dh = 0$。(5)の式で$dh = 0$とし、係数を定義：

$$0 = c_p dT + \left[ v - T \left( \frac{\partial v}{\partial T} \right)_P \right] dP$$
解答
$$\boxed{\mu = \left( \frac{\partial T}{\partial P} \right)_h = \frac{1}{c_p} \left[ T \left( \frac{\partial v}{\partial T} \right)_P - v \right]}$$

(7) 状態式$P \left( v - a + \frac{b}{RT} \right) = RT$に対して逆転温度$T_{\text{inv}}$を求めよ。

方針

逆転温度は$\mu = 0$となる温度です。状態式から$(\partial v/\partial T)_P$を求め、$\mu = 0$の条件を適用します。

計算過程

状態式を変形：

$$v = \frac{RT}{P} + a - \frac{b}{RT}$$

微分：

$$\left( \frac{\partial v}{\partial T} \right)_P = \frac{R}{P} + \frac{b}{R T^2}$$

$\mu = 0$の条件：$T \left( \frac{\partial v}{\partial T} \right)_P - v = 0$

$$T \left( \frac{R}{P} + \frac{b}{R T^2} \right) - \left( \frac{RT}{P} + a - \frac{b}{RT} \right) = 0$$

整理：

$$\frac{RT}{P} + \frac{b}{RT} - \frac{RT}{P} - a + \frac{b}{RT} = 0$$ $$-a + \frac{2b}{RT} = 0$$
解答
$$\boxed{T_{\text{inv}} = \frac{2b}{aR}}$$

(8) 理想気体が絞りを通過した後の温度変化について述べよ。

方針

理想気体では$Pv = RT$であり、(7)の式で$a = b = 0$とした場合を考えます。

解答

理想気体では$a = b = 0$より：

$$T \left( \frac{\partial v}{\partial T} \right)_P - v = T \cdot \frac{R}{P} - \frac{RT}{P} = 0$$

したがって$\mu = 0$

解答

理想気体では絞り通過後も温度変化なし（等エンタルピー変化）

重要公式・関係式
  • 流れ仕事: $Pv$
  • 定常流エネルギー保存則
  • 熱力学基本関係式: $dh = T ds + v dP$
  • マックスウェルの関係式
  • ジュール・トムソン係数の定義
解法のポイント
  • 流れ系でのエネルギー保存則の適用
  • 熱力学関係式の体系的導出
  • 絞り過程の特性理解
  • 実ガス効果の理論的取り扱い
注意点
  • 流れ仕事と軸仕事の区別
  • 偏微分の扱いと関係式
  • 理想気体と実ガスの違い
大問【2】: ボイラのエネルギー損失
高温熱源（1400℃）から2800 kJ/kgの熱量を受けて水が蒸気になる場合を考える。環境温度を0℃とし、以下の問いに答えよ。

(1) この熱量を可逆的に利用した場合のエントロピーと無効エネルギーを求めよ。

方針

可逆利用時のエントロピーは熱源の初期エントロピー、無効エネルギーは低温熱源に関連するエネルギーです。

計算と解答

温度変換：

$T_H = 1400 + 273 = 1673$ K

$T_L = 0 + 273 = 273$ K

可逆利用時のエントロピー：

$$S_{\text{initial}} = \frac{Q}{T_H} = \frac{2800}{1673} = 1.67 \text{ kJ/(kg·K)}$$

無効エネルギー：

$$E_{\text{無効}} = T_L \cdot \frac{Q}{T_H} = 273 \times \frac{2800}{1673} = 456 \text{ kJ/kg}$$
解答
$$\boxed{ \begin{aligned} S_{\text{initial}} &= 1.67 \text{ kJ/(kg·K)} \\ E_{\text{無効}} &= 456 \text{ kJ/kg} \end{aligned} }$$

(2) 不可逆伝熱により水のエントロピーが6.6 kJ/(kg·K)増加した場合のエントロピー増加量と無効エネルギー増加量を求めよ。

方針

熱源と水の系全体でのエントロピー変化を求め、無効エネルギー増加量を計算します。

計算と解答

エントロピー変化：

熱源のエントロピー減少：$-\frac{Q}{T_H} = -\frac{2800}{1673} = -1.67$ kJ/(kg·K)

水のエントロピー増加：$+6.6$ kJ/(kg·K)

全体のエントロピー増加量：

$$\Delta S_{\text{total}} = 6.6 - \frac{2800}{1673} = 6.6 - 1.67 = 4.93 \text{ kJ/(kg·K)}$$

無効エネルギー増加量：

$$\Delta E_{\text{無効}} = T_L \cdot \Delta S_{\text{total}} = 273 \times 4.93 = 1346 \text{ kJ/kg}$$
解答
$$\boxed{ \begin{aligned} \Delta S_{\text{total}} &= 4.93 \text{ kJ/(kg·K)} \\ \Delta E_{\text{無効}} &= 1346 \text{ kJ/kg} \end{aligned} }$$
エクセルギー解析の基本
  • 可逆過程でのエネルギーの有効利用
  • 不可逆性による損失の定量化
  • 環境温度基準の無効エネルギー
  • エントロピー増加と損失の関係
重要な概念
  • 有効エネルギー（エクセルギー）
  • 無効エネルギー（アネルギー）
  • 不可逆性によるエネルギー質の劣化
大問【3】: 蒸気サイクル解析
蒸気表（表1）を用いて蒸気サイクルを解析する。ボイラ出口圧力2 MPa、復水器圧力0.004 MPaとし、以下の問いに答えよ。

(1) 状態4（復水器入口）の乾き度を求めよ。

方針

等エントロピー膨張により状態1から状態4へ変化し、乾き度は$x = (s_4 - s')/(s'' - s')$で求められます。

計算と解答

状態点の設定：

状態1（ボイラ出口）：$P_1 = 2$ MPa, $s_1 = s'' = 6.3$ kJ/(kg·K)

状態4（復水器入口）：$P_4 = 0.004$ MPa, $s_4 = s_1 = 6.3$ kJ/(kg·K)

飽和蒸気表（0.004 MPa）：

$s' = 0.4$ kJ/(kg·K), $s'' = 8.5$ kJ/(kg·K)

乾き度計算：

$$x_4 = \frac{s_4 - s'}{s'' - s'} = \frac{6.3 - 0.4}{8.5 - 0.4} = \frac{5.9}{8.1} = 0.73$$
解答
$$\boxed{x_4 = 0.73}$$

(2) サイクル熱効率を求めよ。

方針

熱効率は$\eta = w_{\text{net}}/q_{\text{in}}$で定義されます。吸熱量と正味仕事を蒸気表から求めます。

計算と解答

吸熱量（ボイラ）：

$$q_{\text{in}} = h_1 - h_2 = 2800 - 900 = 1900 \text{ kJ/kg}$$

状態4のエンタルピー：

$$h_4 = h' + x_4(h'' - h') = 100 + 0.73 \times (2500 - 100) = 100 + 1752 = 1852 \text{ kJ/kg}$$

正味仕事（ポンプ仕事無視）：

$$w_{\text{net}} = h_1 - h_4 = 2800 - 1852 = 948 \text{ kJ/kg}$$

熱効率：

$$\eta = \frac{w_{\text{net}}}{q_{\text{in}}} = \frac{948}{1900} = 0.50$$
解答
$$\boxed{\eta = 0.50 \text{ (50\%)}}$$

(3) 同じ温度範囲でのカルノー効率を求めよ。

方針

カルノー効率は$\eta_{\text{Carnot}} = 1 - T_L/T_H$で計算されます。温度は絶対温度を使用します。

計算と解答

温度設定：

高温熱源温度：$T_H = 200 + 273 = 473$ K

低温熱源温度：$T_L = 30 + 273 = 303$ K

カルノー効率：

$$\eta_{\text{Carnot}} = 1 - \frac{T_L}{T_H} = 1 - \frac{303}{473} = 1 - 0.64 = 0.36$$
解答
$$\boxed{\eta_{\text{Carnot}} = 0.36 \text{ (36\%)}}$$

(4) 実際の原動所で生じる主な損失と不可逆性について述べよ。

解答

実際の原動所で生じる主な損失：

  1. タービン内の不可逆膨張：摩擦や乱流によりエントロピー増加、仕事出力減少
  2. 配管での熱損失：断熱不十分で熱が外部へ逃げる
  3. ポンプの不可逆性：機械損失や内部摩擦により消費電力増加
  4. 復水器の過冷却：設計値より低温で凝縮、利用可能エネルギー損失
  5. 燃焼の不完全性：化学エネルギーから熱エネルギーへの変換効率低下
蒸気サイクル解析の基本
  • 蒸気表の正確な読み取り
  • 湿り蒸気の乾き度計算
  • 等エントロピー過程の仮定
  • エネルギー収支の適用
重要な公式
  • 乾き度: $x = (s - s')/(s'' - s')$
  • 湿り蒸気エンタルピー: $h = h' + x(h'' - h')$
  • 熱効率: $\eta = w_{\text{net}}/q_{\text{in}}$
  • カルノー効率: $\eta_C = 1 - T_L/T_H$
注意点
  • 飽和蒸気表の正確な読み取り
  • 乾き度が1以下であることの確認
  • ポンプ仕事の扱い（通常無視可能）

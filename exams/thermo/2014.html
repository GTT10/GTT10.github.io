大問【1】: シリンダ内の理想気体

シリンダ内の理想気体が定容加熱（0→1）と断熱膨張（1→2）を行う場合について、各状態量とエントロピー変化を求めよ。

(1) 状態1の温度$T_1$と圧力$P_1$を求めよ。

方針

定容加熱では体積一定で熱量を加え、理想気体の状態方程式を適用します。

途中式と解答

過程0→1：定容加熱（体積$V_0$一定）

加熱量：$Q = mc_v(T_1 - T_0)$より

$$T_1 = T_0 + \frac{Q}{mc_v}$$

状態方程式：$P_1V_0 = mRT_1$、$P_0V_0 = mRT_0$より

$$P_1 = P_0 \frac{T_1}{T_0}$$
解答
$$\boxed{T_1 = T_0 + \frac{Q}{mc_v}}$$ $$\boxed{P_1 = P_0 \left(1 + \frac{Q}{mc_vT_0}\right)}$$

(2) 状態0→1のエントロピー変化$\Delta S_{01}$を求めよ。

方針

定容過程のエントロピー変化は$\Delta S = mc_v\ln(T_2/T_1)$で計算します。

解答
$$\boxed{\Delta S_{01} = mc_v \ln\left(1 + \frac{Q}{mc_vT_0}\right)}$$

(3) 状態2の温度$T_2$と体積$V_2$を求めよ。

方針

断熱膨張では系全体が断熱でピストンが外圧$P_0$に抗して動きます。エネルギー保存則を適用します。

途中式と解答

エネルギー保存則：$\Delta U = -W$

$$mc_v(T_2 - T_1) = -P_0(V_2 - V_0)$$

状態方程式：$P_0V_2 = mRT_2$、$P_0V_0 = mRT_0$

これらを連立して解くと：

$$c_v(T_2 - T_1) = -RT_2 + RT_0$$ $$T_2(c_v + R) = c_vT_1 + RT_0$$
解答
$$\boxed{T_2 = \frac{c_v}{c_p}T_1 + \frac{R}{c_p}T_0}$$ $$\boxed{V_2 = \frac{mRT_2}{P_0}}$$

(4) 状態1→2のエントロピー変化$\Delta S_{12}$を求めよ。

方針

理想気体の一般的エントロピー変化式を使用します。

解答

$P_2 = P_0$、$P_1 = P_0T_1/T_0$より：

$$\boxed{\Delta S_{12} = m\left(c_p\ln\frac{T_2}{T_1} + R\ln\frac{T_1}{T_0}\right)}$$

(5) ピストンが外部にした仕事$W$を求めよ。

解答
$$\boxed{W = P_0(V_2 - V_0) = mR(T_2 - T_0)}$$

(6) $P-V$線図と$T-S$線図を描け。

線図の描画と解答

解答

過程の特徴：

  • 0→1: 定容加熱（垂直線、エントロピー増加）
  • 1→2: 断熱膨張（不可逆のためエントロピー増加）
重要公式・関係式
  • 定容過程：$Q = mc_v\Delta T$
  • 状態方程式：$PV = mRT$
  • エネルギー保存：$\Delta U = Q - W$
  • エントロピー変化：$\Delta S = mc_p\ln(T_2/T_1) - mR\ln(P_2/P_1)$
解法のポイント
  • 各過程の特徴把握
  • エネルギー収支の正確な適用
  • 状態方程式の連立解法
  • 不可逆過程でのエントロピー増加
注意点
  • 断熱と等エントロピーの区別
  • 外圧による仕事の計算
  • 比熱の定義（$c_p = c_v + R$）
大問【2】: 再生冷凍サイクル
再生器付き冷凍サイクルについて、T-S線図を描き、各過程の熱量と仕事、成績係数を求めよ。

(1) T-S線図を描き、熱の流れを示せ。

方針

再生サイクルは等温過程と定容過程から構成され、再生器で内部熱交換を行います。

線図と解答

T-S線図

温度 T ↑
|         
| T_H: ●状態1────●状態4
|      |          |
|      |          |
| T_L: ●状態2────●状態3
|      
+---------------------→ エントロピー S
解答

熱の流れ：

  • 吸熱：3→4（等温膨張、$T_L$一定）
  • 放熱：1→2（等温圧縮、$T_H$一定）
  • 再生熱：2→3の放熱 = 4→1の吸熱（内部熱交換）

(2) 2→3の熱量$Q_{23}$と仕事$W_{23}$を求めよ。

方針

定容過程では仕事はゼロで、熱量は内部エネルギー変化に等しくなります。

解答

定容過程（体積一定）：

$$\boxed{Q_{23} = mc_v(T_3 - T_2) = mc_v(T_L - T_H)}$$ $$\boxed{W_{23} = 0}$$

(3) 3→4の熱量$Q_{34}$と仕事$W_{34}$を求めよ。

方針

等温膨張では内部エネルギー変化がゼロのため、熱量と仕事が等しくなります。

解答

等温膨張（$T_L$一定、$\Delta U = 0$）：

$$\boxed{Q_{34} = mRT_L\ln\frac{V_4}{V_3}}$$ $$\boxed{W_{34} = mRT_L\ln\frac{V_4}{V_3}}$$

(4) 成績係数（COP）を求めよ。

方針

成績係数は冷凍効果（吸熱量）を投入仕事で割った値です。再生器により内部熱交換が相殺されます。

途中式と解答

冷凍効果（吸熱量）：

$$Q_{\text{冷凍}} = Q_{34} = mRT_L\ln\frac{V_4}{V_3}$$

正味仕事入力：

1→2（等温圧縮）：$W_{12} = mRT_H\ln\frac{V_2}{V_1} = -mRT_H\ln\frac{V_1}{V_2}$

サイクル構造より：$V_4/V_3 = V_1/V_2 = \varepsilon$

$$W_{\text{net}} = mR\ln\varepsilon(T_L - T_H)$$
解答
$$\text{COP} = \frac{Q_{\text{冷凍}}}{|W_{\text{net}}|} = \frac{T_L\ln\varepsilon}{(T_H - T_L)\ln\varepsilon} = \boxed{\frac{T_L}{T_H - T_L}}$$

これはカルノー冷凍機の成績係数と一致します。

再生サイクルの特徴
  • 内部熱交換により効率向上
  • 等温過程と定容過程の組合せ
  • カルノー効率に近づく
  • 実用的な冷凍システム
計算のポイント
  • 各過程での仕事・熱量の特徴
  • 再生器での熱収支
  • 成績係数の定義
  • 体積比の関係性
大問【3】: ノズル流れ
ノズルを通る流体の臨界条件と音速の関係、適切なノズル形状の選択について解析せよ。

(2) 臨界圧力$P_c$を求めよ。

方針

臨界圧力は入口圧力と比熱比から決まる理論式があります。

解答
$$\boxed{P_c = P_1\left(\frac{2}{\kappa + 1}\right)^{\frac{\kappa}{\kappa - 1}}}$$

(3) 臨界流れの出口速度$w_2$を求めよ。（入口速度$w_1 = 0$）

方針

エネルギー保存則と断熱過程の関係から臨界流れの速度を導出します。

途中式と解答

エネルギー保存則：

$$h_1 = h_2 + \frac{w_2^2}{2}$$

断熱過程：

$$h_1 - h_2 = c_p(T_1 - T_2) = \frac{\kappa RT_1}{\kappa - 1}\left(1 - \frac{T_2}{T_1}\right)$$

臨界条件：$T_2/T_1 = 2/(\kappa + 1)$

解答
$$\boxed{w_2 = \sqrt{\frac{2\kappa RT_1}{\kappa + 1}}}$$

(4) 臨界流れの速度と音速の関係を示せ。

方針

臨界点での音速を計算し、臨界流れ速度と比較します。

途中式と解答

音速の定義：$a = \sqrt{\kappa RT}$

臨界点での温度：$T_2 = T_1 \cdot \frac{2}{\kappa + 1}$

臨界点での音速：

$$a_2 = \sqrt{\kappa RT_2} = \sqrt{\kappa RT_1 \cdot \frac{2}{\kappa + 1}} = \sqrt{\frac{2\kappa RT_1}{\kappa + 1}}$$
解答
$$\boxed{w_2 = a_2}$$

結論：臨界流れの速度は音速に一致します。

(5) 適切なノズル形状を選択せよ。
(a) 空気（$\kappa = 1.40$）：$P_1 = 0.18$ MPa → $P_2 = 0.10$ MPa
(b) 過熱蒸気（$\kappa = 1.30$）：$P_1 = 4.00$ MPa → $P_2 = 2.08$ MPa

方針

圧力比と臨界圧力比を比較してノズル形状を決定します。

計算と解答
(a) 空気の場合：

臨界圧力比：$\left(\frac{2}{\kappa + 1}\right)^{\frac{\kappa}{\kappa - 1}} = \left(\frac{2}{2.4}\right)^{3.5} = 0.5283$

実際の圧力比：$\frac{P_2}{P_1} = \frac{0.10}{0.18} = 0.5556$

$0.5556 > 0.5283$（臨界値以上）

(b) 過熱蒸気の場合：

臨界圧力比：$\left(\frac{2}{2.3}\right)^{\frac{1.3}{0.3}} = 0.5457$

実際の圧力比：$\frac{P_2}{P_1} = \frac{2.08}{4.00} = 0.52$

$0.52 < 0.5457$（臨界値未満）

解答

(a) 空気：先細ノズル

理由：圧力比が臨界値以上で亜音速流れのため

(b) 過熱蒸気：末広ノズル

理由：圧力比が臨界値未満で超音速流れが発生するため

ノズル流れの基本
  • 臨界圧力での流れ変化
  • 音速との関係
  • 亜音速・超音速の区別
  • ノズル形状の選択基準
設計のポイント
  • 圧力比による形状決定
  • 先細：亜音速流れ
  • 末広：超音速流れ
  • 比熱比の流体依存性
重要な関係式
  • 臨界圧力比：$(2/(\kappa+1))^{\kappa/(\kappa-1)}$
  • 臨界速度：$\sqrt{2\kappa RT_1/(\kappa+1)}$
  • 音速：$a = \sqrt{\kappa RT}$
  • 臨界条件：$w = a$

大問【1】: 理想気体サイクル

質量$m$の理想気体が、断熱圧縮（状態1→2）、定容加熱（状態2→3）、等温膨張（状態3→4）、定容冷却（状態4→1）の4つの過程からなるサイクルを行う。

(1) $P-V$線図と$T-S$線図を作成せよ。

方針

各過程の熱力学的特性（断熱・定容・等温）を理解し、対応する曲線形状を描きます。

線図作成

P-V線図

圧力 P ↑
     |    3
     |     \
     |      \(等温膨張: PV=const)
     |       \
     |  2     4
     |  |     |
     |  |(定容) |(定容)
     |  |     |
     |  1-----+------→ 容積 V
     |  ↖(断熱圧縮: PV^k=const)
     |

T-S線図

温度 T ↑
     |  3---------4(等温膨張)
     |  |         |
     |  |(定容)    |(定容)
     |  |         |
     |  2         1
     |  |
     | (断熱: S=const)
     +------------------→ エントロピー S
各過程の特徴：
  • 1→2: 断熱圧縮（急峻な曲線、垂直線）
  • 2→3: 定容加熱（垂直線、曲線）
  • 3→4: 等温膨張（双曲線、水平線）
  • 4→1: 定容冷却（垂直線、曲線）

(2) 熱量$Q_1, Q_2, Q_3$を求めよ。

方針

熱力学第一法則$Q = \Delta U + W$を各過程に適用します。

途中式と解答
$Q_1$（2→3: 定容加熱）：

定容過程では$W = 0$

$$Q_1 = \Delta U = m c_v (T_3 - T_2)$$
$Q_2$（3→4: 等温膨張）：

等温過程では$\Delta U = 0$

$$Q_2 = W = mRT_3 \ln\left(\frac{V_4}{V_3}\right) = mRT_3 \ln(r)$$

ここで、$r = V_1/V_2 = V_4/V_3$（圧縮比）

$Q_3$（4→1: 定容冷却）：

定容過程、$T_4 = T_3$（等温過程より）

$$Q_3 = m c_v (T_1 - T_4) = m c_v (T_1 - T_3)$$

(3) 理論熱効率$\eta$を導出せよ。

方針

熱効率$\eta = 1 - |Q_3|/(Q_1 + Q_2)$を計算し、状態量関係式を用いて整理します。

途中式と解答
関係式の整理：
  • $c_v = R/(k-1)$、$r = V_1/V_2$
  • 断熱過程：$T_2 = T_1 r^{k-1}$
  • 定容過程：$T_3 = T_2 \varepsilon = T_1 r^{k-1} \varepsilon$
解答
$$\eta = 1 - \frac{|T_1 - T_3|}{(T_3 - T_2) + \frac{R}{c_v}T_3 \ln(r)}$$ $$= 1 - \frac{r^{k-1}\varepsilon - 1}{r^{k-1}\left[ (\varepsilon - 1) + (k-1)\varepsilon \ln(r) \right]}$$

(4) $r = 10$, $\varepsilon = 2.0$, $k = 1.4$のときの数値を求めよ。

数値計算
計算過程：
  1. $r^{k-1} = 10^{0.4} = 2.5$
  2. 分子: $2.5 \times 2.0 - 1 = 4.0$
  3. 分母: $(2.0 - 1) + 0.4 \times 2.0 \times \ln(10) = 1.0 + 1.84 = 2.84$
  4. 全体: $2.5 \times 2.84 = 7.1$
$$\eta = 1 - \frac{4.0}{7.1} \approx 0.44 = \boxed{44\%}$$
重要な関係式
  • 断熱過程：$PV^k = \text{const}$
  • 等温過程：$PV = \text{const}$
  • 定容過程：$P/T = \text{const}$
  • 熱効率：$\eta = W_{\text{net}}/Q_{\text{in}}$
解法のポイント
  • 線図作成：過程の特徴理解
  • 熱量計算：第一法則の適用
  • 効率導出：状態量の統一表現
  • 数値計算：近似値の活用
注意点
  • 過程の種類を正確に区別
  • 符号の取り扱いに注意
  • 圧縮比の定義確認
  • 対数計算の精度
大問【2】: 水蒸気サイクル
2MPa飽和液（状態1）から等温加熱で1MPa（状態2）、等エントロピー変化で10℃（状態3）、等温冷却（状態4）、等エントロピー変化で状態1復帰のサイクルを解析せよ。

(1) 状態1（2MPa飽和液）の温度と比エントロピーを求めよ。

方針

蒸気表から2MPaの飽和状態データを読み取ります。

解答

蒸気表より：

  • 温度：$\boxed{212.4°\text{C}}$
  • 比エントロピー：$\boxed{2.447 \text{ kJ/(kg·K)}}$

(2) 状態2の状態を判定せよ。

方針

等温（212.4℃）で1MPaまで加熱したときの状態を判定します。

判定過程

1MPaの飽和温度：179.9℃

加熱温度：212.4℃ > 179.9℃

→ 飽和温度より高い温度で気相

解答
$$\boxed{\text{過熱蒸気}}$$

(3) 状態3の状態と圧力を求めよ。

方針

等エントロピー変化で10℃まで冷却したときの状態を判定します。

判定過程

10℃の飽和蒸気エントロピー：$s_g = 8.150$ kJ/(kg·K)

状態2のエントロピー（1MPa, 212.4℃）：約6.000 kJ/(kg·K)

$6.000 < 8.150$ → 湿り蒸気域

解答
  • 状態：$\boxed{\text{湿り蒸気}}$
  • 圧力：10℃の飽和圧力 = $\boxed{1.228 \text{ kPa}}$

(4) 状態4の状態を判定せよ。

解答

状態3から等温冷却（10℃）で液相へ：

$$\boxed{\text{圧縮液}}$$

(5) $T-s$線図を描き、サイクル名を答えよ。

線図とサイクル名

T-s線図

温度 T ↑

  • 湿り蒸気の判定
  • エントロピーによる状態判定

  • 過熱・湿り・圧縮液の区別
  • T-s線図での位置確認

大問【3】: 絞り現象
直径の大きな管内の気体が狭い通路（絞り）を通過する際の現象について解析せよ。

(1) 絞り通過後の速度$w_2$を求めよ。

方針

エネルギー保存則（断熱・非粘性流れ）を適用します。

途中式と解答
エネルギー保存則：
$$h_1 + \frac{w_1^2}{2} = h_2 + \frac{w_2^2}{2}$$
速度$w_2$の導出：
$$w_2^2 = 2(h_1 - h_2) + w_1^2$$
解答

現実的仮定（$w_1 \approx 0$）により：

$$\boxed{w_2 = \sqrt{2(h_1 - h_2)}}$$

(2) 温度変化とその現象名を答えよ。

方針

絞り現象での温度変化の特徴と物理現象名を説明します。

解答
  • 温度変化：$\boxed{\text{下降する}}$
  • 現象名：$\boxed{\text{ジュール・トムソン効果}}$

条件：絞り前温度が逆転温度以下の場合

(3) 温度計の指示誤差を求めよ。

方針

よどみ点温度の概念を用いて温度測定誤差を求めます。

途中式と解答
よどみ点温度：

流れが完全に停止したときの温度

$$T_0 = T + \frac{w_1^2}{2c_p}$$
解答

温度計の指示誤差：

$$\Delta T = T_0 - T = \boxed{\frac{w_1^2}{2c_p}}$$
絞り現象の特徴
  • エンタルピー一定過程
  • 不可逆な圧力降下
  • 温度変化の発生
  • 実用的な冷却方法
解析のポイント
  • エネルギー保存の適用
  • ジュール・トムソン効果の理解
  • よどみ点温度の概念
  • 測定誤差の物理的意味
重要な関係式
  • エネルギー保存：$h_1 + w_1^2/2 = h_2 + w_2^2/2$
  • よどみ点温度：$T_0 = T + w^2/(2c_p)$
  • ジュール・トムソン係数：$\mu = (\partial T/\partial P)_h$
  • 絞り過程：$h_1 = h_2$

大問【1】

(1) 関数 $y = \sqrt{\frac{x^2 + 1}{x^2 - 1}}$ の微分 $\frac{dy}{dx}$ を求めよ。

考え方のヒント

複雑な形の関数は対数微分法が効果的です！自然対数を取ってから微分すると、根号や分数が扱いやすくなります。対数の性質を活用して計算を簡単にしましょう。

途中式と解答
1. 自然対数を取ります：
$$\ln y = \frac{1}{2}\ln\left(\frac{x^2 + 1}{x^2 - 1}\right) = \frac{1}{2}(\ln(x^2 + 1) - \ln(x^2 - 1))$$
2. 両辺を$x$で微分：
$$\frac{1}{y}\frac{dy}{dx} = \frac{1}{2}\left(\frac{2x}{x^2 + 1} - \frac{2x}{x^2 - 1}\right) = x\left(\frac{1}{x^2 + 1} - \frac{1}{x^2 - 1}\right)$$
3. 右辺を整理：
$$\frac{1}{x^2 + 1} - \frac{1}{x^2 - 1} = \frac{(x^2 - 1) - (x^2 + 1)}{(x^2 + 1)(x^2 - 1)} = \frac{-2}{x^4 - 1}$$
4. $\frac{dy}{dx}$について解く：
$$\frac{dy}{dx} = y \cdot x \cdot \frac{-2}{x^4 - 1} = \sqrt{\frac{x^2 + 1}{x^2 - 1}} \cdot \frac{-2x}{x^4 - 1}$$
5. $x^4 - 1 = (x^2 - 1)(x^2 + 1)$より：
$$\frac{dy}{dx} = \sqrt{\frac{x^2 + 1}{x^2 - 1}} \cdot \frac{-2x}{(x^2 - 1)(x^2 + 1)} = \frac{-2x}{(x^2 - 1)^{1/2}(x^2 + 1)^{3/2}}$$
答え: $\frac{dy}{dx} = -\frac{2x}{(x^{2}-1)^{1/2}(x^{2}+1)^{3/2}}$

(2) 広義積分 $\int_{1}^{\infty} \frac{\log x}{x^3} dx$ を求めよ。

考え方のヒント

この広義積分は、$x \to \infty$での収束性と、分母の$x^3$による減衰がポイントです。$x = 1$以降の区間で特異性はありませんが、$x \to \infty$での挙動を確認し、部分積分を使って計算します。

途中式と解答
1. $\int_{1}^{\infty} \frac{\log x}{x^3} dx$ を部分積分： $u = \log x$，$dv = x^{-3}dx$ とすると
$$du = \frac{1}{x}dx，\quad v = -\frac{1}{2}x^{-2}$$
$$\int_{1}^{\infty} = \left[-\frac{\log x}{2x^2}\right]_{1}^{\infty} + \frac{1}{2}\int_{1}^{\infty} x^{-3}dx = 0 + \frac{1}{2} \cdot \frac{1}{4} = \frac{1}{4}$$
答え: $\frac{1}{4}$
全体のポイント
  • 対数微分法で複雑な関数を簡単に
  • 広義積分は特異点を分けて考える
  • 収束・発散の判定が重要
受験数学のチップス
  • $\ln y' = (\ln y)'$ で対数微分
  • $\int_{0}^{\infty} = \int_{0}^{1} + \int_{1}^{\infty}$ で分割
  • $\log x / x^n$ 型は部分積分が基本
大問【2】

(1) 同次微分方程式 $y' - \frac{x}{x^2 + 1} y = 0$ を解け。

考え方のヒント

これは変数分離形の同次微分方程式です！$\frac{dy}{y} = \frac{x}{x^2 + 1}dx$ として両辺を積分しましょう。右辺は置換積分で簡単に計算できます。

途中式と解答
1. 変数分離：
$$\frac{dy}{y} = \frac{x}{x^2 + 1}dx$$
2. 両辺を積分：
$$\int \frac{dy}{y} = \int \frac{x}{x^2 + 1}dx$$
$$\ln|y| = \frac{1}{2}\ln(x^2 + 1) + C_1$$
3. 指数をとって解く：
$$|y| = e^{C_1} \cdot \sqrt{x^2 + 1}$$
$$y = C\sqrt{x^2 + 1}$$
答え: $y = C\sqrt{x^{2} + 1}$

(2) 非同次微分方程式 $y' - \frac{x}{x^2 + 1} y = x$ を解け。

考え方のヒント

(1)の解を利用して定数変化法を使いましょう！$y = C(x)\sqrt{x^2 + 1}$ とおいて、$C(x)$ を求めます。非同次項が $x$ なので計算は比較的簡単になります。

途中式と解答
1. 定数変化法：$y = C(x)\sqrt{x^2 + 1}$ とおく
2. $y' = C'(x)\sqrt{x^2 + 1} + C(x) \cdot \frac{x}{\sqrt{x^2 + 1}}$ を微分方程式に代入：
$$C'(x)\sqrt{x^2 + 1} + C(x) \cdot \frac{x}{\sqrt{x^2 + 1}} - \frac{x}{x^2 + 1} \cdot C(x)\sqrt{x^2 + 1} = x$$
3. 整理すると：
$$C'(x)\sqrt{x^2 + 1} = x$$
$$C'(x) = \frac{x}{\sqrt{x^2 + 1}}$$
4. $C(x)$ を求める：
$$C(x) = \int \frac{x}{\sqrt{x^2 + 1}}dx = \sqrt{x^2 + 1} + K$$
5. 一般解：
$$y = (\sqrt{x^2 + 1} + K)\sqrt{x^2 + 1} = x^2 + 1 + K\sqrt{x^2 + 1}$$
答え: $y = x^{2} + 1 + K\sqrt{x^{2} + 1}$
全体のポイント
  • 同次方程式は変数分離で解く
  • 非同次方程式は定数変化法が有効
  • 同次解を利用して効率的に解く
受験数学のチップス
  • $\int \frac{x}{x^2+1}dx = \frac{1}{2}\ln(x^2+1) + C$
  • 定数変化法：$y = C(x) \times y_{\text{同次}}$
  • $\int \frac{x}{\sqrt{x^2+1}}dx = \sqrt{x^2+1} + C$
大問【3】

行列 $B = \begin{pmatrix} 3 & -1 & -1 \\ 4 & 0 & 0 \\ -3 & 2 & 2 \end{pmatrix}$ の行列式を求めよ。

考え方のヒント

3×3行列の行列式はサルスの定理や余因子展開で計算できます！第2行に0が多いので、第2行での余因子展開が効率的ですね。計算ミスに注意して丁寧に進めましょう。

途中式と解答
1. 第2行での余因子展開を行います（0が多いため）：
$$|B| = 4 \cdot (-1)^{2+1} \begin{vmatrix} -1 & -1 \\ 2 & 2 \end{vmatrix} + 0 + 0$$
2. 2×2行列式を計算：
$$\begin{vmatrix} -1 & -1 \\ 2 & 2 \end{vmatrix} = (-1) \cdot 2 - (-1) \cdot 2 = -2 + 2 = 0$$
3. したがって：
$$|B| = 4 \cdot (-1) \cdot 0 = 0$$
答え: $|B| = 0$
全体のポイント
  • 0の多い行・列で余因子展開
  • 2×2行列式は暗算で計算
  • 行列式=0は行列が特異（非正則）
受験数学のチップス
  • 余因子展開：$|A| = \sum a_{ij}(-1)^{i+j}M_{ij}$
  • $\begin{vmatrix} a & b \\ c & d \end{vmatrix} = ad - bc$
  • 行列式=0 ⇔ 逆行列が存在しない
大問【4】

(1) 畳み込みのラプラス変換：$\mathcal{L}\left[\int_{0}^{t} f(t-\tau) g(\tau) d\tau\right] = \mathcal{L}[f] \mathcal{L}[g]$ を証明せよ。

考え方のヒント

畳み込みの定義から始めて、積分順序の変更を行います！変数変換 $u = t - \tau$ を使って、最終的に $\mathcal{L}[f] \mathcal{L}[g]$ の形に帰着させましょう。

途中式と解答
1. 畳み込みのラプラス変換を定義に従って計算：
$$\mathcal{L}[(f * g)(t)] = \int_{0}^{\infty} e^{-st} \left(\int_{0}^{t} f(t-\tau) g(\tau) d\tau\right) dt$$
2. 積分順序を変更：
$$= \int_{0}^{\infty} g(\tau) \left(\int_{\tau}^{\infty} e^{-st} f(t-\tau) dt\right) d\tau$$
3. 変数変換 $u = t - \tau$ を適用：
$$= \int_{0}^{\infty} g(\tau) e^{-s\tau} \left(\int_{0}^{\infty} e^{-su} f(u) du\right) d\tau$$
4. 因数分解：
$$= \left(\int_{0}^{\infty} e^{-su} f(u) du\right) \left(\int_{0}^{\infty} g(\tau) e^{-s\tau} d\tau\right) = \mathcal{L}[f] \mathcal{L}[g]$$
証明完了

(2) 積分方程式 $y(t) = \int_{0}^{t} e^{t-\tau} y(\tau) d\tau + t + 1$ を解け。

考え方のヒント

(1)の畳み込み公式を利用しましょう！$\int_{0}^{t} e^{t-\tau} y(\tau) d\tau$ は $e^t$ と $y(t)$ の畳み込みです。ラプラス変換で代数方程式に変換して解きます。

途中式と解答
1. ラプラス変換を適用：
$$Y(s) = \mathcal{L}[e^t] Y(s) + \mathcal{L}[t] + \mathcal{L}[1]$$
$$Y(s) = \frac{1}{s-1} Y(s) + \frac{1}{s^2} + \frac{1}{s}$$
2. $Y(s)$ について整理：
$$Y(s)\left(1 - \frac{1}{s-1}\right) = \frac{s+1}{s^2}$$
$$Y(s) \cdot \frac{s-2}{s-1} = \frac{s+1}{s^2}$$
$$Y(s) = \frac{(s+1)(s-1)}{s^2(s-2)} = \frac{s^2-1}{s^2(s-2)}$$
3. 部分分数分解：
$$\frac{s^2-1}{s^2(s-2)} = \frac{A}{s} + \frac{B}{s^2} + \frac{C}{s-2}$$
計算すると：$A = \frac{1}{4}$，$B = \frac{1}{2}$，$C = \frac{3}{4}$
4. 逆ラプラス変換：
$$y(t) = \frac{1}{4} + \frac{1}{2}t + \frac{3}{4}e^{2t}$$
答え: $y(t) = \frac{1}{4} + \frac{1}{2}t + \frac{3}{4}e^{2t}$
全体のポイント
  • 畳み込み定理は積分方程式の強力な道具
  • 積分順序の変更と変数変換が鍵
  • 部分分数分解で逆変換を簡単に
受験数学のチップス
  • $\mathcal{L}[e^{at}] = \frac{1}{s-a}$，$\mathcal{L}[t^n] = \frac{n!}{s^{n+1}}$
  • 畳み込み：$(f * g)(t) = \int_0^t f(t-\tau)g(\tau)d\tau$
  • 部分分数分解は逆変換の基本技法

問題概要

2014年度の数学入試では、大問【1】で対数微分法と広義積分、大問【2】でクレローの微分方程式（一般解と特異解）、大問【3】で行列の対角化、大問【4】でラプラス変換の遅延定理が出題されました。多彩な分野から基礎から応用まで網羅した良問です。特に微分方程式の特異解や対角化の計算技法をしっかり押さえましょう！

大問【1】

(1) 関数 $y = x^{\sin x}$ ($x > 0$) の微分 $\frac{dy}{dx}$ を求めよ。

考え方のヒント

指数に変数が含まれる関数は対数微分法が最強です！自然対数を取ると積の微分法則が使えて計算が楽になります。$\ln y = \sin x \cdot \ln x$ の形にして微分しましょう。

途中式と解答
1. 自然対数を取ります：
$$\ln y = \sin x \cdot \ln x$$
2. 両辺を$x$で微分（積の微分法則）：
$$\frac{1}{y}\frac{dy}{dx} = \cos x \cdot \ln x + \sin x \cdot \frac{1}{x}$$
3. $y$を代入して整理：
$$\frac{dy}{dx} = y\left(\cos x \cdot \ln x + \frac{\sin x}{x}\right)$$
$$= x^{\sin x}\left(\cos x \cdot \ln x + \frac{\sin x}{x}\right)$$
答え: $\frac{dy}{dx} = x^{\sin x}\left(\cos x \cdot \ln x + \frac{\sin x}{x}\right)$

(2) 広義積分 $\int_{-\infty}^{\infty} \frac{dx}{{\sqrt{1 + x^2}}^3}$ を求めよ。

考え方のヒント

これは無限区間での積分なので 広義積分 です。また偶関数なので，0から無限大までを計算すれば十分．

途中式と解答
1. 被積分関数は偶関数です。すなわち、 $$f(-x) = \frac{1}{(1+(-x)^2)^{3/2}} = \frac{1}{(1+x^2)^{3/2}} = f(x).$$ よって、積分範囲を分割して $$ I = 2 \int_0^\infty \frac{1}{(1+x^2)^{3/2}} dx $$ と書けます。計算が楽になるので、こちらを扱います。
2. 置換 $x = \tan\theta$ を使う：
$x: 0 \to \infty$ ⇒ $\theta: 0 \to \frac{\pi}{2}$
$$dx = \frac{1}{\cos^2 \theta} d\theta$$ 被積分関数は $$ (1+x^2)^{-\frac{3}{2}} = (1 + \tan^2 \theta)^{-\frac{3}{2}} = (\sec^2 \theta)^{-\frac{3}{2}} = \sec^{-3} \theta = \cos^3 \theta. $$ よって積分は $$ \int_0^\infty \frac{dx}{(1+x^2)^{3/2}} = \int_0^{\pi/2} \cos^3 \theta \cdot \sec^2 \theta d\theta = \int_0^{\pi/2} \cos^3 \theta \cdot \frac{1}{\cos^2 \theta} d\theta = \int_0^{\pi/2} \cos \theta d\theta. $$
3. 積分計算
$$ \int_0^{\pi/2} \cos \theta d\theta = \left[ \sin \theta \right]_0^{\pi/2} = \sin \frac{\pi}{2} - \sin 0 = 1 - 0 = 1. $$ 元の積分は偶関数のため $$ I = 2 \times 1 = 2. $$
答え: $\int_{-\infty}^{\infty} \frac{dx}{(1+x^2)^{3/2}} = 2$
全体のポイント
  • 指数に変数が含まれる関数は対数微分
  • 双曲線関数の置換積分が有効
  • 広義積分の発散判定も重要
受験数学のチップス
  • $(\ln y)' = \frac{y'}{y}$ で対数微分の基本
  • $x = \tan\theta$ の置換では、$dx = \sec^2\theta\,d\theta$ となる。
    この微分公式を使うことで、積分区間や被積分関数が三角関数に変換され、計算が簡単になる。
  • 広義積分では $\frac{1}{x^p}$ 型の発散判定や、置換積分の微分公式をしっかり覚えておくと便利。
大問【2】

微分方程式 $y = xy' + \sqrt{1 + y'^2}$ について以下を答えよ。

考え方のヒント

これはクレローの微分方程式です！$p = y'$とおいて両辺を微分すると、$p' = 0$または特定の条件を満たす場合に分かれます。一般解と特異解の両方を求めるのがポイントです。

(1) 両辺を$x$で微分して導出される関係式を示せ。

途中式と解答
1. $p = y'$とおき、両辺を$x$で微分：
$$p = p + xp' + \frac{pp'}{\sqrt{1 + p^2}}$$
2. 整理すると：
$$0 = xp' + \frac{pp'}{\sqrt{1 + p^2}} = p'\left(x + \frac{p}{\sqrt{1 + p^2}}\right)$$
3. したがって：
$$p' = 0 \quad \text{または} \quad x + \frac{p}{\sqrt{1 + p^2}} = 0$$
関係式: $p' = 0$ または $x + \frac{p}{\sqrt{1 + p^2}} = 0$

(2) 一般解を求めよ。

途中式と解答
1. $p' = 0$の場合：$p = C$（定数）
2. $y' = C$を元の方程式に代入：
$$y = xC + \sqrt{1 + C^2}$$
答え: $y = Cx + \sqrt{1 + C^2}$ （$C$は任意定数）

(3) 特異解を求めよ。

途中式と解答
1. $x + \frac{p}{\sqrt{1 + p^2}} = 0$から：
$$p = -\frac{x}{\sqrt{1 - x^2}} \quad (|x| < 1)$$
2. 元の方程式に代入：
$$y = x\left(-\frac{x}{\sqrt{1 - x^2}}\right) + \sqrt{1 + \frac{x^2}{1 - x^2}}$$
3. 計算すると：
$$y = -\frac{x^2}{\sqrt{1 - x^2}} + \sqrt{\frac{1}{1 - x^2}} = \sqrt{1 - x^2}$$
答え: $y = \sqrt{1 - x^2}$

なお$p = +\frac{x}{\sqrt{1 - x^2}}$によって求まる$\frac{x^2+1}{\sqrt{1-x^2}}$は「2乗したことによって生じた無縁解だから」という理由で，正しくないらしい．

全体のポイント
  • クレローの方程式：$y = xy' + f(y')$
  • 一般解と特異解の両方を求める
  • 特異解は包絡線として幾何学的意味
受験数学のチップス
  • $p = y'$とおいて微分するのが定石
  • 特異解は$p$を消去して得られる
  • 幾何学的には直線群の包絡線
大問【3】

行列 $A = \begin{pmatrix} 4 & -3 \\ -1 & 2 \end{pmatrix}$ を対角化する行列 $P$ を求めよ。

考え方のヒント

固有値と固有ベクトルを求めて対角化行列を作ります！特性方程式$\det(A - \lambda I) = 0$を解いて固有値を求め、各固有値に対応する固有ベクトルを計算しましょう。

途中式と解答
1. 固有値を求める（特性方程式）：
$$\det(A - \lambda I) = \begin{vmatrix} 4-\lambda & -3 \\ -1 & 2-\lambda \end{vmatrix}$$
$$= (4-\lambda)(2-\lambda) - 3 = \lambda^2 - 6\lambda + 5 = 0$$
$$\therefore \lambda = 1, 5$$
2. $\lambda = 1$の固有ベクトル：
$$(A - I)\mathbf{v} = \begin{pmatrix} 3 & -3 \\ -1 & 1 \end{pmatrix}\mathbf{v} = \mathbf{0}$$
$3v_1 - 3v_2 = 0$より$v_1 = v_2$なので：$\mathbf{v} = \begin{pmatrix} 1 \\ 1 \end{pmatrix}$
3. $\lambda = 5$の固有ベクトル：
$$(A - 5I)\mathbf{v} = \begin{pmatrix} -1 & -3 \\ -1 & -3 \end{pmatrix}\mathbf{v} = \mathbf{0}$$
$-v_1 - 3v_2 = 0$より$v_1 = -3v_2$なので：$\mathbf{v} = \begin{pmatrix} 3 \\ -1 \end{pmatrix}$
4. 対角化行列$P$を構成：
$$P = \begin{pmatrix} 1 & 3 \\ 1 & -1 \end{pmatrix}$$
答え: $P = \begin{pmatrix} 1 & 3 \\ 1 & -1 \end{pmatrix}$
全体のポイント
  • 特性方程式で固有値を求める
  • 各固有値に対応する固有ベクトル計算
  • 固有ベクトルを列に並べて行列$P$
受験数学のチップス
  • $\det(A - \lambda I) = 0$が特性方程式
  • $(A - \lambda I)\mathbf{v} = \mathbf{0}$で固有ベクトル
  • $P^{-1}AP = D$（$D$は対角行列）
大問【4】

ラプラス変換の遅延定理について以下を証明せよ。

考え方のヒント

遅延定理はラプラス変換の重要な性質です！単位ステップ関数$u(t-a)$を使って、関数の時間シフトがラプラス変換でどう表現されるかを証明します。変数変換が鍵になります。

(1) $\mathcal{L}[f(t-a)u(t-a)] = e^{-as}\mathcal{L}[f(t)]$ を証明せよ。

途中式と解答
1. ラプラス変換の定義から：
$$\mathcal{L}[f(t-a)u(t-a)] = \int_0^{\infty} f(t-a)u(t-a) e^{-st} dt$$
2. $u(t-a) = 0$ $(t < a)$，$u(t-a) = 1$ $(t \geq a)$より：
$$= \int_a^{\infty} f(t-a) e^{-st} dt$$
3. $\tau = t - a$と置換：$t = \tau + a$，$dt = d\tau$
$$= \int_0^{\infty} f(\tau) e^{-s(\tau + a)} d\tau = e^{-as} \int_0^{\infty} f(\tau) e^{-s\tau} d\tau$$
4. 右辺は$\mathcal{L}[f(t)]$そのものです：
$$= e^{-as} \mathcal{L}[f(t)]$$
証明完了

(2) $\mathcal{L}[f(t)u(t-a)] = e^{-as}\mathcal{L}[f(t+a)]$ を証明せよ。

途中式と解答
1. ラプラス変換の定義から：
$$\mathcal{L}[f(t)u(t-a)] = \int_0^{\infty} f(t)u(t-a) e^{-st} dt = \int_a^{\infty} f(t) e^{-st} dt$$
2. $\tau = t - a$と置換：
$$= \int_0^{\infty} f(\tau + a) e^{-s(\tau + a)} d\tau$$
3. 整理すると：
$$= e^{-as} \int_0^{\infty} f(\tau + a) e^{-s\tau} d\tau = e^{-as} \mathcal{L}[f(t+a)]$$
証明完了

(3) 図1の関数のラプラス変換

途中式と解答

図1で示された関数は：

$$ f(t) = \begin{cases} 0, & 0 \leq t < 1 \\ -(t^2 - 4t + 3), & 1 \leq t < 3 \\ 0, & t \geq 3 \end{cases} $$

放物線部分を因数分解すると：

$$-(t^2 - 4t + 3) = -(t-1)(t-3)$$
2. 使用する公式の確認
$$(1) : \quad \mathcal{L}[f(t-a)u(t-a)] = e^{-as}\mathcal{L}[f(t)]$$ $$(2) : \quad \mathcal{L}[f(t)u(t-a)] = e^{-as}\mathcal{L}[f(t+a)]$$

今回は公式(2)を主に使用します。これにより複雑な積分計算を回避できます。

3. 関数を単位ステップ関数で表現

基本形への変換

$$f(t) = [-(t-1)(t-3)] \cdot [u(t-1) - u(t-3)]$$

分解による表現

$$f(t) = g(t)u(t-1) - g(t)u(t-3)$$

ここで $$g(t) = -(t-1)(t-3)$$ とします。

重要ポイント： $$g(t)$$ は元の放物線関数全体を表し、単位ステップ関数が「窓」の役割を果たします。
4. 公式(2)の適用

ラプラス変換の分解

$$\mathcal{L}[f(t)] = \mathcal{L}[g(t)u(t-1)] - \mathcal{L}[g(t)u(t-3)]$$

公式(2)の適用

$$\mathcal{L}[f(t)] = e^{-s}\mathcal{L}[g(t+1)] - e^{-3s}\mathcal{L}[g(t+3)]$$

公式(2)では、$u(t-a)$の a が指数の係数となり、関数内部では t を t+a にシフトします。

5. シフトした関数の計算

g(t+1) の計算

$$g(t) = -(t-1)(t-3)$$
$$g(t+1) = -((t+1)-1)((t+1)-3) = -t(t-2) = -t^2 + 2t$$

g(t+3) の計算

$$g(t+3) = -((t+3)-1)((t+3)-3) = -(t+2)t = -t^2 - 2t$$
計算の確認：
g(t+1): 係数が (-1, +2, 0)
g(t+3): 係数が (-1, -2, 0)
6. 基本ラプラス変換の適用
  • $$\mathcal{L}[1] = \frac{1}{s}$$
  • $$\mathcal{L}[t] = \frac{1}{s^2}$$
  • $$\mathcal{L}[t^2] = \frac{2}{s^3}$$

各シフト関数のラプラス変換

$$\mathcal{L}[g(t+1)] = -\mathcal{L}[t^2] + 2\mathcal{L}[t] = -\frac{2}{s^3} + \frac{2}{s^2} = \frac{2(s-1)}{s^3}$$
$$\mathcal{L}[g(t+3)] = -\mathcal{L}[t^2] - 2\mathcal{L}[t] = -\frac{2}{s^3} - \frac{2}{s^2} = \frac{-2(s+1)}{s^3}$$
7. 最終結果の統合
$$\mathcal{L}[f(t)] = e^{-s} \cdot \frac{2(s-1)}{s^3} + e^{-3s} \cdot \frac{2(s+1)}{s^3}$$
答え：$$\boxed{F(s) = \frac{2e^{-s}(s-1) + 2e^{-3s}(s+1)}{s^3}}$$
全体のポイント
  • 遅延定理は時間シフトの表現
  • 変数変換で積分範囲を調整
  • 単位ステップ関数で区間制限
受験数学のチップス
  • $u(t-a)$ は$t=a$でステップ関数
  • 時間遅延は$e^{-as}$で表現
  • 区間制限された関数は分解して考える

大問【1】

(1) 関数 $F(x,y) = xe^y - y$ について、方程式 $F(x,y) = 0$ によって定まる陰関数 $y = f(x)$ のグラフの概形を $x \geq 0, y \geq 0$ の範囲で描け。

考え方のヒント

陰関数のグラフを描くには、まず陰関数定理が適用できるかを確認し、導関数を求めて増減を調べましょう！特異点（導関数が無限大になる点）も重要なポイントです。$xe^y = y$ という関係式から、どんな曲線になるか想像してみてください。

途中式と解答
1. まず、方程式 $xe^y = y$ を分析します。$x = 0$ のとき $y = 0$ なので、原点$(0, 0)$を通ります。
2. 陰関数微分により導関数を求めます：
$$\frac{dy}{dx} = -\frac{\frac{\partial F}{\partial x}}{\frac{\partial F}{\partial y}} = -\frac{e^y}{xe^y - 1} = \frac{e^y}{1 - xe^y}$$
3. 原点での傾きは：$\frac{dy}{dx}\bigg|_{(0,0)} = \frac{e^0}{1 - 0 \cdot e^0} = 1$
4. 特異点を調べます。分母が0になるのは $xe^y = 1$ のとき：
$$x = \frac{1}{e^y} = e^{-y}$$
これを $xe^y = y$ に代入すると $y = 1$、$x = \frac{1}{e} \approx 0.368$
5. 曲線の挙動： - $x \in [0, \frac{1}{e})$ で単調増加 - $x \to \frac{1}{e}^-$ で $y \to 1^-$、接線は垂直になる - $x > \frac{1}{e}$ では陰関数として定義されない
グラフ: 原点から始まり、点$(\frac{1}{e}, 1)$に向かって増加する曲線

(2) $f'(x)$ を $x$ と $y$ で表せ。

考え方のヒント

(1)で求めた陰関数微分の結果をそのまま使えばOKです！陰関数微分の公式 $\frac{dy}{dx} = -\frac{F_x}{F_y}$ を思い出しましょう。

途中式と解答
(1)で計算した結果をそのまま使います：
$$f'(x) = \frac{dy}{dx} = \frac{e^y}{1 - xe^y}$$
答え: $f'(x) = \frac{e^{y}}{1 - xe^{y}}$
全体のポイント
  • 陰関数定理の適用条件を確認する
  • 特異点（分母=0）で曲線の性質が変わる
  • 陰関数微分は偏微分の比で求める
受験数学のチップス
  • 陰関数のグラフは特異点で注意深く調べる
  • $xe^y = y$ は超越方程式で解析的に解けない
  • 接線の傾きが無限大→垂直漸近線の可能性
大問【2】

(1) 直線 $y = x$ 上に中心をもつ任意の半径の円が満たす微分方程式を求めよ。

考え方のヒント

中心が$(h, h)$、半径$r$の円の方程式から出発しましょう！パラメータ$h$と$r$を消去して、$x$、$y$、$y'$、$y''$だけの関係式にします。円の方程式を2回微分するのがポイントです。

途中式と解答
1. 中心$(h, h)$、半径$r$の円の方程式：
$$(x - h)^2 + (y - h)^2 = r^2$$
2. $x$で微分（$y = y(x)$）：
$$2(x - h) + 2(y - h)y' = 0$$
$$(x - h) + (y - h)y' = 0$$
3. さらに$x$で微分：
$$1 + y'^2 + (y - h)y'' = 0$$
4. 2つ目の式から $(y - h) = -\frac{x - h}{y'}$ なので、これを3つ目の式に代入：
$$1 + y'^2 - \frac{x - h}{y'}y'' = 0$$
5. 2つ目の式から $(x - h) = -(y - h)y'$ を使って整理すると：
$$y''(x - y) = (1 + y'^2)(1 + y')$$
答え: $y''(x - y) = (1 + y'^{2})(1 + y')$

(2) 微分方程式 $\frac{dy}{dx} - x(1 + y^2) = 0$ の一般解と、初期条件 $x = 0, y = 0$ を満たす特解のグラフを描け。

考え方のヒント

これは変数分離形の微分方程式です！$\frac{dy}{1 + y^2} = x dx$ として両辺を積分しましょう。左辺は $\arctan y$ になります。特解では初期条件から積分定数を決定します。

途中式と解答
1. 変数分離を行います：
$$\frac{dy}{dx} = x(1 + y^2)$$
$$\frac{dy}{1 + y^2} = x dx$$
2. 両辺を積分：
$$\int \frac{dy}{1 + y^2} = \int x dx$$
$$\arctan y = \frac{x^2}{2} + C$$
3. 一般解を求めます：
$$y = \tan\left(\frac{x^2}{2} + C\right)$$
4. 初期条件 $x = 0, y = 0$ から：
$$0 = \tan(C) \implies C = 0$$
5. 特解とその性質：
$$y = \tan\left(\frac{x^2}{2}\right)$$
- 定義域：$|x| < \sqrt{\pi}$（$\frac{x^2}{2} < \frac{\pi}{2}$） - $x = \pm\sqrt{\pi}$ で垂直漸近線 - 偶関数（$y(-x) = y(x)$）
一般解: $y = \tan\left(\frac{x^{2}}{2} + C\right)$　特解: $y = \tan\left(\frac{x^{2}}{2}\right)$
全体のポイント
  • 曲線族の微分方程式ではパラメータを消去
  • 変数分離形は両辺を適切に積分
  • $\tan$ 関数は周期的な垂直漸近線をもつ
受験数学のチップス
  • $\int \frac{dy}{1 + y^2} = \arctan y + C$ は基本公式
  • $\tan$ の定義域に注意（$\frac{\pi}{2}$ の奇数倍で発散）
  • 初期条件から積分定数を正確に決定
大問【3】

(1) 関数 $\phi(x,y,z) = \frac{1}{2}x^2 + y^2 + 2z^2$ について、曲面 $\phi(x,y,z) = C$ 上の点における単位法線ベクトルを求めよ。

考え方のヒント

曲面の法線ベクトルは勾配ベクトル $\nabla \phi$ で求まります！勾配ベクトルを計算して、それを大きさで割って単位ベクトルにしましょう。$\phi$ の各変数での偏微分を求めるのがスタートです。

途中式と解答
1. 勾配ベクトルを計算します：
$$\nabla \phi = \left( \frac{\partial \phi}{\partial x}, \frac{\partial \phi}{\partial y}, \frac{\partial \phi}{\partial z} \right) = (x, 2y, 4z)$$
2. 勾配ベクトルの大きさを求めます：
$$\| \nabla \phi \| = \sqrt{x^2 + (2y)^2 + (4z)^2} = \sqrt{x^2 + 4y^2 + 16z^2}$$
3. 単位法線ベクトルは：
$$\mathbf{n} = \frac{\nabla \phi}{\| \nabla \phi \|} = \frac{(x, 2y, 4z)}{\sqrt{x^2 + 4y^2 + 16z^2}}$$
答え: $\mathbf{n} = \frac{(x,\ 2y,\ 4z)}{\sqrt{x^{2} + 4y^{2} + 16z^{2}}}$

(2) 点 $P(1,1,1)$ を通るときの定数 $C$ の値を求めよ。

考え方のヒント

点$P(1,1,1)$が曲面$\phi(x,y,z) = C$上にあるということは、この点の座標を$\phi$に代入すれば$C$の値が分かります！

途中式と解答
点$P(1,1,1)$を$\phi(x,y,z)$に代入：
$$C = \phi(1,1,1) = \frac{1}{2}(1)^2 + (1)^2 + 2(1)^2 = \frac{1}{2} + 1 + 2 = \frac{7}{2}$$
答え: $C = \frac{7}{2}$

(3) 点 $P(1,1,1)$ における直線 $x = y = z$ の方向への方向微分係数を求めよ。

考え方のヒント

方向微分係数は $\nabla \phi \cdot \mathbf{u}$ で求まります！直線$x = y = z$の方向ベクトルは$(1,1,1)$なので、これを単位ベクトルにして内積を計算しましょう。

途中式と解答
1. 点$P(1,1,1)$での勾配ベクトル：
$$\nabla \phi(1,1,1) = (1, 2, 4)$$
2. 直線$x = y = z$の方向ベクトル：$(1,1,1)$
3. 単位方向ベクトル：
$$\mathbf{u} = \frac{(1,1,1)}{\sqrt{1^2 + 1^2 + 1^2}} = \frac{(1,1,1)}{\sqrt{3}}$$
4. 方向微分係数：
$$\nabla \phi \cdot \mathbf{u} = (1, 2, 4) \cdot \frac{(1,1,1)}{\sqrt{3}} = \frac{1 + 2 + 4}{\sqrt{3}} = \frac{7}{\sqrt{3}} = \frac{7\sqrt{3}}{3}$$
答え: $\frac{7\sqrt{3}}{3}$
全体のポイント
  • 曲面の法線ベクトルは勾配ベクトルで求める
  • 単位ベクトルは大きさで割って正規化
  • 方向微分は勾配と方向の内積
受験数学のチップス
  • $\nabla \phi = \left(\frac{\partial \phi}{\partial x}, \frac{\partial \phi}{\partial y}, \frac{\partial \phi}{\partial z}\right)$
  • 方向微分は $\nabla f \cdot \mathbf{u}$（$\mathbf{u}$は単位ベクトル）
  • 内積計算では各成分を正確に掛け合わせる
大問【4】

(1) 関係式 $L\left[\frac{f(t)}{t}\right] = \int_{s}^{\infty} F(x) dx$ を証明せよ。ただし、$F(s) = L[f(t)]$ とする。

考え方のヒント

ラプラス変換の定義を使って、両辺を具体的に計算してみましょう！積分順序の交換がカギになります。$F(s) = \int_0^{\infty} e^{-st}f(t)dt$ から出発して、右辺の積分を計算します。

途中式と解答
1. 左辺を計算します：
$$L\left[\frac{f(t)}{t}\right] = \int_{0}^{\infty} e^{-st} \frac{f(t)}{t} dt$$
2. 右辺を展開します：
$$\int_{s}^{\infty} F(x) dx = \int_{s}^{\infty} \left( \int_{0}^{\infty} e^{-xt} f(t) dt \right) dx$$
3. 積分順序を交換（適当な条件下で）：
$$= \int_{0}^{\infty} f(t) \left( \int_{s}^{\infty} e^{-xt} dx \right) dt$$
4. 内側の積分を計算：
$$\int_{s}^{\infty} e^{-xt} dx = \left[ -\frac{1}{t} e^{-xt} \right]_{x=s}^{\infty} = \frac{1}{t} e^{-st}$$
5. 結果をまとめると：
$$\int_{s}^{\infty} F(x) dx = \int_{0}^{\infty} e^{-st} \frac{f(t)}{t} dt = L\left[\frac{f(t)}{t}\right]$$
証明完了

(2) $g(t) = \frac{1 - e^{at}}{t}$ のラプラス変換を求めよ。

考え方のヒント

(1)の結果を活用しましょう！$f(t) = 1 - e^{at}$ のラプラス変換を求めて、それを$s$から無限大まで積分します。$L[1] = \frac{1}{s}$、$L[e^{at}] = \frac{1}{s-a}$ を使います。

途中式と解答
1. $f(t) = 1 - e^{at}$ のラプラス変換を求めます：
$$F(s) = L[1 - e^{at}] = L[1] - L[e^{at}] = \frac{1}{s} - \frac{1}{s - a}$$
2. (1)の結果を適用：
$$L[g(t)] = \int_{s}^{\infty} F(x) dx = \int_{s}^{\infty} \left( \frac{1}{x} - \frac{1}{x - a} \right) dx$$
3. 積分を計算：
$$= \left[ \ln|x| - \ln|x - a| \right]_{s}^{\infty} = \left[ \ln\left|\frac{x}{x - a}\right| \right]_{s}^{\infty}$$
4. 極限を取ると：
$$= \lim_{b \to \infty} \ln\left(\frac{b}{b - a}\right) - \ln\left(\frac{s}{s - a}\right) = 0 - \ln\left(\frac{s}{s - a}\right) = \ln\left(\frac{s - a}{s}\right)$$
答え: $\ln\left(\frac{s - a}{s}\right)$
全体のポイント
  • 積分順序の交換は慎重に適用
  • 基本的なラプラス変換公式を活用
  • 対数関数の性質を使って計算を簡略化
受験数学のチップス
  • $L[1] = \frac{1}{s}$、$L[e^{at}] = \frac{1}{s-a}$ は基本公式
  • $\ln\left|\frac{a}{b}\right| = \ln|a| - \ln|b|$
  • 無限大での極限では主要項に注目

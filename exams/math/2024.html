大問【1】

(1) \( y = \log x \) の \( x = e \) における接線の方程式を求めよ。

考え方のヒント

接線の方程式は 「点と傾き」 で決まります！まず接点での座標を求めて、次に微分して傾きを求めましょう。あとは接線の公式 \( y - y_1 = m(x - x_1) \) に代入するだけですね。

途中式と解答
1. まず接点の座標を求めます。\( x = e \) のとき \( y = \log e = 1 \) なので、接点は \((e, 1)\) です。
2. 次に傾きを求めましょう。\( y = \log x \) を微分すると \( y' = \frac{1}{x} \) です。\( x = e \) での傾きは \( y' = \frac{1}{e} \) ですね。
3. 接線の方程式に代入します。
$$y - 1 = \frac{1}{e}(x - e) \implies y = \frac{1}{e}x$$
答え: \(y = \frac{1}{e}x\)

(2) \( y = kx^2 \) が \( y = \log x \) に接する \( k > 0 \) の値を求めよ。

考え方のヒント

2つの曲線が接するということは、交点で「値が一致」して「傾きも一致」するということです！この2つの条件を連立方程式にして解いていきましょう。傾きの条件から \( k \) を分離すると計算が楽になりますよ。

途中式と解答
1. 2曲線が接する条件を整理します。
交点で値が等しい：$$kx^2 = \log x \quad \cdots ①$$ 交点で傾きが等しい：$$2kx = \frac{1}{x} \quad \cdots ②$$
2. ②式から \( k \) を分離します。
$$k = \frac{1}{2x^2} \quad \cdots ③$$
3. ③を①に代入して \( x \) を求めます。
$$\frac{1}{2x^2} \cdot x^2 = \log x$$ $$\frac{1}{2} = \log x$$ $$x = e^{1/2} = \sqrt{e}$$
4. 最後に \( k \) を求めます。③に \( x = \sqrt{e} \) を代入すると、
$$k = \frac{1}{2(\sqrt{e})^2} = \frac{1}{2e}$$
答え: \(k = \frac{1}{2e}\)
全体のポイント
  • 接線は「点と傾き」の公式で求められます
  • 接する条件は「値が一致」かつ「微分係数が一致」
  • 計算を楽にするため、適切な順序で文字を消去しましょう
受験数学のチップス
  • 対数関数の微分 \(\frac{d}{dx}\log x = \frac{1}{x}\) は基本中の基本
  • 接する問題では傾きの条件から変数を分離すると計算しやすい
  • 答えは必ず検算して、条件を満たしているか確認しよう
大問【2】

(1) \( y'' + 4y = 0 \) の一般解を求めよ。

考え方のヒント

これは同次の線形微分方程式ですね！特性方程式 \(\lambda^2 + 4 = 0\) を解いて、虚数解が出てきたときの解の形を思い出しましょう。\(\cos\) と \(\sin\) が登場します。

途中式と解答
1. 特性方程式を立てます。
$$\lambda^2 + 4 = 0$$ $$\lambda^2 = -4$$ $$\lambda = \pm 2i$$
2. 虚数解 \(\lambda = \alpha \pm \beta i\) のとき、一般解は \( e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x) \) です。今回は \(\alpha = 0\)、\(\beta = 2\) なので、
$$y = C_1 \cos 2x + C_2 \sin 2x$$
答え: \(y = C_1 \cos 2x + C_2 \sin 2x \quad (C_1, C_2 \text{は任意定数})\)

(2) \( y'' + 4y = \sin 2x + 4x \) の一般解を求めよ。

考え方のヒント

非同次方程式ですね！一般解は「同次解 + 特殊解」になります。非同次項が \(\sin 2x + 4x\) なので、それぞれに対応する特殊解を考えましょう。\(\sin 2x\) は同次解と重複するので要注意です！

途中式と解答
1. 同次解は(1)で求めた結果です。
$$y_h = C_1 \cos 2x + C_2 \sin 2x$$
2. 特殊解を設定します。非同次項を \(4x\) と \(\sin 2x\) に分けて考えます。
\(4x\) に対して ： \( y_{p1} = ax + b \)
\(\sin 2x\) に対して（同次解と重複するので） ： \( y_{p2} = x(c \cos 2x + d \sin 2x) \)
3. \(4x\) に対応する特殊解を求めます。\(y_{p1} = ax + b\) とすると、
$$y_{p1}'' + 4y_{p1} = 0 + 4(ax + b) = 4ax + 4b = 4x$$
係数を比較して \(a = 1\)、\(b = 0\) です。
4. \(\sin 2x\) に対応する特殊解を求めます。\(y_{p2} = x(c \cos 2x + d \sin 2x)\) を微分して代入すると（計算は複雑ですが）、
$$y_{p2}'' + 4y_{p2} = -4c \sin 2x + 4d \cos 2x = \sin 2x$$
係数を比較して \(c = -\frac{1}{4}\)、\(d = 0\) です。
5. 一般解をまとめます。
$$y = y_h + y_{p1} + y_{p2} = C_1 \cos 2x + C_2 \sin 2x + x - \frac{x}{4} \cos 2x$$
答え: \(y = C_1 \cos 2x + C_2 \sin 2x + x - \frac{x}{4} \cos 2x\)
全体のポイント
  • 特性方程式の解が虚数のとき、\(\cos\) と \(\sin\) の形になる
  • 非同次項が同次解と重複するときは \(x\) 倍する
  • 複雑な非同次項は項ごとに分けて特殊解を求める
受験数学のチップス
  • 微分方程式は「特性方程式→同次解→特殊解」の順で解く
  • 特殊解の設定では重複の有無をしっかりチェック
  • 答えが求まったら元の方程式に代入して確認しよう
大問【3】
行列 \(A = \begin{pmatrix} \cos x & \sin x \\ -\sin x & \cos x \end{pmatrix}\) について、以下の問いに答えよ。
考え方のヒント

この行列は回転行列の形をしていますね！三角関数の性質 \(\sin^2 \theta + \cos^2 \theta = 1\) をうまく使って、直交性や固有値、べき乗の性質を調べていきましょう。

(1) \( A \) が直交行列であることを証明せよ。
途中式と解答
直交行列の定義は \( A^T A = I \)（単位行列）です。まず転置行列を求めましょう。
$$A^T = \begin{pmatrix} \cos x & -\sin x \\ \sin x & \cos x \end{pmatrix}$$
\( A^T A \) を計算します。
$$A^T A = \begin{pmatrix} \cos x & -\sin x \\ \sin x & \cos x \end{pmatrix} \begin{pmatrix} \cos x & \sin x \\ -\sin x & \cos x \end{pmatrix}$$ $$= \begin{pmatrix} \cos^2 x + \sin^2 x & 0 \\ 0 & \sin^2 x + \cos^2 x \end{pmatrix} = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$$
よって \( A^T A = I \) なので、\( A \) は直交行列である。
(2) \( A \) の実数の固有値を求めよ。
途中式と解答
固有値方程式 \(\det(A - \lambda I) = 0\) を解きます。
$$A - \lambda I = \begin{pmatrix} \cos x - \lambda & \sin x \\ -\sin x & \cos x - \lambda \end{pmatrix}$$
行列式を計算します。
$$\det(A - \lambda I) = (\cos x - \lambda)^2 + \sin^2 x = \lambda^2 - 2\lambda \cos x + 1$$
判別式を調べます。
$$D = 4\cos^2 x - 4 = -4\sin^2 x \leq 0$$
実数固有値を持つのは \( D \geq 0 \) のとき、つまり \(\sin x = 0\) のときです。
\(\sin x = 0\) となるのは \( x = 0, \pi \) などです。
\(x = 0\) のとき：\(\lambda^2 - 2\lambda + 1 = 0 \implies \lambda = 1\)（重解）
\(x = \pi\) のとき：\(\lambda^2 + 2\lambda + 1 = 0 \implies \lambda = -1\)（重解）
答え: \(\lambda = 1\)（\(x=0\) のとき）、\(\lambda = -1\)（\(x=\pi\) のとき）
(3) \( x = \frac{\pi}{2} \) のとき \( \sum_{k=1}^{n} A^k \) を求めよ。
途中式と解答
\( x = \frac{\pi}{2} \) を代入します。
$$A = \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix}$$
これは90度回転を表す行列です。
\( A \) のべき乗を計算してみましょう。
$$A^1 = \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix}, \quad A^2 = \begin{pmatrix} -1 & 0 \\ 0 & -1 \end{pmatrix} = -I$$ $$A^3 = -A, \quad A^4 = I$$
周期4で循環することがわかります！
\( n \) を4で割った余りごとに場合分けします。
\(n \equiv 0 \pmod{4}\) のとき：\(\sum_{k=1}^{n} A^k = \begin{pmatrix} 0 & 0 \\ 0 & 0 \end{pmatrix}\)
\(n \equiv 1 \pmod{4}\) のとき：\(\sum_{k=1}^{n} A^k = \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix}\)
\(n \equiv 2 \pmod{4}\) のとき：\(\sum_{k=1}^{n} A^k = \begin{pmatrix} -1 & 1 \\ -1 & -1 \end{pmatrix}\)
\(n \equiv 3 \pmod{4}\) のとき：\(\sum_{k=1}^{n} A^k = \begin{pmatrix} -1 & 0 \\ 0 & -1 \end{pmatrix}\)
答え: 上記の場合分けによる結果
全体のポイント
  • 回転行列は直交行列の典型例
  • 固有値は判別式で実数性を判定
  • 行列のべき乗は周期性を利用する
受験数学のチップス
  • 直交行列の判定は \( A^T A = I \) を確認
  • 回転行列の固有値は一般に複素数
  • 周期性のある問題は余りで場合分け
大問【4】
関数 \(f(x) = \begin{cases} 2 - |x| & (|x| \leq 2) \\ 0 & (|x| > 2) \end{cases}\) のフーリエ変換 \(F(\omega) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^{\infty} f(x)e^{-i\omega x}dx\) を求めよ。
考え方のヒント

\( f(x) \) は偶関数（\( f(-x) = f(x) \)）なので、フーリエ変換の計算が簡単になります！偶関数のフーリエ変換は \(\cos\) 項だけ考えればよくて、積分範囲も半分にできるんです。部分積分も使って計算しましょう。

途中式と解答
1. \( f(x) \) が偶関数なので、フーリエ変換は次のように簡単化されます。
$$F(\omega) = \sqrt{\frac{2}{\pi}} \int_{0}^{2} (2 - x) \cos(\omega x) dx$$
2. まず \(\omega = 0\) の場合を計算します。
$$F(0) = \sqrt{\frac{2}{\pi}} \int_{0}^{2} (2 - x) dx = \sqrt{\frac{2}{\pi}} \left[ 2x - \frac{x^2}{2} \right]_{0}^{2} = \sqrt{\frac{2}{\pi}} \cdot 2 = \frac{4}{\sqrt{2\pi}}$$
3. \(\omega \neq 0\) の場合、部分積分を使います。\( u = 2 - x \)、\( dv = \cos(\omega x) dx \) とすると、
$$\int_{0}^{2} (2 - x) \cos(\omega x) dx = \left[ (2 - x) \frac{\sin(\omega x)}{\omega} \right]_{0}^{2} + \frac{1}{\omega} \int_{0}^{2} \sin(\omega x) dx$$
4. 計算を続けると、
$$= 0 + \frac{1}{\omega} \left[ -\frac{\cos(\omega x)}{\omega} \right]_{0}^{2} = \frac{1 - \cos(2\omega)}{\omega^2}$$
倍角公式 \(1 - \cos(2\theta) = 2\sin^2 \theta\) より、
$$= \frac{2\sin^2 \omega}{\omega^2}$$
5. 最終的に、
$$F(\omega) = \sqrt{\frac{2}{\pi}} \cdot \frac{2\sin^2 \omega}{\omega^2} = \frac{4 \sin^2 \omega}{\sqrt{2\pi} \, \omega^2}$$
答え: \(F(\omega) = \begin{cases} \frac{4}{\sqrt{2\pi}} & (\omega = 0) \\ \frac{4 \sin^2 \omega}{\sqrt{2\pi} \, \omega^2} & (\omega \neq 0) \end{cases}\)
全体のポイント
  • 偶関数のフーリエ変換は \(\cos\) 項のみで計算簡略化
  • 部分積分では「微分するとシンプルになる項」を \( u \) に選ぶ
  • 三角関数の恒等式を活用して最終形を整える
受験数学のチップス
  • フーリエ変換では関数の対称性を最大限活用しよう
  • 部分積分は「どちらを微分するか」の判断が重要
  • \(\omega = 0\) の場合は別途計算することが多い

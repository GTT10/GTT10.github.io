大問【1】

(1) $\int \frac{dx}{x(\log x)^4}$ を計算せよ。

考え方のヒント

対数を含む積分では置換積分が効果的です！$u = \log x$ とおくと、$du = \frac{1}{x} dx$ となって、きれいに整理されます。対数の微分公式を思い出しながら計算してみましょう。

途中式と解答
1. 置換積分を使います。$u = \log x$ とおくと、$du = \frac{1}{x} dx$ です。
$$u = \log x, \quad du = \frac{dx}{x}$$
2. 積分を書き換えます。
$$\int \frac{dx}{x(\log x)^4} = \int u^{-4} \, du$$
3. 冪関数の積分公式を使います。
$$\int u^{-4} \, du = \frac{u^{-3}}{-3} + C = -\frac{1}{3u^3} + C$$
4. 元の変数に戻します。
$$-\frac{1}{3(\log x)^3} + C$$
答え: $-\frac{1}{3(\log x)^{3}} + C$

(2) $\int \frac{dx}{(e^x + 1)^2}$ を計算せよ。

考え方のヒント

指数関数を含む積分では $t = e^x$ と置換するのが定石です！その後、部分分数分解を使ってシンプルな分数に分けて積分しましょう。計算は少し長くなりますが、一つずつ丁寧に進めれば大丈夫です。

途中式と解答
1. 置換積分を使います。$t = e^x$ とおくと、$dt = e^x dx$ なので $dx = \frac{dt}{t}$ です。
$$\int \frac{dx}{(e^x + 1)^2} = \int \frac{1}{(t + 1)^2} \cdot \frac{dt}{t} = \int \frac{dt}{t(t + 1)^2}$$
2. 部分分数分解を行います。
$$\frac{1}{t(t + 1)^2} = \frac{A}{t} + \frac{B}{t + 1} + \frac{C}{(t + 1)^2}$$
3. 係数を求めます。両辺に $t(t + 1)^2$ を乗じて：
$$1 = A(t + 1)^2 + Bt(t + 1) + Ct$$
$t = 0$ で $A = 1$、$t = -1$ で $C = -1$、$t^2$ の係数比較で $B = -1$
4. 積分を計算します。
$$\int \left( \frac{1}{t} - \frac{1}{t + 1} - \frac{1}{(t + 1)^2} \right) dt = \ln |t| - \ln |t + 1| + \frac{1}{t + 1} + C$$
5. $t = e^x$ を代入して整理します。
$$x - \ln (e^x + 1) + \frac{1}{e^x + 1} + C$$
答え: $x - \ln (e^{x} + 1) + \frac{1}{e^{x} + 1} + C$
全体のポイント
  • 対数や指数を含む積分では置換積分が基本
  • 部分分数分解は係数決定の順序が大切
  • 計算ミスを防ぐため各ステップを丁寧に確認
受験数学のチップス
  • $\log x$ の積分では $u = \log x$ 置換を試してみる
  • $e^x$ の積分では $t = e^x$ 置換が効果的
  • 部分分数分解では特定値代入法で係数を求める
大問【2】

(1) 微分方程式 $(x^2 + y^2 + x)dx + xy \, dy = 0$ が完全微分形でないことを示し、積分因子を求めよ。

考え方のヒント

完全微分形かどうかは「$\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$」で判定します！もし等しくなければ積分因子を探しましょう。$\mu = \mu(x)$ のタイプかどうかを調べるのがコツです。

途中式と解答
1. 完全微分形の条件を確認します。$M = x^2 + y^2 + x$、$N = xy$ として、
$$\frac{\partial M}{\partial y} = 2y, \quad \frac{\partial N}{\partial x} = y$$
2. $\frac{\partial M}{\partial y} \neq \frac{\partial N}{\partial x}$（$2y \neq y$ for $y \neq 0$）なので完全微分形ではありません。
3. 積分因子 $\mu = \mu(x)$ を仮定します。完全形の条件から：
$$\frac{\partial}{\partial y}(\mu M) = \frac{\partial}{\partial x}(\mu N)$$
$$\mu \cdot 2y = y\mu + xy\frac{d\mu}{dx}$$
4. 整理すると：
$$y\mu = xy\frac{d\mu}{dx} \implies \frac{d\mu}{\mu} = \frac{dx}{x}$$
$$\ln|\mu| = \ln|x| + C \implies \mu = x$$
答え: 完全微分形でない。積分因子 $\mu = x$

(2) 上の微分方程式の一般解を求めよ。

考え方のヒント

(1)で求めた積分因子 $\mu = x$ を元の方程式に掛けて、完全微分方程式にします！そして $F(x, y) = C$ の形で解を求めましょう。

途中式と解答
1. 積分因子 $\mu = x$ を乗じます：
$$x(x^2 + y^2 + x)dx + x \cdot xy \, dy = 0$$
$$(x^3 + xy^2 + x^2)dx + x^2y \, dy = 0$$
2. 完全微分形の確認：$M = x^3 + xy^2 + x^2$、$N = x^2y$ で、
$$\frac{\partial M}{\partial y} = 2xy = \frac{\partial N}{\partial x}$$
完全微分形になりました！
3. 解 $F(x, y) = C$ を求めます。$\frac{\partial F}{\partial y} = x^2y$ より、
$$F = \frac{1}{2}x^2y^2 + g(x)$$
4. $\frac{\partial F}{\partial x} = xy^2 + g'(x) = x^3 + xy^2 + x^2$ より、
$$g'(x) = x^3 + x^2 \implies g(x) = \frac{1}{4}x^4 + \frac{1}{3}x^3$$
5. 一般解をまとめます：
$$\frac{1}{2}x^2y^2 + \frac{1}{4}x^4 + \frac{1}{3}x^3 = C$$
答え: $\frac{1}{2}x^{2}y^{2} + \frac{1}{4}x^{4} + \frac{1}{3}x^{3} = C$
全体のポイント
  • 完全微分形の判定は偏微分の等価性で確認
  • 積分因子は $x$ のみの関数から試してみる
  • 完全微分方程式は $F(x,y) = C$ で解を表現
受験数学のチップス
  • 積分因子の候補：$\mu(x)$、$\mu(y)$、$\mu(x+y)$ など
  • 完全形になったら片方の変数で部分積分
  • 検算として元の方程式に解を代入して確認
大問【3】

(1) 3次元ベクトル $\mathbf{A}$, $\mathbf{B}$ が張る平行四辺形の面積公式を証明せよ。

考え方のヒント

平行四辺形の面積は外積の大きさ $|\mathbf{A} \times \mathbf{B}|$ で求まります！ラグランジュの恒等式 $|\mathbf{A}|^2|\mathbf{B}|^2 - (\mathbf{A} \cdot \mathbf{B})^2 = |\mathbf{A} \times \mathbf{B}|^2$ を使って証明しましょう。

途中式と解答
1. 平行四辺形の面積は外積のノルムです：
$$S = \|\mathbf{A} \times \mathbf{B}\|$$
2. 外積を成分で表すと：
$\mathbf{A} \times \mathbf{B} = (A_yB_z - A_zB_y, A_zB_x - A_xB_z, A_xB_y - A_yB_x)$
3. $S^2$ を計算すると：
$$S^2 = (A_yB_z - A_zB_y)^2 + (A_zB_x - A_xB_z)^2 + (A_xB_y - A_yB_x)^2$$
4. 一方、ラグランジュの恒等式により：
$|\mathbf{A}|^2|\mathbf{B}|^2 - (\mathbf{A} \cdot \mathbf{B})^2$
$= (A_x^2 + A_y^2 + A_z^2)(B_x^2 + B_y^2 + B_z^2) - (A_xB_x + A_yB_y + A_zB_z)^2$
5. 展開すると3. の結果と一致し、証明が完成します。
証明完了：$S = \sqrt{|\mathbf{A}|^2|\mathbf{B}|^2 - (\mathbf{A} \cdot \mathbf{B})^2}$

(2) 球面 $S: |\mathbf{r}| = a$ について次の面積分を計算せよ。

(a) $\iint_S \frac{\mathbf{r} \cdot \mathbf{n}}{r^3} dS$　　(b) $\iint_S (x\mathbf{i} + y\mathbf{j}) \cdot \mathbf{n} \, dS$

ただし、$\mathbf{n}$ は外向き単位法線ベクトルとする。

考え方のヒント

(a)では球面上で $\mathbf{r} \cdot \mathbf{n} = a$、$r = a$ を使います。(b)では発散定理（ガウスの定理）を活用すると簡単に計算できますよ！

途中式と解答
(a) 球面上では $\mathbf{n} = \frac{\mathbf{r}}{a}$、$r = a$ なので：
$\mathbf{r} \cdot \mathbf{n} = \mathbf{r} \cdot \frac{\mathbf{r}}{a} = \frac{r^2}{a} = a$
$\frac{\mathbf{r} \cdot \mathbf{n}}{r^3} = \frac{a}{a^3} = \frac{1}{a^2}$
積分すると：$\iint_S \frac{1}{a^2} \, dS = \frac{1}{a^2} \times 4\pi a^2 = 4\pi$
(b) 発散定理を適用します。$\mathbf{F} = x\mathbf{i} + y\mathbf{j}$ として：
$$\nabla \cdot \mathbf{F} = \frac{\partial x}{\partial x} + \frac{\partial y}{\partial y} = 1 + 1 = 2$$
発散定理により：
$$\iint_S \mathbf{F} \cdot \mathbf{n} \, dS = \iiint_V (\nabla \cdot \mathbf{F}) \, dV = \iiint_V 2 \, dV = 2 \times \frac{4}{3}\pi a^3 = \frac{8}{3}\pi a^3$$
答え: (a) $4\pi$　　(b) $\frac{8}{3}\pi a^{3}$
全体のポイント
  • 外積は平行四辺形の面積を与える重要な概念
  • 球面上では位置ベクトルと法線の関係が単純化
  • 発散定理で面積分を体積分に変換できる
受験数学のチップス
  • ラグランジュ恒等式は外積計算で頻繁に使用
  • 球面の法線ベクトルは $\mathbf{n} = \frac{\mathbf{r}}{|\mathbf{r}|}$
  • 面積分では発散定理・ストークスの定理を考える
大問【4】

ガウス分布 $f(x) = \frac{1}{\sqrt{2\pi b}} \exp\left(-\frac{(x-a)^2}{2b}\right)$ のフーリエ変換を求めよ。

考え方のヒント

ガウス分布のフーリエ変換は「ガウス関数のまま」になる美しい性質があります！指数部を平方完成して、ガウス積分の公式 $\int_{-\infty}^{\infty} e^{-u^2/2b} du = \sqrt{2\pi b}$ を使いましょう。

途中式と解答
1. フーリエ変換の定義に従って積分します：
$$\hat{f}(k) = \int_{-\infty}^{\infty} f(x) e^{-ikx} dx = \frac{1}{\sqrt{2\pi b}} \int_{-\infty}^{\infty} \exp\left(-\frac{(x-a)^2}{2b} - ikx\right) dx$$
2. 指数部を平方完成します：
$-\frac{(x-a)^2}{2b} - ikx$
$= -\frac{1}{2b}\left[x^2 - 2(a - ibk)x + a^2\right]$
$= -\frac{(x - (a - ibk))^2}{2b} - iak - \frac{bk^2}{2}$
3. 積分を分離します：
$$\hat{f}(k) = \frac{1}{\sqrt{2\pi b}} \exp\left(-iak - \frac{bk^2}{2}\right) \int_{-\infty}^{\infty} \exp\left(-\frac{(x - (a - ibk))^2}{2b}\right) dx$$
4. 積分変数を $u = x - (a - ibk)$ と変換し、ガウス積分公式を適用：
$$\int_{-\infty}^{\infty} \exp\left(-\frac{u^2}{2b}\right) du = \sqrt{2\pi b}$$
5. 最終結果をまとめます：
$$\hat{f}(k) = \frac{1}{\sqrt{2\pi b}} \exp\left(-iak - \frac{bk^2}{2}\right) \times \sqrt{2\pi b} = \exp\left(-iak - \frac{bk^2}{2}\right)$$
答え: $\exp\left(-iak - \frac{bk^{2}}{2}\right)$
全体のポイント
  • ガウス分布のフーリエ変換もガウス関数になる
  • 平方完成は複素数を含む場合でも同様に適用
  • ガウス積分公式は数学的に重要な基本公式
受験数学のチップス
  • フーリエ変換では平方完成が頻繁に使われる
  • 積分変数の変換では積分範囲に注意
  • ガウス積分の結果 $\sqrt{2\pi b}$ は暗記必須

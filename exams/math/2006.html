大問【1】

次の広義積分の値を求めよ。

$$\int_{0}^{\infty} \frac{1}{x(1+x^{2})} dx$$
考え方のヒント

この積分は$x=0$で特異点を持つ広義積分です！変数変換$x = e^u$を使うとうまく計算できます。さらに偶関数の性質を利用することで計算がシンプルになりますよ。

途中式と解答
1. 変数変換 $x = e^u$ を行います。$dx = e^u du$、積分区間は $x: 0 \to \infty$ が $u: -\infty \to \infty$ になります。
$$\int_{0}^{\infty} \frac{1}{x(1+x^{2})} dx = \int_{-\infty}^{\infty} \frac{1}{e^u(1+e^{2u})} e^u du = \int_{-\infty}^{\infty} \frac{1}{1 + e^{2u}} du$$
2. 被積分関数 $\frac{1}{1 + e^{2u}}$ は偶関数なので：
$$= 2 \int_{0}^{\infty} \frac{1}{1 + e^{2u}} du$$
3. さらに変数変換 $v = e^{-u}$ を行います。$u = -\ln v$、$du = -\frac{dv}{v}$、積分区間は $u: 0 \to \infty$ が $v: 1 \to 0$ になります。
$$= 2 \int_{1}^{0} \frac{1}{1 + v^2} \left(-\frac{dv}{v}\right) = 2 \int_{0}^{1} \frac{v}{v^2 + 1} dv$$
4. $\int \frac{v}{v^2 + 1} dv$ は $\frac{1}{2}\ln(v^2 + 1)$ なので：
$$= 2 \cdot \frac{1}{2} \left[ \ln(v^2 + 1) \right]_{0}^{1} = \ln(1^2 + 1) - \ln(0^2 + 1) = \ln 2 - \ln 1 = \ln 2$$
答え: $\ln 2$
全体のポイント
  • 特異点を避ける変数変換の選択
  • 偶関数の性質を活用して計算を簡略化
  • 対数積分の基本公式を使用
受験数学のチップス
  • $x = e^u$ 変換は特異点処理に有効
  • 偶関数の積分は $\int_{-a}^{a} = 2\int_0^a$
  • $\int \frac{x}{x^2+1} dx = \frac{1}{2}\ln(x^2+1) + C$
大問【2】

次の微分方程式を解け。

$$y'' + y = \cos x, \quad y(0) = 0, \quad y'(0) = 0$$
考え方のヒント

これは線形非同次微分方程式です！同次解を求めてから特解を見つけましょう。注意：非同次項$\cos x$が同次解と重複するので、特解は$x$をかけた形$y_p = x(C\cos x + D\sin x)$で設定します。

途中式と解答
1. 同次方程式 $y'' + y = 0$ の一般解を求めます。特性方程式は $\lambda^2 + 1 = 0$、解は $\lambda = \pm i$ です。
$$y_h = A \cos x + B \sin x$$
2. 非同次項 $\cos x$ が同次解と重複するため、特解を $y_p = x(C \cos x + D \sin x)$ と設定します。
3. $y_p$ を2回微分します：
$y_p' = (C \cos x + D \sin x) + x(-C \sin x + D \cos x)$
$y_p'' = 2(-C \sin x + D \cos x) + x(-C \cos x - D \sin x)$
4. 微分方程式に代入：
$$y_p'' + y_p = 2(-C \sin x + D \cos x) = \cos x$$
係数比較により：$-2C = 0$ ($C = 0$)、$2D = 1$ ($D = \frac{1}{2}$)
5. 特解は $y_p = \frac{1}{2} x \sin x$、一般解は：
$$y = A \cos x + B \sin x + \frac{1}{2} x \sin x$$
6. 初期条件を適用します： - $y(0) = A = 0$ - $y' = -A \sin x + B \cos x + \frac{1}{2}(\sin x + x \cos x)$ - $y'(0) = B = 0$
答え: $y = \frac{1}{2} x \sin x$
全体のポイント
  • 同次解と非同次項が重複する場合の処理
  • 特解には$x$をかけた形を使用
  • 初期条件から積分定数を決定
受験数学のチップス
  • 重複する場合は特解に$x$を乗ずる
  • 係数比較で未定係数を決定
  • 初期条件は最後に適用する
大問【3】

(1) 次の微分方程式をラプラス変換し、$X(s)$ を求めよ。

$$\frac{d^{2}x(t)}{dt^{2}} + 2\xi \omega \frac{dx(t)}{dt} + \omega^{2}x(t) = 1$$

ただし、$\dot{x}(0) = v_0$、$x(0) = x_0$ とする。

考え方のヒント

ラプラス変換の微分則を使いましょう！$L\{f''(t)\} = s^2F(s) - sf(0) - f'(0)$ です。初期条件も含めて整理すると、$X(s)$ について解けますよ。

途中式と解答
1. ラプラス変換の公式を適用します：
$L\left\{\frac{d^2x}{dt^2}\right\} = s^2 X(s) - s x_0 - v_0$
$L\left\{\frac{dx}{dt}\right\} = s X(s) - x_0$
$L\{1\} = \frac{1}{s}$
2. 微分方程式をラプラス変換：
$$(s^2 X(s) - s x_0 - v_0) + 2\xi \omega (s X(s) - x_0) + \omega^2 X(s) = \frac{1}{s}$$
3. $X(s)$ についてまとめます：
$$(s^2 + 2\xi \omega s + \omega^2) X(s) = \frac{1}{s} + s x_0 + v_0 + 2\xi \omega x_0$$
4. 最終的に：
$$X(s) = \frac{\frac{1}{s} + s x_0 + (v_0 + 2\xi \omega x_0)}{s^2 + 2\xi \omega s + \omega^2}$$
答え: $X(s) = \frac{\frac{1}{s} + s x_{0} + v_{0} + 2\xi \omega x_{0}}{s^{2} + 2\xi \omega s + \omega^{2}}$

(2) $X(s)$ を逆ラプラス変換し、$x(t)$ を求めよ。ただし、$\xi < 1$ とする。

考え方のヒント

$\xi < 1$ なので減衰振動になります！分母を $(s + \xi\omega)^2 + \omega_d^2$ の形に変形し、$\omega_d = \omega\sqrt{1-\xi^2}$ とおいてから逆変換しましょう。

途中式と解答
1. 減衰固有角周波数を定義：$\omega_d = \omega \sqrt{1 - \xi^2}$
2. 分母を完全平方に変形：
$$s^2 + 2\xi \omega s + \omega^2 = (s + \xi \omega)^2 + \omega_d^2$$
3. $X(s)$ を項目別に分解し、標準的な逆変換公式を適用：
$L^{-1}\left\{ \frac{s + \xi \omega}{(s + \xi \omega)^2 + \omega_d^2} \right\} = e^{-\xi \omega t} \cos(\omega_d t)$
$L^{-1}\left\{ \frac{\omega_d}{(s + \xi \omega)^2 + \omega_d^2} \right\} = e^{-\xi \omega t} \sin(\omega_d t)$
4. 各項を逆変換し、整理すると：
$$x(t) = e^{-\xi \omega t} \left( x_0 \cos(\omega_d t) + \frac{v_0 + \xi \omega x_0}{\omega_d} \sin(\omega_d t) \right) + \frac{1}{\omega^2} \left( 1 - e^{-\xi \omega t} \left( \cos(\omega_d t) + \frac{\xi \omega}{\omega_d} \sin(\omega_d t) \right) \right)$$
答え: 上記の式（初期条件応答＋ステップ応答の合成）
全体のポイント
  • ラプラス変換の微分則と初期条件
  • 減衰振動系の標準形への変換
  • 初期条件応答とステップ応答の重ね合わせ
受験数学のチップス
  • $\xi < 1$ は減衰振動、$\xi \geq 1$ は非振動
  • 完全平方への変形が逆変換のキー
  • 物理的意味も理解しておく
大問【4】

(1) 最小二乗法において、残差の二乗和を求めよ。

考え方のヒント

残差は「実測値 - 理論値」です！一次近似では $y_i - (b_0 + b_1 x_i)$ になります。これを二乗して全データ点について和を取りましょう。

途中式と解答
残差 $e_i = y_i - (b_0 + b_1 x_i)$ の二乗和：
$$S = \sum_{i=1}^{n} e_i^2 = \sum_{i=1}^{n} (y_i - b_0 - b_1 x_i)^2$$
答え: $S = \sum_{i=1}^{n} (y_{i} - b_{0} - b_{1} x_{i})^{2}$

(2) 残差の二乗和が最小になる条件を求めよ。

考え方のヒント

$S$ を $b_0$ と $b_1$ で偏微分して0とおきます！これが正規方程式の条件になります。

途中式と解答
1. $S$ を $b_0$ で偏微分：
$$\frac{\partial S}{\partial b_0} = -2 \sum_{i=1}^{n} (y_i - b_0 - b_1 x_i) = 0$$
2. $S$ を $b_1$ で偏微分：
$$\frac{\partial S}{\partial b_1} = -2 \sum_{i=1}^{n} x_i (y_i - b_0 - b_1 x_i) = 0$$
答え: $\sum_{i=1}^{n} (y_{i} - b_{0} - b_{1} x_{i}) = 0$、$\sum_{i=1}^{n} x_{i} (y_{i} - b_{0} - b_{1} x_{i}) = 0$

(3) 行列を用いて(2)の条件を表せ。

考え方のヒント

データを行列形式で表現しましょう！$Y$をデータベクトル、$X$を計画行列、$B$をパラメータベクトルとして正規方程式を書きます。

途中式と解答
行列表現を定義：
$Y = \begin{pmatrix} y_1 \\ y_2 \\ \vdots \\ y_n \end{pmatrix}$、 $X = \begin{pmatrix} 1 & x_1 \\ 1 & x_2 \\ \vdots & \vdots \\ 1 & x_n \end{pmatrix}$、 $B = \begin{pmatrix} b_0 \\ b_1 \end{pmatrix}$
答え: $X^{\top} X B = X^{\top} Y$

(4) $B$ を $Y$ および $X$ を用いて表せ。

考え方のヒント

(3)の正規方程式を$B$について解きましょう！$X^{\top}X$が正則なら逆行列が存在します。

途中式と解答
正規方程式 $X^{\top} X B = X^{\top} Y$ を $B$ について解く：
$$B = (X^{\top} X)^{-1} X^{\top} Y$$
答え: $B = (X^{\top} X)^{-1} X^{\top} Y$
全体のポイント
  • 残差の二乗和を最小化する原理
  • 偏微分による最適化条件の導出
  • 行列による簡潔な表現
受験数学のチップス
  • 正規方程式は最小二乗法の基本
  • $(X^{\top}X)^{-1}X^{\top}$ は疑似逆行列
  • 統計学の基礎となる重要な理論

問題概要

2012年度の数学入試では、大問【1】で逆三角関数の微分とサイクロイドのパラメータ微分、大問【2】で積分因子を用いた微分方程式の解法、大問【3】でベクトルの線形従属性と平面の方程式、大問【4】でラプラス変換とステップ関数を用いた微分方程式が出題されました。基礎から応用まで幅広い分野をカバーし、特にパラメータ表示の微分や積分因子など、計算技法をしっかり身につけることが重要な良問揃いです！

大問【1】

(1) 関数 $y = \cos^{-1} (\sin x)$ の導関数を求めよ。

考え方のヒント

逆三角関数の計算では定義域に注意することが大切ですね！$\sin x = \cos\left(\frac{\pi}{2} - x\right)$ の関係を使うと見通しが良くなります。

途中式と解答
1. 三角関数の関係式を利用：
$$y = \cos^{-1} \left( \cos\left( \frac{\pi}{2} - x \right) \right)$$
2. $x \in \left(-\frac{\pi}{2}, \frac{\pi}{2}\right)$ では $\frac{\pi}{2} - x \in (0, \pi)$ となり、逆余弦の主値範囲にあるので：
$$y = \frac{\pi}{2} - x$$
3. 微分すると：
$$\frac{dy}{dx} = -1$$
答え: $\dfrac{dy}{dx} = -1$

(2) サイクロイド $x = a(\theta - \sin \theta)$, $y = a(1 - \cos \theta)$ の一階導関数と二階導関数を求めよ。

考え方のヒント

パラメータ表示の微分ですね！$\frac{dy}{dx} = \frac{dy/d\theta}{dx/d\theta}$ の公式を使って、段階的に計算していきましょう。二階導関数は商の微分法が鍵になります。

途中式と解答
1. 各パラメータでの微分：
$$\frac{dx}{d\theta} = a(1 - \cos \theta), \quad \frac{dy}{d\theta} = a \sin \theta$$
2. 一階導関数：
$$\frac{dy}{dx} = \frac{dy/d\theta}{dx/d\theta} = \frac{a \sin \theta}{a(1 - \cos \theta)} = \frac{\sin \theta}{1 - \cos \theta}$$
3. 二階導関数（商の微分法）：
$$\frac{d}{d\theta}\left(\frac{\sin \theta}{1 - \cos \theta}\right) = \frac{\cos \theta(1 - \cos \theta) - \sin \theta \cdot \sin \theta}{(1 - \cos \theta)^2}$$
$$= \frac{\cos \theta - \cos^2 \theta - \sin^2 \theta}{(1 - \cos \theta)^2} = \frac{\cos \theta - 1}{(1 - \cos \theta)^2} = \frac{-1}{1 - \cos \theta}$$
4. 最終的に：
$$\frac{d^2y}{dx^2} = \frac{\frac{d}{d\theta}\left(\frac{dy}{dx}\right)}{\frac{dx}{d\theta}} = \frac{-\frac{1}{1 - \cos \theta}}{a(1 - \cos \theta)} = -\frac{1}{a(1 - \cos \theta)^2}$$
答え: $\frac{dy}{dx} = \frac{\sin \theta}{1 - \cos \theta}$, $\frac{d^2y}{dx^2} = -\frac{1}{a(1 - \cos \theta)^2}$
全体のポイント
  • 逆三角関数は定義域と値域に注意
  • パラメータ微分は段階的に計算
  • 商の微分法を正確に適用
受験数学のチップス
  • $\sin x = \cos\left(\frac{\pi}{2} - x\right)$ の関係を活用
  • パラメータ微分: $\frac{dy}{dx} = \frac{dy/d\theta}{dx/d\theta}$
  • 二階導関数: $\frac{d^2y}{dx^2} = \frac{d/d\theta(dy/dx)}{dx/d\theta}$
大問【2】
微分方程式 $\frac{dy}{dx} + \frac{y}{x} = e^x$ について以下を答えよ。
(1) 完全微分形でないことの証明と積分因子を求めよ。
考え方のヒント

微分方程式を $M dx + N dy = 0$ の形に変形して、$\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$ が成り立つかチェックしましょう！成り立たない場合は積分因子を求めます。

途中式と解答
1. 微分方程式を標準形に変形：
$$\left( \frac{y}{x} - e^x \right) dx + dy = 0$$
よって $M = \frac{y}{x} - e^x$, $N = 1$
2. 偏導関数を計算：
$$\frac{\partial M}{\partial y} = \frac{1}{x}, \quad \frac{\partial N}{\partial x} = 0$$
$\frac{1}{x} \neq 0$ より完全微分形でない。
3. 積分因子 $\mu(x)$ を求める：
$$\frac{\partial}{\partial y} (\mu M) = \frac{\partial}{\partial x} (\mu N) \implies \mu \cdot \frac{1}{x} = \frac{d\mu}{dx}$$
変数分離法で $\frac{d\mu}{\mu} = \frac{dx}{x}$ より $\mu = x$
答え: 完全微分形でない、積分因子 $\mu = x$
(2) 一般解を求めよ。
途中式と解答
1. 積分因子 $x$ を乗じた方程式：
$$(y - x e^x) dx + x dy = 0$$
2. ポテンシャル関数 $\psi(x,y)$ を求める：
$$\frac{\partial \psi}{\partial y} = x \implies \psi = xy + h(x)$$
$$\frac{\partial \psi}{\partial x} = y + h'(x) = y - xe^x \implies h'(x) = -xe^x$$
3. $h(x)$ を積分（部分積分）：
$$h(x) = \int -xe^x dx = -xe^x + e^x + C$$
4. 一般解：
$$xy - xe^x + e^x = C \implies xy + e^x(1 - x) = C$$
答え: $xy + e^x(1 - x) = C$
全体のポイント
  • 完全微分方程式の判定方法
  • 積分因子の求め方と適用
  • ポテンシャル関数による解法
受験数学のチップス
  • 完全微分の条件: $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$
  • 積分因子: $\mu \frac{\partial M}{\partial y} = \frac{d\mu}{dx}$ で求める
  • 部分積分: $\int xe^x dx = xe^x - e^x + C$
大問【3】
ベクトル $x_1 = \begin{bmatrix} 1 \\ 2 \\ 3 \end{bmatrix}$, $x_2 = \begin{bmatrix} 3 \\ 1 \\ 2 \end{bmatrix}$, $x_3 = \begin{bmatrix} 5 \\ 0 \\ k \end{bmatrix}$, $x_4 = \begin{bmatrix} -1 \\ 3 \\ 4 \end{bmatrix}$ について以下を答えよ。
考え方のヒント

ベクトルの線形従属性は行列式で判定できますね！3つのベクトルが線形従属 ⇔ それらを列にもつ行列の行列式が0になります。順番に解いていきましょう！

途中式と解答

(1) $x_1, x_2, x_3$ が線形従属となる $k$ の値を求めよ。

行列 $A = \begin{bmatrix} 1 & 3 & 5 \\ 2 & 1 & 0 \\ 3 & 2 & k \end{bmatrix}$ の行列式を計算：
$$\det A = 1 \cdot (k - 0) - 3 \cdot (2k - 0) + 5 \cdot (4 - 3) = k - 6k + 5 = -5k + 5 = 0$$
よって $k = 1$
答え: $k = 1$

(2) $k = 1$ のとき、$x_3$ を $x_1, x_2$ の線形結合で表せ。

$c_1 x_1 + c_2 x_2 = x_3$ とおくと：
$$\begin{cases} c_1 + 3c_2 = 5 \\ 2c_1 + c_2 = 0 \\ 3c_1 + 2c_2 = 1 \end{cases}$$
これを解くと $c_1 = -1$, $c_2 = 2$
答え: $x_3 = -x_1 + 2x_2$

(3) $k = 1$ のとき、原点を通り $x_1, x_2, x_3$ を含む平面の方程式を求めよ。

平面の方程式を $ax + by + cz = 0$ とすると、3点を代入：
$$\begin{cases} a + 2b + 3c = 0 \\ 3a + b + 2c = 0 \\ 5a + c = 0 \end{cases}$$
これを解くと $a = t$, $b = 7t$, $c = -5t$ より
答え: $x + 7y - 5z = 0$

(4) $x_4 = ax_1 + bx_2 + cx_3$ を満たす $a, b, c$ の値を求めよ。

連立方程式を立てる：
$$\begin{cases} a + 3b + 5c = -1 \\ 2a + b = 3 \\ 3a + 2b + c = 4 \end{cases}$$
これを解くと $a = 2$, $b = -1$, $c = 0$
答え: $a = 2$, $b = -1$, $c = 0$
全体のポイント
  • 線形従属性は行列式で判定
  • 連立方程式の解法が基本
  • 平面の方程式は法線ベクトルから
受験数学のチップス
  • 線形従属 ⇔ $\det = 0$
  • 線形結合は連立方程式で求める
  • 平面: $ax + by + cz = 0$ (原点通過)
大問【4】
微分方程式 $y' + y = f(t)$ について以下を答えよ。ここで $f(t)$ は図1で表される関数とする。
考え方のヒント

ラプラス変換とステップ関数の組み合わせ問題ですね！まずは図1の関数を単位ステップ関数で表現し、それをラプラス変換してから微分方程式を解きましょう。時間ずらしの公式が重要です。

(1) $f(t)$ を単位ステップ関数 $u(t)$ を用いて表せ。
途中式と解答
図1より、$f(t)$ は原点と点$(a,1)$を結ぶ直線で、$t \geq a$ で0となる関数：
$$f(t) = \begin{cases} \frac{1}{a}t & (0 \leq t < a) \\ 0 & (t \geq a) \end{cases}$$
これを単位ステップ関数で表現すると：
$$f(t) = \frac{1}{a}t \{u(t) - u(t-a)\}$$
答え: $f(t) = \frac{1}{a}t \{u(t) - u(t-a)\}$
(2) $f(t)$ をラプラス変換せよ。
途中式と解答
$f(t) = \frac{1}{a}t u(t) - \frac{1}{a}t u(t-a)$ として、各項をラプラス変換：
$$\mathcal{L}[\frac{1}{a}t u(t)] = \frac{1}{a} \cdot \frac{1}{s^2} = \frac{1}{as^2}$$
時間ずらしの公式より：
$$\mathcal{L}[\frac{1}{a}t u(t-a)] = \frac{1}{a} e^{-as} \mathcal{L}[t+a] = e^{-as}\left(\frac{1}{as^2} + \frac{1}{s}\right)$$
したがって：
$$F(s) = \frac{1}{as^2} - e^{-as}\left(\frac{1}{as^2} + \frac{1}{s}\right) = \frac{1 - e^{-as} - ase^{-as}}{as^2}$$
答え: $F(s) = \frac{1 - e^{-as} - ase^{-as}}{as^2}$
(3) 微分方程式の解 $y(t)$ を求めよ。ただし $y(0) = 0$ とする。
途中式と解答
微分方程式をラプラス変換：
$$(sY(s) - y(0)) + Y(s) = F(s)$$
$$(s+1)Y(s) = F(s)$$
$$Y(s) = \frac{F(s)}{s+1} = \frac{1 - e^{-as}(1+as)}{as^2(s+1)}$$
$Y(s)$ を逆変換しやすいように整理します。 $$Y(s) = \frac{1}{as^2(s+1)} - e^{-as}\frac{1+as}{as^2(s+1)}$$ 逆ラプラス変換を行うために、部分分数分解を行う：
$$\frac{1}{s^2(s+1)} = \frac{1}{s^2} - \frac{1}{s} + \frac{1}{s+1}$$ $$\frac{1+as}{as^2(s+1)} = \frac{1}{as^2} + \frac{a-1}{as} + \frac{1-a}{a(s+1)}$$
部分分数分解して逆ラプラス変換を行い、場合分けして解を求める：
よって、$Y(s)$ は次のように書き換えられます
$$Y(s) = \frac{1}{a}\left(\frac{1}{s^2} - \frac{1}{s} + \frac{1}{s+1}\right) - e^{-as}\left(\frac{1}{as^2} + \frac{a-1}{as} + \frac{1-a}{a(s+1)}\right)$$
逆ラプラス変換を実行します：
$$g(t) = L^{-1}\left[\frac{1}{a}\left(\frac{1}{s^2} - \frac{1}{s} + \frac{1}{s+1}\right)\right] = \frac{1}{a}(t - 1 + e^{-t})$$ $$h(t) = L^{-1}\left[\frac{1}{as^2} + \frac{a-1}{as} + \frac{1-a}{a(s+1)}\right] = \frac{1}{a}(t + a-1 + (1-a)e^{-t})$$
したがって、$y(t)$ は次のように表されます：
$$y(t) = g(t) - u(t-a)h(t-a)$$ $$\mathcal{L}^{-1}[Y(s)] = \frac{1}{a}\left(t - 1 + e^{-t}\right) - e^{-at}\left(\frac{1}{a}(t + a - 1) + \frac{(a-1)e^{at}}{a} + \frac{(1-a)e^{at}}{a}\right)$$
$$y(t) = \begin{cases} \frac{1}{a}(t - 1 + e^{-t}) & (0 \leq t < a) \\ \frac{e^{-t}}{a}(1 - e^a + ae^a) & (t \geq a) \end{cases}$$
答え: $y(t) = \begin{cases} \frac{1}{a}(t - 1 + e^{-t}) & (0 \leq t < a) \\ \frac{e^{-t}}{a}(1 - e^a + ae^a) & (t \geq a) \end{cases}$

  • 部分分数分解による逆変換

  • 時間ずらし: $\mathcal{L}[f(t)u(t-a)] = e^{-as}\mathcal{L}[f(t+a)]$

大問【1】

積分方程式に関する問題

考え方のヒント

積分方程式は「微分して微分方程式に帰着」が定石です！ライプニッツの積分則と積の微分法を駆使して解きましょう。

(1) $\int_a^x (x-t)f(t)dt$ の$x$微分を求めよ

途中式と解答
1. 積分を分解：
$$\int_a^x (x-t)f(t)dt = x \int_a^x f(t)dt - \int_a^x t f(t)dt$$
2. 各項を微分（積の微分法則）：
$$\frac{d}{dx} \left( x \int_a^x f(t)dt \right) = \int_a^x f(t)dt + x f(x)$$
$$\frac{d}{dx} \left( \int_a^x t f(t)dt \right) = x f(x)$$
3. 結果：
$$\frac{d}{dx} \left( \int_a^x (x-t)f(t)dt \right) = \left( \int_a^x f(t)dt + x f(x) \right) - x f(x) = \int_a^x f(t)dt$$
答え: $\int_a^x f(t) dt$

(2) $(x+1)e^{-x} + b$ の微分を求めよ

途中式と解答
積の微分法則：$(uv)' = u'v + uv'$ を使用
$$\frac{d}{dx} \left( (x+1)e^{-x} \right) = (1) \cdot e^{-x} + (x+1) \cdot (-e^{-x}) = e^{-x} - (x+1)e^{-x} = -x e^{-x}$$
答え: $-x e^{-x}$

(3) $\int_a^x (x-t)f(t)dt = (x+1)e^{-x} + b$ を満たす$f(x)$, $a$, $b$を求めよ

途中式と解答
1. (1)の結果で左辺を微分し、(2)の結果と比較：
$$\int_a^x f(t)dt = -x e^{-x}$$
2. 上式を$x$で微分：
$$f(x) = \frac{d}{dx}(-x e^{-x}) = -(e^{-x} - x e^{-x}) = (x - 1) e^{-x}$$
3. $f(t) = (t-1)e^{-t}$を元の式に代入し、部分積分で計算：
$$\int_a^x (t-1)e^{-t} dt = -x e^{-x}$$
左辺は$[-t e^{-t}]_a^x = -x e^{-x} + a e^{-a}$なので：
$$a e^{-a} = 0 \Rightarrow a = 0$$
4. 元の式で$x=0$を代入：$0 = 1 + b$ より$b = -1$
答え: $f(x) = (x-1)e^{-x}$, $a = 0$, $b = -1$
全体のポイント
  • ライプニッツの積分則を活用
  • 積分方程式は微分で微分方程式に
  • 初期条件で定数を決定
受験数学のチップス
  • 積分上限が変数なら「積分の中身＋上限微分」
  • $(e^{-x})' = -e^{-x}$
  • 部分積分：$\int uv' = uv - \int u'v$
大問【2】

微分方程式: $\left( x - \frac{2y}{x} \right) dx + \left( \frac{e^{2y}}{x} - 2 \right) dy = 0$ $(x>0)$

考え方のヒント

完全微分形でない場合は積分因子を求めます！$\mu = x$, $\mu = e^{ky}$, $\mu = x^k$などを試してみましょう。

(1) 完全微分形でないことを証明せよ

途中式と解答
$P = x - \frac{2y}{x}$, $Q = \frac{e^{2y}}{x} - 2$として：
$$\frac{\partial P}{\partial y} = -\frac{2}{x}, \quad \frac{\partial Q}{\partial x} = -\frac{e^{2y}}{x^2}$$
$-\frac{2}{x} \neq -\frac{e^{2y}}{x^2}$（一般に成立しない）
よって完全微分形ではない

(2) 積分因子を求めよ

途中式と解答
積分因子$\mu = x$を試す：
$$\mu P = x \left( x - \frac{2y}{x} \right) = x^2 - 2y$$
$$\mu Q = x \left( \frac{e^{2y}}{x} - 2 \right) = e^{2y} - 2x$$
検証：
$$\frac{\partial}{\partial y}(\mu P) = -2, \quad \frac{\partial}{\partial x}(\mu Q) = -2$$
$\frac{\partial}{\partial y}(\mu P) = \frac{\partial}{\partial x}(\mu Q) = -2$ なので完全微分形になる
答え: $\mu = x$

(3) 一般解を求めよ

途中式と解答
積分因子$\mu = x$を乗じた式：
$$(x^2 - 2y) dx + (e^{2y} - 2x) dy = 0$$
$\frac{\partial \Phi}{\partial x} = x^2 - 2y$を積分：
$$\Phi(x,y) = \frac{1}{3}x^3 - 2xy + h(y)$$
$\frac{\partial \Phi}{\partial y} = -2x + h'(y) = e^{2y} - 2x$より：
$$h'(y) = e^{2y} \Rightarrow h(y) = \frac{1}{2} e^{2y}$$
答え: $\frac{1}{3}x^3 - 2xy + \frac{1}{2} e^{2y} = C$
全体のポイント
  • 完全微分形の判定は偏導関数の一致
  • 積分因子$\mu = x$を見つける
  • ポテンシャル関数$\Phi$で一般解
受験数学のチップス
  • 完全微分形：$\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$
  • 積分因子は試行錯誤が必要
  • 一般解：$\Phi(x,y) = C$
大問【3】

連立微分方程式をラプラス変換で解け:

$$\begin{cases} x' + y' + y = 0 \\ x' + 2x + 6 \int_0^t y \, d\tau = -2u(t) \end{cases}$$

$x(0) = -5$, $y(0) = 6$

考え方のヒント

ラプラス変換が威力を発揮します！単位ステップ関数$u(t)$は$\frac{1}{s}$、積分項$\int_0^t y d\tau$は$\frac{Y(s)}{s}$に変換されます。

途中式と解答
1. 第1式をラプラス変換：
$$sX(s) - x(0) + sY(s) - y(0) + Y(s) = 0$$
$$sX(s) + 5 + sY(s) - 6 + Y(s) = 0$$
$$sX(s) + (s+1)Y(s) = 1 \quad \text{(A)}$$
2. 第2式をラプラス変換：
$$sX(s) - x(0) + 2X(s) + 6 \frac{Y(s)}{s} = -2 \cdot \frac{1}{s}$$
$$sX(s) + 5 + 2X(s) + \frac{6Y(s)}{s} = -\frac{2}{s}$$
$$(s+2)X(s) + \frac{6Y(s)}{s} = -5 - \frac{2}{s} \quad \text{(B)}$$
3. (A)と(B)を連立して解く： (A)より$X(s) = \frac{1 - (s+1)Y(s)}{s}$を(B)に代入し整理すると：
$$Y(s) = \frac{6s+4}{(s+4)(s-1)} = \frac{4}{s+4} + \frac{2}{s-1}$$
4. 逆ラプラス変換：
$$y(t) = 4e^{-4t} + 2e^{t}$$
5. 第1式から$x(t)$を求める： $x' = -y' - y$より積分して初期条件を適用すると：
$$x(t) = -3e^{-4t} - 4e^{t} + 2$$
答え:
$x(t) = -3e^{-4t} - 4e^{t} + 2$
$y(t) = 4e^{-4t} + 2e^{t}$
全体のポイント
  • ラプラス変換で代数方程式に変換
  • 連立方程式を解いて逆変換
  • 初期条件で未定定数を決定
受験数学のチップス
  • $\mathcal{L}\{f'(t)\} = sF(s) - f(0)$
  • $\mathcal{L}\{\int_0^t f d\tau\} = \frac{F(s)}{s}$
  • $\mathcal{L}\{u(t)\} = \frac{1}{s}$
大問【4】

連立同次方程式の解空間の自由度を求めよ:

$$\begin{cases} x_1 + x_2 + x_3 + x_4 = 0 \\ a x_1 + b x_2 + c x_3 + (a+b+c) x_4 = 0 \\ a^2 x_1 + b^2 x_2 + c^2 x_3 + (a^2 + b^2 + c^2) x_4 = 0 \end{cases}$$

ただし$abc \neq 0$

考え方のヒント

係数行列の階数（ランク）を求めます！解の自由度 = 変数の数 - 階数です。ヴァンデモンド行列の性質を活用しましょう。

途中式と解答
(i) $a$, $b$, $c$が異なる場合
1. 係数行列：
$$A = \begin{pmatrix} 1 & 1 & 1 & 1 \\ a & b & c & a+b+c \\ a^2 & b^2 & c^2 & a^2+b^2+c^2 \end{pmatrix}$$
2. 行基本変形を実行： 第2行 - $a$ × 第1行：
$$(b-a)x_2 + (c-a)x_3 + (b+c)x_4 = 0$$
第3行 - $a^2$ × 第1行：
$$(b^2-a^2)x_2 + (c^2-a^2)x_3 + (b^2+c^2)x_4 = 0$$
3. さらに変形すると、係数行列の左上3×3の部分はヴァンデモンド行列：
$$\det \begin{pmatrix} 1 & 1 & 1 \\ a & b & c \\ a^2 & b^2 & c^2 \end{pmatrix} = (b-a)(c-a)(c-b)$$
4. $a$, $b$, $c$が異なる値のとき、$(b-a)(c-a)(c-b) \neq 0$なので、ランクは3 自由度 = 4 - 3 = 1
(ii) $a = b = c$の場合
5. $a = b = c$を代入すると、係数行列は：
$$A = \begin{pmatrix} 1 & 1 & 1 & 1 \\ a & a & a & 3a \\ a^2 & a^2 & a^2 & 3a^2 \end{pmatrix}$$
6. 第3行 = $a$ × 第2行なので、実質的に独立な行は2行のみ。 よってランクは2、自由度 = 4 - 2 = 2
答え: $a$, $b$, $c$が異なる場合は1、$a = b = c$の場合は2
全体のポイント
  • ヴァンデモンド行列の性質を利用
  • 係数行列の階数が3
  • 解空間の次元は1
受験数学のチップス
  • 自由度 = 変数の数 - ランク
  • ヴァンデモンド行列式 = $\prod (b_j - a_i)$
  • 同次方程式の解空間は部分空間
大問【5】

周期関数のフーリエ級数を求めよ:

$$f(t) = \begin{cases} 1 & (0 < t < 1) \\ 0 & (1 \leq t < 4) \end{cases}$$

周期$T=4$

考え方のヒント

周期$T=4$なので基本振動数$\omega_0 = \frac{2\pi}{4} = \frac{\pi}{2}$です！フーリエ係数を計算し、各周波数成分の振幅を求めましょう。

途中式と解答
1. フーリエ係数の計算：
$$a_0 = \frac{2}{T} \int_0^T f(t) dt = \frac{2}{4} \int_0^1 1 dt = \frac{1}{2}$$
2. $a_n$係数：
$$a_n = \frac{2}{4} \int_0^1 \cos \frac{n\pi t}{2} dt = \frac{1}{n\pi} \sin \frac{n\pi}{2}$$
3. $b_n$係数：
$$b_n = \frac{2}{4} \int_0^1 \sin \frac{n\pi t}{2} dt = -\frac{1}{n\pi} \left( \cos \frac{n\pi}{2} - 1 \right)$$
4. フーリエ級数：
$$f(t) = \frac{1}{4} + \sum_{n=1}^{\infty} \left[ \frac{1}{n\pi} \sin \frac{n\pi}{2} \cos \frac{n\pi t}{2} - \frac{1}{n\pi} \left( \cos \frac{n\pi}{2} - 1 \right) \sin \frac{n\pi t}{2} \right]$$
5. 振動数$\frac{n\pi}{2}$の振幅：
$$c_n = \sqrt{a_n^2 + b_n^2} = \sqrt{ \left( \frac{1}{n\pi} \sin \frac{n\pi}{2} \right)^2 + \left( \frac{1}{n\pi} \left(1 - \cos \frac{n\pi}{2}\right) \right)^2 }$$
フーリエ級数と振幅の公式を導出
全体のポイント
  • 基本振動数$\omega_0 = \frac{2\pi}{T}$
  • 矩形波のフーリエ展開
  • 各周波数成分の振幅計算
受験数学のチップス
  • 振幅 = $\sqrt{a_n^2 + b_n^2}$
  • 矩形波は奇数倍周波数が主成分
  • $\sin\frac{n\pi}{2}$は$n$が奇数で$\pm 1$

大問【1】

次の積分を計算せよ。 $$\int_{0}^{\infty} e^{-ax} \cos bx \, dx \quad (a>0)$$

考え方のヒント

部分積分を2回繰り返すのがポイントです！最初の部分積分で元の積分が再び現れるので、それを利用して方程式として解きます。$a > 0$なので無限遠で収束することが保証されますよ。

途中式と解答
1. まず不定積分 $I = \int e^{-ax} \cos bx \, dx$ を求めます。
2. 第1回部分積分：$u = \cos bx$，$dv = e^{-ax} dx$ とすると
$$du = -b\sin bx \, dx，\quad v = -\frac{1}{a}e^{-ax}$$
$$I = -\frac{1}{a}e^{-ax}\cos bx - \frac{b}{a}\int e^{-ax}\sin bx \, dx$$
3. 第2回部分積分：右辺の積分で $u = \sin bx$，$dv = e^{-ax} dx$ とすると
$$\int e^{-ax}\sin bx \, dx = -\frac{1}{a}e^{-ax}\sin bx + \frac{b}{a}\int e^{-ax}\cos bx \, dx$$
4. 代入して整理：
$$I = -\frac{1}{a}e^{-ax}\cos bx - \frac{b}{a}\left(-\frac{1}{a}e^{-ax}\sin bx + \frac{b}{a}I\right)$$
$$I = -\frac{1}{a}e^{-ax}\cos bx + \frac{b}{a^2}e^{-ax}\sin bx - \frac{b^2}{a^2}I$$
5. $I$について解くと：
$$I + \frac{b^2}{a^2}I = e^{-ax}\left(-\frac{1}{a}\cos bx + \frac{b}{a^2}\sin bx\right)$$
$$I = \frac{e^{-ax}(-a\cos bx + b\sin bx)}{a^2 + b^2}$$
6. 定積分を計算：
$$\int_{0}^{\infty} e^{-ax} \cos bx \, dx = \left[\frac{e^{-ax}(-a\cos bx + b\sin bx)}{a^2 + b^2}\right]_{0}^{\infty}$$
$x \to \infty$で$0$，$x = 0$で$-\frac{a}{a^2 + b^2}$なので：
$$= 0 - \left(-\frac{a}{a^2 + b^2}\right) = \frac{a}{a^2 + b^2}$$
答え: $\frac{a}{a^{2} + b^{2}}$
全体のポイント
  • 部分積分を2回行うと元の積分が再出現
  • 方程式として解く技法が重要
  • $a > 0$が収束の必要条件
受験数学のチップス
  • 指数関数×三角関数は部分積分2回の定番
  • オイラー公式 $e^{ibx} = \cos bx + i\sin bx$ も利用可
  • 無限積分では収束条件を確認する
大問【2】

次の微分方程式の一般解を求めよ。 $$y^2 + (2xy + x^2) \frac{dy}{dx} = 0$$

考え方のヒント

これは同次型微分方程式です！$y = vx$と置換すると変数分離形になります。置換後は部分分数分解が必要になるので、分解の計算を丁寧に行いましょう。

途中式と解答
1. 方程式を整理：
$$\frac{dy}{dx} = -\frac{y^2}{2xy + x^2} = -\frac{y^2}{x(2y + x)}$$
2. 同次型なので$y = vx$と置換：$\frac{dy}{dx} = v + x\frac{dv}{dx}$
$$v + x\frac{dv}{dx} = -\frac{(vx)^2}{x(2vx + x)} = -\frac{v^2x^2}{x^2(2v + 1)} = -\frac{v^2}{2v + 1}$$
3. 整理すると：
$$x\frac{dv}{dx} = -\frac{v^2}{2v + 1} - v = -\frac{v^2 + v(2v + 1)}{2v + 1} = -\frac{3v^2 + v}{2v + 1}$$
4. 変数分離：
$$\frac{2v + 1}{v(3v + 1)}dv = -\frac{1}{x}dx$$
5. 部分分数分解：$\frac{2v + 1}{v(3v + 1)} = \frac{A}{v} + \frac{B}{3v + 1}$ $2v + 1 = A(3v + 1) + Bv$より$A = 1$，$B = -1$
$$\frac{2v + 1}{v(3v + 1)} = \frac{1}{v} - \frac{1}{3v + 1}$$
6. 積分：
$$\int\left(\frac{1}{v} - \frac{1}{3v + 1}\right)dv = -\int\frac{1}{x}dx$$
$$\ln|v| - \frac{1}{3}\ln|3v + 1| = -\ln|x| + C$$
7. $v = \frac{y}{x}$を代入して整理すると：
$$\frac{xy^3}{x + 3y} = C$$
答え: $\frac{xy^{3}}{x + 3y} = C$ （$C$は任意定数）
全体のポイント
  • 同次型は$y = vx$で変数分離に帰着
  • 部分分数分解で積分を簡単に
  • 解は陰関数表示でOK
受験数学のチップス
  • 同次判別：$M(tx,ty) = t^n M(x,y)$
  • 部分分数：$\frac{P(x)}{Q(x)R(x)} = \frac{A}{Q(x)} + \frac{B}{R(x)}$
  • 積分定数は最後にまとめて整理
大問【3】

同一直線上にない3点 $(a_1, a_2, a_3)$，$(b_1, b_2, b_3)$，$(c_1, c_2, c_3)$ を通る平面の方程式を行列式を用いて示せ。

考え方のヒント

3点を通る平面上の点$P(x,y,z)$について考えましょう！点$P$が平面上にある条件は、3つのベクトルが同一平面上にあること、つまりスカラー三重積が0になることです。これを行列式で表現します。

途中式と解答
1. 点$A(a_1, a_2, a_3)$，$B(b_1, b_2, b_3)$，$C(c_1, c_2, c_3)$，$P(x, y, z)$とします。
2. ベクトルを定義：
$\overrightarrow{AP} = (x - a_1, y - a_2, z - a_3)$
$\overrightarrow{AB} = (b_1 - a_1, b_2 - a_2, b_3 - a_3)$
$\overrightarrow{AC} = (c_1 - a_1, c_2 - a_2, c_3 - a_3)$
3. 点$P$が平面上にある条件は、3つのベクトルが同一平面上にあることです：
$$\overrightarrow{AP} \cdot (\overrightarrow{AB} \times \overrightarrow{AC}) = 0$$
4. スカラー三重積は行列式で表現できます：
$$\begin{vmatrix} x - a_1 & y - a_2 & z - a_3 \\ b_1 - a_1 & b_2 - a_2 & b_3 - a_3 \\ c_1 - a_1 & c_2 - a_2 & c_3 - a_3 \end{vmatrix} = 0$$
答え: $\begin{vmatrix} x - a_1 & y - a_2 & z - a_3 \\ b_1 - a_1 & b_2 - a_2 & b_3 - a_3 \\ c_1 - a_1 & c_2 - a_2 & c_3 - a_3 \end{vmatrix} = 0$
全体のポイント
  • 3点が同一平面上⇔スカラー三重積=0
  • 行列式でスカラー三重積を表現
  • 同一直線上でない条件が重要
受験数学のチップス
  • スカラー三重積：$\mathbf{a} \cdot (\mathbf{b} \times \mathbf{c})$
  • 行列式展開で平面方程式$Ax+By+Cz+D=0$
  • 幾何学的直感と代数的表現の対応
大問【4】

次の連立方程式が $0 \leq t$ で成立している。ラプラス変換を用いて $y(t)$ を求めよ。ただし，$y(0)=0$，$z(0)=1$ とする。 $$\begin{cases} \frac{dy(t)}{dt} + y(t) + 2 \frac{dz(t)}{dt} + 3z(t) = 6e^{-2t} \\ 3 \frac{dy(t)}{dt} - y(t) + 4 \frac{dz(t)}{dt} + z(t) = 0 \end{cases}$$

考え方のヒント

連立微分方程式をラプラス変換で代数方程式に変換しましょう！初期条件を適用して、$Y(s) = \mathcal{L}\{y(t)\}$と$Z(s) = \mathcal{L}\{z(t)\}$の連立方程式を作り、クラメールの法則で解きます。

途中式と解答
1. ラプラス変換の準備：$Y(s) = \mathcal{L}\{y(t)\}$，$Z(s) = \mathcal{L}\{z(t)\}$
$\mathcal{L}\left\{\frac{dy}{dt}\right\} = sY(s)$ （$y(0)=0$）
$\mathcal{L}\left\{\frac{dz}{dt}\right\} = sZ(s) - 1$ （$z(0)=1$）
$\mathcal{L}\{e^{-2t}\} = \frac{1}{s+2}$
2. 第1式をラプラス変換：
$$sY(s) + Y(s) + 2(sZ(s) - 1) + 3Z(s) = \frac{6}{s+2}$$
$$(s+1)Y(s) + (2s+3)Z(s) = \frac{6}{s+2} + 2 = \frac{2(s+5)}{s+2}$$
3. 第2式をラプラス変換：
$$3sY(s) - Y(s) + 4(sZ(s) - 1) + Z(s) = 0$$
$$(3s-1)Y(s) + (4s+1)Z(s) = 4$$
4. 連立方程式：
$$\begin{cases} (s+1)Y(s) + (2s+3)Z(s) = \frac{2(s+5)}{s+2} \\ (3s-1)Y(s) + (4s+1)Z(s) = 4 \end{cases}$$
5. クラメールの法則で$Y(s)$を求める： 行列式$D = (s+1)(4s+1) - (2s+3)(3s-1) = -2s^2 - 2s + 4$
分子$D_Y$を計算すると：$D_Y = 14\frac{s-1}{s+2}$
6. $Y(s) = \frac{D_Y}{D}$を計算： $s^2 + s - 2 = (s+2)(s-1)$より
$$Y(s) = \frac{14(s-1)}{(s+2) \cdot (-2)(s+2)(s-1)} = -\frac{7}{(s+2)^2}$$
7. 逆ラプラス変換：
$$\mathcal{L}^{-1}\left\{\frac{1}{(s+2)^2}\right\} = te^{-2t}$$
したがって：
$$y(t) = -7te^{-2t}$$
答え: $y(t) = -7te^{-2t}$
全体のポイント
  • ラプラス変換で微分方程式→代数方程式
  • 初期条件の適用が重要
  • クラメールの法則で効率的に解く
受験数学のチップス
  • $\mathcal{L}\{f'(t)\} = sF(s) - f(0)$ は基本公式
  • $\mathcal{L}^{-1}\left\{\frac{1}{(s+a)^n}\right\} = \frac{t^{n-1}e^{-at}}{(n-1)!}$
  • 連立方程式は行列式で系統的に解く

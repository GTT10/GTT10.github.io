大問【1】

次の広義積分を求めよ。ただし導出過程も示せ。

$$\int_{0}^{1} \frac{\log x}{\sqrt{x}} \, dx$$
考え方のヒント

この積分は「広義積分」と呼ばれるタイプで、$x=0$のところで$\log x$と$\sqrt{x}$が無限大に発散してしまうので、慎重に計算する必要があります！「極限」を使って少しずつ近づけていくのがポイントです。部分積分とロピタルの定理の合わせ技で解いていきましょう。

途中式と解答
1. まず積分区間を $[\varepsilon, 1]$（$\varepsilon$は0に近い小さな正の数）に制限して計算し、最後に $\varepsilon \to 0^+$ の極限を取ります。
2. 部分積分の準備をします。$\frac{\log x}{\sqrt{x}} = \log x \cdot x^{-1/2}$ と見て、部分積分を使います。
$f'(x) = x^{-1/2} \rightarrow f(x) = 2x^{1/2}$
$g(x) = \log x \rightarrow g'(x) = \frac{1}{x}$
3. 部分積分を実行します。
$$\int_{\varepsilon}^{1} \frac{\log x}{\sqrt{x}} \, dx = \left[ 2\sqrt{x} \log x \right]_{\varepsilon}^{1} - \int_{\varepsilon}^{1} 2\sqrt{x} \cdot \frac{1}{x} \, dx$$ $$= \left[ 2\sqrt{x} \log x \right]_{\varepsilon}^{1} - 2 \int_{\varepsilon}^{1} x^{-1/2} \, dx$$
4. 各項を計算します。第1項では $x=1$ で $2 \cdot 1 \cdot \log 1 = 0$、第2項の積分は $\int x^{-1/2} dx = 2x^{1/2}$ なので、
$$= -2\sqrt{\varepsilon}\log\varepsilon - 2(2 - 2\sqrt{\varepsilon}) = -2\sqrt{\varepsilon}\log\varepsilon -4 + 4\sqrt{\varepsilon}$$
5. 極限を取ります（$\varepsilon \to 0^+$）。$\lim_{\varepsilon \to 0^+} 4\sqrt{\varepsilon} = 0$ は明らかですが、$\lim_{\varepsilon \to 0^+} \sqrt{\varepsilon} \log \varepsilon$ は「0×∞」型の不定形です。ロピタルの定理を使うと、
$$\lim_{\varepsilon \to 0^+} \frac{\ln \varepsilon}{\varepsilon^{-1/2}} \xrightarrow{\text{ロピタル}} \lim_{\varepsilon \to 0^+} \frac{1/\varepsilon}{-\frac{1}{2}\varepsilon^{-3/2}} = \lim_{\varepsilon \to 0^+} (-2 \varepsilon^{1/2}) = 0$$
6. 最終的に、$-2 \cdot 0 - 4 + 4 \cdot 0 = -4$ となります。
答え: $\int_{0}^{1} \frac{\log x}{\sqrt{x}} \, dx = -4$
別解（\(y=\sqrt{x}\) と置換） ▶
題意：
$$I=\int_{0}^{1}\frac{\log x}{\sqrt{x}}\;dx$$
この積分を $$y=\sqrt{x}$$ という置換で評価する。
1. 置換 $y=\sqrt{x}$ の準備
置換式：$$x = y^{2}$$
微分：$$dx = 2y\,dy$$
積分区間の変換：
 $x=0$ に対して $y=0$，$x=1$ に対して $y=1$
2. 被積分関数と微分の変換
$$\frac{\log x}{\sqrt{x}}=\frac{\log(y^{2})}{\sqrt{y^{2}}}$$
$\sqrt{y^{2}} = y$（$y\ge0$），$\log(y^{2}) = 2\log y$
よって $$\frac{\log x}{\sqrt{x}} = \frac{2\log y}{y}$$
これに $dx=2y\,dy$ を掛けると $$4\log y \, dy$$
3. 置換後の積分
$$I = \int_{y=0}^{1} 4\log y \; dy = 4\int_{0}^{1} \log y \; dy$$
4. 基本積分 $\int_{0}^{1} \log y \, dy$ を評価
$$\int_{0}^{1}\log y \; dy = -1$$（\(y'\)を無理やりねじこんだ部分積分で示せる）
5. 結果
$$I = 4(-1) = -4$$
答え: $\boxed{\displaystyle \int_{0}^{1}\frac{\log x}{\sqrt{x}}\;dx = -4}$
別解のポイント
  • 置換で積分区間・被積分関数・微分をすべて変換
  • 計算が単なる対数積分へ帰着し，数値評価も容易
全体のポイント
  • 広義積分は極限で定義される特別な積分
  • 部分積分では「微分してシンプルになる項」を選ぶ
  • 不定形の極限はロピタルの定理で解決
受験数学のチップス
  • $\log x$ と冪関数の積分では部分積分を使う
  • $0 \times \infty$ 形は $\frac{\infty}{\infty}$ に変形してロピタル
  • 広義積分は「発散しそうで収束する」面白い例が多い
大問【2】

(1) $y'' - 6y' + 9y = 0$ の一般解を求めよ。

考え方のヒント

これは定数係数線形同次微分方程式ですね！特性方程式 $\lambda^2 - 6\lambda + 9 = 0$ を解いて、重解が出てきたときの解の形を思い出しましょう。重解のときは $x$ をかけた項が追加で必要です。

途中式と解答
1. 特性方程式を立てます。$y = e^{\lambda x}$ と仮定して代入すると、
$$\lambda^2 - 6\lambda + 9 = 0$$ $$(\lambda - 3)^2 = 0$$ $$\lambda = 3 \text{（重解）}$$
2. 重解の場合、解が1つしか出ないので $x$ をかけた解を追加します。一般解は、
$$y = (C_1 + C_2 x)e^{3x} \quad (C_1, C_2 \text{は任意定数})$$
答え: $y = (C_1 + C_2 x)e^{3x}$

(2) $y'' - 6y' + 9y = e^{3x}$ の一般解を求めよ。

考え方のヒント

非同次方程式ですね！一般解は「同次解 + 特殊解」になります。右辺が $e^{3x}$ で、これは特性方程式の重解と同じ形なので、特殊解には $x^2$ をかけた形を試してみましょう。

途中式と解答
1. 同次解は(1)で求めた結果です。
$$y_h = (C_1 + C_2 x)e^{3x}$$
2. 特殊解を設定します。右辺が $e^{3x}$ で、3が特性方程式の2重解なので、
$$y_p = A x^2 e^{3x} \quad (A\text{：未定係数})$$
3. $y_p$ を微分して元の方程式に代入します。
$y_p' = A(2x + 3x^2)e^{3x}$
$y_p'' = A(2 + 12x + 9x^2)e^{3x}$
4. 方程式に代入して整理すると、
$$A e^{3x} [(2 + 12x + 9x^2) - 6(2x + 3x^2) + 9x^2] = e^{3x}$$ $$A e^{3x} \cdot 2 = e^{3x}$$ $$A = \frac{1}{2}$$
5. 一般解をまとめます。
$$y = y_h + y_p = (C_1 + C_2 x)e^{3x} + \frac{1}{2} x^2 e^{3x}$$
答え: $y = \left( C_1 + C_2 x + \frac{1}{2} x^2 \right) e^{3x}$
全体のポイント
  • 重解のとき一般解は $(C_1 + C_2 x) e^{\lambda x}$
  • 非同次項が特性解と重複するときは $x$ の冪をかける
  • 未定係数法で特殊解を見つける
受験数学のチップス
  • 特性方程式は「解の個数と重複度」がキーポイント
  • 重複度に応じて $x$, $x^2$ などをかけて調整
  • 最後は必ず元の方程式に代入して検証
大問【3】

(1) 一周期の区間で、次式で定義される周期関数のフーリエ級数展開式を求めよ。

$$f(t) = \begin{cases} 0 & -\pi < t \leq 0 \\ \sin t & 0 < t \leq \pi \end{cases}$$
考え方のヒント

フーリエ級数は「変な形の周期関数を三角関数の和で表現する」魔法のような技術です！この関数は区間で定義されているので、積分計算では区間を分けて考えましょう。積和公式を使うと計算がスムーズになります。

途中式と解答
1. フーリエ係数の公式を使います。周期 $2\pi$ なので、
$a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} f(t) dt$
$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(t) \cos(nt) dt$
$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(t) \sin(nt) dt$
2. $a_0$ を計算します。
$$a_0 = \frac{1}{\pi} \left[ \int_{-\pi}^{0} 0 \, dt + \int_{0}^{\pi} \sin t \, dt \right] = \frac{1}{\pi} \left[ 0 + [-\cos t]_{0}^{\pi} \right] = \frac{2}{\pi}$$
3. $a_n$ を計算します。積和公式 $\sin t \cos(nt) = \frac{1}{2}[\sin((n+1)t) + \sin((1-n)t)]$ を使って、
$$a_n = \frac{1}{\pi} \int_{0}^{\pi} \sin t \cos(nt) dt$$
計算結果、$n=1$ のときは0、$n \geq 2$ のときも0になります（関数の対称性から）。
4. $b_n$ を計算します。特に重要なのは $n=1$ のときで、
$$b_1 = \frac{1}{\pi} \int_{0}^{\pi} \sin^2 t \, dt = \frac{1}{\pi} \cdot \frac{\pi}{2} = \frac{1}{2}$$
$n \geq 2$ のときは積和公式を使って計算できます。
5. 最終的なフーリエ級数展開は、
$$f(t) = \frac{1}{\pi} + \frac{1}{2} \sin t - \frac{2}{\pi} \sum_{k=1}^{\infty} \frac{\cos(2kt)}{4k^2-1}$$
答え: $f(t) = \frac{1}{\pi} + \frac{1}{2} \sin t - \frac{2}{\pi} \sum_{k=1}^{\infty} \frac{\cos(2kt)}{4k^2-1}$

(2) 次の値を求めよ。

$$\sum_{n=1}^{\infty} \frac{(-1)^n}{4n^2-1}$$
考え方のヒント

(1)のフーリエ級数の結果を活用しましょう！特定の $t$ 値を代入することで、級数の和を求めることができます。$\cos$ の符号が交互に変わる $t$ の値を選ぶのがポイントです。

途中式と解答
お察しの通り、これは**「(1)で求めたフーリエ級数展開の結果を利用して、無限級数の和を求める」**という典型的な問題です。 (1)の式に含まれる級数部分と比較すると、$\cos(2nt)$ が $(-1)^n$ になれば、求めたい級数の形が出てきそうだと分かります。 $$\cos(n\pi) = (-1)^n$$ という性質があります。 これを利用して、 $$ 2nt = n\pi $$ とすればよいので、 $$ t = \frac{\pi}{2} $$ が見つかります。代入して
$$f\left(\frac{\pi}{2}\right) = \frac{1}{\pi} + \frac{1}{2} - \frac{2}{\pi} \sum_{k=1}^{\infty} \frac{(-1)^k}{4k^2-1}$$
2. 関数の定義より $f\left(\frac{\pi}{2}\right) = \sin \frac{\pi}{2} = 1$ なので、
$$1 = \frac{1}{\pi} + \frac{1}{2} - \frac{2}{\pi} \sum_{n=1}^{\infty} \frac{(-1)^n}{4n^2-1}$$
3. この方程式を解くと、
$$\frac{2}{\pi} \sum_{n=1}^{\infty} \frac{(-1)^n}{4n^2-1} = \frac{1}{\pi} + \frac{1}{2} - 1 = \frac{1}{\pi} - \frac{1}{2}$$
4. したがって、
$$\sum_{n=1}^{\infty} \frac{(-1)^n}{4n^2-1} = \frac{\pi}{2} \left( \frac{1}{\pi} - \frac{1}{2} \right) = \frac{1}{2} - \frac{\pi}{4}$$
答え: $\sum_{n=1}^{\infty} \frac{(-1)^n}{4n^2-1} = \frac{1}{2} - \frac{\pi}{4}$
全体のポイント
  • フーリエ級数は積分で各係数を計算する
  • 積和公式で三角関数の積を分解
  • 特定点での値を使って級数の和を求める
受験数学のチップス
  • 区間で定義された関数は積分区間を分けて考える
  • フーリエ級数から級数の和を求める技法は頻出
  • $(-1)^n$ が現れる級数では適切な点を選んで代入

(1) 次の行列の階数を求めよ。

$$A = \begin{bmatrix} 0 & -a & -c \\ a & 0 & -b \\ c & b & 0 \end{bmatrix}$$
考え方のヒント

この行列は歪対称行列（転置すると符号反転）の形をしていますね！階数を求めるには行列式を計算するか、行基本変形を使って独立な行の数を調べましょう。まずは行列式から試してみると良いでしょう。

途中式と解答
1. 行列式をサラスの方法で計算します。
$$\det A = \begin{vmatrix} 0 & -a & -c \\ a & 0 & -b \\ c & b & 0 \end{vmatrix}$$ $$= 0 \cdot 0 \cdot 0 + (-a)(-b)(c) + (-c)(a)(b) - [(-c) \cdot 0 \cdot c + (-a) \cdot a \cdot 0 + 0 \cdot b \cdot (-b)]$$ $$= abc - abc - 0 = 0$$
2. $\det A = 0$ なので階数 < 3です。次に2つの列が独立かどうか調べます。第1列と第2列を見ると、
$\begin{bmatrix} 0 \\ a \\ c \end{bmatrix}$ と $\begin{bmatrix} -a \\ 0 \\ b \end{bmatrix}$
$a, b \neq 0$ なのでこれらは独立です。
3. 第3列が第1,2列の線形結合で表せるか確認します。
$\begin{bmatrix} -c \\ -b \\ 0 \end{bmatrix} = x \begin{bmatrix} 0 \\ a \\ c \end{bmatrix} + y \begin{bmatrix} -a \\ 0 \\ b \end{bmatrix}$
この連立方程式を解くと、$x = -\frac{b}{a}$、$y = \frac{c}{a}$ で確かに成り立ちます。
4. 第3列が第1,2列の線形結合で表せるので、独立な列は2本です。
答え: $\text{rank}(A) = 2$

(2) 次の連立一次方程式の解が存在するかどうか判定し，解が存在する場合には解の自由度を求めよ。

$$\begin{cases} ax_2 - cx_3 = a - c \\ ax_1 - bx_3 = a - b \\ cx_1 + bx_2 = c - b \end{cases}$$
考え方のヒント

連立方程式の解の存在は、係数行列と拡大係数行列の階数を比較して判定します！(1)で求めた行列と関係があるので、その結果を活用しましょう。解の自由度は「変数の数 - 階数」で求められます。

途中式と解答
1. 係数行列は(1)の行列$A$の転置と同じ構造で、階数は2です。拡大係数行列を作ると、
$$[B|\mathbf{b}] = \left[ \begin{array}{ccc|c} 0 & a & -c & a-c \\ a & 0 & -b & a-b \\ c & b & 0 & c-b \end{array} \right]$$
2. 行基本変形で拡大係数行列の階数を調べます。行の交換と消去を行うと、第3行が第1,2行の線形結合になることが確認できます。
3. 具体的に計算すると、第3行 = $-\frac{c}{a} \times$ 第1行 $+ \frac{b}{a} \times$ 第2行 となり、右辺も同様の関係が成り立ちます。
4. $\text{rank}(B) = \text{rank}([B|\mathbf{b}]) = 2$ なので解が存在します。変数の数は3、階数は2なので、解の自由度は $3 - 2 = 1$ です。
答え: 解は存在し、自由度は1
全体のポイント
  • 歪対称行列は特殊な性質を持つ
  • 行列の階数は独立な行・列の数
  • 解の存在は係数行列と拡大係数行列の階数比較
受験数学のチップス
  • 行列式が0なら階数<3、非零なら階数=3
  • 連立方程式は階数を使って解の性質を調べる
  • 自由度 = 変数の数 - 係数行列の階数

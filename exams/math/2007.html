大問【1】

(1) 微分方程式 $\frac{d^2x(t)}{dt^2} + 4x(t) = \delta(t)$ （$\delta(t)$ は単位インパルス関数）を初期条件 $x(0) = 0$，$\left.\frac{dx(t)}{dt}\right|_{t=0} = 1$ のもとでラプラス変換し、$X(s)$ を求めよ。

考え方のヒント

単位インパルス関数のラプラス変換は $\mathcal{L}\{\delta(t)\} = 1$ です！2階微分の変換公式と初期条件を使って、$X(s)$ について解きましょう。インパルス応答は制御理論でとても重要な概念ですよ。

途中式と解答
1. ラプラス変換の公式を適用します：
$\mathcal{L}\left\{\frac{d^2x}{dt^2}\right\} = s^2X(s) - sx(0) - x'(0) = s^2X(s) - 0 - 1 = s^2X(s) - 1$
$\mathcal{L}\{4x(t)\} = 4X(s)$
$\mathcal{L}\{\delta(t)\} = 1$
2. 微分方程式をラプラス変換：
$$s^2X(s) - 1 + 4X(s) = 1$$
3. $X(s)$ について整理：
$$(s^2 + 4)X(s) = 1 + 1 = 2$$
$$X(s) = \frac{2}{s^2 + 4}$$
答え: $X(s) = \frac{2}{s^{2} + 4}$

(2) $X(s)$ を逆ラプラス変換し、$x(t)$ を求めよ。

考え方のヒント

$X(s) = \frac{2}{s^2 + 4}$ の形は標準的なラプラス変換表にありますね！$\mathcal{L}\{\sin(kt)\} = \frac{k}{s^2 + k^2}$ の公式を使えば簡単に逆変換できます。

途中式と解答
1. $X(s) = \frac{2}{s^2 + 4} = \frac{2}{s^2 + 2^2}$ を標準形で表現します。
2. 標準的な逆ラプラス変換公式を適用：
$$\mathcal{L}^{-1}\left\{\frac{k}{s^2 + k^2}\right\} = \sin(kt)$$
ここで $k = 2$ なので：
$$x(t) = \mathcal{L}^{-1}\left\{\frac{2}{s^2 + 4}\right\} = \sin(2t)$$
3. 初期条件の確認： - $x(0) = \sin(0) = 0$ ✓ - $x'(t) = 2\cos(2t)$，$x'(0) = 2\cos(0) = 2$ インパルス効果により速度が瞬間的にジャンプするため整合します。
答え: $x(t) = \sin 2t \quad (t \geq 0)$
全体のポイント
  • インパルス関数のラプラス変換は1
  • 2階微分の変換には初期条件が必要
  • 標準形への変換で逆変換を簡単に
受験数学のチップス
  • $\mathcal{L}\{\delta(t)\} = 1$ は重要公式
  • $\mathcal{L}\{\sin(kt)\} = \frac{k}{s^2 + k^2}$
  • インパルス応答は制御工学で頻出
大問【2】

(1) 不定積分 $\int \sqrt{a^2 + x^2} \, dx$ を導出過程とともに示せ。

考え方のヒント

三角関数による置換積分が有効です！$x = a\tan\theta$ とおくと、$\sqrt{a^2 + x^2} = a\sec\theta$ になります。$\sec^3\theta$ の積分は部分積分で求められますよ。

途中式と解答
1. 三角置換 $x = a\tan\theta$ を行います。$dx = a\sec^2\theta \, d\theta$
2. 平方根を簡単化：
$$\sqrt{a^2 + x^2} = \sqrt{a^2 + a^2\tan^2\theta} = a\sqrt{1 + \tan^2\theta} = a\sec\theta$$
3. 積分を変換：
$$\int \sqrt{a^2 + x^2} \, dx = \int a\sec\theta \cdot a\sec^2\theta \, d\theta = a^2 \int \sec^3\theta \, d\theta$$
4. $\int \sec^3\theta \, d\theta$ を部分積分で計算： $u = \sec\theta$，$dv = \sec^2\theta \, d\theta$ とおくと
$$I = \sec\theta\tan\theta - \int \sec\theta(\sec^2\theta - 1) \, d\theta$$
$$2I = \sec\theta\tan\theta + \ln|\sec\theta + \tan\theta|$$
5. 元の変数に戻す：$\sec\theta = \frac{\sqrt{a^2 + x^2}}{a}$，$\tan\theta = \frac{x}{a}$
$$\int \sqrt{a^2 + x^2} \, dx = \frac{x}{2}\sqrt{a^2 + x^2} + \frac{a^2}{2}\ln\left|x + \sqrt{a^2 + x^2}\right| + C$$
答え: $\frac{x}{2}\sqrt{a^{2} + x^{2}} + \frac{a^{2}}{2}\ln\left|x + \sqrt{a^{2} + x^{2}}\right| + C$

(2) 放物線 $y = x^2$ の原点 $(0,0)$ から点 $(1,1)$ まで放物線に沿って測った長さを求めよ。

考え方のヒント

曲線の長さの公式 $L = \int_a^b \sqrt{1 + \left(\frac{dy}{dx}\right)^2} \, dx$ を使います！(1) の結果を活用できるように、$\sqrt{1 + 4x^2}$ の積分に帰着させましょう。

途中式と解答
1. 曲線の長さの公式を適用： $y = x^2$ より $\frac{dy}{dx} = 2x$
$$L = \int_0^1 \sqrt{1 + (2x)^2} \, dx = \int_0^1 \sqrt{1 + 4x^2} \, dx$$
2. $\sqrt{1 + 4x^2}$ を(1)の形に変換：
$$\sqrt{1 + 4x^2} = \sqrt{4\left(x^2 + \frac{1}{4}\right)} = 2\sqrt{x^2 + \left(\frac{1}{2}\right)^2}$$
3. (1)の結果を $a = \frac{1}{2}$ として適用：
$$L = 2\int_0^1 \sqrt{x^2 + \frac{1}{4}} \, dx$$
$$= 2\left[\frac{x}{2}\sqrt{x^2 + \frac{1}{4}} + \frac{1}{8}\ln\left|x + \sqrt{x^2 + \frac{1}{4}}\right|\right]_0^1$$
4. 定積分を計算： 上端 $x=1$：$\frac{\sqrt{5}}{4} + \frac{1}{8}\ln\left(1 + \frac{\sqrt{5}}{2}\right)$
下端 $x=0$：$0 + \frac{1}{8}\ln\left(\frac{1}{2}\right)$
$$L = \frac{\sqrt{5}}{2} + \frac{1}{4}\ln(2 + \sqrt{5})$$
答え: $L = \frac{\sqrt{5}}{2} + \frac{1}{4}\ln(2 + \sqrt{5})$

大問【3】

変数 $x, y, z$ の間に $\frac{x + y - 2z}{x} = \frac{-x + 2y + z}{y} = \frac{y - z}{z}$ が成り立つとき、$x = y = z = 0$ 以外の場合の $x, y, z$ の比を求めよ。

考え方のヒント

共通の比を $k$ とおいて連立方程式を立てましょう！3つの等式から3つの方程式ができるので、パラメータ $k$ の値によって場合分けして解きます。$x \neq 0$，$y \neq 0$，$z \neq 0$ の条件も忘れずに！

途中式と解答
1. 共通の比を $k$ とおきます：
$$\frac{x + y - 2z}{x} = \frac{-x + 2y + z}{y} = \frac{y - z}{z} = k$$
2. 3つの方程式を立てます：
$(1-k)x + y - 2z = 0$ ... (1)
$-x + (2-k)y + z = 0$ ... (2)
$y - (k+1)z = 0$ ... (3)
3. 式(3)から $y = (k+1)z$ を式(1)に代入：
$$(1-k)x + (k+1)z - 2z = 0$$
$$(1-k)x + (k-1)z = 0$$
$$(1-k)(x - z) = 0$$
4. 場合分けして解きます： ケース1: $k = 1$ のとき - $y = 2z$，式(2)より $-x + 2y + z = 0$ → $x = 3z$ - 比 $x:y:z = 3:2:1$ ケース2: $k \neq 1$ のとき - $x = z$，式(2)より $-z + (2-k)(k+1)z + z = 0$ - $(2-k)(k+1) = 0$ より $k = 2$ または $k = -1$ - $k = 2$：$y = 3z$，$x = z$ → 比 $x:y:z = 1:3:1$ - $k = -1$：$y = 0$（不適、分母に $y$ があるため）
答え: $x:y:z = 3:2:1$ または $1:3:1$
全体のポイント

  • パラメータ $k$ による場合分けが重要
  • 分母≠0の条件を忘れずにチェック

大問【4】

(1) $P(x), Q(x)$ が $x$ の関数のとき、微分方程式 $\frac{dy}{dx} + P(x)y = Q(x)y^n$ ($n \neq 0, 1$) が $z = y^{1-n}$ とおけば線形微分方程式に帰着できることを示せ。

考え方のヒント

これはベルヌーイ型微分方程式の線形化です！$z = y^{1-n}$ を $x$ で微分して、元の方程式に代入してみましょう。$y^n$ の項がうまく消えて線形方程式になるはずです。

途中式と解答
1. $z = y^{1-n}$ を $x$ で微分：
$$\frac{dz}{dx} = (1-n)y^{-n}\frac{dy}{dx}$$
2. 元の方程式から $\frac{dy}{dx} = -P(x)y + Q(x)y^n$ を代入：
$$\frac{dz}{dx} = (1-n)y^{-n}(-P(x)y + Q(x)y^n)$$
$$= (1-n)(-P(x)y^{1-n} + Q(x))$$
3. $y^{1-n} = z$ を代入：
$$\frac{dz}{dx} = (1-n)(-P(x)z + Q(x))$$
$$= (1-n)Q(x) - (1-n)P(x)z$$
4. 線形微分方程式の形に整理：
$$\frac{dz}{dx} + (n-1)P(x)z = (1-n)Q(x)$$
これは $z$ に関する線形微分方程式です。
証明完了

(2) 微分方程式 $\frac{dy}{dx} + y + e^x y^3 = 0$ を解け。

考え方のヒント

(1)の結果を活用しましょう！$n=3$ のベルヌーイ型なので、$z = y^{-2}$ とおいて線形微分方程式に変換します。積分因子法で解いてから $y$ に戻します。

途中式と解答
1. 方程式を標準形に変形：
$$\frac{dy}{dx} + y = -e^x y^3$$
$P(x) = 1$，$Q(x) = -e^x$，$n = 3$
2. $z = y^{1-3} = y^{-2}$ とおき、(1)の結果を適用：
$$\frac{dz}{dx} + 2z = 2e^x$$
3. 積分因子 $\mu(x) = e^{2x}$ を使って解きます：
$$e^{2x}\frac{dz}{dx} + 2e^{2x}z = 2e^{3x}$$
$$\frac{d}{dx}(ze^{2x}) = 2e^{3x}$$
4. 積分：
$$ze^{2x} = \int 2e^{3x} dx = \frac{2}{3}e^{3x} + C$$
$$z = \frac{2}{3}e^x + Ce^{-2x}$$
5. $z = y^{-2}$ より：
$$\frac{1}{y^2} = \frac{2}{3}e^x + Ce^{-2x}$$
$$y = \pm\frac{e^{-x}}{\sqrt{\frac{2}{3} + Ce^{-3x}}}$$
答え: $y = \pm\frac{e^{-x}}{\sqrt{\frac{2}{3} + Ce^{-3x}}}$ （$C$ は定数）

大問【1】ベルヌーイの定理と垂直噴流

非圧縮性流体に関して以下の問いに答えよ。

(1)(a) ベルヌーイの定理は単位（ア）あたりの流体に対するエネルギー保存式である。

考え方のヒント

ベルヌーイの式 $\frac{1}{2}\rho V^{2}+p+\rho gh=一定$ の各項の次元を考えてみましょう。すべて圧力の次元[ML⁻¹T⁻²]を持ちます。

途中式と解答
ベルヌーイの式の各項の次元を確認：
  • $\frac{1}{2}\rho V^{2}$: 単位体積あたりの運動エネルギー
  • $p$: 圧力（単位体積あたりの圧力エネルギー）
  • $\rho gh$: 単位体積あたりの位置エネルギー
答え: 体積

(1)(b) ベルヌーイの定理が成立する条件は、(イ)、(ウ)、(エ)である。

ベルヌーイの定理が成立するための基本条件：
  • 非圧縮性の流れ（密度一定）
  • 非粘性の流れ（完全流体）
  • 定常の流れ（時間変化なし）
答え: (イ)非圧縮性流れ、(ウ)非粘性流れ、(エ)定常流れ

(2)(a) 点Oにおける圧力$p_s$を（オ）または（カ）と呼ぶ。

流れが物体に衝突して速度がゼロになる点を「よどみ点」と呼び、その点における圧力を「よどみ点圧」または「停滞圧」と呼びます。
答え: (オ)よどみ点、(カ)よどみ点圧（または停滞圧）

(2)(b) $p_s$の値を求めよ。

x軸上の中心線に沿った流線で、無限遠方の点と点Oにベルヌーイの定理を適用：
$$\frac{1}{2}\rho V_0^2 + p_0 = \frac{1}{2}\rho (0)^2 + p_s$$
$$p_s = p_0 + \frac{1}{2}\rho V_0^2$$
答え: $p_s = p_0 + \frac{1}{2}\rho V_0^2$

(2)(c) $V_1$の値を求めよ。

噴流の縁に沿った流線で、無限遠方の点と流れ去る点にベルヌーイの定理を適用：
$$\frac{1}{2}\rho V_0^2 + p_0 = \frac{1}{2}\rho V_1^2 + p_0$$
噴流の縁は大気に接しているため圧力は$p_0$のまま
$$V_1 = V_0$$
答え: $V_1 = V_0$
大問1のまとめ

ベルヌーイの定理の基本概念と垂直噴流への適用を理解することが重要です。よどみ点圧の概念と流線に沿った圧力・速度変化の関係を正確に把握しましょう。

重要公式・関係式
  • ベルヌーイの定理: $\frac{1}{2}\rho V^{2}+p+\rho gh=一定$
  • よどみ点圧: $p_s = p_0 + \frac{1}{2}\rho V_0^2$
  • エネルギー保存則の適用
解法のポイント
  • 適切な流線を選択する
  • 境界条件（圧力・速度）を正確に把握する
  • 各項の物理的意味を理解する
注意点・間違いやすいポイント
  • 流線の選択を間違えないこと
  • よどみ点では速度がゼロになることを忘れない
流体力学のチップス
  • 噴流の縁は常に大気圧と等しい
  • よどみ点圧は動圧と静圧の和
大問【2】渦の運動と循環

流体の円形回転運動は一般的に渦と呼ばれる。渦に関して以下の問いに答えよ。ただし、流れは二次元完全流体とし、図2に示すように渦中心はxy座標の原点に位置し、渦中心からの距離をr、x軸からの偏角をθ、回転運動の接線方向速度を$v_θ$とする。

(1) 渦中心からある距離までは$v_θ$がrに比例し、a以上ではrに反比例するような渦のモデルを何と呼ぶか答えよ。
考え方のヒント

渦核（中心部）では強制渦として一体回転し、外側では自由渦として速度が距離に反比例するモデルです。

途中式と解答
このような渦モデルは、内側で強制渦（$v_θ \propto r$）、外側で自由渦（$v_θ \propto 1/r$）の性質を持つ複合渦です。
答え: ランキン渦
(2) $0 \le r \le a$では$v_θ = Ωr^2$、$r > a$における$v_θ$を求めよ。
考え方のヒント

$r = a$で速度が連続でなければなりません。内側と外側の速度を等しくおいて比例定数を決定します。

途中式と解答
$r = a$における内側の速度: $v_θ(a) = Ωa^2$
$r > a$では$v_θ = C/r$とおくと、連続性から：
$$Ωa^2 = \frac{C}{a} \Rightarrow C = Ωa^3$$
答え: $v_θ = \frac{Ωa^3}{r}$
(3) この流れ場において、連続の方程式が満たされることを示せ。
考え方のヒント

極座標での連続の式は$\frac{1}{r}\frac{\partial (rv_r)}{\partial r} + \frac{1}{r}\frac{\partial v_θ}{\partial θ} = 0$です。

途中式と解答
この流れ場では$v_r = 0$であり、$v_θ$は$θ$に依存しないため：
$$\frac{1}{r}\frac{\partial (r \cdot 0)}{\partial r} + \frac{1}{r}\frac{\partial v_θ}{\partial θ} = 0 + 0 = 0$$
答え: 連続の方程式は満たされる
(4) この流れ場の渦度を求めよ。
極座標での渦度: $ω = \frac{1}{r} \frac{\partial(r v_θ)}{\partial r}$
  • $0 ≤ r ≤ a$: $rv_θ = Ωr^3$ → $ω = 3Ωr$
  • $r > a$: $rv_θ = Ωa^3$ (定数) → $ω = 0$
答え: $ω = 3Ωr$ (r ≤ a), $ω = 0$ (r > a)
(5) この流れ場において、半径rの円に沿った循環を求めよ。
循環$Γ = \oint_C \vec{v} \cdot d\vec{s} = \int_0^{2π} v_θ r dθ$
  • $0 ≤ r ≤ a$: $Γ = 2πΩr^3$
  • $r > a$: $Γ = 2πΩa^3$ (一定)
答え: $Γ = 2πΩr^3$ (r ≤ a), $Γ = 2πΩa^3$ (r > a)
まとめ

渦度と循環の概念、および渦の構造を理解することが重要です。内側では有渦流れ、外側では無渦流れという特徴があります。

全体のポイント
  • ランキン渦の特徴を理解する
  • 連続の式の極座標表現
  • 渦度と循環の関係
流体力学のチップス
  • 渦核内では渦度が存在し、外側では無渦流れ
  • 循環は$r > a$で一定値を保つ

2023年度 材料力学大学院入試問題解説

大問【1】

図1は、壁にピンで固定された剛体棒CDが2本のワイヤーAEとBFで吊り下げられ、先端Dに荷重Pが作用している状況を示しています。

(1) つり合い式
考え方のヒント

物体にはたらく力のつり合いを考えます。支持力\(R\)とワイヤーの張力\(T_a, T_b\)を上向き、外力\(P\)を下向きとして、垂直方向の力のつり合い式と点C周りのモーメントのつり合い式を立てます。

途中式と解答
垂直方向の力のつり合い式: \[R + T_a + T_b - P = 0\]
点Cを回転の中心としてモーメントのつり合い式: \[a T_a + b T_b - c P = 0\]
(2) ワイヤーの応力
考え方のヒント

これは不静定問題なので、力のつり合いに加えて変形の適合条件を考慮する必要があります。剛体棒CDが点Cを中心に微小角度 \(\theta\) だけ回転すると、点Aと点Bの変位 \(\delta_a, \delta_b\) はそれぞれワイヤーの伸びに等しくなります。

途中式と解答
変位の幾何学的関係： \[\delta_a = a \theta, \quad \delta_b = b \theta\]
ワイヤーの伸びは材料の性質から： \[\delta_a = \frac{T_a l_a}{E_a S_a}, \quad \delta_b = \frac{T_b l_b}{E_b S_b}\]
変形の適合条件式： \[\frac{\delta_a}{a} = \frac{\delta_b}{b} \implies \frac{T_a l_a}{a E_a S_a} = \frac{T_b l_b}{b E_b S_b}\]
つり合い式と適合条件式を連立： \[T_a = T_b \cdot \frac{a l_b E_a S_a}{b l_a E_b S_b}\] \[a \left( T_b \frac{a l_b E_a S_a}{b l_a E_b S_b} \right) + b T_b = cP\]
張力 \(T_b\) と \(T_a\) を解く： \[T_b = \frac{c P b l_a E_b S_b}{a^2 l_b E_a S_a + b^2 l_a E_b S_b}\] \[T_a = \frac{c P a l_b E_a S_a}{a^2 l_b E_a S_a + b^2 l_a E_b S_b}\]
応力 \(\sigma\) の計算： \[\sigma_a = \frac{T_a}{S_a} = \frac{c P a l_b E_a}{a^2 l_b E_a S_a + b^2 l_a E_b S_b}\] \[\sigma_b = \frac{T_b}{S_b} = \frac{c P b l_a E_b}{a^2 l_b E_a S_a + b^2 l_a E_b S_b}\]
(3) 応力の比
考え方のヒント

変形の適合条件式を応力で書き直し、問題の条件 (\(E_a = E_b\), \(2a = b\), \(l_a = 2l_b\)) を代入して応力の比を求めます。

途中式と解答
適合条件式を応力で表現： \[\frac{\sigma_a l_a}{a E_a} = \frac{\sigma_b l_b}{b E_b}\]
応力の比： \[\frac{\sigma_a}{\sigma_b} = \frac{a E_a l_b}{b E_b l_a}\]
条件を代入： \[\frac{\sigma_a}{\sigma_b} = \frac{a E_a l_b}{(2a) E_a (2l_b)} = \frac{1}{4}\]
答え: \(\sigma_a\) は \(\sigma_b\) の \(\frac{1}{4}\) 倍
全体のポイント
  • 不静定問題では力のつり合いと変形の適合条件の両方を考慮
  • 幾何学的関係と材料特性（フックの法則）を組み合わせて解く
  • ワイヤーの伸びと剛体棒の回転量の関係を明確に定義
材料力学のチップス
  • 不静定問題は「つり合い式」＋「変形の適合条件」が基本
  • 応力計算では断面積で張力を除算
  • 比を求める問題では単位に注意（無次元量になる）

大問【2】

図2は、一端（B）が固定され、他端（A）が支持されたはりに、曲げモーメント \(M_C\) が作用している状況を示しています。

(1) 曲げモーメントとたわみ
考え方のヒント

支持力 \(R_A\) を既知の外力とみなし、点Aを原点としてx軸をとります。区間AC (\(0 \le x \le a\)) と CB (\(a \le x \le l\)) で曲げモーメントの式が異なります。たわみの基本微分方程式 \(EI \frac{d^2v}{dx^2} = M(x)\) を積分し、境界条件と連続条件から積分定数を決定します。

途中式と解答
曲げモーメント \(M(x)\):
  • 区間AC (\(0 \le x \le a\)): \[M_{AC}(x) = R_A x\]
  • 区間CB (\(a \le x \le l\)): \[M_{CB}(x) = R_A x - M_C\]
たわみ \(v(x)\) の微分方程式:
  • 区間AC (\(0 \le x \le a\)): \[EI \frac{d^2v_{AC}}{dx^2} = R_A x\] \[EI \frac{dv_{AC}}{dx} = \frac{1}{2} R_A x^2 + C_1\] \[EI v_{AC}(x) = \frac{1}{6} R_A x^3 + C_1 x + C_2\]
  • 区間CB (\(a \le x \le l\)): \[EI \frac{d^2v_{CB}}{dx^2} = R_A x - M_C\] \[EI \frac{dv_{CB}}{dx} = \frac{1}{2} R_A x^2 - M_C x + C_3\] \[EI v_{CB}(x) = \frac{1}{6} R_A x^3 - \frac{1}{2} M_C x^2 + C_3 x + C_4\]
境界条件と連続条件:
  1. 支持点A (\(x=0\)): \(v_{AC}(0) = 0\)
  2. 固定端B (\(x=l\)): \(v_{CB}(l) = 0\)
  3. 固定端B (\(x=l\)): \(\frac{dv_{CB}}{dx}(l) = 0\)
  4. 点C (\(x=a\)): \(v_{AC}(a) = v_{CB}(a)\)
  5. 点C (\(x=a\)): \(\frac{dv_{AC}}{dx}(a) = \frac{dv_{CB}}{dx}(a)\)
積分定数の決定:
  • 条件1から: \(C_2 = 0\)
  • 条件3, 5から: \(C_1 = M_C(l-a) - \frac{1}{2}R_A l^2\)
たわみの式:
  • 区間AC: \[v_{AC}(x) = \frac{1}{EI} \left[ \frac{1}{6} R_A x^3 + \left( M_C(l-a) - \frac{1}{2}R_A l^2 \right) x \right]\]
  • 区間CB: \[v_{CB}(x) = \frac{1}{EI} \left[ \frac{1}{6} R_A x^3 - \frac{1}{2} M_C x^2 + \left( M_C l - \frac{1}{2}R_A l^2 \right) x - \frac{1}{2}M_C a^2 \right]\]
(2) 支持力 \(R_A\)
考え方のヒント

(1)で用いた境界条件をすべて適用することで、未知数であった支持力 \(R_A\) を決定できます。特に、固定端Bでのたわみ角ゼロの条件から得られる関係式を用います。

途中式と解答
境界条件から得られる関係式: \[-\frac{1}{3}R_A l^3 + \frac{M_C}{2}(l^2-a^2) = 0\]
\(R_A\) について解く: \[R_A = \frac{3 M_C (l^2-a^2)}{2 l^3}\]
答え: \(R_A = \dfrac{3 M_C (l^2-a^2)}{2 l^3}\)
全体のポイント
  • はりの曲げ問題では区間ごとに曲げモーメントを定義
  • たわみの微分方程式を2回積分して一般解を求める
  • 境界条件と連続条件から積分定数を決定
材料力学のチップス
  • 固定端: たわみ=0 & たわみ角=0
  • 支持端: たわみ=0 & 曲げモーメント=0
  • 連続条件: たわみとたわみ角が接続点で連続
  • たわみのn階微分は物理量と対応:
    0階: たわみ, 1階: たわみ角,
    2階: 曲げモーメント, 3階: せん断力
